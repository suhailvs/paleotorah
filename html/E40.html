Exodus Chapter 40

1 and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) to~ >~ SAY (Verb) ( לֵּאמֹר / ley'mor )

and YHWH spoke to Mosheh saying,

2 in~ DAY ( בְּיוֹם / bê'yom ) the~ NEW.MOON ( הַחֹדֶשׁ / ha'hho'desh ) the~ FIRST ( הָרִאשׁוֹן / ha'ri'shon ) in~ UNIT ( בְּאֶחָד / bê'e'hhad ) to~ the~ NEW.MOON ( לַחֹדֶשׁ / la'hho'desh ) [ 643 ] you(ms)~ will~ make~ RISE (Verb) ( תָּקִים / ta'qim ) AT ( אֶת / et ) DWELLING ( מִשְׁכַּן / mish'kan ) TENT ( אֹהֶל / o'hel ) APPOINTED ( מוֹעֵד / mo'eyd )

In the first new moon, on the first day of the new moon, you will make the dwelling of the appointed tent rise,

3 and~ you(ms)~ did~ PLACE (Verb) ( וְשַׂמְתָּ / wê'sam'ta ) THERE ( שָׁם / sham ) AT ( אֵת / eyt ) BOX ( אֲרוֹן / a'ron ) the~ EVIDENCE ( הָעֵדוּת / ha'ey'dut ) and~ you(ms)~ did~ FENCE.AROUND (Verb) ( וְסַכֹּתָ / wê'sa'ko'ta ) UPON ( עַל / al ) the~ BOX ( הָאָרֹן / ha'a'ron ) AT ( אֶת / et ) the~ TENT.CURTAIN ( הַפָּרֹכֶת / ha'pa'ro'khet )

and you will place the box of the evidence there, and you will fence around the box with the tent curtain,

4 and~ you(ms)~ did~ make~ COME (Verb) ( וְהֵבֵאתָ / wê'hey'vey'ta ) AT ( אֶת / et ) the~ TABLE ( הַשֻּׁלְחָן / ha'shul'hhan ) and~ you(ms)~ did~ ARRANGE (Verb) ( וְעָרַכְתָּ / wê'a'rakh'ta ) AT ( אֶת / et ) ARRANGEMENT ~him ( עֶרְכּוֹ / er'ko ) and~ you(ms)~ did~ make~ COME (Verb) ( וְהֵבֵאתָ / wê'hey'vey'ta ) AT ( אֶת / et ) the~ LAMPSTAND ( הַמְּנֹרָה / ham'no'rah ) and~ you(ms)~ did~ make~ GO.UP (Verb) ( וְהַעֲלֵיתָ / wêha'a'ley'ta ) AT ( אֶת / et ) LAMP ~s ~her ( נֵרֹתֶיהָ / ney'ro'te'yah )

and you will bring the table, and you will arrange his arrangement, and you will bring the lampstand, and you will bring up her lamps,

5 and~ you(ms)~ did~ GIVE (Verb) ~& ( וְנָתַתָּה / wê'na'ta'tah ) AT ( אֶת / et ) ALTAR ( מִזְבַּח / miz'bahh ) the~ GOLD ( הַזָּהָב / ha'za'hav ) to~ INCENSE.SMOKE ( לִקְטֹרֶת / liq'to'ret ) to~ FACE ~s ( לִפְנֵי / liph'ney ) BOX ( אֲרוֹן / a'ron ) the~ EVIDENCE ( הָעֵדֻת / ha'ey'dut ) and~ you(ms)~ did~ PLACE (Verb) ( וְשַׂמְתָּ / wê'sam'ta ) AT ( אֶת / et ) CANOPY ( מָסַךְ / ma'sakh ) the~ OPENING ( הַפֶּתַח / ha'pe'tahh ) to~ the~ DWELLING ( לַמִּשְׁכָּן / la'mish'kan )

and you will place the altar of gold for the incense for the face of the box of the evidence, and you will place the canopy of the opening for the dwelling,

6 and~ you(ms)~ did~ GIVE (Verb) ~& ( וְנָתַתָּה / wê'na'ta'tah ) AT ( אֵת / eyt ) ALTAR ( מִזְבַּח / miz'bahh ) the~ ASCENSION.OFFERING ( הָעֹלָה / ha'o'lah ) to~ FACE ~s ( לִפְנֵי / liph'ney ) OPENING ( פֶּתַח / pe'tahh ) DWELLING ( מִשְׁכַּן / mish'kan ) TENT ( אֹהֶל / o'hel ) APPOINTED ( מוֹעֵד / mo'eyd )

and you will place the altar of the ascension offering to the face of the opening of the dwelling of the appointed tent,

7 and~ you(ms)~ did~ GIVE (Verb) ( וְנָתַתָּ / wê'na'ta'ta ) AT ( אֶת / et ) the~ CAULDRON ( הַכִּיֹּר / ha'ki'yor ) BETWEEN ( בֵּין / beyn ) TENT ( אֹהֶל / o'hel ) APPOINTED ( מוֹעֵד / mo'eyd ) and~ BETWEEN ( וּבֵין / u'veyn ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh ) and~ you(ms)~ did~ GIVE (Verb) ( וְנָתַתָּ / wê'na'ta'ta ) THERE ( שָׁם / sham ) WATER ~s2 ( מָיִם / ma'yim )

and you will place the cauldron between the appointed tent and the altar, and you will place waters in there,

8 and~ you(ms)~ did~ PLACE (Verb) ( וְשַׂמְתָּ / wê'sam'ta ) AT ( אֶת / et ) the~ COURTYARD ( הֶחָצֵר / he'hha'tseyr ) ALL.AROUND ( סָבִיב / sa'viv ) and~ you(ms)~ did~ GIVE (Verb) ( וְנָתַתָּ / wê'na'ta'ta ) AT ( אֶת / et ) CANOPY ( מָסַךְ / ma'sakh ) GATE ( שַׁעַר / sha'ar ) the~ COURTYARD ( הֶחָצֵר / he'hha'tseyr )

and you will place the courtyard all around, and you will place the canopy of the gate of the courtyard,

9 and~ you(ms)~ did~ TAKE (Verb) ( וְלָקַחְתָּ / wê'la'qahh'ta ) AT ( אֶת / et ) OIL ( שֶׁמֶן / she'men ) the~ OINTMENT ( הַמִּשְׁחָה / ha'mish'hhah ) and~ you(ms)~ did~ SMEAR (Verb) ( וּמָשַׁחְתָּ / u'ma'shahh'ta ) AT ( אֶת / et ) the~ DWELLING ( הַמִּשְׁכָּן / ha'mish'kan ) and~ AT ( וְאֶת / wê'et ) ALL ( כָּל / kol ) WHICH ( אֲשֶׁר / a'sher ) in~ ~him ( בּוֹ / bo ) and~ you(ms)~ did~ much~ SET.APART (Verb) ( וְקִדַּשְׁתָּ / wê'qi'dash'ta ) AT ~him ( אֹתוֹ / o'to ) and~ AT ( וְאֶת / wê'et ) ALL ( כָּל / kol ) UTENSIL ~s ~him ( כֵּלָיו / key'law ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) SPECIAL ( קֹדֶשׁ / qo'desh )

and you will take the oil of ointment, and you will smear the dwelling and all which is in him, and you will set him, and all his utensils, apart, and he will exist as a special thing,

10 and~ you(ms)~ did~ SMEAR (Verb) ( וּמָשַׁחְתָּ / u'ma'shahh'ta ) AT ( אֶת / et ) ALTAR ( מִזְבַּח / miz'bahh ) the~ ASCENSION.OFFERING ( הָעֹלָה / ha'o'lah ) and~ AT ( וְאֶת / wê'et ) ALL ( כָּל / kol ) UTENSIL ~s ~him ( כֵּלָיו / key'law ) and~ you(ms)~ did~ much~ SET.APART (Verb) ( וְקִדַּשְׁתָּ / wê'qi'dash'ta ) AT ( אֶת / et ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh ) SPECIAL ( קֹדֶשׁ / qo'desh ) SPECIAL ~s ( קָדָשִׁים / qa'da'shim ) [ 644 ]

and you will smear the altar of the ascension offering and all his utensils, and you will set the altar apart, and the altar, a special thing of special things, will exist,

11 and~ you(ms)~ did~ SMEAR (Verb) ( וּמָשַׁחְתָּ / u'ma'shahh'ta ) AT ( אֶת / et ) the~ CAULDRON ( הַכִּיֹּר / ha'ki'yor ) and~ AT ( וְאֶת / wê'et ) BASE ~him ( כַּנּוֹ / ka'no ) and~ you(ms)~ did~ much~ SET.APART (Verb) ( וְקִדַּשְׁתָּ / wê'qi'dash'ta ) AT ~him ( אֹתוֹ / o'to )

and you will smear the cauldron and his base, and you will set him apart,

12 and~ you(ms)~ did~ make~ COME.NEAR (Verb) ( וְהִקְרַבְתָּ / wê'hiq'rav'ta ) AT ( אֶת / et ) Aharon ( אַהֲרֹן / a'ha'ron ) and~ AT ( וְאֶת / wê'et ) SON ~s ~him ( בָּנָיו / ba'naw ) TO ( אֶל / el ) OPENING ( פֶּתַח / pe'tahh ) TENT ( אֹהֶל / o'hel ) APPOINTED ( מוֹעֵד / mo'eyd ) and~ you(ms)~ did~ BATHE (Verb) ( וְרָחַצְתָּ / wê'ra'hhats'ta ) AT ~them(m) ( אֹתָם / o'tam ) in~ the~ WATER ~s2 ( בַּמָּיִם / ba'ma'yim )

and you will bring near Aharon and his sons to the opening of the appointed tent, and you will bathe them in the waters,

13 and~ you(ms)~ did~ make~ WEAR (Verb) ( וְהִלְבַּשְׁתָּ / wê'hil'bash'ta ) AT ( אֶת / et ) Aharon ( אַהֲרֹן / a'ha'ron ) AT ( אֵת / eyt ) GARMENT ~s ( בִּגְדֵי / big'dey ) the~ SPECIAL ( הַקֹּדֶשׁ / ha'qo'desh ) and~ you(ms)~ did~ SMEAR (Verb) ( וּמָשַׁחְתָּ / u'ma'shahh'ta ) AT ~him ( אֹתוֹ / o'to ) and~ you(ms)~ did~ much~ SET.APART (Verb) ( וְקִדַּשְׁתָּ / wê'qi'dash'ta ) AT ~him ( אֹתוֹ / o'to ) and~ he~ did~ much~ ADORN (Verb) ( וְכִהֵן / wê'khi'heyn ) to~ ~me ( לִי / li )

and you will clothe Aharon with the garments of specialness, and you will smear him, and you will set him apart, and he will be adorned for me,

14 and~ AT ( וְאֶת / wê'et ) SON ~s ~him ( בָּנָיו / ba'naw ) you(ms)~ will~ make~ COME.NEAR (Verb) ( תַּקְרִיב / taq'riv ) and~ you(ms)~ did~ make~ WEAR (Verb) ( וְהִלְבַּשְׁתָּ / wê'hil'bash'ta ) AT ~them(m) ( אֹתָם / o'tam ) TUNIC ~s ( כֻּתֳּנֹת / ku'ta'not )

and you will bring near his sons, and you will clothe them with tunics,

15 and~ you(ms)~ did~ SMEAR (Verb) ( וּמָשַׁחְתָּ / u'ma'shahh'ta ) AT ~them(m) ( אֹתָם / o'tam ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) you(ms)~ did~ SMEAR (Verb) ( מָשַׁחְתָּ / ma'shahh'ta ) AT ( אֶת / et ) FATHER ~them(m) ( אֲבִיהֶם / a'vi'hem ) and~ they~ did~ much~ ADORN (Verb) ( וְכִהֲנוּ / wê'khi'ha'nu ) to~ ~me ( לִי / li ) and~ she~ did~ EXIST (Verb) ( וְהָיְתָה / wê'hai'tah ) to~ >~ EXIST (Verb) ( לִהְיֹת / lih'yot ) to~ ~them(m) ( לָהֶם / la'hem ) >~ SMEAR (Verb) ~them(m) ( מָשְׁחָתָם / mash'hha'tam ) to~ ADMINISTRATION ( לִכְהֻנַּת / likh'hu'nat ) DISTANT ( עוֹלָם / o'lam ) to~ GENERATION ~s ~them(m) ( לְדֹרֹתָם / lê'do'ro'tam )

and you will smear them just as you smeared their father, and they will be adorned for me, and their smearing will surely exist for them, for an administration of a distant time, for their generations,

16 and~ he~ will~ DO (Verb) ( וַיַּעַשׂ / wai'ya'as ) Mosheh ( מֹשֶׁה / mo'sheh ) like~ ALL ( כְּכֹל / kê'khol ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ DIRECT (Verb) ( צִוָּה / tsi'wah ) YHWH ( יְהוָה / YHWH ) AT ~him ( אֹתוֹ / o'to ) SO ( כֵּן / keyn ) he~ did~ DO (Verb) ( עָשָׂה / a'sah )

and Mosheh did just like all that YHWH directed him, so he did,

17 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) in~ the~ NEW.MOON ( בַּחֹדֶשׁ / ba'hho'desh ) the~ FIRST ( הָרִאשׁוֹן / ha'ri'shon ) in~ the~ YEAR ( בַּשָּׁנָה / ba'sha'nah ) the~ SECOND ( הַשֵּׁנִית / ha'shey'nit ) in~ UNIT ( בְּאֶחָד / bê'e'hhad ) to~ the~ NEW.MOON ( לַחֹדֶשׁ / la'hho'desh ) he~ did~ be~ make~ RISE (Verb) ( הוּקַם / hu'qam ) the~ DWELLING ( הַמִּשְׁכָּן / ha'mish'kan )

and it came to pass, in the first new moon, in the second year, on the first day of the new moon, the dwelling was made to rise,

18 and~ he~ will~ make~ RISE (Verb) ( וַיָּקֶם / wai'ya'qem ) Mosheh ( מֹשֶׁה / mo'sheh ) AT ( אֶת / et ) the~ DWELLING ( הַמִּשְׁכָּן / ha'mish'kan ) and~ he~ will~ GIVE (Verb) ( וַיִּתֵּן / wai'yi'teyn ) AT ( אֶת / et ) FOOTING ~s ~him ( אֲדָנָיו / a'da'naw ) and~ he~ will~ PLACE (Verb) ( וַיָּשֶׂם / wai'ya'sem ) AT ( אֶת / et ) BOARD ~s ~him ( קְרָשָׁיו / qê'ra'shaw ) and~ he~ will~ GIVE (Verb) ( וַיִּתֵּן / wai'yi'teyn ) AT ( אֶת / et ) WOOD.BAR ~s ~him ( בְּרִיחָיו / bê'ri'hhaw ) and~ he~ will~ make~ RISE (Verb) ( וַיָּקֶם / wai'ya'qem ) AT ( אֶת / et ) PILLAR ~s ~him ( עַמּוּדָיו / a'mu'daw )

and Mosheh made the dwelling rise, and he placed his footings, and he placed his boards, and he placed his wood bars, and he made his pillars rise,

19 and~ he~ will~ SPREAD.OUT (Verb) ( וַיִּפְרֹשׂ / wai'yiph'ros ) AT ( אֶת / et ) the~ TENT ( הָאֹהֶל / ha'o'hel ) UPON ( עַל / al ) the~ DWELLING ( הַמִּשְׁכָּן / ha'mish'kan ) and~ he~ will~ PLACE (Verb) ( וַיָּשֶׂם / wai'ya'sem ) AT ( אֶת / et ) ROOF.COVERING ( מִכְסֵה / mikh'seyh ) the~ TENT ( הָאֹהֶל / ha'o'hel ) UPON ~him ( עָלָיו / a'law ) from~ to~ UPWARD ~unto ( מִלְמָעְלָה / mil'ma'lah ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ much~ DIRECT (Verb) ( צִוָּה / tsi'wah ) YHWH ( יְהוָה / YHWH ) AT ( אֶת / et ) Mosheh ( מֹשֶׁה / mo'sheh )

and he spread out the tent upon the dwelling, and he placed the roof covering of the tent upon the top of him, just as YHWH directed Mosheh,

20 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) and~ he~ will~ GIVE (Verb) ( וַיִּתֵּן / wai'yi'teyn ) AT ( אֶת / et ) the~ EVIDENCE ( הָעֵדֻת / ha'ey'dut ) TO ( אֶל / el ) [ 645 ] the~ BOX ( הָאָרֹן / ha'a'ron ) and~ he~ will~ PLACE (Verb) ( וַיָּשֶׂם / wai'ya'sem ) AT ( אֶת / et ) the~ STRAND ~s ( הַבַּדִּים / ha'ba'dim ) UPON ( עַל / al ) the~ BOX ( הָאָרֹן / ha'a'ron ) and~ he~ will~ GIVE (Verb) ( וַיִּתֵּן / wai'yi'teyn ) AT ( אֶת / et ) the~ LID ( הַכַּפֹּרֶת / ha'ka'po'ret ) UPON ( עַל / al ) the~ BOX ( הָאָרֹן / ha'a'ron ) from~ to~ UPWARD ~unto ( מִלְמָעְלָה / mil'ma'lah )

and he took, and he placed the evidence to the box, and he placed the strands upon the box, and he placed the lid upon the top of the box,

and he brought the box to the dwelling, and he placed the tent curtain of the canopy, and he fenced around upon the box of the evidence, just as YHWH directed Mosheh,

22 and~ he~ will~ GIVE (Verb) ( וַיִּתֵּן / wai'yi'teyn ) AT ( אֶת / et ) the~ TABLE ( הַשֻּׁלְחָן / ha'shul'hhan ) in~ TENT ( בְּאֹהֶל / bê'o'hel ) APPOINTED ( מוֹעֵד / mo'eyd ) UPON ( עַל / al ) MIDSECTION ( יֶרֶךְ / ye'rekh ) the~ DWELLING ( הַמִּשְׁכָּן / ha'mish'kan ) NORTH ~unto ( צָפֹנָה / tsa'pho'nah ) from~ OUTSIDE ( מִחוּץ / mi'hhuts ) to~ the~ TENT.CURTAIN ( לַפָּרֹכֶת / la'pa'ro'khet )

and he placed the table in the appointed tent, upon the midsection of the dwelling, unto the north, outside the tent curtain,

23 and~ he~ will~ ARRANGE (Verb) ( וַיַּעֲרֹךְ / wai'ya'a'rokh ) UPON ~him ( עָלָיו / a'law ) ARRANGEMENT ( עֵרֶךְ / ey'rekh ) BREAD ( לֶחֶם / le'hhem ) to~ FACE ~s ( לִפְנֵי / liph'ney ) YHWH ( יְהוָה / YHWH ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ much~ DIRECT (Verb) ( צִוָּה / tsi'wah ) YHWH ( יְהוָה / YHWH ) AT ( אֶת / et ) Mosheh ( מֹשֶׁה / mo'sheh )

and he arranged upon him the bread for the face of YHWH, just as YHWH directed Mosheh,

24 and~ he~ will~ PLACE (Verb) ( וַיָּשֶׂם / wai'ya'sem ) AT ( אֶת / et ) the~ LAMPSTAND ( הַמְּנֹרָה / ham'no'rah ) in~ TENT ( בְּאֹהֶל / bê'o'hel ) APPOINTED ( מוֹעֵד / mo'eyd ) IN.FRONT ( נֹכַח / no'khahh ) the~ TABLE ( הַשֻּׁלְחָן / ha'shul'hhan ) UPON ( עַל / al ) MIDSECTION ( יֶרֶךְ / ye'rekh ) the~ DWELLING ( הַמִּשְׁכָּן / ha'mish'kan ) SOUTH ~unto ( נֶגְבָּה / neg'bah ) [ 646 ]
