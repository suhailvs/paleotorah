Deuteronomy Chapter 33

1 and~ THIS ( וְזֹאת / wê'zot ) the~ PRESENT ( הַבְּרָכָה / ha'be'ra'kha ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ KNEEL (Verb) ( בֵּרַךְ / bey'rakh ) Mosheh ( מֹשֶׁה / mo'sheh ) MAN ( אִישׁ / ish ) the~ Elohiym ( הָאֱלֹהִים / ha'e'lo'him ) AT ( אֶת / et ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) to~ FACE ~s ( לִפְנֵי / liph'ney ) DEATH ~him ( מוֹתוֹ / mo'to )

and this is the present which Mosheh, the man of Elohiym, respected the sons of Yisra'eyl to the face of his death,

2 and~ he~ will~ SAY (Verb) ( וַיֹּאמַר / wai'yo'mar ) YHWH ( יְהוָה / YHWH ) from~ Sinai ( מִסִּינַי / mi'si'nai ) he~ did~ COME (Verb) ( בָּא / ba ) [ 1067 ] and~ he~ did~ COME.UP (Verb) ( וְזָרַח / wê'za'rahh ) from~ Se'iyr ( מִשֵּׂעִיר / mi'sey'ir ) to~ ~them(m) ( לָמוֹ / la'mo ) he~ did~ make~ BE.BRIGHT (Verb) ( הוֹפִיעַ / ho'phi'a ) from~ HILL ( מֵהַר / me'har ) Paran ( פָּארָן / pa'ran ) and~ he~ did~ ARRIVE (Verb) ( וְאָתָה / wê'a'tah ) from~ MYRIAD ~s ( מֵרִבְבֹת / mey'riv'vot ) SPECIAL ( קֹדֶשׁ / qo'desh ) from~ RIGHT.HAND ~him ( מִימִינוֹ / mi'mi'no ) Eyshdat ( אֵשְׁדָּת / eysh'dat ) [ 1068 ] to~ ~them(m) ( לָמוֹ / la'mo )

and he said, YHWH came from Sinai, and he came up from Se'iyr to them, he shone from the hill of Paran, and he arrived from the myriads of special ones, from his right hand, Eyshdat is for them.

3 MOREOVER ( אַף / aph ) CHERISH (Verb) ~ing(ms) ( חֹבֵב / hho'veyv ) PEOPLE ~s ( עַמִּים / a'mim ) ALL ( כָּל / kol ) UNIQUE ~s ~him ( קְדֹשָׁיו / qê'do'shaw ) in~ HAND ~you(ms) ( בְּיָדֶךָ / bê'ya'de'kha ) and~ THEY(m) ( וְהֵם / wê'heym ) they~ did~ be~ much~ SIT.DOWN (Verb) ( תֻּכּוּ / tu'ku ) to~ FOOT ~you(ms) ( לְרַגְלֶךָ / lê'rag'le'kha ) he~ will~ LIFT.UP (Verb) ( יִשָּׂא / yi'sa ) WORD ~s ~you(ms) ( מִדַּבְּרֹתֶיךָ / mi'da'be'ro'tey'kha )

Moreover, cherishing the peoples, all his unique ones are in your hand, and they, they are sitting down at your foot, he will lift up your words.

4 TEACHING ( תּוֹרָה / to'rah ) he~ did~ much~ DIRECT (Verb) ( צִוָּה / tsi'wah ) to~ ~us ( לָנוּ / la'nu ) Mosheh ( מֹשֶׁה / mo'sheh ) POSSESSION ( מוֹרָשָׁה / mo'ra'shah ) ASSEMBLY ( קְהִלַּת / qê'hi'lat ) Ya'aqov ( יַעֲקֹב / ya'a'qov )

Mosheh directed a teaching to us, a possession of the assembly of Ya'aqov,

5 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) in~ Yeshurun ( בִישֻׁרוּן / vi'shu'run ) KING ( מֶלֶךְ / me'lekh ) in~ >~ self~ GATHER (Verb) ( בְּהִתְאַסֵּף / bê'hit'a'seyph ) HEAD ~s ( רָאשֵׁי / ra'shey ) PEOPLE ( עָם / am ) TOGETHER ( יַחַד / ya'hhad ) STAFF ~s ( שִׁבְטֵי / shiv'tey ) [ 1069 ] Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl )

and he was king in Yeshurun, in the heads of the people gathered together, the staffs of Yisra'eyl.

6 he~ will~ LIVE (Verb) ( יְחִי / yê'hhi ) Re'uven ( רְאוּבֵן / rê'u'veyn ) and~ DO.NOT ( וְאַל / wê'al ) he~ will~ DIE (Verb) ( יָמֹת / ya'mot ) and~ he~ will~ EXIST (Verb) ( וִיהִי / wi'hi ) MORTAL.MAN ~s ~him ( מְתָיו / mê'taw ) NUMBER ( מִסְפָּר / mis'par )

Re'uven will live, and he will not die, and his mortal men will be a number,

7 and~ THIS ( וְזֹאת / wê'zot ) to~ Yehudah ( לִיהוּדָה / li'hu'dah ) and~ he~ will~ SAY (Verb) ( וַיֹּאמַר / wai'yo'mar ) !(ms)~ HEAR (Verb) ( שְׁמַע / shê'ma ) YHWH ( יְהוָה / YHWH ) VOICE ( קוֹל / qol ) Yehudah ( יְהוּדָה / yê'hu'dah ) and~ TO ( וְאֶל / wê'el ) PEOPLE ~him ( עַמּוֹ / a'mo ) you(ms)~ will~ make~ COME (Verb) ~him ( תְּבִיאֶנּוּ / tê'vi'e'nu ) HAND ~s2 ~him ( יָדָיו / ya'daw ) ABUNDANT ( רָב / rav ) to~ ~him ( לוֹ / lo ) and~ HELP ( וְעֵזֶר / wê'ey'zer ) from~ NARROW ~s ~him ( מִצָּרָיו / mi'tsa'raw ) [ 1070 ] you(ms)~ will~ EXIST (Verb) ( תִּהְיֶה / tih'yeh )

and this is for Yehudah, and he will say, hear YHWH the voice of Yehudah, and to his people you will bring him, his hands are abundant for him, and help from his narrow ones, you will exist,

8 and~ to~ Lewi ( וּלְלֵוִי / ul'ley'wi ) he~ did~ SAY (Verb) ( אָמַר / a'mar ) Tumiym ~you(ms) ( תֻּמֶּיךָ / tu'mey'kha ) and~ Uriym ~you(ms) ( וְאוּרֶיךָ / wê'u'rey'kha ) to~ MAN ( לְאִישׁ / lê'ish ) KIND.ONE ~you(ms) ( חֲסִידֶךָ / hha'si'de'kha ) WHICH ( אֲשֶׁר / a'sher ) you(ms)~ did~ much~ TEST (Verb) ~him ( נִסִּיתוֹ / ni'si'to ) in~ Mas'sah ( בְּמַסָּה / bê'ma'sah ) you(ms)~ will~ DISPUTE (Verb) ~him ( תְּרִיבֵהוּ / tê'ri'vey'hu ) UPON ( עַל / al ) WATER ~s2 ( מֵי / mey ) Meriyvah ( מְרִיבָה / mê'ri'vah )

and to Lewi he said, your Tumiym and your Uriym are for each of your kind ones, whom you tested in Mas'sah, you will dispute him upon the waters of Meriyvah.

9 the~ SAY (Verb) ~ing(ms) ( הָאֹמֵר / ha'o'meyr ) to~ FATHER ~him ( לְאָבִיו / lê'a'viw ) and~ to~ MOTHER ~him ( וּלְאִמּוֹ / ul'i'mo ) NOT ( לֹא / lo ) I~ did~ SEE (Verb) ~him ( רְאִיתִיו / rê'i'tiw ) and~ AT ( וְאֶת / wê'et ) BROTHER ~s ~him ( אֶחָיו / e'hhaw ) NOT ( לֹא / lo ) he~ did~ make~ RECOGNIZE (Verb) ( הִכִּיר / hi'kir ) and~ AT ( וְאֶת / wê'et ) SON ~him ( בָּנוֹ / ba'no ) [ 1071 ] NOT ( לֹא / lo ) he~ did~ KNOW (Verb) ( יָדָע / ya'da ) GIVEN.THAT ( כִּי / ki ) they~ did~ SAFEGUARD (Verb) ( שָׁמְרוּ / sham'ru ) SPEECH ~you(ms) ( אִמְרָתֶךָ / im'ra'te'kha ) and~ COVENANT ~you(ms) ( וּבְרִיתְךָ / uv'rit'kha ) they(m)~ will~ PRESERVE (Verb) ( יִנְצֹרוּ / yin'tso'ru )

The one saying to his father and to his mother, I did not see him, and his brother he did not recognize, and his son he did not know, given that they safeguarded your speech, and your covenant they preserved.

10 they(m)~ will~ make~ THROW (Verb) ( יוֹרוּ / yo'ru ) [ 1072 ] DECISION ~s ~you(ms) ( מִשְׁפָּטֶיךָ / mish'pa'tey'kha ) to~ Ya'aqov ( לְיַעֲקֹב / lê'ya'a'qov ) and~ TEACHING ~you(ms) ( וְתוֹרָתְךָ / wê'to'rat'kha ) to~ Yisra'eyl ( לְיִשְׂרָאֵל / lê'yis'ra'eyl ) they(m)~ will~ PLACE (Verb) ( יָשִׂימוּ / ya'si'mu ) INCENSE.SMOKE ( קְטוֹרָה / qê'to'rah ) in~ NOSE ~you(ms) ( בְּאַפֶּךָ / bê'a'pe'kha ) and~ ENTIRELY ( וְכָלִיל / wê'kha'lil ) UPON ( עַל / al ) ALTAR ~you(ms) ( מִזְבְּחֶךָ / miz'be'hhe'kha )

They will throw your decisions to Ya'aqov, and your teachings to Yisra'eyl, they will place incense smoke in your nose, and entirely upon your altar.

11 >~ much~ KNEEL (Verb) ( בָּרֵךְ / ba'reykh ) YHWH ( יְהוָה / YHWH ) FORCE ~him ( חֵילוֹ / hhey'lo ) and~ DEED ( וּפֹעַל / u'pho'al ) HAND ~s2 ~him ( יָדָיו / ya'daw ) you(ms)~ will~ ACCEPT (Verb) ( תִּרְצֶה / tir'tseh ) !(ms)~ STRIKE.THROUGH (Verb) ( מְחַץ / mê'hhats ) WAIST ~s2 ( מָתְנַיִם / mat'na'yim ) RISE (Verb) ~ing(mp) ~him ( קָמָיו / qa'maw ) and~ much~ HATE (Verb) ~ing(mp) ~him ( וּמְשַׂנְאָיו / um'san'aw ) FROM ( מִן / min ) [ 1073 ] they(m)~ will~ RISE (Verb) ~must ( יְקוּמוּן / yê'qu'mun )

Respect YHWH, his force, and the deed of his hands you will accept, strike through the waists of his rising ones, and the ones hating him, they must not rise.

12 to~ Binyamin ( לְבִנְיָמִן / lê'vin'ya'min ) he~ did~ SAY (Verb) ( אָמַר / a'mar ) CHERISHED ( יְדִיד / yê'did ) YHWH ( יְהוֹה / YHWH ) he~ will~ DWELL (Verb) ( יִשְׁכֹּן / yish'kon ) to~ the~ SAFELY ( לָבֶטַח / la've'tahh ) UPON ~him ( עָלָיו / a'law ) BLANKET (Verb) ~ing(ms) ( חֹפֵף / hho'pheyph ) UPON ~him ( עָלָיו / a'law ) ALL ( כָּל / kol ) the~ DAY ( הַיּוֹם / hai'yom ) and~ BETWEEN ( וּבֵין / u'veyn ) SHOULDER.PIECE ~s ~him ( כְּתֵיפָיו / kê'tey'phaw ) he~ did~ DWELL (Verb) ( שָׁכֵן / sha'khen )

To Binyamin he said, the cherished of YHWH will dwell with safety upon him, blanketing upon him all the day, and between his shoulder pieces he dwelled,

13 and~ to~ Yoseph ( וּלְיוֹסֵף / ul'yo'seyph ) he~ did~ SAY (Verb) ( אָמַר / a'mar ) be~ much~ KNEEL (Verb) ~ing(fs) ( מְבֹרֶכֶת / mê'vo're'khet ) YHWH ( יְהוֹה / YHWH ) LAND ~him ( אַרְצוֹ / ar'tso ) from~ PRECIOUS ( מִמֶּגֶד / mi'me'ged ) SKY ~s2 ( שָׁמַיִם / sha'ma'yim ) from~ DEW ( מִטָּל / mi'tal ) and~ from~ DEEP.WATER ( וּמִתְּהוֹם / u'mit'hom ) STRETCH.OUT (Verb) ~ing(fs) ( רֹבֶצֶת / ro've'tset ) UNDER ( תָּחַת / ta'hhat )

and to Yoseph he said, being respected of YHWH is his land, from the precious skies, from the dew, and from the deep water stretching out below,

14 and~ from~ PRECIOUS ( וּמִמֶּגֶד / u'mi'me'ged ) PRODUCTION ~s ( תְּבוּאֹת / tê'vu'ot ) SUN ( שָׁמֶשׁ / sha'mesh ) and~ from~ PRECIOUS ( וּמִמֶּגֶד / u'mi'me'ged ) BROUGHT.OUT ( גֶּרֶשׁ / ge'resh ) MOON ~s ( יְרָחִים / yê'ra'hhim )

and from the precious productions of the sun, and from the precious things brought out of the moons,

15 and~ from~ HEAD ( וּמֵרֹאשׁ / u'mey'rosh ) the~ HILL ~s ( הַרְרֵי / har'rey ) EAST ( קֶדֶם / qe'dem ) and~ from~ PRECIOUS ( וּמִמֶּגֶד / u'mi'me'ged ) KNOLL ~s ( גִּבְעוֹת / giv'ot ) DISTANT ( עוֹלָם / o'lam )

and from the head of the hills of the east, and from the precious distant knolls,

16 and~ from~ PRECIOUS ( וּמִמֶּגֶד / u'mi'me'ged ) LAND ( אֶרֶץ / e'rets ) and~ FILLING ~her ( וּמְלֹאָהּ / um'lo'ah ) and~ SELF-WILL ( וּרְצוֹן / ur'tson ) DWELL (Verb) ~ing(ms) ~me ( שֹׁכְנִי / shokh'ni ) THORN.BUSH ( סְנֶה / sê'neh ) she~ will~ COME (Verb) ~& ( תָּבוֹאתָה / ta'vo'tah ) to~ HEAD ( לְרֹאשׁ / lê'rosh ) Yoseph ( יוֹסֵף / yo'seyph ) and~ to~ TOP.OF.THE.HEAD ( וּלְקָדְקֹד / ul'qad'qod ) DEDICATED ( נְזִיר / nê'zir ) BROTHER ~s ~him ( אֶחָיו / e'hhaw )

and from the precious land and her filling, and the self-will of my dwelling is a thorn bush, she will come to the head of Yoseph, and to the top of the head of the dedicated one of his brothers.

17 FIRSTBORN ( בְּכוֹר / bê'khor ) OX ~him ( שׁוֹרוֹ / sho'ro ) HONOR ( הָדָר / ha'dar ) to~ ~him ( לוֹ / lo ) and~ HORN ~s ( וְקַרְנֵי / wê'qar'ney ) RHINOCEROS ( רְאֵם / rê'eym ) HORN ~s ~him ( קַרְנָיו / qar'naw ) in~ ~them(m) ( בָּהֶם / ba'hem ) PEOPLE ~s ( עַמִּים / a'mim ) he~ will~ much~ GORE (Verb) ( יְנַגַּח / yê'na'gahh ) TOGETHER ( יַחְדָּו / yahh'daw ) FAR.END ~s ( אַפְסֵי / aph'sey ) LAND ( אָרֶץ / a'rets ) and~ THEY(m) ( וְהֵם / wê'heym ) MYRIAD ~s ( רִבְבוֹת / riv'vot ) Ephrayim ( אֶפְרַיִם / eph'ra'yim ) and~ THEY(m) ( וְהֵם / wê'heym ) THOUSAND ~s ( אַלְפֵי / al'phey ) Menasheh ( מְנַשֶּׁה / mê'na'sheh )

The firstborn of his ox is honor for him, and the horns of the rhinoceros are his horns, with them he will gore the people together to the far end of the land, and they are the myriads of Ephrayim, and they are the thousands of Menasheh,

18 and~ to~ Zevulun ( וְלִזְבוּלֻן / wê'liz'vu'lun ) he~ did~ SAY (Verb) ( אָמַר / a'mar ) !(ms)~ REJOICE (Verb) ( שְׂמַח / sê'mahh ) Zevulun ( זְבוּלֻן / zê'vu'lun ) in~ >~ GO.OUT (Verb) ~you(ms) ( בְּצֵאתֶךָ / bê'tsey'te'kha ) and~ Yis'sas'khar ( וְיִשָּׂשכָר / wê'yi'sas'khar ) in~ TENT ~s ~you(ms) ( בְּאֹהָלֶיךָ / bê'o'ha'ley'kha )

and to Zevulun he said, rejoice Zevulun in your going out, and Yis'sas'khar in your tents.

19 PEOPLE ~s ( עַמִּים / a'mim ) HILL ( הַר / har ) they(m)~ will~ CALL.OUT (Verb) ( יִקְרָאוּ / yiq'ra'u ) THERE ( שָׁם / sham ) they(m)~ will~ SACRIFICE (Verb) ( יִזְבְּחוּ / yiz'be'hhu ) SACRIFICE ~s ( זִבְחֵי / zav'hhey ) STEADFAST ( צֶדֶק / tse'deq ) GIVEN.THAT ( כִּי / ki ) ABOUNDING ( שֶׁפַע / she'pha ) SEA ~s ( יַמִּים / ya'mim ) they(m)~ will~ SUCKLE (Verb) ( יִינָקוּ / yi'na'qu ) and~ BOARDED.UP (Verb) ~ing(mp) ( וּשְׂפוּנֵי / us'phu'ney ) SUBMERGE (Verb) ~ed(mp) ( טְמוּנֵי / tê'mu'ney ) SAND ( חוֹל / hhol )

The people of the hill they will call out, there they will sacrifice steadfast sacrifices, given that the abounding seas they will suckle, and boarding up the submerged sand,

20 and~ to~ Gad ( וּלְגָד / ul'gad ) he~ did~ SAY (Verb) ( אָמַר / a'mar ) KNEEL (Verb) ~ed(ms) ( בָּרוּךְ / ba'rukh ) make~ WIDEN (Verb) ~ing(ms) ( מַרְחִיב / mar'hhiv ) Gad ( גָּד / gad ) like~ LIONESS ( כְּלָבִיא / kê'la'vi ) he~ did~ DWELL (Verb) ( שָׁכֵן / sha'khen ) and~ he~ did~ TEAR.INTO.PIECES (Verb) ( וְטָרַף / wê'ta'raph ) ARM ( זְרוֹעַ / zê'ro'a ) MOREOVER ( אַף / aph ) TOP.OF.THE.HEAD ( קָדְקֹד / qad'qod )

and to Gad he said, respected is the one making Gad wide, like a lioness he dwelled, and he will tear into pieces the arm, moreover the top of the head,

21 and~ he~ will~ SEE (Verb) ( וַיַּרְא / wai'yar ) ORIGIN ( רֵאשִׁית / rey'shit ) to~ ~him ( לוֹ / lo ) GIVEN.THAT ( כִּי / ki ) THERE ( שָׁם / sham ) PARCEL ( חֶלְקַת / hhel'qat ) much~ INSCRIBE (Verb) ~ing(ms) ( מְחֹקֵק / mê'hho'qeyq ) BOARDED.UP (Verb) ~ed(ms) ( סָפוּן / sa'phun ) and~ he~ will~ ARRIVE (Verb) ( וַיֵּתֵא / wai'yey'tey ) HEAD ~s ( רָאשֵׁי / ra'shey ) [ 1074 ] PEOPLE ( עָם / am ) STEADFASTNESS ( צִדְקַת / tsid'qat ) YHWH ( יְהוָה / YHWH ) he~ did~ DO (Verb) ( עָשָׂה / a'sah ) and~ DECISION ~s ~him ( וּמִשְׁפָּטָיו / u'mish'pa'taw ) WITH ( עִם / im ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl )

and he saw the origin for himself, given that there the parcel of the inscriber is boarded up, and he arrived with the heads of the people, he did the steadfastness of YHWH, and his decisions with Yisra'eyl,

22 and~ to~ Dan ( וּלְדָן / ul'dan ) he~ did~ SAY (Verb) ( אָמַר / a'mar ) Dan ( דָּן / dan ) WHELP ( גּוּר / gur ) LION ( אַרְיֵה / ar'yeyh ) he~ will~ much~ JUMP (Verb) ( יְזַנֵּק / yê'za'neyq ) FROM ( מִן / min ) the~ Bashan ( הַבָּשָׁן / ha'ba'shan )

and to Dan he said, Dan is a whelp of a lion, he will jump from Bashan,

23 and~ to~ Naphtali ( וּלְנַפְתָּלִי / ul'naph'ta'li ) he~ did~ SAY (Verb) ( אָמַר / a'mar ) Naphtali ( נַפְתָּלִי / naph'ta'li ) PLENTY ( שְׂבַע / sê'va ) SELF-WILL ( רָצוֹן / ra'tson ) and~ FULL ( וּמָלֵא / u'ma'ley ) PRESENT ( בִּרְכַּת / bir'kat ) YHWH ( יְהוָה / YHWH ) SEA ( יָם / yam ) and~ SOUTHERN ( וְדָרוֹם / wê'da'rom ) !(ms)~ POSSESS (Verb) ~& ( יְרָשָׁה / yê'ra'shah )

and to Naphtali he said, Naphtali, plenty of self-will and full of a present of YHWH, possess the sea and the southern,
