Deuteronomy Chapter 1

These are the words which Mosheh spoke to all of Yisra'eyl on the other side of the Yarden, in the wilderness, in the desert, in front of the reeds, between Paran and Tophel, and Lavan and Hhatsarot and Di-Zahav.

2 UNIT ( אַחַד / a'hhad ) TEN ( עָשָׂר / a'sar ) DAY ( יוֹם / yom ) from~ Hhorev ( מֵחֹרֵב / mey'hho'reyv ) ROAD ( דֶּרֶךְ / de'rekh ) HILL ( הַר / har ) Se'iyr ( שֵׂעִיר / sey'ir ) UNTIL ( עַד / ad ) Qadesh ( קָדֵשׁ / qa'deysh ) Barneya ( בַּרְנֵעַ / bar'ney'a )

Eleven days from Hhorev, the road of the hill of Se'iyr, unto Qadesh Barneya,

3 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) in~ FOUR ~s ( בְּאַרְבָּעִים / bê'ar'ba'im ) YEAR ( שָׁנָה / sha'nah ) in~ ONE ( בְּעַשְׁתֵּי / bê'ash'tey ) TEN ( עָשָׂר / a'sar ) NEW.MOON ( חֹדֶשׁ / hho'desh ) in~ UNIT ( בְּאֶחָד / bê'e'hhad ) to~ the~ NEW.MOON ( לַחֹדֶשׁ / la'hho'desh ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) Mosheh ( מֹשֶׁה / mo'sheh ) TO ( אֶל / el ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) like~ ALL ( כְּכֹל / kê'khol ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ DIRECT (Verb) ( צִוָּה / tsi'wah ) YHWH ( יְהוָה / YHWH ) AT ~him ( אֹתוֹ / o'to ) TO ~them(m) ( אֲלֵהֶם / a'ley'hem )

and it came to pass in the fortieth year, in the eleventh new moon, on the first of the new moon, Mosheh spoke to the sons of Yisra'eyl all that YHWH directed him concerning them.

4 AFTER ( אַחֲרֵי / a'hha'rey ) >~ make~ HIT (Verb) ~him ( הַכֹּתוֹ / ha'ko'to ) AT ( אֵת / eyt ) Sihhon ( סִיחֹן / si'hhon ) KING ( מֶלֶךְ / me'lekh ) the~ Emor ~of ( הָאֱמֹרִי / ha'e'mo'ri ) WHICH ( אֲשֶׁר / a'sher ) SETTLE (Verb) ~ing(ms) ( יוֹשֵׁב / yo'sheyv ) in~ Hheshbon ( בְּחֶשְׁבּוֹן / bê'hhesh'bon ) and~ AT ( וְאֵת / wê'eyt ) Og ( עוֹג / og ) KING ( מֶלֶךְ / me'lekh ) the~ Bashan ( הַבָּשָׁן / ha'ba'shan ) WHICH ( אֲשֶׁר / a'sher ) SETTLE (Verb) ~ing(ms) ( יוֹשֵׁב / yo'sheyv ) in~ Ashterot ( בְּעַשְׁתָּרֹת / bê'ash'ta'rot ) in~ Ed're'i ( בְּאֶדְרֶעִי / bê'ed're'i )

After his attacking Sihhon the king of the ones of Emor, which were settling in Hheshbon, and Og the king the Bashan, which were settling in Ashterot in Ed're'i.

5 in~ OTHER.SIDE ( בְּעֵבֶר / bê'ey'ver ) the~ Yarden ( הַיַּרְדֵּן / hai'yar'deyn ) in~ LAND ( בְּאֶרֶץ / bê'e'rets ) Mo'av ( מוֹאָב / mo'av ) he~ did~ make~ TAKE.UPON (Verb) ( הוֹאִיל / ho'il ) Mosheh ( מֹשֶׁה / mo'sheh ) he~ did~ much~ EXPLAIN (Verb) ( בֵּאֵר / bey'eyr ) AT ( אֶת / et ) the~ TEACHING ( הַתּוֹרָה / ha'to'rah ) the~ THIS ( הַזֹּאת / ha'zot ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor )

On the other side of the Yarden, in the land of Mo'av, Mosheh agreed, he explained this teaching saying,

6 YHWH ( יְהוָה / YHWH ) Elohiym ~us ( אֱלֹהֵינוּ / e'lo'hey'nu ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) TO ~us ( אֵלֵינוּ / ey'ley'nu ) in~ Hhorev ( בְּחֹרֵב / bê'hho'reyv ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) ABUNDANT ( רַב / rav ) to~ ~you(mp) ( לָכֶם / la'khem ) >~ SETTLE (Verb) ( שֶׁבֶת / she'vet ) in~ the~ HILL ( בָּהָר / ba'har ) the~ THIS ( הַזֶּה / ha'zeh ) [ 935 ]

YHWH our Elohiym spoke to us in Hhorev saying, abundant for you settling on this hill.

7 !(mp)~ TURN (Verb) ( פְּנוּ / pê'nu ) and~ !(mp)~ JOURNEY (Verb) ( וּסְעוּ / us'u ) to~ ~you(mp) ( לָכֶם / la'khem ) and~ !(mp)~ COME (Verb) ( וּבֹאוּ / u'vo'u ) HILL ( הַר / har ) the~ Emor ~of ( הָאֱמֹרִי / ha'e'mo'ri ) and~ TO ( וְאֶל / wê'el ) ALL ( כָּל / kol ) DWELLER ~s ~him ( שְׁכֵנָיו / shê'khey'naw ) in~ the~ DESERT ( בָּעֲרָבָה / ba'a'ra'vah ) in~ the~ HILL ( בָהָר / va'har ) and~ in~ the~ LOWLAND ( וּבַשְּׁפֵלָה / u'vash'phey'lah ) and~ in~ the~ SOUTH ( וּבַנֶּגֶב / u'va'ne'gev ) and~ in~ SHORE ( וּבְחוֹף / uv'hhoph ) the~ SEA ( הַיָּם / hai'yam ) LAND ( אֶרֶץ / e'rets ) the~ Kena'an ~of ( הַכְּנַעֲנִי / ha'ke'na'a'ni ) and~ the~ Levanon ( וְהַלְּבָנוֹן / wê'hal'va'non ) UNTIL ( עַד / ad ) the~ RIVER ( הַנָּהָר / ha'na'har ) the~ GREAT ( הַגָּדֹל / ha'ga'dol ) RIVER ( נְהַר / nê'har ) Perat ( פְּרָת / pê'rat )

Turn and journey for you, and come to the hill of the ones of Emor, and to all his dwellers, in the desert, on the hill, and in the lowland, and in the south, and in the shore of the sea, the land of the ones of Kena'an and the Levanon, unto the great river, the river of Perat.

8 !(ms)~ SEE (Verb) ( רְאֵה / rê'eyh ) I~ did~ GIVE (Verb) ( נָתַתִּי / na'ta'ti ) to~ FACE ~s ~you(mp) ( לִפְנֵיכֶם / liph'ney'khem ) AT ( אֶת / et ) the~ LAND ( הָאָרֶץ / ha'a'rets ) !(mp)~ COME (Verb) ( בֹּאוּ / bo'u ) and~ !(mp)~ POSSESS (Verb) ( וּרְשׁוּ / ur'shu ) AT ( אֶת / et ) the~ LAND ( הָאָרֶץ / ha'a'rets ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ be~ SWEAR (Verb) ( נִשְׁבַּע / nish'ba ) YHWH ( יְהוָה / YHWH ) to~ FATHER ~s ~you(mp) ( לַאֲבֹתֵיכֶם / la'a'vo'tey'khem ) to~ Avraham ( לְאַבְרָהָם / lê'av'ra'ham ) to~ Yits'hhaq ( לְיִצְחָק / lê'yits'hhaq ) and~ to~ Ya'aqov ( וּלְיַעֲקֹב / ul'ya'a'qov ) to~ >~ GIVE (Verb) ( לָתֵת / la'teyt ) to~ ~them(m) ( לָהֶם / la'hem ) and~ to~ SEED ~them(m) ( וּלְזַרְעָם / ul'zar'am ) AFTER ~them(m) ( אַחֲרֵיהֶם / a'hha'rey'hem )

See, I will give to your faces the land, come and possess the land which YHWH swore to your fathers, to Avraham, to Yits'hhaq, and to Ya'aqov, to give to them and to their seed after them,

9 and~ I~ will~ SAY (Verb) ( וָאֹמַר / wa'o'mar ) TO ~you(mp) ( אֲלֵכֶם / a'ley'khem ) in~ the~ APPOINTED.TIME ( בָּעֵת / ba'eyt ) the~ SHE ( הַהִוא / ha'hi ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) NOT ( לֹא / lo ) I~ will~ BE.ABLE (Verb) ( אוּכַל / u'khal ) to~ STRAND ~me ( לְבַדִּי / lê'va'di ) >~ LIFT.UP (Verb) ( שְׂאֵת / sê'eyt ) AT ~you(mp) ( אֶתְכֶם / et'khem )

and I will say to you in that appointed time saying, I will not be able to lift you up myself.

YHWH your Elohiym increased you, and look, today you are like the stars of the skies for an abundance.

11 YHWH ( יְהוָה / YHWH ) Elohiym ( אֱלֹהֵי / e'lo'hey ) FATHER ~s ~you(mp) ( אֲבוֹתֵכֶם / a'vo'tey'khem ) he~ will~ make~ ADD (Verb) ( יֹסֵף / yo'seyph ) UPON ~you(mp) ( עֲלֵיכֶם / a'ley'khem ) like~ ~you(mp) ( כָּכֶם / ka'khem ) THOUSAND ( אֶלֶף / e'leph ) FOOTSTEP ~s ( פְּעָמִים / pê'a'mim ) and~ he~ will~ much~ KNEEL (Verb) ( וִיבָרֵךְ / wi'va'reykh ) AT ~you(mp) ( אֶתְכֶם / et'khem ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) to~ ~you(mp) ( לָכֶם / la'khem )

YHWH Elohiym of your fathers will make add upon you a thousand times like you, and he will exalt you just as he spoke to you.

12 HOW ( אֵיכָה / ey'khah ) I~ will~ LIFT.UP (Verb) ( אֶשָּׂא / e'sa ) to~ STRAND ~me ( לְבַדִּי / lê'va'di ) HEAVY.BURDEN ~you(mp) ( טָרְחֲכֶם / tar'hha'khem ) and~ LOAD ~you(mp) ( וּמַשַּׂאֲכֶם / u'ma'sa'a'khem ) and~ DISPUTE ~you(mp) ( וְרִיבְכֶם / wê'riv'khem )

How will I lift up by myself your heavy burden and your load and your dispute.

13 !(mp)~ PROVIDE (Verb) ( הָבוּ / ha'vu ) to~ ~you(mp) ( לָכֶם / la'khem ) MAN ~s ( אֲנָשִׁים / a'na'shim ) SKILLED.ONE ~s ( חֲכָמִים / hha'kha'mim ) and~ be~ UNDERSTAND (Verb) ~ing(mp) ( וּנְבֹנִים / un'vo'nim ) and~ KNOW (Verb) ~ed(mp) ( וִידֻעִים / wi'du'im ) to~ STAFF ~s ~you(mp) ( לְשִׁבְטֵיכֶם / lê'shiv'tey'khem ) [ 936 ] and~ I~ will~ PLACE (Verb) ~them(m) ( וַאֲשִׂימֵם / wa'a'si'meym ) in~ HEAD ~s ~you(mp) ( בְּרָאשֵׁיכֶם / bê'ra'shey'khem )

Provide for you skilled men, and understanding and knowing to your staffs, and I will place them as your heads,

and you answered me, and you said, the word which you spoke to do is functional,

15 and~ I~ will~ TAKE (Verb) ( וָאֶקַּח / wa'e'qahh ) AT ( אֶת / et ) HEAD ~s ( רָאשֵׁי / ra'shey ) STAFF ~s ~you(mp) ( שִׁבְטֵיכֶם / shiv'tey'khem ) [ 937 ] MAN ~s ( אֲנָשִׁים / a'na'shim ) SKILLED.ONE ~s ( חֲכָמִים / hha'kha'mim ) and~ KNOW (Verb) ~ed(mp) ( וִידֻעִים / wi'du'im ) and~ I~ will~ GIVE (Verb) ( וָאֶתֵּן / wa'e'teyn ) AT ~them(m) ( אֹתָם / o'tam ) HEAD ~s ( רָאשִׁים / ra'shim ) UPON ~you(mp) ( עֲלֵיכֶם / a'ley'khem ) NOBLE ~s ( שָׂרֵי / sa'rey ) THOUSAND ~s ( אֲלָפִים / a'la'phim ) and~ NOBLE ~s ( וְשָׂרֵי / wê'sa'rey ) HUNDRED ~s ( מֵאוֹת / mey'ot ) and~ NOBLE ~s ( וְשָׂרֵי / wê'sa'rey ) FIVE ~s ( חֲמִשִּׁים / hha'mi'shim ) [ 938 ] and~ NOBLE ~s ( וְשָׂרֵי / wê'sa'rey ) TEN ~s ( עֲשָׂרֹת / a'sa'rot ) [ 939 ] and~ DOMINATE (Verb) ~s ( וְשֹׁטְרִים / wê'shot'rim ) to~ STAFF ~s ~you(mp) ( לְשִׁבְטֵיכֶם / lê'shiv'tey'khem ) [ 940 ]

and I took the heads of your staffs, skilled and knowing men, and I made them heads upon you, nobles of thousands, nobles of hundreds, and nobles of fifty, and nobles of twenty, and dominators for your staffs,

You will not pay attention to faces in the decision, like the small, like the great, you must hear, you will not be afraid of the face of man, given that the decision belongs to Elohiym, and the word which will be hard for you, you must come near to me and I will hear him,

18 and~ I~ will~ much~ DIRECT (Verb) ( וָאֲצַוֶּה / wa'a'tsa'weh ) AT ~you(mp) ( אֶתְכֶם / et'khem ) in~ the~ APPOINTED.TIME ( בָּעֵת / ba'eyt ) the~ SHE ( הַהִוא / ha'hi ) AT ( אֵת / eyt ) ALL ( כָּל / kol ) the~ WORD ~s ( הַדְּבָרִים / ha'de'va'rim ) WHICH ( אֲשֶׁר / a'sher ) you(ms)~ will~ DO (Verb) ~must ( תַּעֲשׂוּן / ta'a'sun )

and I directed you in that appointed time all the words which you must do,
