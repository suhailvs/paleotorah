13 to~ BRANCH ( לְמַטֵּה / lê'ma'teyh ) Asher ( אָשֵׁר / a'sheyr ) Setur ( סְתוּר / sê'tur ) SON ( בֶּן / ben ) Mika'eyl ( מִיכָאֵל / mi'kha'eyl )
