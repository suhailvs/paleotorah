Deuteronomy Chapter 6

1 and~ THIS ( וְזֹאת / wê'zot ) the~ DIRECTIVE ( הַמִּצְוָה / ha'mits'wah ) the~ CUSTOM ~s ( הַחֻקִּים / ha'hhu'qim ) and~ the~ DECISION ~s ( וְהַמִּשְׁפָּטִים / wê'ha'mish'pa'tim ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ DIRECT (Verb) ( צִוָּה / tsi'wah ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(mp) ( אֱלֹהֵיכֶם / e'lo'hey'khem ) to~ >~ much~ LEARN (Verb) ( לְלַמֵּד / lê'la'meyd ) AT ~you(mp) ( אֶתְכֶם / et'khem ) to~ >~ DO (Verb) ( לַעֲשׂוֹת / la'a'sot ) in~ the~ LAND ( בָּאָרֶץ / ba'a'rets ) WHICH ( אֲשֶׁר / a'sher ) YOU(MP) ( אַתֶּם / a'tem ) CROSS.OVER (Verb) ~ing(mp) ( עֹבְרִים / ov'rim ) THERE ~unto ( שָׁמָּה / sha'mah ) to~ >~ POSSESS (Verb) ~her ( לְרִשְׁתָּהּ / lê'rish'tah )

and this is the directive, the customs, and the decisions, which YHWH your Elohiym directed, to teach you to do in the land, which you are crossing over unto there to possess her.

2 to~ THAT ( לְמַעַן / lê'ma'an ) you(ms)~ will~ FEAR (Verb) ( תִּירָא / ti'ra ) AT ( אֶת / et ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) to~ >~ SAFEGUARD (Verb) ( לִשְׁמֹר / lish'mor ) AT ( אֶת / et ) ALL ( כָּל / kol ) CUSTOM ~s ~him ( חֻקֹּתָיו / hhu'qo'taw ) and~ DIRECTIVE ~s ~him ( וּמִצְוֹתָיו / u'mits'o'taw ) WHICH ( אֲשֶׁר / a'sher ) I ( אָנֹכִי / a'no'khi ) much~ DIRECT (Verb) ~ing(ms) ~you(ms) ( מְצַוֶּךָ / mê'tsa'we'kha ) YOU(MS) ( אַתָּה / a'tah ) and~ SON ~you(ms) ( וּבִנְךָ / u'vin'kha ) and~ SON ( וּבֶן / u'ven ) SON ~you(ms) ( בִּנְךָ / bin'kha ) ALL ( כֹּל / kol ) DAY ~s ( יְמֵי / yê'mey ) LIVING ~s ~you(ms) ( חַיֶּיךָ / hhei'yey'kha ) and~ to~ THAT ( וּלְמַעַן / ul'ma'an ) they(m)~ will~ make~ PROLONG (Verb) ~must ( יַאֲרִכֻן / ya'a'ri'khun ) DAY ~s ~you(ms) ( יָמֶיךָ / ya'mey'kha )

So that you will fear YHWH your Elohiym, to safeguard all his customs and his directives, which I am directing you, you and your son and the son of your son, all the days of your life, and so that you will make your days prolonged,

3 and~ you(ms)~ did~ HEAR (Verb) ( וְשָׁמַעְתָּ / wê'sha'ma'ta ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) and~ you(ms)~ did~ SAFEGUARD (Verb) ( וְשָׁמַרְתָּ / wê'sha'mar'ta ) to~ >~ DO (Verb) ( לַעֲשׂוֹת / la'a'sot ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ DO.WELL (Verb) ( יִיטַב / yi'tav ) to~ ~you(ms) ( לְךָ / lê'kha ) and~ WHICH ( וַאֲשֶׁר / wa'a'sher ) you(mp)~ will~ INCREASE (Verb) ~must ( תִּרְבּוּן / tir'bun ) MANY ( מְאֹד / mê'od ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) YHWH ( יְהוָה / YHWH ) Elohiym ( אֱלֹהֵי / e'lo'hey ) FATHER ~s ~you(ms) ( אֲבֹתֶיךָ / a'vo'tey'kha ) to~ ~you(fs) ( לָךְ / lakh ) LAND ( אֶרֶץ / e'rets ) ISSUE (Verb) ~ing(fs) ( זָבַת / za'vat ) FAT ( חָלָב / hha'lav ) and~ HONEY ( וּדְבָשׁ / ud'vash )

and you will hear, Yisra'eyl, and you will safeguard to do, that it will go well for you, and that you will certainly greatly increase, just as YHWH the Elohiym of your fathers spoke to you, a land issuing fat and honey.

4 !(ms)~ HEAR (Verb) ( שְׁמַע / shê'ma ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) YHWH ( יְהוָה / YHWH ) Elohiym ~us ( אֱלֹהֵינוּ / e'lo'hey'nu ) YHWH ( יְהוָה / YHWH ) UNIT ( אֶחָד / e'hhad ) [ 960 ]

Hear Yisra'eyl, YHWH our Elohiym, YHWH a unit,

5 and~ you(ms)~ did~ LOVE (Verb) ( וְאָהַבְתָּ / wê'a'hav'ta ) AT ( אֵת / eyt ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) in~ ALL ( בְּכָל / bê'khol ) HEART ~you(ms) ( לְבָבְךָ / lê'vav'kha ) and~ in~ ALL ( וּבְכָל / uv'khol ) SOUL ~you(ms) ( נַפְשְׁךָ / naph'she'kha ) and~ in~ ALL ( וּבְכָל / uv'khol ) MANY ~you(ms) ( מְאֹדֶךָ / mê'o'de'kha )

and you will love YHWH your Elohiym with all your heart and with all your soul and with all your everything,

6 and~ they~ did~ EXIST (Verb) ( וְהָיוּ / wê'hai'u ) the~ WORD ~s ( הַדְּבָרִים / ha'de'va'rim ) the~ THESE ( הָאֵלֶּה / ha'ey'leh ) WHICH ( אֲשֶׁר / a'sher ) I ( אָנֹכִי / a'no'khi ) much~ DIRECT (Verb) ~ing(ms) ~you(ms) ( מְצַוְּךָ / mê'tsa'we'kha ) the~ DAY ( הַיּוֹם / hai'yom ) UPON ( עַל / al ) HEART ~you(ms) ( לְבָבֶךָ / lê'va've'kha )

and these words, which I am directing you today, will exist upon your heart,

7 and~ you(ms)~ did~ much~ WHET (Verb) ~them(m) ( וְשִׁנַּנְתָּם / wê'shi'nan'tam ) to~ SON ~s ~you(ms) ( לְבָנֶיךָ / lê'va'ney'kha ) and~ you(ms)~ did~ much~ SPEAK (Verb) ( וְדִבַּרְתָּ / wê'di'bar'ta ) in~ ~them(m) ( בָּם / bam ) in~ >~ SETTLE (Verb) ~you(ms) ( בְּשִׁבְתְּךָ / bê'shiv'te'kha ) in~ HOUSE ~you(ms) ( בְּבֵיתֶךָ / bê'vey'te'kha ) and~ in~ >~ WALK (Verb) ~you(ms) ( וּבְלֶכְתְּךָ / uv'lekh'te'kha ) in~ the~ ROAD ( בַדֶּרֶךְ / va'de'rekh ) and~ in~ >~ LIE.DOWN (Verb) ~you(ms) ( וּבְשָׁכְבְּךָ / uv'shakh'be'kha ) and~ in~ >~ RISE (Verb) ~you(ms) ( וּבְקוּמֶךָ / uv'qu'me'kha )

and you will whet them for your sons, and you will speak with them in your settlings, in your house and in your walking in the road, and your lying down, and in your rising,

8 and~ you(ms)~ did~ TIE (Verb) ~them(m) ( וּקְשַׁרְתָּם / uq'shar'tam ) to~ SIGN ( לְאוֹת / lê'ot ) UPON ( עַל / al ) HAND ~you(ms) ( יָדֶךָ / ya'de'kha ) and~ they~ did~ EXIST (Verb) ( וְהָיוּ / wê'hai'u ) to~ MARKER ~s ( לְטֹטָפֹת / lê'to'ta'phot ) BETWEEN ( בֵּין / beyn ) EYE ~s2 ~you(ms) ( עֵינֶיךָ / ey'ney'kha )

and you will tie them for a sign upon your hand, and they will exist for markers between your eyes,

9 and~ you(ms)~ did~ WRITE (Verb) ~them(m) ( וּכְתַבְתָּם / ukh'tav'tam ) UPON ( עַל / al ) DOORPOST ~s ( מְזוּזֹת / mê'zu'zot ) HOUSE ~you(ms) ( בֵּיתֶךָ / bey'te'kha ) and~ in~ GATE ~s ~you(ms) ( וּבִשְׁעָרֶיךָ / u'vish'a'rey'kha )

and you will write them upon the doorposts of your house and on your gates,

10 and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) GIVEN.THAT ( כִּי / ki ) he~ will~ make~ COME (Verb) ~you(ms) ( יְבִיאֲךָ / yê'vi'a'kha ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) TO ( אֶל / el ) the~ LAND ( הָאָרֶץ / ha'a'rets ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ be~ SWEAR (Verb) ( נִשְׁבַּע / nish'ba ) to~ FATHER ~s ~you(ms) ( לַאֲבֹתֶיךָ / la'a'vo'tey'kha ) to~ Avraham ( לְאַבְרָהָם / lê'av'ra'ham ) to~ Yits'hhaq ( לְיִצְחָק / lê'yits'hhaq ) and~ to~ Ya'aqov ( וּלְיַעֲקֹב / ul'ya'a'qov ) to~ >~ GIVE (Verb) ( לָתֶת / la'tet ) to~ ~you(fs) ( לָךְ / lakh ) CITY ~s ( עָרִים / a'rim ) GREAT ~s ( גְּדֹלֹת / gê'do'lot ) and~ FUNCTIONAL ~s ( וְטֹבֹת / wê'to'vot ) WHICH ( אֲשֶׁר / a'sher ) NOT ( לֹא / lo ) you(ms)~ did~ BUILD (Verb) ( בָנִיתָ / va'ni'ta )

and it came to pass, given that YHWH your Elohiym will bring you to the land which he swore to your fathers, to Avraham, to Yits'hhaq, and to Ya'aqov, to give to you great and functional cities, which you did not build,

11 and~ HOUSE ~s ( וּבָתִּים / u'va'tim ) FULL ~s ( מְלֵאִים / mê'ley'im ) ALL ( כָּל / kol ) FUNCTIONAL ( טוּב / tuv ) WHICH ( אֲשֶׁר / a'sher ) NOT ( לֹא / lo ) you(ms)~ did~ much~ FILL (Verb) ( מִלֵּאתָ / mi'ley'ta ) and~ CISTERN ~s ( וּבֹרֹת / u'vo'rot ) DIVIDE (Verb) ~ed(mp) ( חֲצוּבִים / hha'tsu'vim ) WHICH ( אֲשֶׁר / a'sher ) NOT ( לֹא / lo ) you(ms)~ did~ HEW (Verb) ( חָצַבְתָּ / hha'tsav'ta ) VINEYARD ~s ( כְּרָמִים / kê'ra'mim ) and~ OLIVE ~s ( וְזֵיתִים / wê'zey'tim ) WHICH ( אֲשֶׁר / a'sher ) NOT ( לֹא / lo ) you(ms)~ did~ PLANT (Verb) ( נָטָעְתָּ / na'ta'ta ) and~ you(ms)~ did~ EAT (Verb) ( וְאָכַלְתָּ / wê'a'khal'ta ) and~ you(ms)~ did~ BE.SATISFIED (Verb) ( וְשָׂבָעְתָּ / wê'sa'va'ta )

and houses full of every functional thing which you did not fill, and divided cisterns which you did not hew, vineyards and olives which you did not plant, and you will eat and you will be satisfied.

12 !(ms)~ be~ SAFEGUARD (Verb) ( הִשָּׁמֶר / hi'sha'mer ) to~ ~you(ms) ( לְךָ / lê'kha ) OTHERWISE ( פֶּן / pen ) you(ms)~ will~ FORGET (Verb) ( תִּשְׁכַּח / tish'kahh ) AT ( אֶת / et ) YHWH ( יְהוָה / YHWH ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ make~ GO.OUT (Verb) ~you(ms) ( הוֹצִיאֲךָ / ho'tsi'a'kha ) from~ LAND ( מֵאֶרֶץ / mey'e'rets ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) from~ HOUSE ( מִבֵּית / mi'beyt ) SERVANT ~s ( עֲבָדִים / a'va'dim )

Safeguard yourselves, otherwise you will forget YHWH who brought you out from the land of Mits'rayim, from the house of servants.

13 AT ( אֶת / et ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) you(ms)~ will~ FEAR (Verb) ( תִּירָא / ti'ra ) and~ AT ~him ( וְאֹתוֹ / wê'o'to ) you(ms)~ will~ SERVE (Verb) ( תַעֲבֹד / ta'a'vod ) and~ in~ TITLE ~him ( וּבִשְׁמוֹ / u'vish'mo ) you(ms)~ will~ be~ SWEAR (Verb) ( תִּשָּׁבֵעַ / ti'sha'vey'a )

YHWH your Elohiym you will fear, and him you will serve, and in his title you will swear.

14 NOT ( לֹא / lo ) you(mp)~ will~ WALK (Verb) ~must ( תֵלְכוּן / teyl'khun ) AFTER ( אַחֲרֵי / a'hha'rey ) Elohiym ( אֱלֹהִים / e'lo'him ) OTHER ~s ( אֲחֵרִים / a'hhey'rim ) from~ Elohiym ( מֵאֱלֹהֵי / mey'e'lo'hey ) the~ PEOPLE ~s ( הָעַמִּים / ha'a'mim ) WHICH ( אֲשֶׁר / a'sher ) ALL.AROUND ~s ~you(mp) ( סְבִיבוֹתֵיכֶם / sê'vi'vo'tey'khem )

You must not walk after other Elohiym from the Elohiym of the people which are all around you.

15 GIVEN.THAT ( כִּי / ki ) MIGHTY.ONE ( אֵל / eyl ) ZEALOUS ( קַנָּא / qa'na ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) in~ INSIDE ~you(ms) ( בְּקִרְבֶּךָ / bê'qir'be'kha ) OTHERWISE ( פֶּן / pen ) he~ will~ FLARE.UP (Verb) ( יֶחֱרֶה / ye'hhe'reh ) NOSE ( אַף / aph ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) in~ ~you(fs) ( בָּךְ / bakh ) and~ he~ did~ make~ DESTROY (Verb) ~you(ms) ( וְהִשְׁמִידְךָ / wê'hish'mid'kha ) from~ UPON ( מֵעַל / mey'al ) FACE ~s ( פְּנֵי / pê'ney ) the~ GROUND ( הָאֲדָמָה / ha'a'da'mah )

Given that YHWH your Elohiym is a zealous mighty one in among you, otherwise, YHWH your Elohiym will flare up the nose with you, and he will destroy you from upon the face of the ground.

16 NOT ( לֹא / lo ) you(mp)~ will~ TEST (Verb) ( תְנַסּוּ / tê'na'su ) AT ( אֶת / et ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(mp) ( אֱלֹהֵיכֶם / e'lo'hey'khem ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) you(mp)~ did~ much~ TEST (Verb) ( נִסִּיתֶם / ni'si'tem ) in~ Mas'sah ( בַּמַּסָּה / ba'ma'sah ) [ 961 ]

You will not test YHWH your Elohiym, just as you tested in Mas'sah.

17 >~ SAFEGUARD (Verb) ( שָׁמוֹר / sha'mor ) you(mp)~ will~ SAFEGUARD (Verb) ~must ( תִּשְׁמְרוּן / tish'me'run ) AT ( אֶת / et ) DIRECTIVE ~s ( מִצְוֹת / mits'ot ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(mp) ( אֱלֹהֵיכֶם / e'lo'hey'khem ) and~ WITNESS ~s ~him ( וְעֵדֹתָיו / wê'ey'do'taw ) and~ CUSTOM ~s ~him ( וְחֻקָּיו / wê'hhu'qaw ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ DIRECT (Verb) ~you(ms) ( צִוָּךְ / tsi'wakh )

You must certainly safeguard the directives of YHWH your Elohiym, and his witnesses and his customs, which he directed you,

18 and~ you(ms)~ did~ DO (Verb) ( וְעָשִׂיתָ / wê'a'si'ta ) the~ STRAIGHT ( הַיָּשָׁר / hai'ya'shar ) and~ the~ FUNCTIONAL ( וְהַטּוֹב / wê'ha'tov ) in~ EYE ~s2 ( בְּעֵינֵי / bê'ey'ney ) YHWH ( יְהוָה / YHWH ) to~ THAT ( לְמַעַן / lê'ma'an ) he~ will~ DO.WELL (Verb) ( יִיטַב / yi'tav ) to~ ~you(fs) ( לָךְ / lakh ) and~ you(ms)~ did~ COME (Verb) ( וּבָאתָ / u'va'ta ) and~ you(ms)~ did~ POSSESS (Verb) ( וְיָרַשְׁתָּ / wê'ya'rash'ta ) AT ( אֶת / et ) the~ LAND ( הָאָרֶץ / ha'a'rets ) the~ FUNCTIONAL ( הַטֹּבָה / ha'to'vah ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ be~ SWEAR (Verb) ( נִשְׁבַּע / nish'ba ) YHWH ( יְהוָה / YHWH ) to~ FATHER ~s ~you(ms) ( לַאֲבֹתֶיךָ / la'a'vo'tey'kha )

and you did the straight and functional thing in the eyes of YHWH, so that he will do well for you, and you will come and you will possess the functional land, which YHWH sword to your fathers.

19 to~ >~ EXPEL (Verb) ( לַהֲדֹף / la'ha'doph ) AT ( אֶת / et ) ALL ( כָּל / kol ) ATTACK (Verb) ~ing(mp) ~s ~you(ms) ( אֹיְבֶיךָ / oi'vey'kha ) from~ FACE ~s ~you(ms) ( מִפָּנֶיךָ / mi'pa'ney'kha ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) YHWH ( יְהוָה / YHWH )

Given that your son will inquire of you tomorrow saying, what are the witnesses, and the customs, and the decisions, which YHWH our Elohiym directed you,
