Deuteronomy Chapter 4

and now Yisra'eyl listen to the customs and to the decisions which I am teaching you to do so that you will live, and you will come and you will possess the land which YHWH, the Elohiym of your fathers, is giving to you.

2 NOT (לֹא / lo) you(mp)~will~make~ADD(Verb) (תֹסִפוּ / to'si'phu) UPON (עַל / al) the~WORD (הַדָּבָר / ha'da'var) WHICH (אֲשֶׁר / a'sher) I (אָנֹכִי / a'no'khi) much~DIRECT(Verb)~ing(ms) (מְצַוֶּה / mê'tsa'weh) AT~you(mp) (אֶתְכֶם / et'khem) and~NOT (וְלֹא / wê'lo) you(mp)~will~TAKE.AWAY(Verb) (תִגְרְעוּ / tig're'u) FROM~him (מִמֶּנּוּ / mi'me'nu) to~>~SAFEGUARD(Verb) (לִשְׁמֹר / lish'mor) AT (אֶת / et) DIRECTIVE~s (מִצְוֹת / mits'ot) YHWH (יְהוָה / YHWH) Elohiym~you(mp) (אֱלֹהֵיכֶם / e'lo'hey'khem) WHICH (אֲשֶׁר / a'sher) I (אָנֹכִי / a'no'khi) much~DIRECT(Verb)~ing(ms) (מְצַוֶּה / mê'tsa'weh) AT~you(mp) (אֶתְכֶם / et'khem)

You will not add upon the word, which I am directing you, and you will not take away from him, safeguard the directives of YHWH your Elohiym, which I am directing you.

3 EYE~s2~you(mp) (עֵינֵיכֶם / ey'ney'khem) the~SEE(Verb)~ing(fp) (הָרֹאֹת / ha'ro'ot) AT (אֵת / eyt) WHICH (אֲשֶׁר / a'sher) he~did~DO(Verb) (עָשָׂה / a'sah) YHWH (יְהוָה / YHWH) in~Ba'al-Pe'or (בְּבַעַל פְּעוֹר / bê'va'al Pê'or) GIVEN.THAT (כִּי / ki) ALL (כָל / khol) the~MAN (הָאִישׁ / ha'ish) WHICH (אֲשֶׁר / a'sher) he~did~WALK(Verb) (הָלַךְ / ha'lakh) AFTER (אַחֲרֵי / a'hha'rey) Ba'al-Pe'or (בַעַל פְּעוֹר / va'al pê'or) he~did~make~DESTROY(Verb)~him (הִשְׁמִידוֹ / hish'mi'do) YHWH (יְהוָה / YHWH) Elohiym~you(ms) (אֱלֹהֶיךָ / e'lo'hey'kha) from~INSIDE~you(ms) (מִקִּרְבֶּךָ / mi'qir'be'kha)

Your eyes are seeing what YHWH did in Ba'al Pe'or, given that every man that walked after Ba'al Pe'or YHWH your Elohiym destroyed from inside you,

4 and~YOU(MP) (וְאַתֶּם / wê'a'tem) the~FASTENER~s (הַדְּבֵקִים / ha'de'vey'qim) in~YHWH (בַּיהוָה / ba'YHWH) Elohiym~you(mp) (אֱלֹהֵיכֶם / e'lo'hey'khem) LIVING~s (חַיִּים / hhai'yim) ALL~you(mp) (כֻּלְּכֶם / kul'khem) the~DAY (הַיּוֹם / hai'yom)

and you, the fasteners with YHWH your Elohiym, all of you are living today.

5 !(ms)~SEE(Verb) (רְאֵה / rê'eyh) I~did~much~LEARN(Verb) (לִמַּדְתִּי / lim'de'ti) AT~you(mp) (אֶתְכֶם / et'khem) CUSTOM~s (חֻקִּים / hhu'qim) and~DECISION~s (וּמִשְׁפָּטִים / u'mish'pa'tim) like~WHICH (כַּאֲשֶׁר / ka'a'sheyr) he~did~much~DIRECT(Verb)~me (צִוַּנִי / tsi'wa'ni) YHWH (יְהוָה / YHWH) Elohiym~me (אֱלֹהָי / e'lo'hai) to~>~DO(Verb) (לַעֲשׂוֹת / la'a'sot) SO (כֵּן / keyn) in~INSIDE (בְּקֶרֶב / bê'qe'rev) the~LAND (הָאָרֶץ / ha'a'rets) WHICH (אֲשֶׁר / a'sher) YOU(MP) (אַתֶּם / a'tem) COME(Verb)~ing(mp) (בָּאִים / ba'im) THERE~unto (שָׁמָּה / sha'mah) to~>~POSSESS(Verb)~her (לְרִשְׁתָּהּ / lê'rish'tah)

See, I taught you customs and decisions, just as YHWH my Elohiym did, so it is inside the land when you come unto there to possess her,

6 and~you(mp)~did~SAFEGUARD(Verb) (וּשְׁמַרְתֶּם / ush'mar'tem) and~you(mp)~did~DO(Verb) (וַעֲשִׂיתֶם / wa'a'si'tem) GIVEN.THAT (כִּי / ki) SHE (הִוא / hi) SKILL~you(mp) (חָכְמַתְכֶם / hhakh'mat'khem) and~UNDERSTANDING~you(mp) (וּבִינַתְכֶם / u'vi'nat'khem) to~EYE~s2 (לְעֵינֵי / lê'ey'ney) the~PEOPLE~s (הָעַמִּים / ha'a'mim) WHICH (אֲשֶׁר / a'sher) they(m)~will~HEAR(Verb)~must (יִשְׁמְעוּן / yish'me'un) AT (אֵת / eyt) ALL (כָּל / kol) the~CUSTOM~s (הַחֻקִּים / ha'hhu'qim) the~THESE (הָאֵלֶּה / ha'ey'leh) and~they~did~SAY(Verb) (וְאָמְרוּ / wê'am'ru) ONLY (רַק / raq) PEOPLE (עַם / am) SKILLED.ONE (חָכָם / hha'kham) and~be~UNDERSTAND(Verb)~ing(ms) (וְנָבוֹן / wê'na'von) the~NATION (הַגּוֹי / ha'goi) the~GREAT (הַגָּדוֹל / ha'ga'dol) the~THIS (הַזֶּה / ha'zeh)

and you will safeguard, and you will do, given that she is your skill and your understanding to the eyes of the peoples, who must hear all these customs, and they will say, only a people skilled and with understanding is this great nation.

7 GIVEN.THAT (כִּי / ki) WHO (מִי / mi) NATION (גוֹי / goy) GREAT (גָּדוֹל / ga'dol) WHICH (אֲשֶׁר / a'sher) to~~him (לוֹ / lo) Elohiym (אֱלֹהִים / e'lo'him) NEAR~s (קְרֹבִים / qê'ro'vim) TO~him (אֵלָיו / ey'law) like~YHWH (כַּיהוָה / ka'YHWH) Elohiym~us (אֱלֹהֵינוּ / e'lo'hey'nu) in~ALL (בְּכָל / bê'khol) >~CALL.OUT(Verb)~us (קָרְאֵנוּ / qar'ey'nu) TO~him (אֵלָיו / ey'law)

Given that what great nation that belongs to him an Elohiym near to him like our Elohiym YHWH, in all we called out to him,

8 and~WHO (וּמִי / u'mi) NATION (גּוֹי / goy) GREAT (גָּדוֹל / ga'dol) WHICH (אֲשֶׁר / a'sher) to~~him (לוֹ / lo) CUSTOM~s (חֻקִּים / hhu'qim) and~DECISION~s (וּמִשְׁפָּטִים / u'mish'pa'tim) STEADFAST.ONE~s (צַדִּיקִם / tsa'di'qim) like~ALL (כְּכֹל / kê'khol) the~TEACHING (הַתּוֹרָה / ha'to'rah) the~THIS (הַזֹּאת / ha'zot) WHICH (אֲשֶׁר / a'sher) I (אָנֹכִי / a'no'khi) GIVE(Verb)~ing(ms) (נֹתֵן / no'teyn) to~FACE~s~you(mp) (לִפְנֵיכֶם / liph'ney'khem) the~DAY (הַיּוֹם / hai'yom)

and what great nation that belongs to him steadfast customs and decisions like all this teaching that I am giving to your faces today.

9 ONLY (רַק / raq) !(ms)~be~SAFEGUARD(Verb) (הִשָּׁמֶר / hi'sha'mer) to~~you(ms) (לְךָ / lê'kha) and~>~SAFEGUARD(Verb) (וּשְׁמֹר / ush'mor) SOUL~you(ms) (נַפְשְׁךָ / naph'she'kha) MANY (מְאֹד / mê'od) OTHERWISE (פֶּן / pen) you(ms)~will~FORGET(Verb) (תִּשְׁכַּח / tish'kahh) AT (אֶת / et) the~WORD~s (הַדְּבָרִים / ha'de'va'rim) WHICH (אֲשֶׁר / a'sher) they~did~SEE(Verb) (רָאוּ / ra'u) EYE~s2~you(ms) (עֵינֶיךָ / ey'ney'kha) and~OTHERWISE (וּפֶן / u'phen) they(m)~will~TURN.ASIDE(Verb) (יָסוּרוּ / ya'su'ru) from~HEART~you(ms) (מִלְּבָבְךָ / mil'vav'kha) ALL (כֹּל / kol) DAY~s (יְמֵי / yê'mey) LIVING~s~you(ms) (חַיֶּיךָ / hhei'yey'kha) and~you(mp)~did~make~KNOW(Verb)~them(m) (וְהוֹדַעְתָּם / wê'ho'da'tam) to~SON~s~you(ms) (לְבָנֶיךָ / lê'va'ney'kha) and~to~SON~s (וְלִבְנֵי / wê'liv'ney) SON~s~you(ms) (בָנֶיךָ / va'ney'kha)

Be safeguarded for yourself and safeguard your soul much, otherwise you will forget the things which your eyes saw, and otherwise they will turn aside from your heart all the days of your life, and you will make them known to your sons and to the sons of your sons.

10 DAY (יוֹם / yom) WHICH (אֲשֶׁר / a'sher) you(ms)~did~STAND(Verb) (עָמַדְתָּ / a'mad'ta) to~FACE~s (לִפְנֵי / liph'ney) YHWH (יְהוָה / YHWH) Elohiym~you(ms) (אֱלֹהֶיךָ / e'lo'hey'kha) in~Hhorev (בְּחֹרֵב / bê'hho'reyv) in~>~SAY(Verb) (בֶּאֱמֹר / be'e'mor) YHWH (יְהוָה / YHWH) TO~me (אֵלַי / ey'lai) !(mp)~make~ASSEMBLE(Verb) (הַקְהֶל / haq'hel) to~~me (לִי / li) AT (אֶת / et) the~PEOPLE (הָעָם / ha'am) and~I~will~make~HEAR(Verb)~them(m) (וְאַשְׁמִעֵם / wê'ash'mi'eym) AT (אֶת / et) WORD~s~me (דְּבָרָי / dê'va'rai) WHICH (אֲשֶׁר / a'sher) they(m)~will~LEARN(Verb)~must (יִלְמְדוּן / yil'me'dun) to~>~FEAR(Verb) (לְיִרְאָה / lê'yir'ah) AT~me (אֹתִי / o'ti) ALL (כָּל / kol) the~DAY~s (הַיָּמִים / hai'ya'mim) WHICH (אֲשֶׁר / a'sher) THEY(m) (הֵם / heym) LIVING~s (חַיִּים / hhai'yim) UPON (עַל / al) the~GROUND (הָאֲדָמָה / ha'a'da'mah) and~AT (וְאֶת / wê'et) SON~s~them(m) (בְּנֵיהֶם / bê'ney'hem) they(m)~will~much~LEARN(Verb)~must (יְלַמֵּדוּן / yê'la'mey'dun)

The day when you stood to the face of YHWH your Elohiym in Hhorev, with YHWH saying to me, assemble for me the people, and I will make them hear my words, because they must learn to fear me all the days which they are living upon the ground, and their sons they must teach,

11 and~you(mp)~will~COME.NEAR(Verb)~must (וַתִּקְרְבוּן / wa'tiq're'vun) and~you(mp)~will~STAND(Verb)~must (וַתַּעַמְדוּן / wa'ta'am'dun) UNDER (תַּחַת / ta'hhat) the~HILL (הָהָר / ha'har) and~the~HILL (וְהָהָר / wê'ha'har) BURN(Verb)~ing(ms) (בֹּעֵר / bo'eyr) in~the~FIRE (בָּאֵשׁ / ba'eysh) UNTIL (עַד / ad) HEART (לֵב / leyv) the~SKY~s2 (הַשָּׁמַיִם / ha'sha'ma'yim) DARKNESS (חֹשֶׁךְ / hho'shekh) CLOUD (עָנָן / a'nan) and~THICK.DARKNESS (וַעֲרָפֶל / wa'a'ra'phel)

and you must come near, and you must stand under the hill, and the hill is burning with fire unto the heart of the skies, a dark cloud and thick darkness,

12 and~he~will~much~SPEAK(Verb) (וַיְדַבֵּר / wai'da'beyr) YHWH (יְהוָה / YHWH) TO~you(mp) (אֲלֵיכֶם / a'ley'khem) from~MIDST (מִתּוֹךְ / mi'tokh) the~FIRE (הָאֵשׁ / ha'eysh) VOICE (קוֹל / qol) WORD~s (דְּבָרִים / dê'va'rim) YOU(MP) (אַתֶּם / a'tem) HEAR(Verb)~ing(mp) (שֹׁמְעִים / shom'im) and~RESEMBLANCE (וּתְמוּנָה / ut'mu'nah) WITHOUT~you(mp) (אֵינְכֶם / eyn'khem) SEE(Verb)~ing(mp) (רֹאִים / ro'im) WITH.THE.EXCEPTION~of (זוּלָתִי / zu'la'ti) VOICE (קוֹל / qol)

and YHWH spoke to you from the midst of the fire, a voice of words you are hearing, and a resemblance you are not seeing, with the exception of the voice,

13 and~he~will~make~BE.FACE.TO.FACE(Verb) (וַיַּגֵּד / wai'ya'geyd) to~~you(mp) (לָכֶם / la'khem) AT (אֶת / et) COVENANT~him (בְּרִיתוֹ / bê'ri'to) WHICH (אֲשֶׁר / a'sher) he~did~much~DIRECT(Verb) (צִוָּה / tsi'wah) AT~you(mp) (אֶתְכֶם / et'khem) to~>~DO(Verb) (לַעֲשׂוֹת / la'a'sot) TEN (עֲשֶׂרֶת / a'se'ret) the~WORD~s (הַדְּבָרִים / ha'de'va'rim) and~he~will~WRITE(Verb)~them(m) (וַיִּכְתְּבֵם / wai'yikh'te'veym) UPON (עַל / al) TWO (שְׁנֵי / shê'ney) SLAB~s (לֻחוֹת / lu'hhot) STONE~s (אֲבָנִים / a'va'nim)

and he told you his covenant which he directed you to do, the ten words, and he wrote them upon two slabs of stone,

14 and~AT~me (וְאֹתִי / wê'o'ti) he~did~much~DIRECT(Verb) (צִוָּה / tsi'wah) YHWH (יְהוָה / YHWH) in~the~APPOINTED.TIME (בָּעֵת / ba'eyt) the~SHE (הַהִוא / ha'hi) to~>~much~LEARN(Verb) (לְלַמֵּד / lê'la'meyd) AT~you(mp) (אֶתְכֶם / et'khem) CUSTOM~s (חֻקִּים / hhu'qim) and~DECISION~s (וּמִשְׁפָּטִים / u'mish'pa'tim) to~>~DO(Verb)~you(mp) (לַעֲשֹׂתְכֶם / la'a'sot'khem) AT~them(m) (אֹתָם / o'tam) in~the~LAND (בָּאָרֶץ / ba'a'rets) WHICH (אֲשֶׁר / a'sher) YOU(MP) (אַתֶּם / a'tem) CROSS.OVER(Verb)~ing(mp) (עֹבְרִים / ov'rim) THERE~unto (שָׁמָּה / sha'mah) to~>~POSSESS(Verb)~her (לְרִשְׁתָּהּ / lê'rish'tah)

and YHWH directed me in that appointed time to teach you customs and decisions, for you to do them in the land which you are crossing over unto there to possess her,

15 and~you(mp)~will~be~SAFEGUARD(Verb) (וְנִשְׁמַרְתֶּם / wê'nish'mar'tem) MANY (מְאֹד / mê'od) to~SOUL~s~you(mp) (לְנַפְשֹׁתֵיכֶם / lê'naph'sho'tey'khem) GIVEN.THAT (כִּי / ki) NOT (לֹא / lo) you(mp)~did~SEE(Verb) (רְאִיתֶם / rê'i'tem) ALL (כָּל / kol) RESEMBLANCE (תְּמוּנָה / tê'mu'nah) in~DAY (בְּיוֹם / bê'yom) he~did~much~SPEAK(Verb) (דִּבֶּר / di'ber) YHWH (יְהוָה / YHWH) TO~you(mp) (אֲלֵיכֶם / a'ley'khem) in~Hhorev (בְּחֹרֵב / bê'hho'reyv) from~MIDST (מִתּוֹךְ / mi'tokh) the~FIRE (הָאֵשׁ / ha'eysh)

and you will carefully safeguard your souls, given that you did not see any resemblance in the day YHWH spoke to you in Hhorev from the midst of the fire.

16 OTHERWISE (פֶּן / pen) you(mp)~will~make~DAMAGE(Verb)~must (תַּשְׁחִתוּן / tash'hhi'tun) and~you(mp)~did~DO(Verb) (וַעֲשִׂיתֶם / wa'a'si'tem) to~~you(mp) (לָכֶם / la'khem) SCULPTURE (פֶּסֶל / pe'sel) RESEMBLANCE (תְּמוּנַת / tê'mu'nat) ALL (כָּל / kol) FIGURE (סָמֶל / sa'mel) PATTERN (תַּבְנִית / tav'nit) MALE (זָכָר / za'khar) OR (אוֹ / o) FEMALE (נְקֵבָה / nê'qey'vah)

Otherwise, you will cause damage, and you will make for yourselves a sculpture of resemblance of every figure, a pattern of male or female.

17 PATTERN (תַּבְנִית / tav'nit) ALL (כָּל / kol) BEAST (בְּהֵמָה / bê'hey'mah) WHICH (אֲשֶׁר / a'sher) in~the~LAND (בָּאָרֶץ / ba'a'rets) PATTERN (תַּבְנִית / tav'nit) ALL (כָּל / kol) BIRD (צִפּוֹר / tsi'por) WING (כָּנָף / ka'naph) WHICH (אֲשֶׁר / a'sher) she~will~FLY(Verb) (תָּעוּף / ta'uph) in~the~SKY~s2 (בַּשָּׁמָיִם / ba'sha'ma'yim)

A pattern of every beast which is in the land, a pattern of every bird of the wing which will fly in the skies.
