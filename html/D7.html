Given that YHWH your Elohiym brought you to the land, which you came unto there to possess, and he cast off the abundant nations from your face, the ones of Hhet, and the ones of Girgash, and the ones of Emor, and the ones of Kena'an, and the ones of Perez, and the ones of Hhiw, and the ones of Yevus, seven nations more abundant and numerous nations than you,

2 and~ he~ did~ GIVE (Verb) ~them(m) ( וּנְתָנָם / un'ta'nam ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) to~ FACE ~s ~you(ms) ( לְפָנֶיךָ / lê'pha'ney'kha ) and~ you(ms)~ did~ make~ HIT (Verb) ~them(m) ( וְהִכִּיתָם / wê'hi'ki'tam ) >~ make~ PERFORATE (Verb) ( הַחֲרֵם / ha'hha'reym ) you(ms)~ will~ make~ PERFORATE (Verb) ( תַּחֲרִים / ta'hha'rim ) AT ~them(m) ( אֹתָם / o'tam ) NOT ( לֹא / lo ) you(ms)~ will~ CUT (Verb) ( תִכְרֹת / tikh'rot ) to~ ~them(m) ( לָהֶם / la'hem ) COVENANT ( בְּרִית / bê'rit ) and~ NOT ( וְלֹא / wê'lo ) you(ms)~ will~ PROVIDE.PROTECTION (Verb) ~them(m) ( תְחָנֵּם / tê'hha'neym )

and YHWH your Elohiym will give them to your face, and you will attack them, you will completely perforate them, you will not cut them a covenant, and you will not provide them protection,

3 and~ NOT ( וְלֹא / wê'lo ) you(ms)~ will~ self~ BE.AN.IN-LAW (Verb) ( תִתְחַתֵּן / tit'hha'teyn ) in~ ~them(m) ( בָּם / bam ) DAUGHTER ~you(ms) ( בִּתְּךָ / bit'kha ) NOT ( לֹא / lo ) you(ms)~ will~ GIVE (Verb) ( תִתֵּן / ti'teyn ) to~ SON ~him ( לִבְנוֹ / liv'no ) and~ DAUGHTER ~him ( וּבִתּוֹ / u'vi'to ) NOT ( לֹא / lo ) you(ms)~ will~ TAKE (Verb) ( תִקַּח / ti'qahh ) to~ SON ~you(ms) ( לִבְנֶךָ / liv'ne'kha )

and you will not make yourselves be in-laws with them, you will not give your daughters to his sons, and his daughters you will not take for your sons.

4 GIVEN.THAT ( כִּי / ki ) he~ will~ make~ TURN.ASIDE (Verb) ( יָסִיר / ya'sir ) AT ( אֶת / et ) SON ~you(ms) ( בִּנְךָ / bin'kha ) from~ AFTER ~me ( מֵאַחֲרַי / mey'a'hha'rai ) and~ they~ did~ SERVE (Verb) ( וְעָבְדוּ / wê'av'du ) Elohiym ( אֱלֹהִים / e'lo'him ) OTHER ~s ( אֲחֵרִים / a'hhey'rim ) and~ he~ did~ FLARE.UP (Verb) ( וְחָרָה / wê'hha'rah ) NOSE ( אַף / aph ) YHWH ( יְהוָה / YHWH ) in~ ~you(mp) ( בָּכֶם / ba'khem ) and~ he~ did~ make~ DESTROY (Verb) ~you(ms) ( וְהִשְׁמִידְךָ / wê'hish'mid'kha ) QUICKLY ( מַהֵר / ma'heyr )

Given that he will make your son turn aside from after me, and they will serve other Elohiym, and the nose of YHWH will flare up with you, and he will quickly destroy you.

5 GIVEN.THAT ( כִּי / ki ) IF ( אִם / im ) IN.THIS.WAY ( כֹּה / koh ) you(mp)~ will~ DO (Verb) ( תַעֲשׂוּ / ta'a'su ) to~ ~them(m) ( לָהֶם / la'hem ) ALTAR ~s ~them(m) ( מִזְבְּחֹתֵיהֶם / miz'be'hho'tey'hem ) you(mp)~ will~ BREAK.DOWN (Verb) ( תִּתֹּצוּ / ti'to'tsu ) and~ MONUMENT ~s ~them(m) ( וּמַצֵּבֹתָם / u'ma'tsey'vo'tam ) you(mp)~ will~ much~ CRACK (Verb) ( תְּשַׁבֵּרוּ / tê'sha'bey'ru ) and~ GROVE ~them(m) ( וַאֲשֵׁירֵהֶם / wa'a'shey'rey'hem ) you(mp)~ will~ much~ CUT.DOWN (Verb) ~must ( תְּגַדֵּעוּן / tê'ga'dey'un ) and~ SCULPTURE ~s ~them(m) ( וּפְסִילֵיהֶם / uph'si'ley'hem ) you(mp)~ will~ CREMATE (Verb) ~must ( תִּשְׂרְפוּן / tis're'phun ) in~ the~ FIRE ( בָּאֵשׁ / ba'eysh )

Instead, in this way, you will do to them, their altars you will break down, and their monuments you will crack, and their groves you must cut down, and their sculptures you must cremate in the fire.

6 GIVEN.THAT ( כִּי / ki ) PEOPLE ( עַם / am ) UNIQUE ( קָדוֹשׁ / qa'dosh ) YOU(MS) ( אַתָּה / a'tah ) to~ YHWH ( לַיהוָה / la'YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) in~ ~you(ms) ( בְּךָ / bê'kha ) he~ did~ CHOOSE (Verb) ( בָּחַר / ba'hhar ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) to~ >~ EXIST (Verb) ( לִהְיוֹת / lih'yot ) to~ ~him ( לוֹ / lo ) to~ PEOPLE ( לְעַם / lê'am ) JEWEL ( סְגֻלָּה / sê'gu'lah ) from~ ALL ( מִכֹּל / mi'kol ) the~ PEOPLE ~s ( הָעַמִּים / ha'a'mim ) WHICH ( אֲשֶׁר / a'sher ) UPON ( עַל / al ) FACE ~s ( פְּנֵי / pê'ney ) the~ GROUND ( הָאֲדָמָה / ha'a'da'mah )

Given that you are a unique people for YHWH your Elohiym, with you YHWH your Elohiym chose to exist, to be for himself for a people, a jewel from all the peoples, which are upon the face of the ground.

7 NOT ( לֹא / lo ) from~ >~ INCREASE.IN.NUMBER (Verb) ~you(mp) ( מֵרֻבְּכֶם / mey'ru'be'khem ) from~ ALL ( מִכָּל / mi'kol ) the~ PEOPLE ~s ( הָעַמִּים / ha'a'mim ) he~ did~ ATTACH (Verb) ( חָשַׁק / hha'shaq ) YHWH ( יְהוָה / YHWH ) in~ ~you(mp) ( בָּכֶם / ba'khem ) and~ he~ will~ CHOOSE (Verb) ( וַיִּבְחַר / wai'yiv'hhar ) in~ ~you(mp) ( בָּכֶם / ba'khem ) GIVEN.THAT ( כִּי / ki ) YOU(MP) ( אַתֶּם / a'tem ) the~ SMALL.AMOUNT ( הַמְעַט / ha'me'at ) from~ ALL ( מִכָּל / mi'kol ) the~ PEOPLE ~s ( הָעַמִּים / ha'a'mim )

Not because you are great in number, more than all the peoples, YHWH attached with you, and he chose you, given that you are the fewest from all the peoples.

8 GIVEN.THAT ( כִּי / ki ) from~ AFFECTION ( מֵאַהֲבַת / mey'a'ha'vat ) YHWH ( יְהוָה / YHWH ) AT ~you(mp) ( אֶתְכֶם / et'khem ) and~ >~ SAFEGUARD (Verb) ~him ( וּמִשָּׁמְרוֹ / u'mi'sham'ro ) AT ( אֶת / et ) the~ SWEARING ( הַשְּׁבֻעָה / hash'vu'ah ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ be~ SWEAR (Verb) ( נִשְׁבַּע / nish'ba ) to~ FATHER ~s ~you(mp) ( לַאֲבֹתֵיכֶם / la'a'vo'tey'khem ) he~ did~ make~ GO.OUT (Verb) ( הוֹצִיא / ho'tsi ) YHWH ( יְהוָה / YHWH ) AT ~you(mp) ( אֶתְכֶם / et'khem ) in~ HAND ( בְּיָד / bê'yad ) FORCEFUL ( חֲזָקָה / hha'za'qah ) and~ he~ will~ RANSOM (Verb) ~you(ms) ( וַיִּפְדְּךָ / wai'yiph'de'kha ) from~ HOUSE ( מִבֵּית / mi'beyt ) SERVANT ~s ( עֲבָדִים / a'va'dim ) from~ HAND ( מִיַּד / mi'yad ) Paroh ( פַּרְעֹה / par'oh ) KING ( מֶלֶךְ / me'lekh ) Mits'rayim ( מִצְרָיִם / mits'ra'yim )

Given that from the affection of YHWH for you, and his safeguarding the swearing that he swore to your fathers, YHWH brought you out with a forceful hand, and he ransomed you from the house of servants, from the hand of Paroh, the king of Mits'rayim,

and you knew that YHWH your Elohiym, he is the mighty Elohiym, the one being secure, safeguarding the covenant, and kindness to the ones loving him and safeguarding his directives to a thousand generations,

10 and~ much~ MAKE.RESTITUTION (Verb) ~ing(ms) ( וּמְשַׁלֵּם / um'sha'leym ) [ 963 ] to~ HATE (Verb) ~ing(mp) ~him ( לְשֹׂנְאָיו / lê'son'aw ) TO ( אֶל / el ) FACE ~s ~him ( פָּנָיו / pa'naw ) to~ >~ make~ PERISH (Verb) ~him ( לְהַאֲבִידוֹ / lê'ha'a'vi'do ) NOT ( לֹא / lo ) he~ will~ much~ DELAY (Verb) ( יְאַחֵר / yê'a'hheyr ) to~ HATE (Verb) ~ing(ms) ~him ( לְשֹׂנְאוֹ / lê'son'o ) TO ( אֶל / el ) FACE ~s ~him ( פָּנָיו / pa'naw ) he~ will~ much~ MAKE.RESTITUTION (Verb) ( יְשַׁלֶּם / yê'sha'lem ) to~ ~him ( לוֹ / lo )

and making restitution to the ones hating him to his face, to make him perish, he will not delay to the one hating him to his face, he will make restitution for him,

11 and~ you(ms)~ did~ SAFEGUARD (Verb) ( וְשָׁמַרְתָּ / wê'sha'mar'ta ) AT ( אֶת / et ) the~ DIRECTIVE ( הַמִּצְוָה / ha'mits'wah ) and~ AT ( וְאֶת / wê'et ) the~ CUSTOM ~s ( הַחֻקִּים / ha'hhu'qim ) and~ AT ( וְאֶת / wê'et ) the~ DECISION ~s ( הַמִּשְׁפָּטִים / ha'mish'pa'tim ) WHICH ( אֲשֶׁר / a'sher ) I ( אָנֹכִי / a'no'khi ) much~ DIRECT (Verb) ~ing(ms) ~you(ms) ( מְצַוְּךָ / mê'tsa'we'kha ) the~ DAY ( הַיּוֹם / hai'yom ) to~ >~ DO (Verb) ~them(m) ( לַעֲשׂוֹתָם / la'a'so'tam )

and you will safeguard the directives and the customs and the decisions, which I am directing you today to do them,

and therefore, it will come to pass, you must hear these decisions, and you will safeguard, and you will do them, then YHWH your Elohiym will safeguard you, the covenant and the kindness, which he swore to your fathers,

and he loved you, and he exalted you, and he made you increase, and he exalted the produce of your womb, and the produce of your ground, your cereal and your fresh wine and your fresh oil, the birth of your bovine, and the young sheep of your flocks upon the ground, which he swore to your fathers to give you.

Exalted you will be, more than all the peoples, sterility will not exist in you or in your beast,
