and he will support his hand upon the head of his donation and he will slay him at the opening of the appointed tent, and the sons of Aharon, the administrators, will sprinkle the blood upon the altar all around,

3 and~ he~ did~ make~ COME.NEAR (Verb) ( וְהִקְרִיב / wê'hiq'riv ) from~ SACRIFICE ( מִזֶּבַח / mi'ze'vahh ) the~ OFFERING.OF.RESTITUTION ~s ( הַשְּׁלָמִים / ha'she'la'mim ) FIRE.OFFERING ( אִשֶּׁה / i'sheh ) to~ YHWH ( לַיהוָה / la'YHWH ) AT ( אֶת / et ) the~ FAT ( הַחֵלֶב / ha'hhey'lev ) the~ much~ COVER.OVER (Verb) ~ing(ms) ( הַמְכַסֶּה / ham'kha'seh ) AT ( אֶת / et ) the~ INSIDE ( הַקֶּרֶב / ha'qe'rev ) and~ AT ( וְאֵת / wê'eyt ) ALL ( כָּל / kol ) the~ FAT ( הַחֵלֶב / ha'hhey'lev ) WHICH ( אֲשֶׁר / a'sher ) UPON ( עַל / al ) the~ INSIDE ( הַקֶּרֶב / ha'qe'rev )

and he will bring near, from the sacrifice of the offerings of restitution, a fire offering for YHWH, the fat covering, the inside and all the fat which is upon the inside,

and the two kidneys and the fat, which is upon them, which is upon the hips, and the lobe upon the liver and upon the kidneys he will remove,

5 and~ they~ did~ make~ BURN.INCENSE (Verb) ( וְהִקְטִירוּ / wê'hiq'ti'ru ) AT ~him ( אֹתוֹ / o'to ) SON ~s ( בְנֵי / vê'ney ) Aharon ( אַהֲרֹן / a'ha'ron ) the~ ALTAR ~unto ( הַמִּזְבֵּחָה / ha'miz'bey'hhah ) UPON ( עַל / al ) the~ ASCENSION.OFFERING ( הָעֹלָה / ha'o'lah ) WHICH ( אֲשֶׁר / a'sher ) UPON ( עַל / al ) the~ TREE ~s ( הָעֵצִים / ha'eytsim ) WHICH ( אֲשֶׁר / a'sher ) UPON ( עַל / al ) the~ FIRE ( הָאֵשׁ / ha'eysh ) FIRE.OFFERING ( אִשֵּׁה / i'sheyh ) AROMA ( רֵיחַ / rey'ahh ) SWEET ( נִיחֹחַ / ni'hho'ahh ) to~ YHWH ( לַיהוָה / la'YHWH )

and the sons of Aharon will burn him as incense upon the altar upon the ascension offering, which is upon the wood, which is upon the fire, a fire offering, a sweet aroma to YHWH,

6 and~ IF ( וְאִם / wê'im ) FROM ( מִן / min ) the~ FLOCKS ( הַצֹּאן / ha'tson ) DONATION ~him ( קָרְבָּנוֹ / qar'ba'no ) to~ SACRIFICE ( לְזֶבַח / lê'ze'vahh ) OFFERING.OF.RESTITUTION ~s ( שְׁלָמִים / shê'la'mim ) to~ YHWH ( לַיהוָה / la'YHWH ) MALE ( זָכָר / za'khar ) OR ( אוֹ / o ) FEMALE ( נְקֵבָה / nê'qey'vah ) WHOLE ( תָּמִים / ta'mim ) he~ will~ make~ COME.NEAR (Verb) ~him ( יַקְרִיבֶנּוּ / yaq'riy've'nu )

and if his donation is from the flocks for a sacrifice of offerings of restitution to YHWH, it will be a whole male or female, he will bring him near.
