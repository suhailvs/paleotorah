1 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) !(ms)~ SCULPT (Verb) ( פְּסָל / pê'sal ) to~ ~you(ms) ( לְךָ / lê'kha ) TWO ( שְׁנֵי / shê'ney ) SLAB ~s ( לֻחֹת / lu'hhot ) STONE ~s ( אֲבָנִים / a'va'nim ) like~ FIRST ~s ( כָּרִאשֹׁנִים / ka'ri'sho'nim ) and~ I~ did~ WRITE (Verb) ( וְכָתַבְתִּי / wê'kha'tav'ti ) UPON ( עַל / al ) the~ SLAB ~s ( הַלֻּחֹת / ha'lu'hhot ) AT ( אֶת / et ) the~ WORD ~s ( הַדְּבָרִים / ha'de'va'rim ) WHICH ( אֲשֶׁר / a'sher ) they~ did~ EXIST (Verb) ( הָיוּ / hai'u ) UPON ( עַל / al ) the~ SLAB ~s ( הַלֻּחֹת / ha'lu'hhot ) like~ FIRST ~s ( כָּרִאשֹׁנִים / ka'ri'sho'nim ) WHICH ( אֲשֶׁר / a'sher ) you(ms)~ did~ CRACK (Verb) ( שִׁבַּרְתָּ / shi'bar'ta )

and YHWH said to Mosheh, sculpt for yourself two slabs of stone like the first ones, and I will write upon the slabs the words, which existed upon the slabs, like the first ones, which you cracked,

2 and~ !(ms)~ EXIST (Verb) ( וֶהְיֵה / weh'yey ) be~ PREPARE (Verb) ~ing(ms) ( נָכוֹן / na'khon ) to~ the~ MORNING ( לַבֹּקֶר / la'bo'qer ) and~ you(ms)~ did~ GO.UP (Verb) ( וְעָלִיתָ / wê'a'li'ta ) in~ the~ MORNING ( בַבֹּקֶר / va'bo'qer ) TO ( אֶל / el ) HILL ( הַר / har ) Sinai ( סִינַי / si'nai ) and~ you(ms)~ did~ be~ STAND.UP (Verb) ( וְנִצַּבְתָּ / wê'ni'tsav'ta ) to~ ~me ( לִי / li ) THERE ( שָׁם / sham ) UPON ( עַל / al ) HEAD ( רֹאשׁ / rosh ) the~ HILL ( הָהָר / ha'har )

and be ready for the morning, and you will go up in the morning to the hill of Sinai, and you will stand up to me there upon the head of the hill,

3 and~ MAN ( וְאִישׁ / wê'ish ) NOT ( לֹא / lo ) he~ will~ GO.UP (Verb) ( יַעֲלֶה / ya'a'leh ) WITH ~you(fs) ( עִמָּךְ / i'makh ) and~ ALSO ( וְגַם / wê'gam ) MAN ( אִישׁ / ish ) DO.NOT ( אַל / al ) he~ will~ be~ SEE (Verb) ( יֵרָא / yey'ra ) in~ ALL ( בְּכָל / bê'khol ) the~ HILL ( הָהָר / ha'har ) ALSO ( גַּם / gam ) the~ FLOCKS ( הַצֹּאן / ha'tson ) and~ the~ CATTLE ( וְהַבָּקָר / wê'ha'ba'qar ) DO.NOT ( אַל / al ) they(m)~ will~ FEED (Verb) ( יִרְעוּ / yir'u ) TO ( אֶל / el ) FOREFRONT ( מוּל / mul ) the~ HILL ( הָהָר / ha'har ) the~ HE ( הַהוּא / ha'hu )

and no man will go up with you, and also, do not let a man appear in all the hill, also do not let the flocks and the cattle feed on the forefront of that hill,

4 and~ he~ will~ SCULPT (Verb) ( וַיִּפְסֹל / wai'yiph'sol ) TWO ( שְׁנֵי / shê'ney ) SLAB ~s ( לֻחֹת / lu'hhot ) STONE ~s ( אֲבָנִים / a'va'nim ) like~ FIRST ~s ( כָּרִאשֹׁנִים / ka'ri'sho'nim ) and~ he~ will~ make~ DEPART.EARLY (Verb) ( וַיַּשְׁכֵּם / wai'yash'keym ) Mosheh ( מֹשֶׁה / mo'sheh ) in~ the~ MORNING ( בַבֹּקֶר / va'bo'qer ) and~ he~ will~ make~ GO.UP (Verb) ( וַיַּעַל / wai'ya'al ) TO ( אֶל / el ) HILL ( הַר / har ) Sinai ( סִינַי / si'nai ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ much~ DIRECT (Verb) ( צִוָּה / tsi'wah ) YHWH ( יְהוָה / YHWH ) AT ~him ( אֹתוֹ / o'to ) and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) in~ HAND ~him ( בְּיָדוֹ / bê'ya'do ) TWO ( שְׁנֵי / shê'ney ) SLAB ~s ( לֻחֹת / lu'hhot ) STONE ~s ( אֲבָנִים / a'va'nim )

and he sculpted two slabs of stone like the first ones, and Mosheh departed early in the morning, and he went up to the hill of Sinai, just as YHWH directed him, and he took in his hand the two slabs of stone,

5 and~ he~ will~ GO.DOWN (Verb) ( וַיֵּרֶד / wai'yey'red ) YHWH ( יְהוָה / YHWH ) in~ CLOUD ( בֶּעָנָן / be'a'nan ) and~ he~ will~ self~ STATION (Verb) ( וַיִּתְיַצֵּב / wai'yit'ya'tseyv ) WITH ~him ( עִמּוֹ / i'mo ) THERE ( שָׁם / sham ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) in~ TITLE ( בְשֵׁם / vê'sheym ) YHWH ( יְהוָה / YHWH ) [ 565 ]

and YHWH went down in the cloud, and he stationed himself with him there, and he called out YHWH by title,

6 and~ he~ will~ CROSS.OVER (Verb) ( וַיַּעֲבֹר / wai'ya'a'vor ) YHWH ( יְהוָה / YHWH ) UPON ( עַל / al ) FACE ~s ~him ( פָּנָיו / pa'naw ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) YHWH ( יְהוָה / YHWH ) YHWH ( יְהוָה / YHWH ) MIGHTY.ONE ( אֵל / eyl ) [ 566 ] COMPASSIONATE ( רַחוּם / ra'hhum ) and~ PROTECTIVE ( וְחַנּוּן / wê'hha'nun ) SLOW ( אֶרֶךְ / e'rekh ) NOSE ~s2 ( אַפַּיִם / a'pa'yim ) [ 567 ] and~ ABUNDANT ( וְרַב / wê'rav ) KINDNESS ( חֶסֶד / hhe'sed ) and~ TRUTH ( וֶאֱמֶת / we'e'met )

and YHWH crossed over upon his face, and he called out, YHWH, YHWH, the mighty one, compassionate and gracious, slow of nostrils, and abundant in kindness and truth.

7 PRESERVE (Verb) ~ing(ms) ( נֹצֵר / no'tseyr ) KINDNESS ( חֶסֶד / hhe'sed ) to~ the~ THOUSAND ~s ( לָאֲלָפִים / la'a'la'phim ) LIFT.UP (Verb) ~ing(ms) ( נֹשֵׂא / no'sey ) [ 568 ] TWISTEDNESS ( עָו‍ֹן / a'won ) and~ OFFENSE ( וָפֶשַׁע / wa'phe'sha ) and~ FAILURE ( וְחַטָּאָה / wê'hha'ta'ah ) and~ >~ much~ ACQUIT (Verb) ( וְנַקֵּה / wê'na'qeyh ) NOT ( לֹא / lo ) he~ will~ much~ ACQUIT (Verb) ( יְנַקֶּה / yê'na'qeh ) [ 569 ] REGISTER (Verb) ~ing(ms) ( פֹּקֵד / po'qeyd ) TWISTEDNESS ( עֲו‍ֹן / a'won ) FATHER ~s ( אָבוֹת / a'vot ) UPON ( עַל / al ) SON ~s ( בָּנִים / ba'nim ) and~ UPON ( וְעַל / wê'al ) SON ~s ( בְּנֵי / bê'ney ) SON ~s ( בָנִים / va'nim ) UPON ( עַל / al ) THIRD.GENERATION ~s ( שִׁלֵּשִׁים / shi'ley'shim ) and~ UPON ( וְעַל / wê'al ) FOURTH.GENERATION ~s ( רִבֵּעִים / ri'bey'im )

Preserving kindness to the thousands, lifting up twistedness and offense and failure, but he will not completely acquit, registering the twistedness of the fathers upon the sons and upon the sons of the sons, upon the third generations and upon the fourth generation,

8 and~ he~ will~ much~ HURRY (Verb) ( וַיְמַהֵר / wai'ma'heyr ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ he~ will~ BOW.THE.HEAD (Verb) ( וַיִּקֹּד / wai'yi'qod ) LAND ~unto ( אַרְצָה / ar'tsah ) and~ he~ will~ self~ BEND.DOWN (Verb) ( וַיִּשְׁתָּחוּ / wai'yish'ta'hhu )

and Mosheh hurried and he bowed the head unto the land and he bent himself down,

9 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) IF ( אִם / im ) PLEASE ( נָא / na ) I~ did~ FIND (Verb) ( מָצָאתִי / ma'tsa'ti ) BEAUTY ( חֵן / hheyn ) [ 570 ] in~ EYE ~s2 ~you(ms) ( בְּעֵינֶיךָ / bê'ey'ney'kha ) Adonai ( אֲדֹנָי / a'do'nai ) he~ will~ WALK (Verb) ( יֵלֶךְ / yey'lekh ) PLEASE ( נָא / na ) Adonai ( אֲדֹנָי / a'do'nai ) in~ INSIDE ~us ( בְּקִרְבֵּנוּ / bê'qir'bey'nu ) GIVEN.THAT ( כִּי / ki ) PEOPLE ( עַם / am ) HARD ( קְשֵׁה / qê'sheyh ) NECK ( עֹרֶף / o'reph ) HE ( הוּא / hu ) and~ you(ms)~ will~ FORGIVE (Verb) ( וְסָלַחְתָּ / wê'sa'lahh'ta ) to~ TWISTEDNESS ~us ( לַעֲו‍ֹנֵנוּ / la'a'o'ney'nu ) and~ to~ FAILURE ~us ( וּלְחַטָּאתֵנוּ / ul'hha'ta'tey'nu ) and~ you(ms)~ did~ INHERIT (Verb) ~us ( וּנְחַלְתָּנוּ / un'hhal'ta'nu )

and he said, please, if I found beauty in your eyes Adonai, please, Adonai will walk inside us, given that he is a hard necked people, and you will forgive our twistedness and our failure, and you will inherit us,

10 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) LOOK ( הִנֵּה / hin'neyh ) I ( אָנֹכִי / a'no'khi ) CUT (Verb) ~ing(ms) ( כֹּרֵת / ko'reyt ) COVENANT ( בְּרִית / bê'rit ) OPPOSITE ( נֶגֶד / ne'ged ) ALL ( כָּל / kol ) PEOPLE ~you(ms) ( עַמְּךָ / am'kha ) I~ will~ DO (Verb) ( אֶעֱשֶׂה / e'e'seh ) be~ PERFORM (Verb) ~ing(fp) ( נִפְלָאֹת / niph'la'ot ) WHICH ( אֲשֶׁר / a'sher ) NOT ( לֹא / lo ) they~ did~ be~ SHAPE (Verb) ( נִבְרְאוּ / niv're'u ) [ 571 ] in~ ALL ( בְכָל / vê'khol ) the~ LAND ( הָאָרֶץ / ha'a'rets ) and~ in~ ALL ( וּבְכָל / uv'khol ) the~ NATION ~s ( הַגּוֹיִם / ha'go'yim ) and~ he~ did~ SEE (Verb) ( וְרָאָה / wê'ra'ah ) ALL ( כָל / khol ) the~ PEOPLE ( הָעָם / ha'am ) WHICH ( אֲשֶׁר / a'sher ) YOU(MS) ( אַתָּה / a'tah ) in~ INSIDE ~him ( בְקִרְבּוֹ / vê'qir'bo ) AT ( אֶת / et ) WORK ( מַעֲשֵׂה / ma'a'seyh ) YHWH ( יְהוָה / YHWH ) GIVEN.THAT ( כִּי / ki ) be~ FEAR (Verb) ~ing(ms) ( נוֹרָא / no'ra ) HE ( הוּא / hu ) WHICH ( אֲשֶׁר / a'sher ) I ( אֲנִי / a'ni ) DO (Verb) ~ing(ms) ( עֹשֶׂה / o'seh ) WITH ~you(fs) ( עִמָּךְ / i'makh )

and he said, look, I am cutting a covenant before all your people, I will do performances, which has not been shaped in all the land and in all the nations, and all the people, which you are inside, will see the work of YHWH, given that what I am doing with you, is to be feared.

11 !(ms)~ SAFEGUARD (Verb) ( שְׁמָר / shê'mar ) to~ ~you(ms) ( לְךָ / lê'kha ) AT ( אֵת / eyt ) WHICH ( אֲשֶׁר / a'sher ) I ( אָנֹכִי / a'no'khi ) much~ DIRECT (Verb) ~ing(ms) ~you(ms) ( מְצַוְּךָ / mê'tsa'we'kha ) the~ DAY ( הַיּוֹם / hai'yom ) LOOK ~me ( הִנְנִי / hin'ni ) CAST.OUT (Verb) ~ing(ms) ( גֹרֵשׁ / go'reysh ) from~ FACE ~s ~you(ms) ( מִפָּנֶיךָ / mi'pa'ney'kha ) [ 572 ] AT ( אֶת / et ) the~ Emor ~of ( הָאֱמֹרִי / ha'e'mo'ri ) and~ the~ Kena'an ~of ( וְהַכְּנַעֲנִי / wê'hak'na'a'ni ) and~ the~ Hhet ~of ( וְהַחִתִּי / wê'ha'hhi'ti ) and~ the~ Perez ~of ( וְהַפְּרִזִּי / wê'ha'pe'ri'zi ) and~ the~ Hhiw ~of ( וְהַחִוִּי / wê'ha'hhi'wi ) and~ the~ Yevus ~of ( וְהַיְבוּסִי / wê'hai'vu'si )

Safeguard for you what I am directing you today, look at me casting out from your face the one of Emor and the one of Kena'an and the one of Hhet and the one of Perez and the one of Hhiw and the one of Yevus.

12 !(ms)~ be~ SAFEGUARD (Verb) ( הִשָּׁמֶר / hi'sha'mer ) to~ ~you(ms) ( לְךָ / lê'kha ) OTHERWISE ( פֶּן / pen ) you(ms)~ will~ CUT (Verb) ( תִּכְרֹת / tikh'rot ) COVENANT ( בְּרִית / bê'rit ) to~ SETTLE (Verb) ~ing(ms) ( לְיוֹשֵׁב / lê'yo'sheyv ) the~ LAND ( הָאָרֶץ / ha'a'rets ) WHICH ( אֲשֶׁר / a'sher ) YOU(MS) ( אַתָּה / a'tah ) he~ did~ COME (Verb) ( בָּא / ba ) UPON ~her ( עָלֶיהָ / a'ley'ah ) OTHERWISE ( פֶּן / pen ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) to~ SNARE ( לְמוֹקֵשׁ / lê'mo'qeysh ) in~ INSIDE ~you(ms) ( בְּקִרְבֶּךָ / bê'qir'be'kha )

Be safeguarded to yourself, otherwise you will cut a covenant for a settler of the land which you are coming upon, otherwise he will exist for a snare in among you.

13 GIVEN.THAT ( כִּי / ki ) AT ( אֶת / et ) ALTAR ~s ~them(m) ( מִזְבְּחֹתָם / miz'be'hho'tam ) you(mp)~ will~ BREAK.DOWN (Verb) ~must ( תִּתֹּצוּן / ti'to'tsun ) and~ AT ( וְאֶת / wê'et ) MONUMENT ~s ~them(m) ( מַצֵּבֹתָם / ma'tsey'vo'tam ) you(mp)~ will~ much~ CRACK (Verb) ~must ( תְּשַׁבֵּרוּן / tê'sha'bey'run ) and~ AT ( וְאֶת / wê'et ) GROVE ~s ~him ( אֲשֵׁרָיו / a'shey'raw ) you(mp)~ will~ CUT (Verb) ~must ( תִּכְרֹתוּן / tikh'ro'tun )

Given that their altars you must break down, and their monuments you must shatter, and his groves you must cut.

14 GIVEN.THAT ( כִּי / ki ) NOT ( לֹא / lo ) you(ms)~ will~ self~ BEND.DOWN (Verb) ( תִשְׁתַּחֲוֶה / tish'ta'hha'weh ) to~ MIGHTY.ONE ( לְאֵל / lê'eyl ) OTHER ( אַחֵר / a'hheyr ) GIVEN.THAT ( כִּי / ki ) YHWH ( יְהוָה / YHWH ) ZEALOUS ( קַנָּא / qa'na ) TITLE ~him ( שְׁמוֹ / shê'mo ) MIGHTY.ONE ( אֵל / eyl ) ZEALOUS ( קַנָּא / qa'na ) HE ( הוּא / hu )

Given that you will not bend yourself down to another mighty one, given that YHWH is zealous, his title is the mighty one, he is zealous.

15 OTHERWISE ( פֶּן / pen ) you(ms)~ will~ CUT (Verb) ( תִּכְרֹת / tikh'rot ) COVENANT ( בְּרִית / bê'rit ) to~ SETTLE (Verb) ~ing(ms) ( לְיוֹשֵׁב / lê'yo'sheyv ) the~ LAND ( הָאָרֶץ / ha'a'rets ) and~ they~ did~ BE.A.HARLOT (Verb) ( וְזָנוּ / wê'za'nu ) AFTER ( אַחֲרֵי / a'hha'rey ) Elohiym ~them(m) ( אֱלֹהֵיהֶם / e'lo'hey'hem ) and~ they~ did~ SACRIFICE (Verb) ( וְזָבְחוּ / wê'zav'hhu ) to~ Elohiym ~them(m) ( לֵאלֹהֵיהֶם / ley'lo'hey'hem ) and~ he~ did~ CALL.OUT (Verb) ( וְקָרָא / wê'qa'ra ) to~ ~you(ms) ( לְךָ / lê'kha ) and~ you(ms)~ did~ EAT (Verb) ( וְאָכַלְתָּ / wê'a'khal'ta ) ALTAR ~him ( מִזִּבְחוֹ / mi'ziv'hho )

Otherwise you will cut a covenant for a settler of the land, and they will be a whore after their Elohiym, and they will sacrifice to their Elohiym, and he will call out to you and you will eat of his altar,

16 and~ you(ms)~ did~ TAKE (Verb) ( וְלָקַחְתָּ / wê'la'qahh'ta ) from~ DAUGHTER ~s ~him ( מִבְּנֹתָיו / mi'be'no'taw ) to~ SON ~s ~you(ms) ( לְבָנֶיךָ / lê'va'ney'kha ) and~ they~ did~ BE.A.HARLOT (Verb) ( וְזָנוּ / wê'za'nu ) DAUGHTER ~s ~him ( בְנֹתָיו / vê'no'taw ) AFTER ( אַחֲרֵי / a'hha'rey ) Elohiym ~them(f) ( אֱלֹהֵיהֶן / e'lo'hey'hen ) and~ they~ did~ make~ BE.A.HARLOT (Verb) ( וְהִזְנוּ / wê'hiz'nu ) AT ( אֶת / et ) SON ~s ~you(ms) ( בָּנֶיךָ / ba'ney'kha ) AFTER ( אַחֲרֵי / a'hha'rey ) Elohiym ~them(f) ( אֱלֹהֵיהֶן / e'lo'hey'hen )

and you will take from his daughters for your sons, and his daughters will be a whore after their Elohiym, and they will make your sons be a whore after their Elohiym.

17 Elohiym ( אֱלֹהֵי / e'lo'hey ) CAST.IMAGE ( מַסֵּכָה / ma'sey'khah ) NOT ( לֹא / lo ) you(ms)~ will~ DO (Verb) ( תַעֲשֶׂה / ta'a'seh ) to~ ~you(fs) ( לָּךְ / lakh )

You will not make for you an Elohiym of a cast image.

18 AT ( אֶת / et ) FEAST ( חַג / hhag ) the~ UNLEAVENED.BREAD ~s ( הַמַּצּוֹת / ha'ma'tsot ) you(ms)~ will~ SAFEGUARD (Verb) ( תִּשְׁמֹר / tish'mor ) SEVEN ( שִׁבְעַת / shiv'at ) DAY ~s ( יָמִים / ya'mim ) you(ms)~ will~ EAT (Verb) ( תֹּאכַל / to'khal ) UNLEAVENED.BREAD ~s ( מַצּוֹת / ma'tsot ) WHICH ( אֲשֶׁר / a'sher ) I~ did~ much~ DIRECT (Verb) ~you(ms) ( צִוִּיתִךָ / tsi'wi'ti'kha ) to~ APPOINTED ( לְמוֹעֵד / lê'mo'eyd ) NEW.MOON ( חֹדֶשׁ / hho'desh ) the~ GREEN.GRAIN ( הָאָבִיב / ha'a'viv ) GIVEN.THAT ( כִּי / ki ) in~ NEW.MOON ( בְּחֹדֶשׁ / bê'hho'desh ) the~ GREEN.GRAIN ( הָאָבִיב / ha'a'viv ) you(ms)~ did~ GO.OUT (Verb) ( יָצָאתָ / ya'tsa'ta ) from~ Mits'rayim ( מִמִּצְרָיִם / mi'mits'ra'yim )

You will safeguard the feast of the unleavened bread, seven days you will eat the unleavened bread which I directed you for an appointed time in the month of the green grain, given that in the month of the green grain you went out from Mits'rayim.
