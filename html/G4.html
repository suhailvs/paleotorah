1 and~ the~ HUMAN ( וְהָאָדָם / wê'ha'a'dam ) he~ did~ KNOW (Verb) ( יָדַע / ya'da ) AT ( אֶת / et ) Hhawah ( חַוָּה / hha'wah ) WOMAN ~him ( אִשְׁתּוֹ / ish'to ) and~ she~ will~ CONCEIVE (Verb) ( וַתַּהַר / wa'ta'har ) and~ she~ will~ BRING.FORTH (Verb) ( וַתֵּלֶד / wa'tey'led ) AT ( אֶת / et ) [ 27 ] Qayin ( קַיִן / qa'yin ) and~ she~ will~ SAY (Verb) ( וַתֹּאמֶר / wa'to'mer ) I~ did~ PURCHASE (Verb) ( קָנִיתִי / qa'ni'ti ) MAN ( אִישׁ / ish ) AT ( אֶת / et ) YHWH ( יְהוָה / YHWH )

and the human had known Hhawah his woman and she conceived and she brought forth Qayin, and she said, I purchased a man with YHWH,

2 and~ she~ will~ make~ ADD (Verb) ( וַתֹּסֶף / wa'to'seph ) to~ >~ BRING.FORTH (Verb) ( לָלֶדֶת / la'le'det ) AT ( אֶת / et ) BROTHER ~him ( אָחִיו / a'hhiw ) AT ( אֶת / et ) Hevel ( הָבֶל / ha'vel ) and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) Hevel ( הֶבֶל / he'vel ) FEED (Verb) ~ing(ms) ( רֹעֵה / ro'eyh ) FLOCKS ( צֹאן / tson ) and~ Qayin ( וְקַיִן / wê'qa'yin ) he~ did~ EXIST (Verb) ( הָיָה / hai'yah ) SERVE (Verb) ~ing(ms) ( עֹבֵד / o'veyd ) GROUND ( אֲדָמָה / a'da'mah )

and she brought forth again his brother Hevel, and Hevel existed as a feeder of the flocks, and Qayin existed as a server of the ground,

3 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) from~ CONCLUSION ( מִקֵּץ / mi'qeyts ) [ 28 ] DAY ~s ( יָמִים / ya'mim ) and~ he~ will~ make~ COME (Verb) ( וַיָּבֵא / wai'ya'vey ) Qayin ( קַיִן / qa'yin ) from~ PRODUCE ( מִפְּרִי / mi'pe'ri ) the~ GROUND ( הָאֲדָמָה / ha'a'da'mah ) DEPOSIT ( מִנְחָה / min'hhah ) to~ YHWH ( לַיהוָה / la'YHWH )

and it came to pass at the conclusion of days, Qayin brought from the produce of the ground, a deposit to YHWH,

4 and~ Hevel ( וְהֶבֶל / wê'he'vel ) he~ did~ make~ COME (Verb) ( הֵבִיא / hey'vi ) ALSO ( גַם / gam ) HE ( הוּא / hu ) from~ FIRSTBORN.FEMALE ~s ( מִבְּכֹרוֹת / mi'be'kho'rot ) FLOCKS ~him ( צֹאנוֹ / tso'no ) and~ from~ FAT ~them(f) ( וּמֵחֶלְבֵהֶן / u'mey'hhel'vey'hen ) and~ he~ will~ LOOK.WITH.RESPECT (Verb) ( וַיִּשַׁע / wai'yi'sha ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Hevel ( הֶבֶל / he'vel ) and~ TO ( וְאֶל / wê'el ) DEPOSIT ~him ( מִנְחָתוֹ / min'hha'to )

and Hevel had brought also, he from the firstborn females of his flocks and from their fat, and YHWH looked with respect to Hevel and to his deposit,

5 and~ TO ( וְאֶל / wê'el ) Qayin ( קַיִן / qa'yin ) and~ TO ( וְאֶל / wê'el ) DEPOSIT ~him ( מִנְחָתוֹ / min'hha'to ) NOT ( לֹא / lo ) he~ did~ LOOK.WITH.RESPECT (Verb) ( שָׁעָה / sha'ah ) and~ he~ will~ FLARE.UP (Verb) ( וַיִּחַר / wai'yi'hhar ) [ 29 ] to~ Qayin ( לְקַיִן / lê'qa'yin ) MANY ( מְאֹד / mê'od ) and~ they(m)~ will~ FALL (Verb) ( וַיִּפְּלוּ / wai'yip'lu ) FACE ~s ~him ( פָּנָיו / pa'naw ) [ 30 ]

and to Qayin and to his deposit he did not look with respect, and Qayin flared up and his face fell,

6 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Qayin ( קָיִן / qa'yin ) to~ WHAT ( לָמָּה / la'mah ) he~ did~ FLARE.UP (Verb) ( חָרָה / hha'rah ) to~ ~you(fs) ( לָךְ / lakh ) and~ to~ WHAT ( וְלָמָּה / wê'la'mah ) they~ did~ FALL (Verb) ( נָפְלוּ / naph'lu ) FACE ~s ~you(ms) ( פָנֶיךָ / pha'ney'kha )

and YHWH said to Qayin, why were you flared up, and why is your face fallen.

7 ?~ NOT ( הֲלוֹא / ha'lo ) IF ( אִם / im ) you(ms)~ will~ make~ DO.WELL (Verb) ( תֵּיטִיב / tey'tiv ) >~ LIFT.UP (Verb) ( שְׂאֵת / sê'eyt ) and~ IF ( וְאִם / wê'im ) NOT ( לֹא / lo ) you(ms)~ will~ make~ DO.WELL (Verb) ( תֵיטִיב / tey'tiv ) to~ OPENING ( לַפֶּתַח / la'pe'tahh ) FAILURE ( חַטָּאת / hha'tat ) STRETCH.OUT (Verb) ~ing(ms) ( רֹבֵץ / ro'veyts ) and~ TO ~you(ms) ( וְאֵלֶיךָ / wê'ey'ley'kha ) FOLLOWING ~him ( תְּשׁוּקָתוֹ / tê'shu'qa'to ) [ 31 ] and~ YOU(MS) ( וְאַתָּה / wê'a'tah ) you(ms)~ will~ REGULATE (Verb) ( תִּמְשָׁל / tim'shal ) in~ ~him ( בּוֹ / bo )

If you do well, will there not be a lifting up, but if you do not do well, there will be an opening of failure stretching out, and to you is his following and you will regulate in him,

8 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) [ 32 ] Qayin ( קַיִן / qa'yin ) TO ( אֶל / el ) Hevel ( הֶבֶל / he'vel ) BROTHER ~him ( אָחִיו / a'hhiw ) and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) in~ >~ EXIST (Verb) ~them(m) ( בִּהְיוֹתָם / bih'yo'tam ) in~ the~ FIELD ( בַּשָּׂדֶה / ba'sa'deh ) and~ he~ will~ RISE (Verb) ( וַיָּקָם / wai'ya'qam ) Qayin ( קַיִן / qa'yin ) TO ( אֶל / el ) Hevel ( הֶבֶל / he'vel ) BROTHER ~him ( אָחִיו / a'hhiw ) and~ he~ will~ KILL (Verb) ~him ( וַיַּהַרְגֵהוּ / wai'ya'har'gey'hu )

and Qayin said to Hevel his brother, let us go out into the field, and it came to pass in their existing in the field, and Qayin rose to Hevel his brother and he killed him,

9 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Qayin ( קַיִן / qa'yin ) WHERE ( אֵי / ey ) Hevel ( הֶבֶל / he'vel ) BROTHER ~you(ms) ( אָחִיךָ / a'hhi'kha ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) NOT ( לֹא / lo ) I~ did~ KNOW (Verb) ( יָדַעְתִּי / ya'da'ti ) ?~ SAFEGUARD (Verb) ~ing(ms) ( הֲשֹׁמֵר / ha'sho'meyr ) BROTHER ~me ( אָחִי / a'hhi ) I ( אָנֹכִי / a'no'khi )

and YHWH said to Qayin, where is Hevel your brother, and he said, I do not know, am I the guardian of my brother,

10 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) WHAT ( מֶה / meh ) you(ms)~ did~ DO (Verb) ( עָשִׂיתָ / a'si'ta ) VOICE ( קוֹל / qol ) BLOOD ~s ( דְּמֵי / dê'mey ) BROTHER ~you(ms) ( אָחִיךָ / a'hhi'kha ) CRY.OUT (Verb) ~ing(mp) ( צֹעֲקִים / tso'a'qim ) TO ~me ( אֵלַי / ey'lai ) FROM ( מִן / min ) the~ GROUND ( הָאֲדָמָה / ha'a'da'mah )

and he said, what did you do, the voice of the bloodshed of your brother is crying out to me from the ground,

11 and~ NOW ( וְעַתָּה / wê'a'tah ) SPIT.UPON (Verb) ~ed(ms) ( אָרוּר / a'rur ) [ 33 ] YOU(MS) ( אָתָּה / a'tah ) FROM ( מִן / min ) the~ GROUND ( הָאֲדָמָה / ha'a'da'mah ) WHICH ( אֲשֶׁר / a'sher ) she~ did~ PART (Verb) ( פָּצְתָה / pats'tah ) AT ( אֶת / et ) MOUTH ~her ( פִּיהָ / pi'ah ) to~ >~ TAKE (Verb) ( לָקַחַת / la'qa'hhat ) AT ( אֶת / et ) BLOOD ~s ( דְּמֵי / dê'mey ) BROTHER ~you(ms) ( אָחִיךָ / a'hhi'kha ) from~ HAND ~you(ms) ( מִיָּדֶךָ / mi'ya'de'kha )

and now, you are spitted upon from the ground which parted her mouth to take the bloodshed of your brother from your hand.

12 GIVEN.THAT ( כִּי / ki ) you(ms)~ will~ SERVE (Verb) ( תַעֲבֹד / ta'a'vod ) AT ( אֶת / et ) the~ GROUND ( הָאֲדָמָה / ha'a'da'mah ) NOT ( לֹא / lo ) she~ will~ make~ ADD (Verb) ( תֹסֵף / to'seyph ) >~ GIVE (Verb) ( תֵּת / teyt ) STRENGTH ~her ( כֹּחָהּ / ko'hhah ) to~ ~you(fs) ( לָךְ / lakh ) STAGGER (Verb) ~ing(ms) ( נָע / na ) and~ NOD (Verb) ~ing(ms) ( וָנָד / wa'nad ) [ 34 ] you(ms)~ will~ EXIST (Verb) ( תִּהְיֶה / tih'yeh ) in~ the~ LAND ( בָאָרֶץ / va'a'rets )

Given that you will serve the ground, she will not again give her strength to you, you will exist in the land staggering and nodding,

13 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Qayin ( קַיִן / qa'yin ) TO ( אֶל / el ) YHWH ( יְהוָה / YHWH ) GREAT ( גָּדוֹל / ga'dol ) TWISTEDNESS ~me ( עֲו‍ֹנִי / a'o'ni ) from~ >~ LIFT.UP (Verb) ( מִנְּשֹׂא / mi'ne'so )

and Qayin said to YHWH, great is my twistedness, it is more than I can lift up.

14 THOUGH ( הֵן / heyn ) you(ms)~ did~ much~ CAST.OUT (Verb) ( גֵּרַשְׁתָּ / gey'rash'ta ) AT ~me ( אֹתִי / o'ti ) the~ DAY ( הַיּוֹם / hai'yom ) from~ UPON ( מֵעַל / mey'al ) FACE ~s ( פְּנֵי / pê'ney ) the~ GROUND ( הָאֲדָמָה / ha'a'da'mah ) and~ from~ FACE ~s ~you(ms) ( וּמִפָּנֶיךָ / u'mi'pa'ney'kha ) I~ will~ be~ HIDE (Verb) ( אֶסָּתֵר / e'sa'teyr ) and~ I~ did~ EXIST (Verb) ( וְהָיִיתִי / wê'hai'yi'ti ) STAGGER (Verb) ~ing(ms) ( נָע / na ) and~ NOD (Verb) ~ing(ms) ( וָנָד / wa'nad ) in~ the~ LAND ( בָּאָרֶץ / ba'a'rets ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) ALL ( כָל / khol ) FIND (Verb) ~ing(ms) ~me ( מֹצְאִי / mots'i ) he~ will~ KILL (Verb) ~me ( יַהַרְגֵנִי / ya'har'gey'ni )

Though you cast me out today from upon the face of the ground, and I will be hidden from your face, but I will exist in the land staggering and nodding, and it will come to pass, all the ones finding me will kill me,

15 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ ~him ( לוֹ / lo ) YHWH ( יְהוָה / YHWH ) to~ SO ( לָכֵן / la'kheyn ) ALL ( כָּל / kol ) KILL (Verb) ~ing(ms) ( הֹרֵג / ho'reyg ) Qayin ( קַיִן / qa'yin ) SEVENTH.TIME ~s2 ( שִׁבְעָתַיִם / shiv'a'ta'yim ) he~ will~ be~ make~ AVENGE (Verb) ( יֻקָּם / yu'qam ) and~ he~ will~ PLACE (Verb) ( וַיָּשֶׂם / wai'ya'sem ) YHWH ( יְהוָה / YHWH ) to~ Qayin ( לְקַיִן / lê'qa'yin ) SIGN ( אוֹת / ot ) to~ EXCEPT ( לְבִלְתִּי / lê'vil'ti ) >~ make~ HIT (Verb) ( הַכּוֹת / ha'kot ) AT ~him ( אֹתוֹ / o'to ) ALL ( כָּל / kol ) FIND (Verb) ~ing(ms) ~him ( מֹצְאוֹ / mots'o )

and YHWH said to him, because of this, all the ones killing Qayin will be avenged sevenfold, and YHWH placed for Qayin a sign for all the ones finding him to not attack him,

16 and~ he~ will~ GO.OUT (Verb) ( וַיֵּצֵא / wai'yey'tsey ) Qayin ( קַיִן / qa'yin ) from~ to~ FACE ~s ( מִלִּפְנֵי / mi'liph'ney ) YHWH ( יְהוָה / YHWH ) and~ he~ will~ SETTLE (Verb) ( וַיֵּשֶׁב / wai'yey'shev ) in~ LAND ( בְּאֶרֶץ / bê'e'rets ) Nod ( נוֹד / nod ) EASTWARD ( קִדְמַת / qid'mat ) Eden ( עֵדֶן / ey'den )

and Qayin went out from before the face of YHWH, and he settled in the land of Nod, eastward of Eden,

17 and~ he~ will~ KNOW (Verb) ( וַיֵּדַע / wai'yey'da ) Qayin ( קַיִן / qa'yin ) AT ( אֶת / et ) WOMAN ~him ( אִשְׁתּוֹ / ish'to ) and~ she~ will~ CONCEIVE (Verb) ( וַתַּהַר / wa'ta'har ) and~ she~ will~ BRING.FORTH (Verb) ( וַתֵּלֶד / wa'tey'led ) AT ( אֶת / et ) Hhanokh ( חֲנוֹךְ / hha'nokh ) and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) BUILD (Verb) ~ing(ms) ( בֹּנֶה / bo'neh ) CITY ( עִיר / ir ) [ 35 ] and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) TITLE ( שֵׁם / sheym ) the~ CITY ( הָעִיר / ha'ir ) like~ TITLE ( כְּשֵׁם / kê'sheym ) SON ~him ( בְּנוֹ / bê'no ) Hhanokh ( חֲנוֹךְ / hha'nokh )

and Qayin knew his woman and she conceived and she brought forth Hhanokh, and he existed as a builder of a city, and called out the title of the city Hhanokh, like the title of his son,

18 and~ he~ will~ be~ BRING.FORTH (Verb) ( וַיִּוָּלֵד / wai'yi'wa'leyd ) to~ Hhanokh ( לַחֲנוֹךְ / la'hha'nokh ) AT ( אֶת / et ) Irad ( עִירָד / i'rad ) and~ Irad ( וְעִירָד / wê'i'rad ) he~ did~ BRING.FORTH (Verb) ( יָלַד / ya'lad ) AT ( אֶת / et ) Mehhuya'el ( מְחוּיָאֵל / mê'hhu'ya'eyl ) and~ Mehhuya'el ( וּמְחִיָּיאֵל / um'hhi'yai'eyl ) he~ did~ BRING.FORTH (Verb) ( יָלַד / ya'lad ) AT ( אֶת / et ) Metusha'el ( מְתוּשָׁאֵל / mê'tu'sha'eyl ) and~ Metusha'el ( וּמְתוּשָׁאֵל / um'tu'sha'eyl ) he~ did~ BRING.FORTH (Verb) ( יָלַד / ya'lad ) AT ( אֶת / et ) Lamekh ( לָמֶךְ / la'mekh )

and Irad was brought forth to Hhanokh, and Irad had brought forth Mehhuya'el and Mehhuya'el had brought forth Metusha'el and Metusha'el had brought forth Lamekh,

19 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) to~ ~him ( לוֹ / lo ) Lamekh ( לֶמֶךְ / le'mekh ) TWO ( שְׁנֵי / shê'ney ) WOMAN ~s ( נָשִׁים / na'shim ) TITLE ( שֵׁם / sheym ) the~ UNIT ( הָאַחַת / ha'a'hhat ) Adah ( עָדָה / a'dah ) and~ TITLE ( וְשֵׁם / wê'sheym ) the~ SECOND ( הַשֵּׁנִית / ha'shey'nit ) Tsilah ( צִלָּה / tsi'lah )

and Lamekh took for himself two women, the title of the one is Adah and the title of the second is Tsilah,

20 and~ she~ will~ BRING.FORTH (Verb) ( וַתֵּלֶד / wa'tey'led ) Adah ( עָדָה / a'dah ) AT ( אֶת / et ) Yaval ( יָבָל / ya'val ) HE ( הוּא / hu ) he~ did~ EXIST (Verb) ( הָיָה / hai'yah ) FATHER ~of ( אֲבִי / a'vi ) [ 36 ] SETTLE (Verb) ~ing(ms) ( יֹשֵׁב / yo'sheyv ) TENT ( אֹהֶל / o'hel ) and~ LIVESTOCK ( וּמִקְנֶה / u'miq'neh ) [ 37 ]

and Adah brought forth Yaval, he existed as the father of tent and livestock settlers,

21 and~ TITLE ( וְשֵׁם / wê'sheym ) BROTHER ~him ( אָחִיו / a'hhiw ) Yuval ( יוּבָל / yu'val ) HE ( הוּא / hu ) he~ did~ EXIST (Verb) ( הָיָה / hai'yah ) FATHER ~of ( אֲבִי / a'vi ) ALL ( כָּל / kol ) SEIZE.HOLD (Verb) ~ing(ms) ( תֹּפֵשׂ / to'pheys ) [ 38 ] HARP ( כִּנּוֹר / ki'nor ) and~ REED.PIPE ( וְעוּגָב / wê'u'gav )

and the title of his brother is Yuval, he existed as the father of all the ones seizing hold of the harp and reed-pipe,

22 and~ Tsilah ( וְצִלָּה / wê'tsi'lah ) ALSO ( גַם / gam ) SHE ( הִוא / hi ) she~ did~ BRING.FORTH (Verb) ( יָלְדָה / yal'dah ) AT ( אֶת / et ) Tuval-Qayin ( תּוּבַל־קַיִן / tu'val-qa'yin ) SHARPEN (Verb) ~ing(ms) ( לֹטֵשׁ / lo'teysh ) [ 39 ] ALL ( כָּל / kol ) CRAFTSMAN ( חֹרֵשׁ / hho'reysh ) COPPER ( נְחֹשֶׁת / nê'hho'shet ) and~ IRON ( וּבַרְזֶל / u'var'zel ) and~ SISTER ( וַאֲחוֹת / wa'a'hhot ) Tuval-Qayin ( תּוּבַל־קַיִן / tu'val-qa'yin ) Na'amah ( נַעֲמָה / na'a'mah )

and Tsilah also had brought forth Tuval-Qayin, a sharpener of all the craftsmen of copper and iron, and the sister of Tuval-Qayin is Na'amah,
