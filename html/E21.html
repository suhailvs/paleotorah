Exodus Chapter 21

1 and~ THESE ( וְאֵלֶּה / wê'ey'leh ) the~ DECISION ~s ( הַמִּשְׁפָּטִים / ha'mish'pa'tim ) WHICH ( אֲשֶׁר / a'sher ) you(ms)~ will~ PLACE (Verb) ( תָּשִׂים / ta'sim ) to~ FACE ~s ~them(m) ( לִפְנֵיהֶם / liph'ney'hem )

and these are the decisions which you will place to their faces.

2 GIVEN.THAT ( כִּי / ki ) you(ms)~ will~ PURCHASE (Verb) ( תִקְנֶה / tiq'neh ) SERVANT ( עֶבֶד / e'ved ) Ever ~of ( עִבְרִי / iv'ri ) SIX ( שֵׁשׁ / sheysh ) YEAR ~s ( שָׁנִים / sha'nim ) he~ will~ SERVE (Verb) ( יַעֲבֹד / ya'a'vod ) and~ in~ the~ SEVENTH ( וּבַשְּׁבִעִת / u'vash'vi'it ) he~ will~ GO.OUT (Verb) ( יֵצֵא / yey'tsey ) to~ FREE ( לַחָפְשִׁי / la'hhaph'shi ) FREELY ( חִנָּם / hhi'nam )

If you will purchase a servant of Ever, he will serve six years, and in the seventh he will go out freely to freedom.

3 IF ( אִם / im ) in~ ARCH ~him ( בְּגַפּוֹ / bê'ga'po ) he~ will~ COME (Verb) ( יָבֹא / ya'vo ) in~ ARCH ~him ( בְּגַפּוֹ / bê'ga'po ) he~ will~ GO.OUT (Verb) ( יֵצֵא / yey'tsey ) IF ( אִם / im ) MASTER ( בַּעַל / ba'al ) WOMAN ( אִשָּׁה / i'shah ) HE ( הוּא / hu ) and~ she~ did~ GO.OUT (Verb) ( וְיָצְאָה / wê'yats'ah ) WOMAN ~him ( אִשְׁתּוֹ / ish'to ) WITH ~him ( עִמּוֹ / i'mo )

If he comes by himself, he will go out by himself, if he is the master of a woman, then his woman will go out with him.

4 IF ( אִם / im ) LORD ~s ~him ( אֲדֹנָיו / a'do'naw ) he~ will~ GIVE (Verb) ( יִתֶּן / yi'ten ) to~ ~him ( לוֹ / lo ) WOMAN ( אִשָּׁה / i'shah ) and~ she~ did~ BRING.FORTH (Verb) ( וְיָלְדָה / wê'yal'dah ) to~ ~him ( לוֹ / lo ) SON ~s ( בָנִים / va'nim ) OR ( אוֹ / o ) DAUGHTER ~s ( בָנוֹת / va'not ) the~ WOMAN ( הָאִשָּׁה / ha'i'shah ) and~ BOY ~s ~her ( וִילָדֶיהָ / wi'la'dey'ah ) [ 389 ] she~ will~ EXIST (Verb) ( תִּהְיֶה / tih'yeh ) to~ LORD ~s ~her ( לַאדֹנֶיהָ / la'do'ney'ah ) and~ HE ( וְהוּא / wê'hu ) he~ will~ GO.OUT (Verb) ( יֵצֵא / yey'tsey ) in~ ARCH ~him ( בְגַפּוֹ / vê'ga'po )

If his lord will give him a woman, and she will bring forth for him sons or daughters, the woman and her boys will exist for her lord, and he, he will go out by himself,

5 and~ IF ( וְאִם / wê'im ) >~ SAY (Verb) ( אָמֹר / a'mor ) he~ will~ SAY (Verb) ( יֹאמַר / yo'mar ) the~ SERVANT ( הָעֶבֶד / ha'e'ved ) I~ did~ LOVE (Verb) ( אָהַבְתִּי / a'hav'ti ) AT ( אֶת / et ) LORD ~me ( אֲדֹנִי / a'do'ni ) AT ( אֶת / et ) WOMAN ~me ( אִשְׁתִּי / ish'ti ) and~ AT ( וְאֶת / wê'et ) SON ~s ~me ( בָּנָי / ba'nai ) NOT ( לֹא / lo ) I~ will~ GO.OUT (Verb) ( אֵצֵא / ey'tsey ) FREE ( חָפְשִׁי / hhaph'shi )

but if the servant will say, I love my lord, my woman and my sons, I will not go out free,

6 and~ he~ did~ make~ DRAW.NEAR (Verb) ~him ( וְהִגִּישׁוֹ / wê'hi'gi'sho ) LORD ~s ~him ( אֲדֹנָיו / a'do'naw ) TO ( אֶל / el ) the~ Elohiym ( הָאֱלֹהִים / ha'e'lo'him ) [ 390 ] and~ he~ did~ make~ DRAW.NEAR (Verb) ~him ( וְהִגִּישׁוֹ / wê'hi'gi'sho ) TO ( אֶל / el ) the~ DOOR ( הַדֶּלֶת / ha'de'let ) OR ( אוֹ / o ) TO ( אֶל / el ) the~ DOORPOST ( הַמְּזוּזָה / ham'zu'zah ) and~ he~ did~ BORE.THROUGH (Verb) ( וְרָצַע / wê'ra'tsa ) LORD ~s ~him ( אֲדֹנָיו / a'do'naw ) AT ( אֶת / et ) EAR ~him ( אָזְנוֹ / az'no ) in~ the~ AWL ( בַּמַּרְצֵעַ / ba'mar'tsey'a ) and~ he~ did~ SERVE (Verb) ~him ( וַעֲבָדוֹ / wa'a'vad'o ) to~ DISTANT ( לְעֹלָם / lê'o'lam )

then his lord will make him draw near to the Elohiym, and he will make him draw near to the door, or to the doorpost, and his lord will bore through his ear with the awl and he will serve him to a distant time,

7 and~ GIVEN.THAT ( וְכִי / wê'khi ) he~ will~ SELL (Verb) ( יִמְכֹּר / yim'kor ) MAN ( אִישׁ / ish ) AT ( אֶת / et ) DAUGHTER ~him ( בִּתּוֹ / bi'to ) to~ BONDWOMAN ( לְאָמָה / lê'a'mah ) NOT ( לֹא / lo ) she~ will~ GO.OUT (Verb) ( תֵצֵא / tey'tsey ) like~ >~ GO.OUT (Verb) ( כְּצֵאת / kê'tseyt ) the~ SERVANT ~s ( הָעֲבָדִים / ha'a'va'dim )

and if a man will sell his daughter as a bondwoman, she will not go out like the going out of the servants.

8 IF ( אִם / im ) DYSFUNCTIONAL ( רָעָה / ra'ah ) in~ EYE ~s2 ( בְּעֵינֵי / bê'ey'ney ) LORD ~s ~her ( אֲדֹנֶיהָ / a'do'ney'ah ) WHICH ( אֲשֶׁר / a'sher ) NOT ( לֹא / lo ) [ 391 ] he~ did~ APPOINT (Verb) ~her ( יְעָדָהּ / yê'a'dah ) [ 392 ] and~ he~ did~ make~ RANSOM (Verb) ~her ( וְהֶפְדָּהּ / wê'heph'dah ) to~ PEOPLE ( לְעַם / lê'am ) FOREIGN ( נָכְרִי / nakh'ri ) NOT ( לֹא / lo ) he~ will~ REGULATE (Verb) ( יִמְשֹׁל / yim'shol ) [ 393 ] to~ >~ SELL (Verb) ~her ( לְמָכְרָהּ / lê'makh'rah ) in~ GARMENT ~him ( בְּבִגְדוֹ / bê'vig'do ) in~ ~her ( בָהּ / vah )

If she is dysfunctional in the eyes of her lord, which he did not appoint, then he will ransom her, he will not regulate to sell her in his treacherous act with her,

9 and~ IF ( וְאִם / wê'im ) to~ SON ~him ( לִבְנוֹ / liv'no ) he~ will~ APPOINT (Verb) ~her ( יִיעָדֶנָּה / yi'a'de'nah ) like~ DECISION ( כְּמִשְׁפַּט / kê'mish'pat ) [ 394 ] the~ DAUGHTER ~s ( הַבָּנוֹת / ha'ba'not ) he~ will~ DO (Verb) ( יַעֲשֶׂה / ya'a'seh ) to~ ~her ( לָּהּ / lah )

and if he will appoint her to his son, he will do to her just like the decision of the daughters.

10 IF ( אִם / im ) OTHER ( אַחֶרֶת / a'hhe'ret ) he~ will~ TAKE (Verb) ( יִקַּח / yi'qahh ) to~ ~him ( לוֹ / lo ) REMAINS ~her ( שְׁאֵרָהּ / shê'ey'rah ) RAIMENT ~her ( כְּסוּתָהּ / kê'su'tah ) and~ COHABITATION ~her ( וְעֹנָתָהּ / wê'o'na'tah ) NOT ( לֹא / lo ) he~ will~ TAKE.AWAY (Verb) ( יִגְרָע / yig'ra )

If he will take another, he will not take away her remains, her raiment and her habitation,

11 and~ IF ( וְאִם / wê'im ) THREE ( שְׁלָשׁ / shê'lash ) THESE ( אֵלֶּה / ey'leh ) NOT ( לֹא / lo ) he~ will~ DO (Verb) ( יַעֲשֶׂה / ya'a'seh ) to~ ~her ( לָהּ / lah ) and~ she~ did~ GO.OUT (Verb) ( וְיָצְאָה / wê'yats'ah ) FREELY ( חִנָּם / hhi'nam ) WITHOUT ( אֵין / eyn ) SILVER ( כָּסֶף / ka'seph )

and if he will not do these three to her, then she will go out freely without silver.

12 make~ HIT (Verb) ~ing(ms) ( מַכֵּה / ma'keyh ) MAN ( אִישׁ / ish ) and~ he~ did~ DIE (Verb) ( וָמֵת / wa'meyt ) >~ DIE (Verb) ( מוֹת / mot ) he~ will~ be~ make~ DIE (Verb) ( יוּמָת / yu'mat )

The one attacking a man, and dies, he will be killed,

13 and~ WHICH ( וַאֲשֶׁר / wa'a'sher ) NOT ( לֹא / lo ) he~ did~ LAY.IN.WAIT (Verb) ( צָדָה / tsa'dah ) and~ the~ Elohiym ( וְהָאֱלֹהִים / wê'ha'e'lo'him ) [ 395 ] he~ did~ much~ APPROACH (Verb) ( אִנָּה / i'nah ) to~ HAND ~him ( לְיָדוֹ / lê'ya'do ) and~ I~ did~ PLACE (Verb) ( וְשַׂמְתִּי / wê'sam'ti ) to~ ~you(ms) ( לְךָ / lê'kha ) AREA ( מָקוֹם / ma'qom ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ FLEE (Verb) ( יָנוּס / ya'nus ) THERE ~unto ( שָׁמָּה / sha'mah )

and when he did not lay in wait, and the Elohiym delivers him to his hand, then I will place an area for you where he will flee unto,

14 and~ GIVEN.THAT ( וְכִי / wê'khi ) he~ will~ make~ SIMMER (Verb) ( יָזִד / ya'zid ) [ 396 ] MAN ( אִישׁ / ish ) UPON ( עַל / al ) COMPANION ~him ( רֵעֵהוּ / rey'ey'hu ) to~ >~ KILL (Verb) ~him ( לְהָרְגוֹ / lê'har'go ) in~ SUBTLETY ( בְעָרְמָה / vê'ar'mah ) from~ WITH ( מֵעִם / mey'im ) ALTAR ~me ( מִזְבְּחִי / miz'be'hhi ) you(ms)~ will~ TAKE (Verb) ~him ( תִּקָּחֶנּוּ / ti'qa'hhe'nu ) to~ >~ DIE (Verb) ( לָמוּת / la'mut )

and if a man simmers upon his companion to kill him with subtlety, you will take him from my altar to die,

15 and~ make~ HIT (Verb) ~ing(ms) ( וּמַכֵּה / u'ma'keyh ) FATHER ~him ( אָבִיו / a'viw ) and~ MOTHER ~him ( וְאִמּוֹ / wê'i'mo ) >~ DIE (Verb) ( מוֹת / mot ) he~ will~ be~ make~ DIE (Verb) ( יוּמָת / yu'mat )

and the one attacking his father or his mother, will surely be killed,

16 and~ STEAL (Verb) ~ing(ms) ( וְגֹנֵב / wê'go'neyv ) MAN ( אִישׁ / ish ) and~ he~ did~ SELL (Verb) ~him ( וּמְכָרוֹ / um'kha'ro ) and~ he~ did~ be~ FIND (Verb) ( וְנִמְצָא / wê'nim'tsa ) in~ HAND ~him ( בְיָדוֹ / vê'ya'do ) [ 397 ] >~ DIE (Verb) ( מוֹת / mot ) he~ will~ be~ make~ DIE (Verb) ( יוּמָת / yu'mat )

and anyone stealing a man and sells him or is found in his hand, he will surely be killed,

17 and~ much~ BELITTLE (Verb) ~ing(ms) ( וּמְקַלֵּל / um'qa'leyl ) FATHER ~him ( אָבִיו / a'viw ) and~ MOTHER ~him ( וְאִמּוֹ / wê'i'mo ) >~ DIE (Verb) ( מוֹת / mot ) he~ will~ be~ make~ DIE (Verb) ( יוּמָת / yu'mat )

and anyone belittling his father or his mother, he will surely be killed,

18 and~ GIVEN.THAT ( וְכִי / wê'khi ) they(m)~ will~ DISPUTE (Verb) ~must ( יְרִיבֻן / yê'ri'vun ) MAN ~s ( אֲנָשִׁים / a'na'shim ) and~ he~ did~ make~ HIT (Verb) ( וְהִכָּה / wê'hi'kah ) MAN ( אִישׁ / ish ) AT ( אֶת / et ) COMPANION ~him ( רֵעֵהוּ / rey'ey'hu ) in~ STONE ( בְּאֶבֶן / bê'e'ven ) OR ( אוֹ / o ) in~ FIST ( בְאֶגְרֹף / ve'eg'roph ) and~ NOT ( וְלֹא / wê'lo ) he~ will~ DIE (Verb) ( יָמוּת / ya'mut ) and~ he~ did~ FALL (Verb) ( וְנָפַל / wê'na'phal ) to~ LYING.PLACE ( לְמִשְׁכָּב / lê'mish'kav )

and if men must dispute, and a man will attack his companion with a stone, or with a fist, and he does not die, then he will fall to the lying place.

19 IF ( אִם / im ) he~ will~ RISE (Verb) ( יָקוּם / ya'qum ) and~ he~ did~ self~ WALK (Verb) ( וְהִתְהַלֵּךְ / wê'hit'ha'leykh ) in~ the~ OUTSIDE ( בַּחוּץ / ba'hhuts ) UPON ( עַל / al ) STAVE ~him ( מִשְׁעַנְתּוֹ / mish'a'ne'to ) and~ he~ did~ be~ ACQUIT (Verb) ( וְנִקָּה / wê'ni'qah ) make~ HIT (Verb) ~ing(ms) ( הַמַּכֶּה / ha'ma'keh ) ONLY ( רַק / raq ) CEASING ~him ( שִׁבְתּוֹ / shiv'to ) he~ will~ GIVE (Verb) ( יִתֵּן / yi'teyn ) [ 398 ] and~ >~ much~ HEAL (Verb) ( וְרַפֹּא / wê'ra'po ) he~ will~ much~ HEAL (Verb) ( יְרַפֵּא / yê'ra'pey ) [ 399 ]

If he will rise, and he walks himself to the outside upon his stave, then the one attacking him will be acquitted, only his ceasing he will give him and he will be completely healed,

20 and~ GIVEN.THAT ( וְכִי / wê'khi ) he~ will~ make~ HIT (Verb) ( יַכֶּה / ya'keh ) MAN ( אִישׁ / ish ) AT ( אֶת / et ) SERVANT ~him ( עַבְדּוֹ / av'do ) OR ( אוֹ / o ) AT ( אֶת / et ) BONDWOMAN ~him ( אֲמָתוֹ / a'ma'to ) in~ the~ STAFF ( בַּשֵּׁבֶט / ba'shey'vet ) and~ he~ did~ DIE (Verb) ( וּמֵת / u'meyt ) UNDER ( תַּחַת / ta'hhat ) HAND ~him ( יָדוֹ / ya'do ) >~ AVENGE (Verb) ( נָקֹם / na'qom ) he~ will~ be~ AVENGE (Verb) ( יִנָּקֵם / yi'na'qeym )

and if a man attacks his servant or his bondwoman with the staff, and he dies by his hand, he will surely be avenged.

21 SURELY ( אַךְ / akh ) IF ( אִם / im ) DAY ( יוֹם / yom ) OR ( אוֹ / o ) DAY ~s2 ( יוֹמַיִם / yo'ma'yim ) he~ will~ STAND (Verb) ( יַעֲמֹד / ya'a'mod ) NOT ( לֹא / lo ) he~ will~ be~ make~ AVENGE (Verb) ( יֻקַּם / yu'qam ) GIVEN.THAT ( כִּי / ki ) SILVER ~him ( כַסְפּוֹ / khas'po ) HE ( הוּא / hu )

However, if he will stand for a day or two days, he will not be avenged, given that he is his silver,

22 and~ GIVEN.THAT ( וְכִי / wê'khi ) they(m)~ will~ be~ STRUGGLE (Verb) ( יִנָּצוּ / yi'na'tsu ) MAN ~s ( אֲנָשִׁים / a'na'shim ) and~ they~ did~ SMITE (Verb) ( וְנָגְפוּ / wê'nag'phu ) WOMAN ( אִשָּׁה / i'shah ) PREGNANT ( הָרָה / ha'rah ) and~ they~ did~ GO.OUT (Verb) ( וְיָצְאוּ / wê'yats'u ) BOY ~s ~her ( יְלָדֶיהָ / yê'la'dey'ah ) [ 400 ] and~ NOT ( וְלֹא / wê'lo ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) HARM ( אָסוֹן / a'son ) >~ FINE (Verb) ( עָנוֹשׁ / a'nosh ) he~ will~ be~ FINE (Verb) ( יֵעָנֵשׁ / yey'a'neysh ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ will~ SET.DOWN (Verb) ( יָשִׁית / ya'shit ) UPON ~him ( עָלָיו / a'law ) MASTER ( בַּעַל / ba'al ) the~ WOMAN ( הָאִשָּׁה / ha'i'shah ) and~ he~ did~ GIVE (Verb) ( וְנָתַן / wê'na'tan ) in~ JUDGE ~s ( בִּפְלִלִים / biph'li'lim )

and if men struggle, and they smite a pregnant woman, and her boys go out, but harm did not exist, he will surely be fined just as the master of the woman will set down upon him, and he will give the judgments,

23 and~ IF ( וְאִם / wê'im ) HARM ( אָסוֹן / a'son ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) and~ you(ms)~ did~ GIVE (Verb) ~& ( וְנָתַתָּה / wê'na'ta'tah ) SOUL ( נֶפֶשׁ / ne'phesh ) UNDER ( תַּחַת / ta'hhat ) SOUL ( נָפֶשׁ / na'phesh )

but if harm does exist, then you will give a soul in place of a soul.

24 EYE ( עַיִן / a'yin ) UNDER ( תַּחַת / ta'hhat ) EYE ( עַיִן / a'yin ) TOOTH ( שֵׁן / sheyn ) UNDER ( תַּחַת / ta'hhat ) TOOTH ( שֵׁן / sheyn ) HAND ( יָד / yad ) UNDER ( תַּחַת / ta'hhat ) HAND ( יָד / yad ) FOOT ( רֶגֶל / re'gel ) UNDER ( תַּחַת / ta'hhat ) FOOT ( רָגֶל / ra'gel )

An eye in place of an eye, a tooth in place of a tooth, a hand in place of a hand, a foot in place of a foot.

25 SINGEING ( כְּוִיָּה / kê'wi'yah ) UNDER ( תַּחַת / ta'hhat ) SINGEING ( כְּוִיָּה / kê'wi'yah ) WOUND ( פֶּצַע / pe'tsa ) UNDER ( תַּחַת / ta'hhat ) WOUND ( פָּצַע / pa'tsa ) STRIPED.BRUISE ( חַבּוּרָה / hha'bu'rah ) UNDER ( תַּחַת / ta'hhat ) STRIPED.BRUISE ( חַבּוּרָה / hha'bu'rah )

A singeing in place of a singeing, a bruise in place of a bruise, a striped bruise in place of a striped bruise,

26 and~ GIVEN.THAT ( וְכִי / wê'khi ) he~ will~ make~ HIT (Verb) ( יַכֶּה / ya'keh ) MAN ( אִישׁ / ish ) AT ( אֶת / et ) EYE ( עֵין / eyn ) SERVANT ~him ( עַבְדּוֹ / av'do ) OR ( אוֹ / o ) AT ( אֶת / et ) EYE ( עֵין / eyn ) BONDWOMAN ~him ( אֲמָתוֹ / a'ma'to ) and~ he~ did~ much~ DAMAGE (Verb) ~her ( וְשִׁחֲתָהּ / wê'shi'hha'tah ) to~ FREE ( לַחָפְשִׁי / la'hhaph'shi ) he~ will~ much~ SEND (Verb) ~him ( יְשַׁלְּחֶנּוּ / yê'shal'hhe'nu ) UNDER ( תַּחַת / ta'hhat ) EYE ~him ( עֵינוֹ / ey'no )

but if a man will attack the eye of his servant, or the eye of his bondwoman, and he damages her, he will send him to freedom in place of his eye,
