Genesis Chapter 15

1 AFTER ( אַחַר / a'hhar ) the~ WORD ~s ( הַדְּבָרִים / ha'de'va'rim ) the~ THESE ( הָאֵלֶּה / ha'ey'leh ) he~ did~ EXIST (Verb) ( הָיָה / hai'yah ) WORD ( דְבַר / dê'var ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Avram ( אַבְרָם / av'ram ) in~ the~ VISION ( בַּמַּחֲזֶה / ba'ma'hha'zeh ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) DO.NOT ( אַל / al ) you(ms)~ will~ FEAR (Verb) ( תִּירָא / ti'ra ) Avram ( אַבְרָם / av'ram ) I ( אָנֹכִי / a'no'khi ) SHIELD ( מָגֵן / ma'geyn ) to~ ~you(fs) ( לָךְ / lakh ) WAGE ~you(ms) ( שְׂכָרְךָ / sê'khar'kha ) >~ make~ INCREASE (Verb) ( הַרְבֵּה / har'beyh ) MANY ( מְאֹד / mê'od )

After these words, the word of YHWH existed for Avram in the vision saying, do not fear Avram, I am a shield for you, your wages will increase greatly,

2 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Avram ( אַבְרָם / av'ram ) Adonai ( אֲדֹנָי / a'do'nai ) YHWH ( יְהוִה / YHWH ) WHAT ( מַה / mah ) you(ms)~ will~ GIVE (Verb) ( תִּתֶּן / ti'ten ) to~ ~me ( לִי / li ) and~ I ( וְאָנֹכִי / wê'a'no'khi ) WALK (Verb) ~ing(ms) ( הוֹלֵךְ / ho'leykh ) BARREN ( עֲרִירִי / a'ri'ri ) and~ SON ( וּבֶן / u'ven ) ACQUISITION ( מֶשֶׁק / me'sheq ) HOUSE ~me ( בֵּיתִי / bey'ti ) HE ( הוּא / hu ) Dameseq ( דַּמֶּשֶׂק / da'me'seq ) Eli'ezer ( אֱלִיעֶזֶר / e'li'e'zer ) [ 107 ]

and Avram said, Adonai of YHWH, what will you give to me as I am walking barren and the son of acquisition of my house is Eli'ezer of Dameseq,

3 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Avram ( אַבְרָם / av'ram ) THOUGH ( הֵן / heyn ) to~ ~me ( לִי / li ) NOT ( לֹא / lo ) you(ms)~ did~ GIVE (Verb) ( נָתַתָּה / na'ta'tah ) SEED ( זָרַע / za'ra ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) SON ( בֶן / ven ) HOUSE ~me ( בֵּיתִי / bey'ti ) POSSESS (Verb) ~ing(ms) ( יוֹרֵשׁ / yo'reysh ) AT ~me ( אֹתִי / o'ti )

and Avram said, though you did not give me seed, look, a son of my house is possessing me,

4 and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) WORD ( דְבַר / dê'var ) YHWH ( יְהוָה / YHWH ) TO ~him ( אֵלָיו / ey'law ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) NOT ( לֹא / lo ) he~ will~ POSSESS (Verb) ~you(ms) ( יִירָשְׁךָ / yi'rash'kha ) THIS ( זֶה / zeh ) GIVEN.THAT ( כִּי / ki ) IF ( אִם / im ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ GO.OUT (Verb) ( יֵצֵא / yey'tsey ) from~ ABDOMEN ~s ~you(ms) ( מִמֵּעֶיךָ / mi'mey'ey'kha ) HE ( הוּא / hu ) he~ will~ POSSESS (Verb) ~you(ms) ( יִירָשֶׁךָ / yi'ra'she'kha )

and look, the word of YHWH was for him saying, this one will not possess you, instead he which will go out from your abdomens, he will possess you,

5 and~ he~ will~ make~ GO.OUT (Verb) ( וַיּוֹצֵא / wai'yo'tsey ) AT ~him ( אֹתוֹ / o'to ) the~ OUTSIDE ~unto ( הַחוּצָה / ha'hhu'tsah ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) !(ms)~ make~ STARE (Verb) ( הַבֶּט / ha'bet ) PLEASE ( נָא / na ) the~ SKY ~s2 ~unto ( הַשָּׁמַיְמָה / ha'sha'mai'mah ) and~ !(ms)~ COUNT (Verb) ( וּסְפֹר / us'phor ) the~ STAR ~s ( הַכּוֹכָבִים / ha'ko'kha'vim ) IF ( אִם / im ) you(ms)~ will~ BE.ABLE (Verb) ( תּוּכַל / tu'khal ) to~ >~ COUNT (Verb) ( לִסְפֹּר / lis'por ) AT ~them(m) ( אֹתָם / o'tam ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ ~him ( לוֹ / lo ) IN.THIS.WAY ( כֹּה / koh ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) SEED ~you(ms) ( זַרְעֶךָ / zar'e'kha )

and he brought him out unto the outside and he said, please, stare unto the skies and count the stars, if you are able to count them, and he said to him, in this way your seed will exist,

6 and~ he~ will~ make~ SECURE (Verb) ( וְהֶאֱמִן / wê'he'e'min ) in~ YHWH ( בַּיהוָה / ba'YHWH ) and~ he~ will~ THINK (Verb) ~her ( וַיַּחְשְׁבֶהָ / wai'yahh'she've'ah ) to~ ~him ( לּוֹ / lo ) STEADFASTNESS ( צְדָקָה / tsê'da'qah )

and he supported in YHWH and he thought it was steadfastness for him,

7 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) TO ~him ( אֵלָיו / ey'law ) I ( אֲנִי / a'ni ) YHWH ( יְהוָה / YHWH ) WHICH ( אֲשֶׁר / a'sher ) I~ did~ make~ GO.OUT (Verb) ~you(ms) ( הוֹצֵאתִיךָ / ho'tsey'ti'kha ) from~ Ur ( מֵאוּר / mey'ur ) Kesed ~s ( כַּשְׂדִּים / kas'dim ) to~ >~ GIVE (Verb) ( לָתֶת / la'tet ) to~ ~you(ms) ( לְךָ / lê'kha ) AT ( אֶת / et ) the~ LAND ( הָאָרֶץ / ha'a'rets ) the~ THIS ( הַזֹּאת / ha'zot ) to~ >~ POSSESS (Verb) ~her ( לְרִשְׁתָּהּ / lê'rish'tah )

and he said to him, I am YHWH who brought you out from Ur of the ones of Kesed, to give to you this land to possess her,

8 and~ he~ will~ SAY (Verb) ( וַיֹּאמַר / wai'yo'mar ) Adonai ( אֲדֹנָי / a'do'nai ) YHWH ( יְהוִה / YHWH ) in~ WHAT ( בַּמָּה / ba'mah ) I~ will~ KNOW (Verb) ( אֵדַע / ey'da ) GIVEN.THAT ( כִּי / ki ) I~ will~ POSSESS (Verb) ~her ( אִירָשֶׁנָּה / i'ra'she'nah )

and he said, Adonai of YHWH how will I know that I will inherit her,

9 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) TO ~him ( אֵלָיו / ey'law ) !(ms)~ TAKE (Verb) ~& ( קְחָה / qê'hhah ) to~ ~me ( לִי / li ) HEIFER ( עֶגְלָה / eg'lah ) much~ BE.THREEFOLD (Verb) ~ing(fs) ( מְשֻׁלֶּשֶׁת / mê'shu'le'shet ) [ 108 ] and~ GOAT ( וְעֵז / wê'eyz ) much~ BE.THREEFOLD (Verb) ~ing(fs) ( מְשֻׁלֶּשֶׁת / mê'shu'le'shet ) and~ BUCK ( וְאַיִל / wê'a'yil ) much~ BE.THREEFOLD (Verb) ~ing(ms) ( מְשֻׁלָּשׁ / mê'shu'lash ) and~ TURTLEDOVE ( וְתֹר / wê'tor ) and~ YOUNG.PIGEON ( וְגוֹזָל / wê'go'zal )

and he said to him, take for me a heifer being threefold, and a goat being a threefold, and a buck being a threefold and a turtledove and a young pigeon,

10 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) to~ ~him ( לוֹ / lo ) AT ( אֶת / et ) ALL ( כָּל / kol ) THESE ( אֵלֶּה / ey'leh ) and~ he~ will~ CUT.IN.TWO (Verb) ( וַיְבַתֵּר / wa'ye'va'teyr ) AT ~them(m) ( אֹתָם / o'tam ) in~ the~ MIDST ( בַּתָּוֶךְ / ba'ta'wekh ) [ 109 ] and~ he~ will~ GIVE (Verb) ( וַיִּתֵּן / wai'yi'teyn ) MAN ( אִישׁ / ish ) CUT.PIECE ~him ( בִּתְרוֹ / bit'ro ) to~ >~ MEET (Verb) ( לִקְרַאת / liq'rat ) COMPANION ~him ( רֵעֵהוּ / rey'ey'hu ) and~ AT ( וְאֶת / wê'et ) the~ BIRD ( הַצִּפֹּר / ha'tsi'por ) NOT ( לֹא / lo ) he~ did~ CUT.IN.TWO (Verb) ( בָתָר / va'tar )

and he took for himself all these, and he cut them in two in the midst, and he gave each cut piece of him to meet his companion, and the bird he had not cut in two,

11 and~ he~ will~ GO.DOWN (Verb) ( וַיֵּרֶד / wai'yey'red ) the~ BIRD.OF.PREY ( הָעַיִט / ha'a'yit ) UPON ( עַל / al ) the~ CORPSE ~s ( הַפְּגָרִים / ha'pe'ga'rim ) and~ he~ will~ make~ GUST (Verb) ( וַיַּשֵּׁב / wai'ya'sheyv ) AT ~them(m) ( אֹתָם / o'tam ) Avram ( אַבְרָם / av'ram )

and the bird of prey went down upon the corpses, and Avram made a gust at them,

12 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) the~ SUN ( הַשֶּׁמֶשׁ / ha'she'mesh ) to~ >~ COME (Verb) ( לָבוֹא / la'vo ) and~ TRANCE ( וְתַרְדֵּמָה / wê'tar'dey'mah ) she~ did~ FALL (Verb) ( נָפְלָה / naph'lah ) UPON ( עַל / al ) Avram ( אַבְרָם / av'ram ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) TERROR ( אֵימָה / ey'mah ) DARK ( חֲשֵׁכָה / hha'shey'khah ) GREAT ( גְדֹלָה / gê'do'lah ) FALL (Verb) ~ing(fs) ( נֹפֶלֶת / no'phe'let ) UPON ~him ( עָלָיו / a'law )

and it came to pass, the sun came, and a trance had fallen upon Avram, and look, a terror of a great darkness was falling upon him,

13 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ Avram ( לְאַבְרָם / lê'av'ram ) >~ KNOW (Verb) ( יָדֹעַ / ya'do'a ) you(ms)~ will~ KNOW (Verb) ( תֵּדַע / tey'da ) GIVEN.THAT ( כִּי / ki ) IMMIGRANT ( גֵר / geyr ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) SEED ~you(ms) ( זַרְעֲךָ / zar'a'kha ) in~ LAND ( בְּאֶרֶץ / bê'e'rets ) NOT ( לֹא / lo ) to~ ~them(m) ( לָהֶם / la'hem ) and~ they~ did~ SERVE (Verb) ~them(m) ( וַעֲבָדוּם / wa'a'va'dum ) and~ they~ did~ much~ AFFLICT (Verb) ( וְעִנּוּ / wê'i'nu ) AT ~them(m) ( אֹתָם / o'tam ) FOUR ( אַרְבַּע / ar'ba ) HUNDRED ~s ( מֵאוֹת / mey'ot ) YEAR ( שָׁנָה / sha'nah )

and he said to Avram, you will surely know that your seed will exist as an immigrant in a land not belonging to them, and they will serve them and they will afflict them four hundred years,

14 and~ ALSO ( וְגַם / wê'gam ) AT ( אֶת / et ) the~ NATION ( הַגּוֹי / ha'goi ) WHICH ( אֲשֶׁר / a'sher ) they(m)~ will~ SERVE (Verb) ( יַעֲבֹדוּ / ya'a'vo'du ) MODERATE (Verb) ~ing(ms) ( דָּן / dan ) I ( אָנֹכִי / a'no'khi ) and~ AFTER ( וְאַחֲרֵי / wê'a'hha'rey ) SO ( כֵן / kheyn ) they(m)~ will~ GO.OUT (Verb) ( יֵצְאוּ / yeyts'u ) in~ GOODS ( בִּרְכֻשׁ / bir'khush ) GREAT ( גָּדוֹל / ga'dol )

and also, the nation which they will serve, I am moderating, and afterward they will go out with great goods,

15 and~ YOU(MS) ( וְאַתָּה / wê'a'tah ) you(ms)~ will~ COME (Verb) ( תָּבוֹא / ta'vo ) TO ( אֶל / el ) FATHER ~s ~you(ms) ( אֲבֹתֶיךָ / a'vo'tey'kha ) in~ COMPLETENESS ( בְּשָׁלוֹם / bê'sha'lom ) you(ms)~ will~ be~ BURY (Verb) ( תִּקָּבֵר / ti'qa'veyr ) in~ GRAY-HEADED ( בְּשֵׂיבָה / bê'sey'vah ) FUNCTIONAL ( טוֹבָה / to'vah )

and you will come to your fathers in completeness, you will be buried with a functional gray-head,

16 and~ GENERATION ( וְדוֹר / wê'dor ) FOURTH ( רְבִיעִי / rê'vi'i ) they(m)~ will~ TURN.BACK (Verb) ( יָשׁוּבוּ / ya'shu'vu ) TO.THIS.POINT ( הֵנָּה / heyn'nah ) GIVEN.THAT ( כִּי / ki ) NOT ( לֹא / lo ) COMPLETENESS ( שָׁלֵם / sha'leym ) TWISTEDNESS ( עֲו‍ֹן / a'won ) the~ Emor ~of ( הָאֱמֹרִי / ha'e'mo'ri ) UNTIL ( עַד / ad ) TO.THIS.POINT ( הֵנָּה / heyn'nah ) [ 110 ]

and the fourth generation will turn back to this point, given that the twistedness of the ones of Emor are not complete at this point,

17 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) the~ SUN ( הַשֶּׁמֶשׁ / ha'she'mesh ) she~ did~ COME (Verb) ( בָּאָה / ba'ah ) and~ TWILIGHT ( וַעֲלָטָה / wa'a'la'tah ) he~ did~ EXIST (Verb) ( הָיָה / hai'yah ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) OVEN ( תַנּוּר / ta'nur ) SMOKE ( עָשָׁן / a'shan ) and~ TORCH ( וְלַפִּיד / wê'la'pid ) FIRE ( אֵשׁ / eysh ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ CROSS.OVER (Verb) ( עָבַר / a'var ) BETWEEN ( בֵּין / beyn ) the~ DIVIDED.PART ~s ( הַגְּזָרִים / hag'za'rim ) the~ THESE ( הָאֵלֶּה / ha'ey'leh )

and it came to pass, the sun came and it was twilight, and look, an oven of smoke and a torch of fire that crossed over between these divided parts.

18 in~ the~ DAY ( בַּיּוֹם / ba'yom ) the~ HE ( הַהוּא / ha'hu ) he~ did~ CUT (Verb) ( כָּרַת / ka'rat ) YHWH ( יְהוָה / YHWH ) AT ( אֶת / et ) Avram ( אַבְרָם / av'ram ) COVENANT ( בְּרִית / bê'rit ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) to~ SEED ~you(ms) ( לְזַרְעֲךָ / lê'zar'a'kha ) I~ did~ GIVE (Verb) ( נָתַתִּי / na'ta'ti ) AT ( אֶת / et ) the~ LAND ( הָאָרֶץ / ha'a'rets ) the~ THIS ( הַזֹּאת / ha'zot ) from~ RIVER ( מִנְּהַר / mi'ne'har ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) UNTIL ( עַד / ad ) the~ RIVER ( הַנָּהָר / ha'na'har ) the~ GREAT ( הַגָּדֹל / ha'ga'dol ) RIVER ( נְהַר / nê'har ) Perat ( פְּרָת / pê'rat )

In that day YHWH cut with Avram a covenant saying, to your seed I gave this land, from the river of Mits'rayim as far as the great river, the river Perat.

19 AT ( אֶת / et ) the~ Qayin ~of ( הַקֵּינִי / ha'qey'ni ) and~ AT ( וְאֶת / wê'et ) the~ Qenaz ~of ( הַקְּנִזִּי / haq'ni'zi ) and~ AT ( וְאֵת / wê'eyt ) the~ Qadmon ~of ( הַקַּדְמֹנִי / ha'qad'mo'ni )

The ones of Qayin, and the ones of Qenaz, and the ones of Qadmon,

20 and~ AT ( וְאֶת / wê'et ) the~ Hhet ~of ( הַחִתִּי / ha'hhi'ti ) and~ AT ( וְאֶת / wê'et ) the~ Perez ~of ( הַפְּרִזִּי / ha'pe'ri'zi ) and~ AT ( וְאֶת / wê'et ) the~ Rapha ~s ( הָרְפָאִים / ha're'pha'im )

and the ones of Hhet, and the one of Perez, and the one of Rapha,

21 and~ AT ( וְאֶת / wê'et ) the~ Emor ~of ( הָאֱמֹרִי / ha'e'mo'ri ) and~ AT ( וְאֶת / wê'et ) the~ Kena'an ~of ( הַכְּנַעֲנִי / ha'ke'na'a'ni ) and~ AT ( וְאֶת / wê'et ) the~ Girgash ~of ( הַגִּרְגָּשִׁי / ha'gir'ga'shi ) and~ AT ( וְאֶת / wê'et ) the~ Yevus ~of ( הַיְבוּסִי / hai'vu'si )

and the ones of Emor, and the one of Kena'an, and the ones of Girgash, and the one of Yevus,
