Deuteronomy Chapter 18

1 NOT ( לֹא / lo ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) to~ the~ ADMINISTRATOR ~s ( לַכֹּהֲנִים / la'ko'ha'nim ) the~ Lewi ~s ( הַלְוִיִּם / hal'wi'yim ) ALL ( כָּל / kol ) STAFF ( שֵׁבֶט / shey'vet ) [ 1003 ] Lewi ( לֵוִי / ley'wi ) DISTRIBUTION ( חֵלֶק / hhey'leq ) and~ INHERITANCE ( וְנַחֲלָה / wê'na'hha'lah ) WITH ( עִם / im ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) FIRE.OFFERING ~s ( אִשֵּׁי / i'shey ) YHWH ( יְהוָה / YHWH ) and~ INHERITANCE ~him ( וְנַחֲלָתוֹ / wê'na'hha'la'to ) he~ will~ EAT (Verb) ~must ( יֹאכֵלוּן / yo'khey'lun )

A distribution and an inheritance with Yisra'eyl will not exist for the administrators, the ones of Lewi, any staff of Lewi, fire offerings of YHWH and his inheritance he must eat,

2 and~ INHERITANCE ( וְנַחֲלָה / wê'na'hha'lah ) NOT ( לֹא / lo ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) to~ ~him ( לּוֹ / lo ) in~ INSIDE ( בְּקֶרֶב / bê'qe'rev ) BROTHER ~s ~him ( אֶחָיו / e'hhaw ) YHWH ( יְהוָה / YHWH ) HE ( הוּא / hu ) INHERITANCE ~him ( נַחֲלָתוֹ / na'hha'la'to ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) to~ ~him ( לוֹ / lo )

and an inheritance will not exist for him in among his brothers, YHWH, he is his inheritance just as he spoke to him,

3 and~ THIS ( וְזֶה / wê'zeh ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) DECISION ( מִשְׁפַּט / mish'pat ) the~ ADMINISTRATOR ~s ( הַכֹּהֲנִים / ha'ko'ha'nim ) from~ AT ( מֵאֵת / mey'eyt ) the~ PEOPLE ( הָעָם / ha'am ) from~ AT ( מֵאֵת / mey'eyt ) SACRIFICE (Verb) ~ing(mp) ( זֹבְחֵי / zov'hhey ) the~ SACRIFICE ( הַזֶּבַח / ha'ze'vahh ) IF ( אִם / im ) OX ( שׁוֹר / shor ) IF ( אִם / im ) RAM ( שֶׂה / seh ) and~ he~ did~ GIVE (Verb) ( וְנָתַן / wê'na'tan ) to~ the~ ADMINISTRATOR ( לַכֹּהֵן / la'ko'heyn ) the~ ARM ( הַזְּרֹעַ / haz'ro'a ) and~ the~ JAW ~s2 ( וְהַלְּחָיַיִם / wê'hal'hhai'ya'yim ) and~ the~ STOMACH ( וְהַקֵּבָה / wê'ha'qey'vah )

and this will be the decision of the administrators from the people, from ones sacrificing the sacrifice, if an ox, if a ram, and he will give it to the administrator the arm and the jaws and the stomach.

4 ORIGIN ( רֵאשִׁית / rey'shit ) [ 1004 ] CEREAL ~you(ms) ( דְּגָנְךָ / dê'gan'kha ) FRESH.WINE ~you(ms) ( תִּירֹשְׁךָ / ti'rosh'kha ) and~ FRESH.OIL ~you(ms) ( וְיִצְהָרֶךָ / wê'yits'ha're'kha ) and~ ORIGIN ( וְרֵאשִׁית / wê'rey'shit ) FLEECE ( גֵּז / geyz ) FLOCKS ~you(ms) ( צֹאנְךָ / tson'kha ) you(ms)~ will~ GIVE (Verb) ( תִּתֶּן / ti'ten ) to~ ~him ( לּוֹ / lo )

The origin of your cereal, your fresh wine, and the origin of the fleece of your flocks, you will give to him.

5 GIVEN.THAT ( כִּי / ki ) in~ ~him ( בוֹ / vo ) he~ did~ CHOOSE (Verb) ( בָּחַר / ba'hhar ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) from~ ALL ( מִכָּל / mi'kol ) STAFF ~s ~you(ms) ( שְׁבָטֶיךָ / shê'va'tey'kha ) [ 1005 ] to~ >~ STAND (Verb) ( לַעֲמֹד / la'a'mod ) to~ >~ much~ MINISTER (Verb) ( לְשָׁרֵת / lê'sha'reyt ) in~ TITLE ( בְּשֵׁם / bê'sheym ) YHWH ( יְהוָה / YHWH ) HE ( הוּא / hu ) and~ SON ~s ~him ( וּבָנָיו / u'va'naw ) ALL ( כָּל / kol ) the~ DAY ~s ( הַיָּמִים / hai'ya'mim )

Given that in him YHWH your Elohiym chose from all your staffs, to stand, to minister in the title of YHWH, he and his sons, all the days,

6 and~ GIVEN.THAT ( וְכִי / wê'khi ) he~ will~ COME (Verb) ( יָבֹא / ya'vo ) the~ Lewi ( הַלֵּוִי / ha'ley'wi ) from~ UNIT ( מֵאַחַד / mey'a'hhad ) GATE ~s ~you(ms) ( שְׁעָרֶיךָ / shê'a'rey'kha ) from~ ALL ( מִכָּל / mi'kol ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) WHICH ( אֲשֶׁר / a'sher ) HE ( הוּא / hu ) he~ did~ IMMIGRATE (Verb) ( גָּר / gar ) THERE ( שָׁם / sham ) and~ he~ did~ COME (Verb) ( וּבָא / u'va ) in~ ALL ( בְּכָל / bê'khol ) DESIRE ( אַוַּת / a'wat ) SOUL ~him ( נַפְשׁוֹ / naph'sho ) TO ( אֶל / el ) the~ AREA ( הַמָּקוֹם / ha'ma'qom ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ CHOOSE (Verb) ( יִבְחַר / yiv'hhar ) YHWH ( יְהוָה / YHWH )

and when the Lewi will come from one of your gates from all Yisra'eyl, where he immigrated there, and he will come with all the desire of his soul to the area which YHWH chose,

7 and~ he~ did~ much~ MINISTER (Verb) ( וְשֵׁרֵת / wê'shey'reyt ) in~ TITLE ( בְּשֵׁם / bê'sheym ) YHWH ( יְהוָה / YHWH ) Elohiym ~him ( אֱלֹהָיו / e'lo'haw ) like~ ALL ( כְּכָל / kê'khol ) BROTHER ~s ~him ( אֶחָיו / e'hhaw ) the~ Lewi ~s ( הַלְוִיִּם / hal'wi'yim ) the~ STAND (Verb) ~ing(mp) ( הָעֹמְדִים / ha'om'dim ) THERE ( שָׁם / sham ) to~ FACE ~s ( לִפְנֵי / liph'ney ) YHWH ( יְהוָה / YHWH )

and he will minister in the title of YHWH his Elohiym, like all his brothers, the ones of Lewi, the ones standing there to the face of YHWH.

8 DISTRIBUTION ( חֵלֶק / hhey'leq ) like~ DISTRIBUTION ( כְּחֵלֶק / kê'hhey'leq ) they(m)~ will~ EAT (Verb) ( יֹאכֵלוּ / yo'khey'lu ) to~ STRAND ( לְבַד / lê'vad ) MERCHANDISE ~s ~him ( מִמְכָּרָיו / mi'me'ka'raw ) UPON ( עַל / al ) the~ FATHER ~s ( הָאָבוֹת / ha'a'vot )

Distribution like distribution they will eat, apart from his merchandise with the fathers.

9 GIVEN.THAT ( כִּי / ki ) YOU(MS) ( אַתָּה / a'tah ) he~ did~ COME (Verb) ( בָּא / ba ) TO ( אֶל / el ) the~ LAND ( הָאָרֶץ / ha'a'rets ) WHICH ( אֲשֶׁר / a'sher ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) GIVE (Verb) ~ing(ms) ( נֹתֵן / no'teyn ) to~ ~you(fs) ( לָךְ / lakh ) NOT ( לֹא / lo ) you(ms)~ will~ LEARN (Verb) ( תִלְמַד / til'mad ) to~ >~ DO (Verb) ( לַעֲשׂוֹת / la'a'sot ) like~ DISGUSTING ~s ( כְּתוֹעֲבֹת / kê'tu'a'vot ) the~ NATION ~s ( הַגּוֹיִם / ha'go'yim ) the~ THEY(m) ( הָהֵם / ha'heym )

Given that you came to the land which YHWH your Elohiym is giving to you, you will not learn to do the disgusting things of those nations.

10 NOT ( לֹא / lo ) he~ will~ be~ FIND (Verb) ( יִמָּצֵא / yi'ma'tsey ) in~ ~you(ms) ( בְךָ / vê'kha ) make~ CROSS.OVER (Verb) ~ing(ms) ( מַעֲבִיר / ma'a'vir ) SON ~him ( בְּנוֹ / bê'no ) and~ DAUGHTER ~him ( וּבִתּוֹ / u'vi'to ) in~ the~ FIRE ( בָּאֵשׁ / ba'eysh ) DIVINE (Verb) ~ing(ms) ( קֹסֵם / qo'seym ) DIVINATION ~s ( קְסָמִים / qê'sa'mim ) much~ CONJURE (Verb) ~ing(ms) ( מְעוֹנֵן / mê'o'neyn ) and~ much~ PREDICT (Verb) ~ing(ms) ( וּמְנַחֵשׁ / um'na'hheysh ) and~ much~ DO.SORCERY (Verb) ~ing(ms) ( וּמְכַשֵּׁף / um'kha'sheyph )

One will not be found among you making his son or his daughter to cross over in the fire, a divining one of divinations, conjurer or a predictor or a sorcerer.

11 and~ COUPLE (Verb) ~ing(ms) ( וְחֹבֵר / wê'hho'veyr ) COUPLE ( חָבֶר / hha'ver ) and~ INQUIRE (Verb) ~ing(ms) ( וְשֹׁאֵל / wê'sho'eyl ) NECROMANCER ( אוֹב / o'yeyv ) and~ KNOWER ( וְיִדְּעֹנִי / wê'yid'o'ni ) and~ SEEK (Verb) ~ing(ms) ( וְדֹרֵשׁ / wê'do'reysh ) TO ( אֶל / el ) the~ DIE (Verb) ~ing(mp) ( הַמֵּתִים / ha'mey'tim ) [ 1006 ]

Or a coupler of coupling or inquirer, necromancer or knower or seeker to the dying.

12 GIVEN.THAT ( כִּי / ki ) DISGUSTING ( תוֹעֲבַת / to'a'vat ) YHWH ( יְהוָה / YHWH ) ALL ( כָּל / kol ) DO (Verb) ~ing(ms) ( עֹשֵׂה / o'seyh ) THESE ( אֵלֶּה / ey'leh ) and~ in~ ON.ACCOUNT.OF ( וּבִגְלַל / u'vig'lal ) the~ DISGUSTING ~s ( הַתּוֹעֵבֹת / ha'to'ey'vot ) the~ THESE ( הָאֵלֶּה / ha'ey'leh ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) make~ POSSESS (Verb) ~ing(ms) ( מוֹרִישׁ / mo'rish ) AT ~them(m) ( אוֹתָם / o'tam ) from~ FACE ~s ~you(ms) ( מִפָּנֶיךָ / mi'pa'ney'kha )

Given that all of these doings are the disgusting things of YHWH, and on account of these disgusting things YHWH your Elohiym is dispossessing them from your face.

13 WHOLE ( תָּמִים / ta'mim ) you(ms)~ will~ EXIST (Verb) ( תִּהְיֶה / tih'yeh ) WITH ( עִם / im ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha )

You will exist whole with YHWH your Elohiym.

14 GIVEN.THAT ( כִּי / ki ) the~ NATION ~s ( הַגּוֹיִם / ha'go'yim ) the~ THESE ( הָאֵלֶּה / ha'ey'leh ) WHICH ( אֲשֶׁר / a'sher ) YOU(MS) ( אַתָּה / a'tah ) POSSESS (Verb) ~ing(ms) ( יוֹרֵשׁ / yo'reysh ) AT ~them(m) ( אוֹתָם / o'tam ) TO ( אֶל / el ) much~ CONJURE (Verb) ~ing(mp) ( מְעֹנְנִים / mê'on'nim ) and~ TO ( וְאֶל / wê'el ) DIVINE (Verb) ~ing(mp) ( קֹסְמִים / qos'mim ) they(m)~ will~ HEAR (Verb) ( יִשְׁמָעוּ / yish'ma'u ) and~ YOU(MS) ( וְאַתָּה / wê'a'tah ) NOT ( לֹא / lo ) SO ( כֵן / kheyn ) he~ did~ GIVE (Verb) ( נָתַן / na'tan ) to~ ~you(ms) ( לְךָ / lê'kha ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha )

Given that these nations, which you are possessing them, listened to the conjurer and to the diviner, and you, not so did YHWH your Elohiym give to you.

15 ANNOUNCER ( נָבִיא / na'vi ) from~ INSIDE ~you(ms) ( מִקִּרְבְּךָ / mi'qir'be'kha ) from~ BROTHER ~s ~you(ms) ( מֵאַחֶיךָ / mey'a'hhey'kha ) like~ THAT.ONE ~me ( כָּמֹנִי / ka'mo'ni ) he~ will~ make~ RISE (Verb) ( יָקִים / ya'qim ) to~ ~you(ms) ( לְךָ / lê'kha ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) TO ~him ( אֵלָיו / ey'law ) you(mp)~ will~ HEAR (Verb) ~must ( תִּשְׁמָעוּן / tish'ma'un )

An announcer from among you, from your brothers, like one of me, YHWH your Elohiym will make rise, to him you must listen.

16 like~ ALL ( כְּכֹל / kê'khol ) WHICH ( אֲשֶׁר / a'sher ) you(ms)~ did~ INQUIRE (Verb) ( שָׁאַלְתָּ / sha'al'ta ) from~ WITH ( מֵעִם / mey'im ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) in~ Hhorev ( בְּחֹרֵב / bê'hho'reyv ) in~ DAY ( בְּיוֹם / bê'yom ) the~ ASSEMBLY ( הַקָּהָל / ha'qa'hal ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) NOT ( לֹא / lo ) ADD (Verb) ~ing(ms) ( אֹסֵף / o'seyph ) to~ >~ HEAR (Verb) ( לִשְׁמֹעַ / lish'mo'a ) AT ( אֶת / et ) VOICE ( קוֹל / qol ) YHWH ( יְהוָה / YHWH ) Elohiym ~me ( אֱלֹהָי / e'lo'hai ) and~ AT ( וְאֶת / wê'et ) the~ FIRE ( הָאֵשׁ / ha'eysh ) the~ GREAT ( הַגְּדֹלָה / hag'do'lah ) the~ THIS ( הַזֹּאת / ha'zot ) NOT ( לֹא / lo ) I~ will~ SEE (Verb) ( אֶרְאֶה / er'eh ) YET.AGAIN ( עוֹד / od ) and~ NOT ( וְלֹא / wê'lo ) I~ will~ DIE (Verb) ( אָמוּת / a'mut )

Just as all you inquired from YHWH your Elohiym in Hhorev, in the day of assembly, saying, not again to hear the voice of YHWH my Elohiym, and this great fire I will not see again, and I will not die,

17 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ~me ( אֵלָי / ey'lai ) they~ did~ make~ DO.WELL (Verb) ( הֵיטִיבוּ / hey'ti'vu ) WHICH ( אֲשֶׁר / a'sher ) they~ did~ much~ SPEAK (Verb) ( דִּבֵּרוּ / di'bey'ru )

and YHWH said to me, they did well which they spoke.

An announcer I will make rise for them from among their brothers, like one of you, and I will give my words in his mouth, and he will speak to them all which I will direct him,

19 and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) the~ MAN ( הָאִישׁ / ha'ish ) WHICH ( אֲשֶׁר / a'sher ) NOT ( לֹא / lo ) he~ will~ HEAR (Verb) ( יִשְׁמַע / yish'ma ) TO ( אֶל / el ) WORD ~s ~me ( דְּבָרַי / dê'va'rai ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ much~ SPEAK (Verb) ( יְדַבֵּר / yê'da'beyr ) in~ TITLE ~me ( בִּשְׁמִי / bish'mi ) I ( אָנֹכִי / a'no'khi ) I~ will~ SEEK (Verb) ( אֶדְרֹשׁ / ed'rosh ) from~ WITH ~him ( מֵעִמּוֹ / mey'i'mo )
