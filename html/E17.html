Exodus Chapter 17

1 and~ they(m)~ will~ JOURNEY (Verb) ( וַיִּסְעוּ / wai'yis'u ) ALL ( כָּל / kol ) COMPANY ( עֲדַת / a'dat ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) from~ WILDERNESS ( מִמִּדְבַּר / mi'mid'bar ) Sin ( סִין / sin ) to~ JOURNEY ~s ~them(m) ( לְמַסְעֵיהֶם / lê'mas'ey'hem ) UPON ( עַל / al ) MOUTH ( פִּי / pi ) YHWH ( יְהוָה / YHWH ) and~ they(m)~ will~ CAMP (Verb) ( וַיַּחֲנוּ / wai'ya'hha'nu ) in~ Rephiydiym ( בִּרְפִידִים / bir'phi'dim ) and~ WITHOUT ( וְאֵין / wê'eyn ) WATER ~s2 ( מַיִם / ma'yim ) to~ >~ GULP (Verb) ( לִשְׁתֹּת / lish'tot ) the~ PEOPLE ( הָעָם / ha'am )

and all the company of the sons of Yisra'eyl journeyed from the wilderness of Sin, according to their breaking camps by the mouth of YHWH, and they camped in Rephiydiym, and the people were without waters to gulp,

2 and~ he~ will~ DISPUTE (Verb) ( וַיָּרֶב / wai'ya'rev ) the~ PEOPLE ( הָעָם / ha'am ) WITH ( עִם / im ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ they(m)~ will~ SAY (Verb) ( וַיֹּאמְרוּ / wai'yom'ru ) !(mp)~ GIVE (Verb) ( תְּנוּ / tê'nu ) to~ ~us ( לָנוּ / la'nu ) WATER ~s2 ( מַיִם / ma'yim ) and~ we~ will~ GULP (Verb) ( וְנִשְׁתֶּה / wê'nish'teh ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ ~them(m) ( לָהֶם / la'hem ) Mosheh ( מֹשֶׁה / mo'sheh ) WHAT ( מַה / mah ) you(mp)~ will~ DISPUTE (Verb) ~must ( תְּרִיבוּן / tê'ri'vun ) BY ~me ( עִמָּדִי / i'ma'di ) WHAT ( מַה / mah ) you(mp)~ will~ TEST (Verb) ~must ( תְּנַסּוּן / tê'na'sun ) AT ( אֶת / et ) YHWH ( יְהוָה / YHWH )

and the people disputed with Mosheh and they said, give us waters and we will gulp, and Mosheh said to them, why must you dispute with me, why must you test YHWH,

3 and~ he~ will~ THIRST (Verb) ( וַיִּצְמָא / wai'yits'ma ) THERE ( שָׁם / sham ) the~ PEOPLE ( הָעָם / ha'am ) to~ WATER ~s2 ( לַמַּיִם / la'ma'yim ) and~ he~ will~ STAY.THE.NIGHT (Verb) ( וַיָּלֶן / wai'ya'len ) the~ PEOPLE ( הָעָם / ha'am ) UPON ( עַל / al ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ WHAT ( לָמָּה / la'mah ) THIS ( זֶּה / zeh ) you(ms)~ did~ make~ GO.UP (Verb) ~us ( הֶעֱלִיתָנוּ / he'e'li'ta'nu ) from~ Mits'rayim ( מִמִּצְרַיִם / mi'mits'ra'yim ) to~ >~ make~ DIE (Verb) ( לְהָמִית / lê'ha'mit ) AT ~me ( אֹתִי / o'ti ) and~ AT ( וְאֶת / wê'et ) SON ~s ~me ( בָּנַי / ba'nai ) and~ AT ( וְאֶת / wê'et ) ACQUIRED ~s ~me ( מִקְנַי / miq'nai ) in~ the~ THIRST ( בַּצָּמָא / ba'tsa'ma )

and the people thirsted there for waters, and the people murmured upon Mosheh, and he said, for what reason did you make us go up from Mits'rayim, to kill me and my sons, and my acquirings with the thirst,

4 and~ he~ will~ CRY.OUT (Verb) ( וַיִּצְעַק / wai'yits'aq ) Mosheh ( מֹשֶׁה / mo'sheh ) TO ( אֶל / el ) YHWH ( יְהוָה / YHWH ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) WHAT ( מָה / mah ) I~ will~ DO (Verb) ( אֶעֱשֶׂה / e'e'seh ) to~ the~ PEOPLE ( לָעָם / la'am ) the~ THIS ( הַזֶּה / ha'zeh ) YET.AGAIN ( עוֹד / od ) SMALL.AMOUNT ( מְעַט / mê'at ) [ 367 ] and~ they~ did~ STONE (Verb) ~me ( וּסְקָלֻנִי / us'qa'lu'ni )

and Mosheh cried out to YHWH saying, what will I do for this people, in a moment they will stone me,

5 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) !(ms)~ CROSS.OVER (Verb) ( עֲבֹר / a'vor ) to~ FACE ~s ( לִפְנֵי / liph'ney ) the~ PEOPLE ( הָעָם / ha'am ) and~ !(ms)~ TAKE (Verb) ( וְקַח / wê'qahh ) AT ~you(ms) ( אִתְּךָ / it'kha ) from~ BEARD ~s ( מִזִּקְנֵי / mi'ziq'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) and~ BRANCH ~you(ms) ( וּמַטְּךָ / u'mat'kha ) WHICH ( אֲשֶׁר / a'sher ) you(ms)~ did~ make~ HIT (Verb) ( הִכִּיתָ / hi'ki'ta ) in~ ~him ( בּוֹ / bo ) AT ( אֶת / et ) the~ STREAM ( הַיְאֹר / hai'or ) !(ms)~ TAKE (Verb) ( קַח / qahh ) in~ HAND ~you(ms) ( בְּיָדְךָ / bê'yad'kha ) and~ you(ms)~ did~ WALK (Verb) ( וְהָלָכְתָּ / wê'ha'lakh'ta )

and YHWH said to Mosheh, cross over to the face of the people and take with you from the bearded ones of Yisra'eyl, and your branch, which you attacked the stream with, take it in your hand and you will walk.

6 LOOK ~me ( הִנְנִי / hin'ni ) STAND (Verb) ~ing(ms) ( עֹמֵד / o'meyd ) to~ FACE ~s ~you(ms) ( לְפָנֶיךָ / lê'pha'ney'kha ) THERE ( שָּׁם / sham ) UPON ( עַל / al ) the~ BOULDER ( הַצּוּר / ha'tsur ) in~ Hhorev ( בְּחֹרֵב / bê'hho'reyv ) and~ you(ms)~ did~ make~ HIT (Verb) ( וְהִכִּיתָ / wê'hi'ki'ta ) in~ the~ BOULDER ( בַצּוּר / va'tsur ) and~ they~ did~ GO.OUT (Verb) ( וְיָצְאוּ / wê'yats'u ) FROM ~him ( מִמֶּנּוּ / mi'me'nu ) WATER ~s2 ( מַיִם / ma'yim ) and~ he~ did~ GULP (Verb) ( וְשָׁתָה / wê'sha'tah ) the~ PEOPLE ( הָעָם / ha'am ) and~ he~ will~ DO (Verb) ( וַיַּעַשׂ / wai'ya'as ) SO ( כֵּן / keyn ) Mosheh ( מֹשֶׁה / mo'sheh ) to~ EYE ~s2 ( לְעֵינֵי / lê'ey'ney ) BEARD ~s ( זִקְנֵי / ziq'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl )

Here, I am standing to your face, there upon the boulder in Hhorev, and you will attack the boulder and waters will go out from him, and the people will gulp, and Mosheh did so to the eyes of the bearded ones of Yisra'eyl,

7 and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) TITLE ( שֵׁם / sheym ) the~ AREA ( הַמָּקוֹם / ha'ma'qom ) Mas'sah ( מַסָּה / ma'sah ) and~ Meriyvah ( וּמְרִיבָה / um'ri'vah ) UPON ( עַל / al ) DISPUTE ( רִיב / riv ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) and~ UPON ( וְעַל / wê'al ) >~ much~ TEST (Verb) ~them(m) ( נַסֹּתָם / na'so'tam ) AT ( אֶת / et ) YHWH ( יְהוָה / YHWH ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) ?~ THERE.IS ( הֲיֵשׁ / ha'yeysh ) YHWH ( יְהוָה / YHWH ) in~ INSIDE ~us ( בְּקִרְבֵּנוּ / bê'qir'bey'nu ) IF ( אִם / im ) WITHOUT ( אָיִן / a'yin )

and he called out the title of the area, Mas'sah and Meriyvah, because of the dispute of the sons of Yisra'eyl, and because of their testing YHWH saying, is YHWH inside us or not,

8 and~ he~ will~ COME (Verb) ( וַיָּבֹא / wai'ya'vo ) Amaleq ( עֲמָלֵק / a'ma'leyq ) and~ he~ will~ be~ FIGHT (Verb) ( וַיִּלָּחֶם / wai'yi'la'hhem ) WITH ( עִם / im ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) in~ Rephiydiym ( בִּרְפִידִם / bir'phi'dim )

and Amaleq came and he waged war with Yisra'eyl in Rephiydiym,

9 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Mosheh ( מֹשֶׁה / mo'sheh ) TO ( אֶל / el ) Yehoshu'a ( יְהוֹשֻׁעַ / yê'ho'shu'a ) !(ms)~ CHOOSE (Verb) ( בְּחַר / bê'hhar ) to~ ~us ( לָנוּ / la'nu ) MAN ~s ( אֲנָשִׁים / a'na'shim ) and~ !(ms)~ GO.OUT (Verb) ( וְצֵא / wê'tsey ) !(ms)~ be~ FIGHT (Verb) ( הִלָּחֵם / hi'la'hheym ) in~ the~ Amaleq ( בַּעֲמָלֵק / ba'a'ma'leyq ) TOMORROW ( מָחָר / ma'hhar ) I ( אָנֹכִי / a'no'khi ) be~ STAND.UP (Verb) ~ing(ms) ( נִצָּב / ni'tsav ) UPON ( עַל / al ) HEAD ( רֹאשׁ / rosh ) the~ KNOLL ( הַגִּבְעָה / ha'giv'ah ) and~ BRANCH ( וּמַטֵּה / u'ma'teyh ) the~ Elohiym ( הָאֱלֹהִים / ha'e'lo'him ) in~ HAND ~me ( בְּיָדִי / bê'ya'di )

and Mosheh said to Yehoshu'a, choose for us men and go out, wage war with the Amaleq, tomorrow I will be standing up upon the head of the knoll, and the branch of the Elohiym will be in my hand,

10 and~ he~ will~ DO (Verb) ( וַיַּעַשׂ / wai'ya'as ) Yehoshu'a ( יְהוֹשֻׁעַ / yê'ho'shu'a ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ SAY (Verb) ( אָמַר / a'mar ) to~ ~him ( לוֹ / lo ) Mosheh ( מֹשֶׁה / mo'sheh ) to~ >~ be~ FIGHT (Verb) ( לְהִלָּחֵם / lê'hi'la'hheym ) in~ the~ Amaleq ( בַּעֲמָלֵק / ba'a'ma'leyq ) and~ Mosheh ( וּמֹשֶׁה / u'mo'sheh ) Aharon ( אַהֲרֹן / a'ha'ron ) and~ Hhur ( וְחוּר / wê'hhur ) they~ did~ GO.UP (Verb) ( עָלוּ / a'lu ) HEAD ( רֹאשׁ / rosh ) the~ KNOLL ( הַגִּבְעָה / ha'giv'ah )

and Yehoshu'a did just as Mosheh said to him, to wage war with the Amaleq, and Mosheh, Aharon and Hhur, had gone up to the head of the knoll,

11 and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ will~ make~ RAISE.UP (Verb) ( יָרִים / ya'rim ) Mosheh ( מֹשֶׁה / mo'sheh ) HAND ~him ( יָדוֹ / ya'do ) and~ he~ did~ OVERCOME (Verb) ( וְגָבַר / wê'ga'var ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) and~ like~ WHICH ( וְכַאֲשֶׁר / wê'kha'a'sher ) he~ will~ make~ REST (Verb) ( יָנִיחַ / ya'ni'ahh ) HAND ~him ( יָדוֹ / ya'do ) and~ he~ did~ OVERCOME (Verb) ( וְגָבַר / wê'ga'var ) Amaleq ( עֲמָלֵק / a'ma'leyq )

and it came to pass, just as Mosheh made his hand rise, then Yisra'eyl will overcome, and just as he made his hand rest, then Amaleq will overcome,

12 and~ HAND ~s2 ( וִידֵי / wi'dey ) Mosheh ( מֹשֶׁה / mo'sheh ) HEAVY ~s ( כְּבֵדִים / kê'vey'dim ) and~ they(m)~ will~ TAKE (Verb) ( וַיִּקְחוּ / wai'yiq'hhu ) STONE ( אֶבֶן / e'ven ) and~ they(m)~ will~ PLACE (Verb) ( וַיָּשִׂימוּ / wai'ya'si'mu ) UNDER ~s ~him ( תַחְתָּיו / tahh'taw ) and~ he~ will~ SETTLE (Verb) ( וַיֵּשֶׁב / wai'yey'shev ) UPON ~her ( עָלֶיהָ / a'ley'ah ) and~ Aharon ( וְאַהֲרֹן / wê'a'ha'ron ) and~ Hhur ( וְחוּר / wê'hhur ) they~ did~ UPHOLD (Verb) ( תָּמְכוּ / tam'khu ) in~ HAND ~s2 ~him ( בְיָדָיו / vê'ya'daw ) from~ THIS ( מִזֶּה / mi'zeh ) UNIT ( אֶחָד / e'hhad ) and~ from~ THIS ( וּמִזֶּה / u'mi'zeh ) UNIT ( אֶחָד / e'hhad ) [ 368 ] and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) HAND ~s2 ~him ( יָדָיו / ya'daw ) FIRMNESS ( אֱמוּנָה / e'mu'nah ) UNTIL ( עַד / ad ) !(ms)~ COME (Verb) ( בֹּא / bo ) [ 369 ] the~ SUN ( הַשָּׁמֶשׁ / ha'sha'mesh )

and the hands of Mosheh were heavy, and they took a stone and they placed it underneath him, and he settled upon her, and Aharon and Hhur upheld his hands, from this one and from that one, and his hands were firm until the coming of the sun,

13 and~ he~ will~ WEAKEN (Verb) ( וַיַּחֲלֹשׁ / wai'ya'hha'losh ) Yehoshu'a ( יְהוֹשֻׁעַ / yê'ho'shu'a ) AT ( אֶת / et ) Amaleq ( עֲמָלֵק / a'ma'leyq ) and~ AT ( וְאֶת / wê'et ) PEOPLE ~him ( עַמּוֹ / a'mo ) to~ MOUTH ( לְפִי / lê'phi ) SWORD ( חָרֶב / hha'rev )

and Yehoshu'a weakened Amaleq and his people by the mouth of the sword,

14 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) !(ms)~ WRITE (Verb) ( כְּתֹב / kê'tov ) THIS ( זֹאת / zot ) REMEMBRANCE ( זִכָּרוֹן / zi'ka'ron ) in~ the~ SCROLL ( בַּסֵּפֶר / ba'sey'pher ) and~ !(ms)~ PLACE (Verb) ( וְשִׂים / wê'sim ) in~ EAR ~s2 ( בְּאָזְנֵי / bê'az'ney ) [ 370 ] Yehoshu'a ( יְהוֹשֻׁעַ / yê'ho'shu'a ) GIVEN.THAT ( כִּי / ki ) >~ WIPE.AWAY (Verb) ( מָחֹה / ma'hhoh ) I~ will~ WIPE.AWAY (Verb) ( אֶמְחֶה / em'hheh ) AT ( אֶת / et ) MEMORY ( זֵכֶר / zey'kher ) Amaleq ( עֲמָלֵק / a'ma'leyq ) from~ UNDER ( מִתַּחַת / mi'ta'hhat ) the~ SKY ~s2 ( הַשָּׁמָיִם / ha'sha'ma'yim )

and YHWH said to Mosheh, write this remembrance in the scroll and place it in the ears of Yehoshu'a, given that I will surely wipe away the memorial of Amaleq from under the skies,

15 and~ he~ will~ BUILD (Verb) ( וַיִּבֶן / wai'yi'ven ) Mosheh ( מֹשֶׁה / mo'sheh ) ALTAR ( מִזְבֵּחַ / miz'bey'ahh ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) TITLE ~him ( שְׁמוֹ / shê'mo ) YHWH-Nisiy ( יְהוָה־נִסִּי / YHWH-ni'si )

and Mosheh built an altar and he called out his title YHWH-Nisiy,

16 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) GIVEN.THAT ( כִּי / ki ) HAND ( יָד / yad ) UPON ( עַל / al ) STOOL ( כֵּס / keys ) Yah ( יָהּ / yah ) [ 371 ] BATTLE ( מִלְחָמָה / mil'hha'mah ) to~ YHWH ( לַיהוָה / la'YHWH ) in~ the~ Amaleq ( בַּעֲמָלֵק / ba'a'ma'leyq ) from~ GENERATION ( מִדֹּר / mi'dor ) GENERATION ( דֹּר / dor ) [ 372 ]
