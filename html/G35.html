Genesis Chapter 35

1 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Elohiym ( אֱלֹהִים / e'lo'him ) TO ( אֶל / el ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) !(ms)~ RISE (Verb) ( קוּם / qum ) !(ms)~ GO.UP (Verb) ( עֲלֵה / a'leyh ) Beyt-El ( בֵית־אֵל / veyt-eyl ) [ 177 ] and~ !(ms)~ SETTLE (Verb) ( וְשֶׁב / wê'shev ) THERE ( שָׁם / sham ) and~ !(ms)~ DO (Verb) ( וַעֲשֵׂה / wa'a'seyh ) THERE ( שָׁם / sham ) ALTAR ( מִזְבֵּחַ / miz'bey'ahh ) to~ MIGHTY.ONE ( לָאֵל / la'eyl ) the~ be~ SEE (Verb) ~ing(ms) ( הַנִּרְאֶה / ha'nir'eh ) TO ~you(ms) ( אֵלֶיךָ / ey'ley'kha ) in~ >~ FLEE.AWAY (Verb) ~you(ms) ( בְּבָרְחֲךָ / bê'var'hha'kha ) from~ FACE ~s ( מִפְּנֵי / mip'ney ) Esaw ( עֵשָׂו / ey'saw ) BROTHER ~you(ms) ( אָחִיךָ / a'hhi'kha )

and Elohiym said to Ya'aqov, rise, go up to Beyt-El and settle there and make an altar there to the mighty one, the one appearing to you in your fleeing away from the face of Esaw your brother,

2 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) TO ( אֶל / el ) HOUSE ~him ( בֵּיתוֹ / bey'to ) and~ TO ( וְאֶל / wê'el ) ALL ( כָּל / kol ) WHICH ( אֲשֶׁר / a'sher ) WITH ~him ( עִמּוֹ / i'mo ) !(mp)~ make~ TURN.ASIDE (Verb) ( הָסִרוּ / ha'si'ru ) AT ( אֶת / et ) Elohiym ( אֱלֹהֵי / e'lo'hey ) the~ FOREIGNER ( הַנֵּכָר / ha'ney'khar ) WHICH ( אֲשֶׁר / a'sher ) in~ MIDST ~you(mp) ( בְּתֹכְכֶם / bê'tokh'khem ) and~ !(mp)~ make~ BE.CLEAN (Verb) ( וְהִטַּהֲרוּ / wê'hi'ta'ha'ru ) and~ !(mp)~ make~ PASS.OVER (Verb) ( וְהַחֲלִיפוּ / wê'ha'hha'li'phu ) APPAREL ~s ~you(mp) ( שִׂמְלֹתֵיכֶם / sim'lo'tey'khem )

and Ya'aqov said to his house and to all which were with him, remove the Elohiym of the foreign one which is in the midst of you and be clean and change your apparels,

3 and~ we~ will~ RISE (Verb) ~& ( וְנָקוּמָה / wê'na'qu'mah ) and~ we~ will~ GO.UP (Verb) ~& ( וְנַעֲלֶה / wê'na'a'leh ) Beyt-El ( בֵּית־אֵל / beyt-eyl ) [ 178 ] and~ I~ will~ DO (Verb) ~& ( וְאֶעֱשֶׂה / wê'e'e'seh ) THERE ( שָּׁם / sham ) ALTAR ( מִזְבֵּחַ / miz'bey'ahh ) to~ MIGHTY.ONE ( לָאֵל / la'eyl ) the~ ANSWER (Verb) ~ing(ms) ( הָעֹנֶה / ha'o'neh ) AT ~me ( אֹתִי / o'ti ) in~ DAY ( בְּיוֹם / bê'yom ) PERSECUTION ~me ( צָרָתִי / tsa'ra'ti ) and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) BY ~me ( עִמָּדִי / i'ma'di ) in~ the~ ROAD ( בַּדֶּרֶךְ / ba'de'rekh ) WHICH ( אֲשֶׁר / a'sher ) I~ did~ WALK (Verb) ( הָלָכְתִּי / ha'lakh'ti )

and we will rise and we will go up to Beyt-El and I will make an altar there to the mighty one, the one answering me in the day of my trouble and he existed by me in the road which I walked,

4 and~ they(m)~ will~ GIVE (Verb) ( וַיִּתְּנוּ / wai'yit'nu ) TO ( אֶל / el ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) AT ( אֵת / eyt ) ALL ( כָּל / kol ) Elohiym ( אֱלֹהֵי / e'lo'hey ) the~ FOREIGNER ( הַנֵּכָר / ha'ney'khar ) WHICH ( אֲשֶׁר / a'sher ) in~ HAND ~them(m) ( בְּיָדָם / bê'ya'dam ) and~ AT ( וְאֶת / wê'et ) the~ ORNAMENTAL.RING ~s ( הַנְּזָמִים / han'za'mim ) WHICH ( אֲשֶׁר / a'sher ) in~ EAR ~s2 ~them(m) ( בְּאָזְנֵיהֶם / bê'az'ney'hem ) and~ he~ will~ SUBMERGE (Verb) ( וַיִּטְמֹן / wai'yit'mon ) AT ~them(m) ( אֹתָם / o'tam ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) UNDER ( תַּחַת / ta'hhat ) the~ OAK ( הָאֵלָה / ha'ey'lah ) WHICH ( אֲשֶׁר / a'sher ) WITH ( עִם / im ) Shekhem ( שְׁכֶם / shê'khem )

and they gave to Ya'aqov all the Elohiym of the foreign one which was in their hand and the rings which were in their ears and Ya'aqov submerged them under the oak which was with Shekhem,

5 and~ they(m)~ will~ JOURNEY (Verb) ( וַיִּסָּעוּ / wai'yi'sa'u ) and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) DREAD ( חִתַּת / hhi'tat ) Elohiym ( אֱלֹהִים / e'lo'him ) UPON ( עַל / al ) the~ CITY ~s ( הֶעָרִים / he'a'rim ) WHICH ( אֲשֶׁר / a'sher ) ALL.AROUND ~s ~them(m) ( סְבִיבֹתֵיהֶם / sê'vi'vo'tey'hem ) and~ NOT ( וְלֹא / wê'lo ) they~ did~ PURSUE (Verb) ( רָדְפוּ / rad'phu ) AFTER ( אַחֲרֵי / a'hha'rey ) SON ~s ( בְּנֵי / bê'ney ) Ya'aqov ( יַעֲקֹב / ya'a'qov )

and they journeyed and the dread of Elohiym existed upon the cities which were around them and they did not pursue after the sons of Ya'aqov,

6 and~ he~ will~ COME (Verb) ( וַיָּבֹא / wai'ya'vo ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) Luz ~unto ( לוּזָה / lu'zah ) WHICH ( אֲשֶׁר / a'sher ) in~ LAND ( בְּאֶרֶץ / bê'e'rets ) Kena'an ( כְּנַעַן / kê'na'an ) SHE ( הִוא / hi ) Beyt-El ( בֵּית־אֵל / beyt-eyl ) HE ( הוּא / hu ) and~ ALL ( וְכָל / wê'khol ) the~ PEOPLE ( הָעָם / ha'am ) WHICH ( אֲשֶׁר / a'sher ) WITH ~him ( עִמּוֹ / i'mo )

and Ya'aqov came unto Luz which was in the land of Kena'an, she is Beyt-El, he and all the people which were with him,

7 and~ he~ will~ BUILD (Verb) ( וַיִּבֶן / wai'yi'ven ) THERE ( שָׁם / sham ) ALTAR ( מִזְבֵּחַ / miz'bey'ahh ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) to~ the~ AREA ( לַמָּקוֹם / la'ma'qom ) El-Beyt-El ( אֵל־בֵּית־אֵל / eyl-beyt-eyl ) GIVEN.THAT ( כִּי / ki ) THERE ( שָׁם / sham ) they~ did~ be~ REMOVE.THE.COVER (Verb) ( נִגְלוּ / nig'lu ) TO ~him ( אֵלָיו / ey'law ) the~ Elohiym ( הָאֱלֹהִים / ha'e'lo'him ) in~ >~ FLEE.AWAY (Verb) ~him ( בְּבָרְחוֹ / bê'var'hho ) from~ FACE ~s ( מִפְּנֵי / mip'ney ) BROTHER ~him ( אָחִיו / a'hhiw )

and he built an altar there and called out to the place El-Beyt-El given that there the Elohiym were uncovered to him in his fleeing away from the face of his brother,

8 and~ she~ will~ DIE (Verb) ( וַתָּמָת / wa'ta'mat ) Devorah ( דְּבֹרָה / dê'vo'rah ) make~ SUCKLE (Verb) ~ing(fs) ( מֵינֶקֶת / mey'ne'qet ) Rivqah ( רִבְקָה / riv'qah ) and~ she~ will~ be~ BURY (Verb) ( וַתִּקָּבֵר / wa'ti'qa'veyr ) from~ UNDER ( מִתַּחַת / mi'ta'hhat ) to~ Beyt-El ( לְבֵית־אֵל / lê'veyt-eyl ) UNDER ( תַּחַת / ta'hhat ) the~ GREAT.TREE ( הָאַלּוֹן / ha'a'lon ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) TITLE ~him ( שְׁמוֹ / shê'mo ) Alon-Bakhut ( אַלּוֹן־בָּכוּת / a'lon-ba'khut )

and Devorah, the nurse of Rivqah, died and she was buried under Beyt-El, under the great tree and he called out his title Alon-Bakhut,

9 and~ he~ will~ be~ SEE (Verb) ( וַיֵּרָא / wai'yey'ra ) Elohiym ( אֱלֹהִים / e'lo'him ) TO ( אֶל / el ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) YET.AGAIN ( עוֹד / od ) in~ >~ COME (Verb) ~him ( בְּבֹאוֹ / bê'vo'o ) from~ Padan-Aram ( מִפַּדַּן־אֲרָם / mi'pa'dan-a'ram ) and~ he~ will~ much~ KNEEL (Verb) ( וַיְבָרֶךְ / wai'va'rekh ) AT ~him ( אֹתוֹ / o'to )

and Elohiym appeared to Ya'aqov yet again in his coming from Padan-Aram and he exalted him,

10 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ ~him ( לוֹ / lo ) Elohiym ( אֱלֹהִים / e'lo'him ) TITLE ~you(ms) ( שִׁמְךָ / shim'kha ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) NOT ( לֹא / lo ) he~ will~ be~ CALL.OUT (Verb) ( יִקָּרֵא / yi'qa'rey ) TITLE ~you(ms) ( שִׁמְךָ / shim'kha ) YET.AGAIN ( עוֹד / od ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) GIVEN.THAT ( כִּי / ki ) IF ( אִם / im ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) TITLE ~you(ms) ( שְׁמֶךָ / shê'me'kha ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) AT ( אֶת / et ) TITLE ~him ( שְׁמוֹ / shê'mo ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl )

and Elohiym said to him, your title is Ya'aqov, your title will not be called yet again Ya'aqov instead your title will exist as Yisra'eyl and he called out his title Yisra'eyl,

11 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ ~him ( לוֹ / lo ) Elohiym ( אֱלֹהִים / e'lo'him ) I ( אֲנִי / a'ni ) MIGHTY.ONE ( אֵל / eyl ) Shaddai ( שַׁדַּי / sha'dai ) !(ms)~ REPRODUCE (Verb) ( פְּרֵה / pê'reyh ) and~ !(ms)~ INCREASE (Verb) ( וּרְבֵה / ur'veyh ) NATION ( גּוֹי / goy ) and~ ASSEMBLY ( וּקְהַל / uq'hal ) NATION ~s ( גּוֹיִם / go'yim ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) FROM ~you(ms) ( מִמֶּךָּ / mi'me'ka ) and~ KING ~s ( וּמְלָכִים / um'la'khim ) from~ LOINS ~you(ms) ( מֵחֲלָצֶיךָ / mey'hha'la'tsey'kha ) they(m)~ will~ GO.OUT (Verb) ( יֵצֵאוּ / yey'tsey'u )

and Elohiym said to him, I am the mighty one of Shaddai, reproduce and increase, a nation and assembled flock of nations will exist from you and kings will go out from your loins,

12 and~ AT ( וְאֶת / wê'et ) the~ LAND ( הָאָרֶץ / ha'a'rets ) WHICH ( אֲשֶׁר / a'sher ) I~ did~ GIVE (Verb) ( נָתַתִּי / na'ta'ti ) to~ Avraham ( לְאַבְרָהָם / lê'av'ra'ham ) and~ to~ Yits'hhaq ( וּלְיִצְחָק / ul'yits'hhaq ) to~ ~you(ms) ( לְךָ / lê'kha ) I~ will~ GIVE (Verb) ~her ( אֶתְּנֶנָּה / et'ne'nah ) and~ to~ SEED ~you(ms) ( וּלְזַרְעֲךָ / ul'zar'a'kha ) AFTER ~you(ms) ( אַחֲרֶיךָ / a'hha'rey'kha ) I~ will~ GIVE (Verb) ( אֶתֵּן / e'teyn ) AT ( אֶת / et ) the~ LAND ( הָאָרֶץ / ha'a'rets )

and the land which I gave to Avraham and to Yits'hhaq I will give her to you, and to your seed after you I will give the land,

13 and~ he~ will~ make~ GO.UP (Verb) ( וַיַּעַל / wai'ya'al ) from~ UPON ~him ( מֵעָלָיו / mey'a'law ) Elohiym ( אֱלֹהִים / e'lo'him ) in~ the~ AREA ( בַּמָּקוֹם / ba'ma'qom ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) AT ~him ( אִתּוֹ / i'to )

and Elohiym went up from upon him in the place which he spoke to him,

14 and~ he~ will~ make~ STAND.UP (Verb) ( וַיַּצֵּב / wai'ya'tseyv ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) MONUMENT ( מַצֵּבָה / ma'tsey'vah ) in~ the~ AREA ( בַּמָּקוֹם / ba'ma'qom ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) AT ~him ( אִתּוֹ / i'to ) MONUMENT ( מַצֶּבֶת / ma'tse'vet ) STONE ( אָבֶן / a'ven ) and~ he~ will~ make~ POUR (Verb) ( וַיַּסֵּךְ / wai'ya'seykh ) UPON ~her ( עָלֶיהָ / a'ley'ah ) POURING ( נֶסֶךְ / ne'sekh ) and~ he~ will~ POUR.DOWN (Verb) ( וַיִּצֹק / wai'yi'tsoq ) UPON ~her ( עָלֶיהָ / a'ley'ah ) OIL ( שָׁמֶן / sha'men )

and Ya'aqov made a monument stand erect in the place which he spoke to him, a monument of stone, and he poured upon her a pouring, and he poured down oil upon her,

15 and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) AT ( אֶת / et ) TITLE ( שֵׁם / sheym ) the~ AREA ( הַמָּקוֹם / ha'ma'qom ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) AT ~him ( אִתּוֹ / i'to ) THERE ( שָׁם / sham ) Elohiym ( אֱלֹהִים / e'lo'him ) Beyt-El ( בֵּית־אֵל / beyt-eyl )

and Ya'aqov called out the title of the place which Elohiym spoke to him there, Beyt-El,

16 and~ they(m)~ will~ JOURNEY (Verb) ( וַיִּסְעוּ / wai'yis'u ) from~ Beyt-El ( מִבֵּית־אֵל / mi'beyt-eyl ) and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) YET.AGAIN ( עוֹד / od ) SHORT ( כִּבְרַת / kiv'rat ) the~ LAND ( הָאָרֶץ / ha'a'rets ) to~ >~ COME (Verb) ( לָבוֹא / la'vo ) Ephrat ~unto ( אֶפְרָתָה / eph'ra'tah ) and~ she~ will~ BRING.FORTH (Verb) ( וַתֵּלֶד / wa'tey'led ) Rahhel ( רָחֵל / ra'hheyl ) and~ she~ will~ much~ BE.HARD (Verb) ( וַתְּקַשׁ / wat'qash ) in~ >~ BRING.FORTH (Verb) ~her ( בְּלִדְתָּהּ / bê'lid'tah )

and they journeyed from Beyt-El and a short distance yet existed to come unto Ephrat and Rahhel brought forth and she was very hard in her bringing forth,

17 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) in~ >~ make~ BE.HARD (Verb) ~her ( בְהַקְשֹׁתָהּ / vê'haq'sho'tah ) in~ >~ BRING.FORTH (Verb) ~her ( בְּלִדְתָּהּ / bê'lid'tah ) and~ she~ will~ SAY (Verb) ( וַתֹּאמֶר / wa'to'mer ) to~ ~her ( לָהּ / lah ) the~ much~ BRING.FORTH (Verb) ~ing(fs) ( הַמְיַלֶּדֶת / ham'ya'le'det ) DO.NOT ( אַל / al ) you(fs)~ will~ FEAR (Verb) ( תִּירְאִי / tir'i ) GIVEN.THAT ( כִּי / ki ) ALSO ( גַם / gam ) THIS ( זֶה / zeh ) to~ ~you(fs) ( לָךְ / lakh ) SON ( בֵּן / beyn )

and it came to pass with her bringing forth being hard, the midwife said to her, you will not fear given that this also is to you a son,

18 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) in~ >~ GO.OUT (Verb) ( בְּצֵאת / bê'tseyt ) SOUL ~her ( נַפְשָׁהּ / naph'shah ) GIVEN.THAT ( כִּי / ki ) she~ did~ DIE (Verb) ( מֵתָה / mey'tah ) and~ she~ will~ CALL.OUT (Verb) ( וַתִּקְרָא / wa'tiq'ra ) TITLE ~him ( שְׁמוֹ / shê'mo ) Ben-Oni ( בֶּן־אוֹנִי / ben-o'ni ) and~ FATHER ~him ( וְאָבִיו / wê'a'viw ) he~ did~ CALL.OUT (Verb) ( קָרָא / qa'ra ) to~ ~him ( לוֹ / lo ) Binyamin ( בִנְיָמִין / vin'ya'min )

and it came to pass, with the going out of her soul, that she died and she called out his title Ben-Oni and his father called out to him Binyamin,

19 and~ she~ will~ DIE (Verb) ( וַתָּמָת / wa'ta'mat ) Rahhel ( רָחֵל / ra'hheyl ) and~ she~ will~ be~ BURY (Verb) ( וַתִּקָּבֵר / wa'ti'qa'veyr ) in~ ROAD ( בְּדֶרֶךְ / bê'de'rekh ) Ephrat ~unto ( אֶפְרָתָה / eph'ra'tah ) SHE ( הִוא / hi ) Beyt-Lehhem ( בֵּית־לָחֶם / beyt'-la'hhem )

and Rahhel died and she was buried in the road unto Ephrat, she is Beyt-Lehhem,

20 and~ he~ will~ make~ STAND.UP (Verb) ( וַיַּצֵּב / wai'ya'tseyv ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) MONUMENT ( מַצֵּבָה / ma'tsey'vah ) UPON ( עַל / al ) BURIAL.PLACE ~her ( קְבֻרָתָהּ / qê'vu'ra'tah ) SHE ( הִוא / hi ) MONUMENT ( מַצֶּבֶת / ma'tse'vet ) BURIAL.PLACE ( קְבֻרַת / qê'vu'rat ) Rahhel ( רָחֵל / ra'hheyl ) UNTIL ( עַד / ad ) the~ DAY ( הַיּוֹם / hai'yom )

and Ya'aqov made a monument stand erect upon her burial place, she is the monument of the burial place of Rahhel also today,
