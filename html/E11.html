Exodus Chapter 11

1 and~he~will~SAY(Verb) (וַיֹּאמֶר / wai'yo'mer) YHWH (יְהוָה / YHWH) TO (אֶל / el) Mosheh (מֹשֶׁה / mo'sheh) YET.AGAIN (עוֹד / od) TOUCH (נֶגַע / ne'ga) UNIT (אֶחָד / e'hhad) I~will~make~COME(Verb) (אָבִיא / a'vi) UPON (עַל / al) Paroh (פַּרְעֹה / par'oh) and~UPON (וְעַל / wê'al) Mits'rayim (מִצְרַיִם / mits'ra'yim) AFTER (אַחֲרֵי / a'hha'rey) SO (כֵן / kheyn) he~will~much~SEND(Verb) (יְשַׁלַּח / yê'sha'lahh) AT~you(mp) (אֶתְכֶם / et'khem) from~THIS (מִזֶּה / mi'zeh) like~>~much~SEND(Verb)~him (כְּשַׁלְּחוֹ / kê'shal'hho) COMPLETION (כָּלָה / ka'lah) >~much~CAST.OUT(Verb) (גָּרֵשׁ / ga'reysh) he~will~much~CAST.OUT(Verb) (יְגָרֵשׁ / yê'ga'reysh) AT~you(mp) (אֶתְכֶם / et'khem) from~THIS (מִזֶּה / mi'zeh)

and YHWH said to Mosheh, I will bring one more plague upon Paroh and upon Mits'rayim, afterward he will send you from this place, as he is completely sending you, he will surely cast you out from this place.

2 >~much~SPEAK(Verb) (דַּבֶּר / da'ber) PLEASE (נָא / na) in~EAR~s2 (בְּאָזְנֵי / bê'az'ney) the~PEOPLE (הָעָם / ha'am) and~they(m)~will~INQUIRE(Verb) (וְיִשְׁאֲלוּ / wê'yish'a'lu) MAN (אִישׁ / ish) from~AT (מֵאֵת / mey'eyt) COMPANION~him (רֵעֵהוּ / rey'ey'hu) and~WOMAN (וְאִשָּׁה / wê'i'shah) from~AT (מֵאֵת / mey'eyt) FRIEND~her (רְעוּתָהּ / rê'u'tah) UTENSIL~s (כְּלֵי / kê'ley) SILVER (כֶסֶף / khe'seph) and~UTENSIL~s (וּכְלֵי / ukh'ley) GOLD (זָהָב / za'hav)

Please speak in the ears of the people and each will inquire, from his companion and each from her friend, utensils of silver and utensils of gold,

3 and~he~will~GIVE(Verb) (וַיִּתֵּן / wai'yi'teyn) YHWH (יְהוָה / YHWH) AT (אֶת / et) BEAUTY (חֵן / hheyn) [291] the~PEOPLE (הָעָם / ha'am) in~EYE~s2 (בְּעֵינֵי / bê'ey'ney) Mits'rayim (מִצְרָיִם / mits'ra'yim) ALSO (גַּם / gam) the~MAN (הָאִישׁ / ha'ish) Mosheh (מֹשֶׁה / mo'sheh) GREAT (גָּדוֹל / ga'dol) MANY (מְאֹד / mê'od) in~LAND (בְּאֶרֶץ / bê'e'rets) Mits'rayim (מִצְרַיִם / mits'ra'yim) in~EYE~s2 (בְּעֵינֵי / bê'ey'ney) SERVANT~s (עַבְדֵי / av'dey) Paroh (פַרְעֹה / phar'oh) and~in~EYE~s2 (וּבְעֵינֵי / uv'ey'ney) the~PEOPLE (הָעָם / ha'am)

and YHWH placed the beauty of the people in the eyes of Mits'rayim, also the man Mosheh was very great in the land of Mits'rayim in the eyes of the servants of Paroh and in the eyes of the people,

4 and~he~will~SAY(Verb) (וַיֹּאמֶר / wai'yo'mer) Mosheh (מֹשֶׁה / mo'sheh) IN.THIS.WAY (כֹּה / koh) he~did~SAY(Verb) (אָמַר / a'mar) YHWH (יְהוָה / YHWH) like~CENTER (כַּחֲצֹת / ka'hha'tsot) the~NIGHT (הַלַּיְלָה / ha'lai'lah) [292] I (אֲנִי / a'ni) GO.OUT(Verb)~ing(ms) (יוֹצֵא / yo'tsey) in~MIDST (בְּתוֹךְ / bê'tokh) Mits'rayim (מִצְרָיִם / mits'ra'yim)

and Mosheh said, in this way YHWH said, about the center of the night, I am going out in the midst of Mits'rayim,

5 and~he~did~DIE(Verb) (וּמֵת / u'meyt) ALL (כָּל / kol) FIRSTBORN (בְּכוֹר / bê'khor) in~LAND (בְּאֶרֶץ / bê'e'rets) Mits'rayim (מִצְרַיִם / mits'ra'yim) from~FIRSTBORN (מִבְּכוֹר / mi'be'khor) Paroh (פַּרְעֹה / par'oh) the~SETTLE(Verb)~ing(ms) (הַיֹּשֵׁב / hai'yo'sheyv) UPON (עַל / al) SEAT~him (כִּסְאוֹ / kis'o) UNTIL (עַד / ad) FIRSTBORN (בְּכוֹר / bê'khor) the~MAID (הַשִּׁפְחָה / ha'shiph'hhah) WHICH (אֲשֶׁר / a'sher) AFTER (אַחַר / a'hhar) the~MILLSTONE~s (הָרֵחָיִם / ha'rey'hha'yim) and~ALL (וְכֹל / wê'khol) FIRSTBORN (בְּכוֹר / bê'khor) BEAST (בְּהֵמָה / bê'hey'mah)

and all the firstborn in the land of Mits'rayim will die, from the firstborn of Paroh, the one settling upon his seat, unto the firstborn of the maid which is behind the millstones, and all the firstborn beasts,

6 and~she~did~EXIST(Verb) (וְהָיְתָה / wê'hai'tah) CRY (צְעָקָה / tsê'a'qah) GREAT (גְדֹלָה / gê'do'lah) in~ALL (בְּכָל / bê'khol) LAND (אֶרֶץ / e'rets) Mits'rayim (מִצְרָיִם / mits'ra'yim) WHICH (אֲשֶׁר / a'sher) like~THAT.ONE~him (כָּמֹהוּ / ka'mo'hu) NOT (לֹא / lo) she~did~be~EXIST(Verb) (נִהְיָתָה / nih'ya'tah) and~like~THAT.ONE~him (וְכָמֹהוּ / wê'kha'mo'hu) NOT (לֹא / lo) she~will~make~ADD(Verb) (תֹסִף / to'siph)

and a great cry will exist in all the land of Mits'rayim, which had not existed like this and will not again exist like this,

7 and~to~ALL (וּלְכֹל / ul'khol) SON~s (בְּנֵי / bê'ney) Yisra'eyl (יִשְׂרָאֵל / yis'ra'eyl) NOT (לֹא / lo) he~will~CUT.SHARPLY(Verb) (יֶחֱרַץ / ye'hhe'rats) DOG (כֶּלֶב / ke'lev) TONGUE~him (לְשֹׁנוֹ / lê'sho'no) [293] to~from~MAN (לְמֵאִישׁ / lê'mey'ish) and~UNTIL (וְעַד / wê'ad) BEAST (בְּהֵמָה / bê'hey'mah) to~THAT (לְמַעַן / lê'ma'an) you(mp)~will~KNOW(Verb)~must (תֵּדְעוּן / teyd'un) WHICH (אֲשֶׁר / a'sher) he~will~make~BE.DISTINCT(Verb) (יַפְלֶה / yaph'leh) YHWH (יְהוָה / YHWH) BETWEEN (בֵּין / beyn) Mits'rayim (מִצְרַיִם / mits'ra'yim) and~BETWEEN (וּבֵין / u'veyn) Yisra'eyl (יִשְׂרָאֵל / yis'ra'eyl)

and to all of the sons of Yisra'eyl, the dog will not cut his tongue sharply, from the man and even the beast, so that you must know that YHWH will make a distinction between Mits'rayim and Yisra'eyl,

8 and~they~will~GO.DOWN(Verb) (וְיָרְדוּ / wê'yar'du) ALL (כָל / khol) SERVANT~s~you(ms) (עֲבָדֶיךָ / a'va'dey'kha) THESE (אֵלֶּה / ey'leh) TO~me (אֵלַי / ey'lai) and~they~did~self~BEND.DOWN(Verb) (וְהִשְׁתַּחֲווּ / wê'hish'ta'hha'wu) to~~me (לִי / li) to~>~SAY(Verb) (לֵאמֹר / ley'mor) !(ms)~GO.OUT(Verb) (צֵא / tsey) YOU(MS) (אַתָּה / a'tah) and~ALL (וְכָל / wê'khol) the~PEOPLE (הָעָם / ha'am) WHICH (אֲשֶׁר / a'sher) in~FOOT~s~you(ms) (בְּרַגְלֶיךָ / bê'rag'ley'kha) [294] and~AFTER (וְאַחֲרֵי / wê'a'hha'rey) SO (כֵן / kheyn) I~will~GO.OUT(Verb) (אֵצֵא / ey'tsey) and~he~will~GO.OUT(Verb) (וַיֵּצֵא / wai'yey'tsey) from~WITH (מֵעִם / mey'im) Paroh (פַּרְעֹה / par'oh) in~the~FLAMING (בָּחֳרִי / ba'hha'ri) NOSE (אָף / aph) [295]

and all these, your servants, will go down to me and they will bend themselves down to me saying, go out, you and all the people which are with your feet, and afterward I will go out, and he went out away from Paroh with the flaming nose,

9 and~he~will~SAY(Verb) (וַיֹּאמֶר / wai'yo'mer) YHWH (יְהוָה / YHWH) TO (אֶל / el) Mosheh (מֹשֶׁה / mo'sheh) NOT (לֹא / lo) he~will~HEAR(Verb) (יִשְׁמַע / yish'ma) TO~you(mp) (אֲלֵיכֶם / a'ley'khem) Paroh (פַּרְעֹה / par'oh) to~THAT (לְמַעַן / lê'ma'an) >~INCREASE(Verb) (רְבוֹת / rê'vot) WONDER~s~me (מוֹפְתַי / moph'tai) in~LAND (בְּאֶרֶץ / bê'e'rets) Mits'rayim (מִצְרָיִם / mits'ra'yim)

and YHWH said to Mosheh, Paroh will not listen to you so that my wonders will increase in the land of Mits'rayim,

10 and~Mosheh (וּמֹשֶׁה / u'mo'sheh) and~Aharon (וְאַהֲרֹן / wê'a'ha'ron) they~did~DO(Verb) (עָשׂוּ / a'su) AT (אֶת / et) ALL (כָּל / kol) the~WONDER~s (הַמֹּפְתִים / ha'moph'tim) the~THESE (הָאֵלֶּה / ha'ey'leh) to~FACE~s (לִפְנֵי / liph'ney) Paroh (פַרְעֹה / phar'oh) and~he~will~much~SEIZE(Verb) (וַיְחַזֵּק / wai'hha'zeyq) YHWH (יְהוָה / YHWH) AT (אֶת / et) HEART (לֵב / leyv) Paroh (פַּרְעֹה / par'oh) and~NOT (וְלֹא / wê'lo) he~did~much~SEND(Verb) (שִׁלַּח / shi'lahh) AT (אֶת / et) SON~s (בְּנֵי / bê'ney) Yisra'eyl (יִשְׂרָאֵל / yis'ra'eyl) from~LAND~him (מֵאַרְצוֹ / mey'ar'tso)

and Mosheh and Aharon did all these wonders to the face of Paroh, and YHWH strengthened the heart of Paroh and he did not send the sons of Yisra'eyl from his land,
