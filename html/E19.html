Exodus Chapter 19

1 in~ the~ NEW.MOON ( בַּחֹדֶשׁ / ba'hho'desh ) the~ THIRD ( הַשְּׁלִישִׁי / hash'li'shi ) to~ >~ GO.OUT (Verb) ( לְצֵאת / lê'tseyt ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) from~ LAND ( מֵאֶרֶץ / mey'e'rets ) Mits'rayim ( מִצְרָיִם / mits'ra'yim ) in~ the~ DAY ( בַּיּוֹם / ba'yom ) the~ THIS ( הַזֶּה / ha'zeh ) they~ did~ COME (Verb) ( בָּאוּ / ba'u ) WILDERNESS ( מִדְבַּר / mid'bar ) Sinai ( סִינָי / si'nai )

In the third new moon to the going out of the sons of Yisra'eyl from the land of Mits'rayim, in this day they came to the wilderness of Sinai,

2 and~ they(m)~ will~ JOURNEY (Verb) ( וַיִּסְעוּ / wai'yis'u ) from~ Rephiydiym ( מֵרְפִידִים / meyr'phi'dim ) and~ they(m)~ will~ COME (Verb) ( וַיָּבֹאוּ / wai'ya'vo'u ) WILDERNESS ( מִדְבַּר / mid'bar ) Sinai ( סִינַי / si'nai ) and~ they(m)~ will~ CAMP (Verb) ( וַיַּחֲנוּ / wai'ya'hha'nu ) in~ the~ WILDERNESS ( בַּמִּדְבָּר / ba'mid'bar ) and~ he~ will~ CAMP (Verb) ( וַיִּחַן / wai'yi'hhan ) THERE ( שָׁם / sham ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) OPPOSITE ( נֶגֶד / ne'ged ) the~ HILL ( הָהָר / ha'har )

and they journeyed from Rephiydiym, and they came to the wilderness of Sinai, and they camped in the wilderness, and Yisra'eyl camped there, opposite the hill,

3 and~ Mosheh ( וּמֹשֶׁה / u'mo'sheh ) he~ did~ GO.UP (Verb) ( עָלָה / a'lah ) TO ( אֶל / el ) the~ Elohiym ( הָאֱלֹהִים / ha'e'lo'him ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) TO ~him ( אֵלָיו / ey'law ) YHWH ( יְהוָה / YHWH ) FROM ( מִן / min ) the~ HILL ( הָהָר / ha'har ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) IN.THIS.WAY ( כֹּה / koh ) you(ms)~ will~ SAY (Verb) ( תֹאמַר / to'mar ) to~ HOUSE ( לְבֵית / lê'veyt ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) and~ you(ms)~ will~ make~ BE.FACE.TO.FACE (Verb) ( וְתַגֵּיד / wê'ta'geyd ) to~ SON ~s ( לִבְנֵי / liv'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl )

and Mosheh had gone up to the Elohiym and YHWH called out to him from the hill saying, in this way you will say to the house of Ya'aqov, and you will tell to the sons of Yisra'eyl.

4 YOU(MP) ( אַתֶּם / a'tem ) you(mp)~ did~ SEE (Verb) ( רְאִיתֶם / rê'i'tem ) WHICH ( אֲשֶׁר / a'sher ) I~ did~ DO (Verb) ( עָשִׂיתִי / a'si'ti ) to~ Mits'rayim ( לְמִצְרָיִם / lê'mits'ra'yim ) and~ I~ will~ LIFT.UP (Verb) ( וָאֶשָּׂא / wa'e'sa ) AT ~you(mp) ( אֶתְכֶם / et'khem ) UPON ( עַל / al ) WING ~s ( כַּנְפֵי / kan'phey ) EAGLE ~s ( נְשָׁרִים / nê'sha'rim ) [ 380 ] and~ I~ will~ make~ COME (Verb) ( וָאָבִא / wa'a'vi ) AT ~you(mp) ( אֶתְכֶם / et'khem ) TO ~me ( אֵלָי / ey'lai )

You saw what I did to Mits'rayim, and I will lift you up upon the wings of the eagle, and I will bring you to me,

5 and~ NOW ( וְעַתָּה / wê'a'tah ) IF ( אִם / im ) >~ HEAR (Verb) ( שָׁמוֹעַ / sha'mo'a ) you(mp)~ will~ HEAR (Verb) ( תִּשְׁמְעוּ / tish'me'u ) in~ VOICE ~me ( בְּקֹלִי / bê'qo'li ) and~ you(mp)~ did~ SAFEGUARD (Verb) ( וּשְׁמַרְתֶּם / ush'mar'tem ) AT ( אֶת / et ) COVENANT ~me ( בְּרִיתִי / bê'ri'ti ) and~ you(mp)~ did~ EXIST (Verb) ( וִהְיִיתֶם / wih'yi'tem ) to~ ~me ( לִי / li ) JEWEL ( סְגֻלָּה / sê'gu'lah ) from~ ALL ( מִכָּל / mi'kol ) the~ PEOPLE ~s ( הָעַמִּים / ha'a'mim ) GIVEN.THAT ( כִּי / ki ) to~ ~me ( לִי / li ) ALL ( כָּל / kol ) the~ LAND ( הָאָרֶץ / ha'a'rets )

and now, if you will carefully hear my voice, and you will safeguard my covenant, then you will exist for me as a jewel more than all the peoples, given that all the lands belong to me,

6 and~ YOU(MP) ( וְאַתֶּם / wê'a'tem ) you(mp)~ will~ EXIST (Verb) ( תִּהְיוּ / tih'yu ) to~ ~me ( לִי / li ) KINGDOM ( מַמְלֶכֶת / mam'le'khet ) ADMINISTRATOR ~s ( כֹּהֲנִים / ko'ha'nim ) and~ NATION ( וְגוֹי / wê'goy ) UNIQUE ( קָדוֹשׁ / qa'dosh ) THESE ( אֵלֶּה / ey'leh ) the~ WORD ~s ( הַדְּבָרִים / ha'de'va'rim ) WHICH ( אֲשֶׁר / a'sher ) you(ms)~ will~ much~ SPEAK (Verb) ( תְּדַבֵּר / tê'da'beyr ) TO ( אֶל / el ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl )

and you will exist for me as a kingdom of administrators and a unique nation, these are the words which you will speak to the sons of Yisra'eyl,

7 and~ he~ will~ COME (Verb) ( וַיָּבֹא / wai'ya'vo ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) to~ BEARD ~s ( לְזִקְנֵי / lê'ziq'ney ) the~ PEOPLE ( הָעָם / ha'am ) and~ he~ will~ PLACE (Verb) ( וַיָּשֶׂם / wai'ya'sem ) to~ FACE ~s ~them(m) ( לִפְנֵיהֶם / liph'ney'hem ) AT ( אֵת / eyt ) ALL ( כָּל / kol ) the~ WORD ~s ( הַדְּבָרִים / ha'de'va'rim ) the~ THESE ( הָאֵלֶּה / ha'ey'leh ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ DIRECT (Verb) ~him ( צִוָּהוּ / tsi'wa'hu ) YHWH ( יְהוָה / YHWH )

and Mosheh came and he called out to the bearded ones of the people, and he placed all these words, which YHWH directed him, to their faces,

8 and~ they(m)~ will~ ANSWER (Verb) ( וַיַּעֲנוּ / wai'ya'a'nu ) ALL ( כָל / khol ) the~ PEOPLE ( הָעָם / ha'am ) TOGETHER ( יַחְדָּו / yahh'daw ) and~ they(m)~ will~ SAY (Verb) ( וַיֹּאמְרוּ / wai'yom'ru ) ALL ( כֹּל / kol ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) YHWH ( יְהוָה / YHWH ) we~ will~ DO (Verb) ( נַעֲשֶׂה / na'a'seh ) and~ he~ will~ make~ TURN.BACK (Verb) ( וַיָּשֶׁב / wai'ya'shev ) Mosheh ( מֹשֶׁה / mo'sheh ) AT ( אֶת / et ) WORD ~s ( דִּבְרֵי / div'rey ) the~ PEOPLE ( הָעָם / ha'am ) TO ( אֶל / el ) YHWH ( יְהוָה / YHWH )

and all the people answered together, and they said, all which YHWH spoke, we will do, and Mosheh returned the words of the people to YHWH,

9 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) LOOK ( הִנֵּה / hin'neyh ) I ( אָנֹכִי / a'no'khi ) he~ did~ COME (Verb) ( בָּא / ba ) TO ~you(ms) ( אֵלֶיךָ / ey'ley'kha ) in~ THICK ( בְּעַב / bê'av ) the~ CLOUD ( הֶעָנָן / he'a'nan ) in~ the~ CROSS.OVER (Verb) ~ed(ms) ( בַּעֲבוּר / ba'a'vur ) he~ will~ HEAR (Verb) ( יִשְׁמַע / yish'ma ) the~ PEOPLE ( הָעָם / ha'am ) in~ >~ much~ SPEAK (Verb) ~me ( בְּדַבְּרִי / bê'da'be'ri ) WITH ~you(fs) ( עִמָּךְ / i'makh ) and~ ALSO ( וְגַם / wê'gam ) in~ ~you(ms) ( בְּךָ / bê'kha ) they(m)~ will~ make~ SECURE (Verb) ( יַאֲמִינוּ / ya'a'mi'nu ) to~ DISTANT ( לְעוֹלָם / lê'o'lam ) and~ he~ will~ make~ BE.FACE.TO.FACE (Verb) ( וַיַּגֵּד / wai'ya'geyd ) Mosheh ( מֹשֶׁה / mo'sheh ) AT ( אֶת / et ) WORD ~s ( דִּבְרֵי / div'rey ) the~ PEOPLE ( הָעָם / ha'am ) TO ( אֶל / el ) YHWH ( יְהוָה / YHWH )

and YHWH said to Mosheh, look, I am coming to you in the thick of the cloud, in order that the people will hear me speaking with you, and also, they will support you for a distant time, and Mosheh told the words of the people to YHWH,

10 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) !(ms)~ WALK (Verb) ( לֵךְ / leykh ) TO ( אֶל / el ) the~ PEOPLE ( הָעָם / ha'am ) and~ you(ms)~ did~ much~ SET.APART (Verb) ~them(m) ( וְקִדַּשְׁתָּם / wê'qi'dash'tam ) the~ DAY ( הַיּוֹם / hai'yom ) and~ TOMORROW ( וּמָחָר / u'ma'hhar ) and~ they~ did~ much~ WASH (Verb) ( וְכִבְּסוּ / wê'khi'be'su ) APPAREL ~s ~them(m) ( שִׂמְלֹתָם / sim'lo'tam )

and YHWH said to Mosheh, walk to the people and you will set them apart today and tomorrow, and they will wash their apparel,

11 and~ they~ did~ EXIST (Verb) ( וְהָיוּ / wê'hai'u ) be~ PREPARE (Verb) ~ing(mp) ( נְכֹנִים / nê'kho'nim ) to~ the~ DAY ( לַיּוֹם / lai'yom ) the~ THIRD ( הַשְּׁלִישִׁי / hash'li'shi ) GIVEN.THAT ( כִּי / ki ) in~ the~ DAY ( בַּיּוֹם / ba'yom ) the~ THIRD ( הַשְּׁלִישִׁי / hash'li'shi ) he~ will~ GO.DOWN (Verb) ( יֵרֵד / yey'reyd ) YHWH ( יְהוָה / YHWH ) to~ EYE ~s2 ( לְעֵינֵי / lê'ey'ney ) ALL ( כָל / khol ) the~ PEOPLE ( הָעָם / ha'am ) UPON ( עַל / al ) HILL ( הַר / har ) Sinai ( סִינָי / si'nai )

and they will be ready for the third day, given that in the third day YHWH will go down to the eyes of all the people upon the hill of Sinai,

12 and~ you(ms)~ did~ make~ BOUND (Verb) ( וְהִגְבַּלְתָּ / wê'hig'bal'ta ) AT ( אֶת / et ) the~ PEOPLE ( הָעָם / ha'am ) ALL.AROUND ( סָבִיב / sa'viv ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) !(mp)~ be~ SAFEGUARD (Verb) ( הִשָּׁמְרוּ / hi'sham'ru ) to~ ~you(mp) ( לָכֶם / la'khem ) >~ GO.UP (Verb) ( עֲלוֹת / a'lot ) in~ the~ HILL ( בָּהָר / ba'har ) [ 381 ] and~ >~ TOUCH (Verb) ( וּנְגֹעַ / un'go'a ) in~ EXTREMITY ~him ( בְּקָצֵהוּ / bê'qa'tsey'hu ) ALL ( כָּל / kol ) the~ TOUCH (Verb) ~ing(ms) ( הַנֹּגֵעַ / ha'no'gey'a ) in~ the~ HILL ( בָּהָר / ba'har ) >~ DIE (Verb) ( מוֹת / mot ) he~ will~ be~ make~ DIE (Verb) ( יוּמָת / yu'mat )

and you will make bounds all around the people saying, safeguard yourselves, go up in the hill, and touch his extremity, all the ones touching the hill will surely be killed.

13 NOT ( לֹא / lo ) she~ will~ TOUCH (Verb) ( תִגַּע / ti'ga ) in~ ~him ( בּוֹ / bo ) HAND ( יָד / yad ) GIVEN.THAT ( כִּי / ki ) >~ STONE (Verb) ( סָקוֹל / sa'qol ) he~ will~ be~ STONE (Verb) ( יִסָּקֵל / yi'sa'qeyl ) OR ( אוֹ / o ) >~ THROW (Verb) ( יָרֹה / ya'roh ) he~ will~ be~ THROW (Verb) ( יִיָּרֶה / yi'ya'reh ) IF ( אִם / im ) BEAST ( בְּהֵמָה / bê'hey'mah ) IF ( אִם / im ) MAN ( אִישׁ / ish ) NOT ( לֹא / lo ) he~ will~ LIVE (Verb) ( יִחְיֶה / yihh'yeh ) in~ >~ DRAW (Verb) ( בִּמְשֹׁךְ / bim'shokh ) the~ JUBILEE ( הַיֹּבֵל / hai'yo'veyl ) THEY(m) ( הֵמָּה / hey'mah ) they(m)~ will~ GO.UP (Verb) ( יַעֲלוּ / ya'a'lu ) in~ the~ HILL ( בָהָר / va'har )

The hand will not touch him, given that he will surely be stoned or he will surely be thrown, if it is a beast, if it is a man, he will not live, with the drawing of the jubilee horn, they, they will go up in the hill,

14 and~ he~ will~ GO.DOWN (Verb) ( וַיֵּרֶד / wai'yey'red ) Mosheh ( מֹשֶׁה / mo'sheh ) FROM ( מִן / min ) the~ HILL ( הָהָר / ha'har ) TO ( אֶל / el ) the~ PEOPLE ( הָעָם / ha'am ) and~ he~ will~ much~ SET.APART (Verb) ( וַיְקַדֵּשׁ / wai'qa'deysh ) AT ( אֶת / et ) the~ PEOPLE ( הָעָם / ha'am ) and~ they(m)~ will~ much~ WASH (Verb) ( וַיְכַבְּסוּ / wai'kha'be'su ) APPAREL ~s ~them(m) ( שִׂמְלֹתָם / sim'lo'tam )

and Mosheh went down from the hill to the people, and he set apart the people, and they washed their apparel,

15 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) TO ( אֶל / el ) the~ PEOPLE ( הָעָם / ha'am ) !(mp)~ EXIST (Verb) ( הֱיוּ / he'yu ) be~ PREPARE (Verb) ~ing(mp) ( נְכֹנִים / nê'kho'nim ) to~ THREE ( לִשְׁלֹשֶׁת / lish'lo'shet ) DAY ~s ( יָמִים / ya'mim ) DO.NOT ( אַל / al ) you(mp)~ will~ DRAW.NEAR (Verb) ( תִּגְּשׁוּ / tig'shu ) TO ( אֶל / el ) WOMAN ( אִשָּׁה / i'shah )

and he said to the people, be ready for three days, do not draw near to a woman,

16 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) in~ the~ DAY ( בַיּוֹם / vai'yom ) the~ THIRD ( הַשְּׁלִישִׁי / hash'li'shi ) in~ >~ EXIST (Verb) ( בִּהְיֹת / bih'yot ) the~ MORNING ( הַבֹּקֶר / ha'bo'qer ) and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) VOICE ~s ( קֹלֹת / qo'lot ) and~ FLASH ~s ( וּבְרָקִים / uv'ra'qim ) and~ CLOUD ( וְעָנָן / wê'a'nan ) HEAVY ( כָּבֵד / ka'veyd ) UPON ( עַל / al ) the~ HILL ( הָהָר / ha'har ) and~ VOICE ( וְקֹל / wê'qol ) RAM.HORN ( שֹׁפָר / sho'phar ) FORCEFUL ( חָזָק / hha'zaq ) MANY ( מְאֹד / mê'od ) and~ he~ will~ TREMBLE (Verb) ( וַיֶּחֱרַד / wai'ye'hhe'rad ) ALL ( כָּל / kol ) the~ PEOPLE ( הָעָם / ha'am ) WHICH ( אֲשֶׁר / a'sher ) in~ the~ CAMP ( בַּמַּחֲנֶה / ba'ma'hha'neh )

and it came to pass in the third day, in the existing of the morning, and it came to pass, thunder and flashes and a heavy cloud were upon the hill, and the voice of the ram horn was very forceful, and all the people, which were in the camp, trembled,

17 and~ he~ will~ make~ GO.OUT (Verb) ( וַיּוֹצֵא / wai'yo'tsey ) Mosheh ( מֹשֶׁה / mo'sheh ) AT ( אֶת / et ) the~ PEOPLE ( הָעָם / ha'am ) to~ >~ MEET (Verb) ( לִקְרַאת / liq'rat ) the~ Elohiym ( הָאֱלֹהִים / ha'e'lo'him ) FROM ( מִן / min ) the~ CAMP ( הַמַּחֲנֶה / ha'ma'hha'neh ) and~ they(m)~ will~ be~ make~ STAND.UP (Verb) ( וַיִּתְיַצְּבוּ / wai'yit'yats'vu ) in~ LOWER.PART ( בְּתַחְתִּית / bê'tahh'tit ) the~ HILL ( הָהָר / ha'har )

and Mosheh made the people go out from the camp to meet the Elohiym, and they were made to stand up in the lower part of the hill,

18 and~ HILL ( וְהַר / wê'har ) Sinai ( סִינַי / si'nai ) he~ did~ SMOKE (Verb) ( עָשַׁן / a'shan ) ALL ~him ( כֻּלּוֹ / ku'lo ) from~ FACE ~s ( מִפְּנֵי / mip'ney ) [ 382 ] WHICH ( אֲשֶׁר / a'sher ) he~ did~ GO.DOWN (Verb) ( יָרַד / ya'rad ) UPON ~him ( עָלָיו / a'law ) YHWH ( יְהוָה / YHWH ) in~ the~ FIRE ( בָּאֵשׁ / ba'eysh ) and~ he~ will~ make~ GO.UP (Verb) ( וַיַּעַל / wai'ya'al ) SMOKE ~him ( עֲשָׁנוֹ / a'sha'no ) like~ SMOKE ( כְּעֶשֶׁן / kê'e'shen ) the~ FURNACE ( הַכִּבְשָׁן / ha'kiv'shan ) and~ he~ will~ TREMBLE (Verb) ( וַיֶּחֱרַד / wai'ye'hhe'rad ) ALL ( כָּל / kol ) the~ HILL ( הָהָר / ha'har ) MANY ( מְאֹד / mê'od )

and all of the hill of Sinai had smoked, all his face, because YHWH went down upon him in the fire, and his smoke went up like the smoke of the furnace, and all the hill trembled greatly,
