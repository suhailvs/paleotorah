and Elohiym came to Aviymelekh in the dream in the night and he said to him, look at you, dying because of the woman whom you took and she is the married of a master,

4 and~ Aviymelekh ( וַאֲבִימֶלֶךְ / wa'a'vi'me'lekh ) NOT ( לֹא / lo ) he~ did~ COME.NEAR (Verb) ( קָרַב / qa'rav ) TO ~her ( אֵלֶיהָ / ey'ley'ah ) and~ he~ will~ SAY (Verb) ( וַיֹּאמַר / wai'yo'mar ) Adonai ( אֲדֹנָי / a'do'nai ) ?~ NATION ( הֲגוֹי / ha'goy ) ALSO ( גַּם / gam ) STEADFAST.ONE ( צַדִּיק / tsa'diq ) you(ms)~ will~ KILL (Verb) ( תַּהֲרֹג / ta'ha'rog )

Did he not say to me she is my sister and she also said he is my brother, in the maturity of my heart and in the innocence of my palms I did this,

6 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) TO ~him ( אֵלָיו / ey'law ) the~ Elohiym ( הָאֱלֹהִים / ha'e'lo'him ) in~ the~ DREAM ( בַּחֲלֹם / ba'hha'lom ) ALSO ( גַּם / gam ) I ( אָנֹכִי / a'no'khi ) I~ did~ KNOW (Verb) ( יָדַעְתִּי / ya'da'ti ) GIVEN.THAT ( כִּי / ki ) in~ MATURE ( בְתָם / vê'tam ) HEART ~you(ms) ( לְבָבְךָ / lê'vav'kha ) you(ms)~ did~ DO (Verb) ( עָשִׂיתָ / a'si'ta ) THIS ( זֹּאת / zot ) and~ I~ will~ KEEP.BACK (Verb) ( וָאֶחְשֹׂךְ / wa'ehh'sokh ) ALSO ( גַּם / gam ) I ( אָנֹכִי / a'no'khi ) AT ~you(ms) ( אוֹתְךָ / ot'kha ) from~ FAILURE ~him ( מֵחֲטוֹ / mey'hha'to ) [ 143 ] to~ ~me ( לִי / li ) UPON ( עַל / al ) SO ( כֵּן / keyn ) NOT ( לֹא / lo ) I~ did~ GIVE (Verb) ~you(ms) ( נְתַתִּיךָ / nê'ta'ti'kha ) to~ >~ TOUCH (Verb) ( לִנְגֹּעַ / lin'go'a ) TO ~her ( אֵלֶיהָ / ey'ley'ah )

and the Elohiym said to him in the dream, also I, I knew, given that in the maturity of your heart, you did this, and I also kept you back from his failure to me therefore, I did not give you to touch her,

7 and~ NOW ( וְעַתָּה / wê'a'tah ) !(ms)~ make~ TURN.BACK (Verb) ( הָשֵׁב / ha'sheyv ) WOMAN ( אֵשֶׁת / ey'shet ) the~ MAN ( הָאִישׁ / ha'ish ) GIVEN.THAT ( כִּי / ki ) ANNOUNCER ( נָבִיא / na'vi ) HE ( הוּא / hu ) and~ he~ will~ self~ PLEAD (Verb) ( וְיִתְפַּלֵּל / wê'yit'pa'leyl ) in~ UNTIL ~you(ms) ( בַּעַדְךָ / ba'ad'kha ) and~ !(ms)~ LIVE (Verb) ( וֶחְיֵה / we'hhe'yeyh ) and~ IF ( וְאִם / wê'im ) WITHOUT ~you(ms) ( אֵינְךָ / eyn'kha ) make~ TURN.BACK (Verb) ~ing(ms) ( מֵשִׁיב / mey'shiv ) !(ms)~ KNOW (Verb) ( דַּע / da ) GIVEN.THAT ( כִּי / ki ) >~ DIE (Verb) ( מוֹת / mot ) you(ms)~ will~ DIE (Verb) ( תָּמוּת / ta'mut ) YOU(MS) ( אַתָּה / a'tah ) and~ ALL ( וְכָל / wê'khol ) WHICH ( אֲשֶׁר / a'sher ) to~ ~you(fs) ( לָךְ / lakh )

and Aviymelekh departed early in the morning and he called out to all of his servants and he spoke all of these words in their ears and the men greatly feared,

and Aviymelekh called out to Avraham and said, to him what did you do to us and how did I fail to you given that you brought upon me and upon my kingdom place a great failure, works which were not done, you did by me,

10 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Aviymelekh ( אֲבִימֶלֶךְ / a'vi'me'lekh ) TO ( אֶל / el ) Avraham ( אַבְרָהָם / av'ra'ham ) WHAT ( מָה / mah ) you(ms)~ did~ SEE (Verb) ( רָאִיתָ / ra'i'ta ) GIVEN.THAT ( כִּי / ki ) you(ms)~ did~ DO (Verb) ( עָשִׂיתָ / a'si'ta ) AT ( אֶת / et ) the~ WORD ( הַדָּבָר / ha'da'var ) the~ THIS ( הַזֶּה / ha'zeh )

and Aviymelekh said to Avraham, what did you see, given that you did this word,

and Avraham said, given that I said, fearfulness of Elohiym is not at all in this place and they will kill me because of the word of my woman,

and indeed she is my sister, daughter of my father, surely not the daughter of my mother and she exists to me for a woman,

13 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) they~ did~ make~ WANDER (Verb) ( הִתְעוּ / hit'u ) AT ~me ( אֹתִי / o'ti ) Elohiym ( אֱלֹהִים / e'lo'him ) from~ HOUSE ( מִבֵּית / mi'beyt ) FATHER ~me ( אָבִי / a'vi ) and~ I~ will~ SAY (Verb) ( וָאֹמַר / wa'o'mar ) to~ ~her ( לָהּ / lah ) THIS ( זֶה / zeh ) KINDNESS ~you(fs) ( חַסְדֵּךְ / hhas'deykh ) WHICH ( אֲשֶׁר / a'sher ) you(fs)~ will~ DO (Verb) ( תַּעֲשִׂי / ta'a'si ) BY ~me ( עִמָּדִי / i'ma'di ) TO ( אֶל / el ) ALL ( כָּל / kol ) the~ AREA ( הַמָּקוֹם / ha'ma'qom ) WHICH ( אֲשֶׁר / a'sher ) we~ will~ COME (Verb) ( נָבוֹא / na'vo ) THERE ~unto ( שָׁמָּה / sha'mah ) !(fs)~ SAY (Verb) ( אִמְרִי / im'ri ) to~ ~me ( לִי / li ) BROTHER ~me ( אָחִי / a'hhi ) HE ( הוּא / hu )

and it came to pass just as Elohiym caused me to wander from the house of my father and I said to her, this is your kindness which you will do by me to all of the places which we will come unto, say for me he is my brother,

14 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) Aviymelekh ( אֲבִימֶלֶךְ / a'vi'me'lekh ) FLOCKS ( צֹאן / tson ) and~ CATTLE ( וּבָקָר / u'va'qar ) and~ SERVANT ~s ( וַעֲבָדִים / wa'a'va'dim ) and~ MAID ~s ( וּשְׁפָחֹת / ush'pha'hhot ) and~ he~ will~ GIVE (Verb) ( וַיִּתֵּן / wai'yi'teyn ) to~ Avraham ( לְאַבְרָהָם / lê'av'ra'ham ) and~ he~ will~ make~ TURN.BACK (Verb) ( וַיָּשֶׁב / wai'ya'shev ) to~ ~him ( לוֹ / lo ) AT ( אֵת / eyt ) Sarah ( שָׂרָה / sa'rah ) WOMAN ~him ( אִשְׁתּוֹ / ish'to )

15 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Aviymelekh ( אֲבִימֶלֶךְ / a'vi'me'lekh ) LOOK ( הִנֵּה / hin'neyh ) LAND ~me ( אַרְצִי / ar'tsi ) to~ FACE ~s ~you(ms) ( לְפָנֶיךָ / lê'pha'ney'kha ) in~ the~ FUNCTIONAL ( בַּטּוֹב / ba'tov ) in~ EYE ~s2 ~you(ms) ( בְּעֵינֶיךָ / bê'ey'ney'kha ) !(ms)~ SETTLE (Verb) ( שֵׁב / sheyv )
