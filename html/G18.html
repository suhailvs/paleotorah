1 and~ he~ will~ be~ SEE (Verb) ( וַיֵּרָא / wai'yey'ra ) TO ~him ( אֵלָיו / ey'law ) YHWH ( יְהוָה / YHWH ) in~ GREAT.TREE ~s ( בְּאֵלֹנֵי / bê'ey'lo'ney ) Mamre ( מַמְרֵא / mam'rey ) and~ HE ( וְהוּא / wê'hu ) SETTLE (Verb) ~ing(ms) ( יֹשֵׁב / yo'sheyv ) OPENING ( פֶּתַח / pe'tahh ) the~ TENT ( הָאֹהֶל / ha'o'hel ) like~ HOT ( כְּחֹם / kê'hhom ) the~ DAY ( הַיּוֹם / hai'yom )

and YHWH appeared to him in the great trees of Mamre, and he was settling in the opening of the tent as the day was hot,

and he lifted up his eyes and he saw and look, three men were standing upon him, and he saw, and he ran from the opening of the tent to meet them, he bent himself down unto the land,

3 and~ he~ will~ SAY (Verb) ( וַיֹּאמַר / wai'yo'mar ) Adonai ( אֲדֹנָי / a'do'nai ) [ 124 ] IF ( אִם / im ) PLEASE ( נָא / na ) I~ did~ FIND (Verb) ( מָצָאתִי / ma'tsa'ti ) BEAUTY ( חֵן / hheyn ) in~ EYE ~s2 ~you(ms) ( בְּעֵינֶיךָ / bê'ey'ney'kha ) DO.NOT ( אַל / al ) PLEASE ( נָא / na ) you(ms)~ will~ CROSS.OVER (Verb) ( תַעֲבֹר / ta'a'vor ) from~ UPON ( מֵעַל / mey'al ) SERVANT ~you(ms) ( עַבְדֶּךָ / av'de'kha )

and he said, Adonai, please, if I find beauty in your eyes, please do not cross over from upon your servant.

4 he~ will~ be~ much~ TAKE (Verb) ( יֻקַּח / yu'qahh ) PLEASE ( נָא / na ) SMALL.AMOUNT ( מְעַט / mê'at ) WATER ~s2 ( מַיִם / ma'yim ) and~ !(mp)~ BATHE (Verb) ( וְרַחֲצוּ / wê'ra'hha'tsu ) FOOT ~s2 ~you(mp) ( רַגְלֵיכֶם / rag'ley'khem ) and~ !(ms)~ be~ LEAN (Verb) ( וְהִשָּׁעֲנוּ / wê'hi'sha'a'nu ) UNDER ( תַּחַת / ta'hhat ) the~ TREE ( הָעֵץ / ha'eyts )

Please, a small amount of waters will be taken and bathe your feet and lean under the tree,

5 and~ I~ will~ TAKE (Verb) ~her ( וְאֶקְחָה / wê'eq'hhah ) FRAGMENT ( פַת / phat ) BREAD ( לֶחֶם / le'hhem ) and~ !(mp)~ HOLD.UP (Verb) ( וְסַעֲדוּ / wê'sa'a'du ) [ 125 ] HEART ~you(mp) ( לִבְּכֶם / li'be'khem ) AFTER ( אַחַר / a'hhar ) you(mp)~ will~ CROSS.OVER (Verb) ( תַּעֲבֹרוּ / ta'a'vo'ru ) GIVEN.THAT ( כִּי / ki ) UPON ( עַל / al ) SO ( כֵּן / keyn ) [ 126 ] you(mp)~ did~ CROSS.OVER (Verb) ( עֲבַרְתֶּם / a'var'tem ) UPON ( עַל / al ) SERVANT ~you(mp) ( עַבְדְּכֶם / av'de'khem ) and~ they(m)~ will~ SAY (Verb) ( וַיֹּאמְרוּ / wai'yom'ru ) SO ( כֵּן / keyn ) you(ms)~ will~ DO (Verb) ( תַּעֲשֶׂה / ta'a'seh ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) you(ms)~ did~ much~ SPEAK (Verb) ( דִּבַּרְתָּ / di'bar'ta )

and I will take a fragment of bread and hold up your heart, afterward you will cross over since you crossed over upon your servant, and they said, you will do so just as you spoke,

6 and~ he~ will~ much~ HURRY (Verb) ( וַיְמַהֵר / wai'ma'heyr ) Avraham ( אַבְרָהָם / av'ra'ham ) the~ TENT ~unto ( הָאֹהֱלָה / ha'o'he'lah ) TO ( אֶל / el ) Sarah ( שָׂרָה / sa'rah ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) !(fs)~ much~ HURRY (Verb) ( מַהֲרִי / me'ha'ri ) THREE ( שְׁלֹשׁ / shê'losh ) SE'AH ~s ( סְאִים / sê'im ) GRAIN.FLOUR ( קֶמַח / qe'mahh ) FLOUR ( סֹלֶת / so'let ) !(fs)~ KNEAD (Verb) ( לוּשִׁי / lu'shi ) and~ !(fs)~ DO (Verb) ( וַעֲשִׂי / wa'a'si ) BAKED.BREAD ~s ( עֻגוֹת / u'got )

and Avraham much hurried unto the tent to Sarah and he said, hurry, knead three se'ahs of grain flour and make baked breads,

7 and~ TO ( וְאֶל / wê'el ) the~ CATTLE ( הַבָּקָר / ha'ba'qar ) he~ did~ RUN (Verb) ( רָץ / rats ) Avraham ( אַבְרָהָם / av'ra'ham ) and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) SON ( בֶּן / ben ) CATTLE ( בָּקָר / ba'qar ) TENDER ( רַךְ / rakh ) and~ FUNCTIONAL ( וָטוֹב / wa'tov ) and~ he~ will~ GIVE (Verb) ( וַיִּתֵּן / wai'yi'teyn ) TO ( אֶל / el ) the~ YOUNG.MAN ( הַנַּעַר / ha'na'ar ) and~ he~ will~ much~ HURRY (Verb) ( וַיְמַהֵר / wai'ma'heyr ) to~ >~ DO (Verb) ( לַעֲשׂוֹת / la'a'sot ) AT ~him ( אֹתוֹ / o'to )

and Avraham ran to the cattle and he took a son of the cattle, tender and functional, and he gave it to the young man, and he hurried to make him,

8 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) CHEESE ( חֶמְאָה / hhem'ah ) and~ FAT ( וְחָלָב / wê'hha'lav ) and~ SON ( וּבֶן / u'ven ) the~ CATTLE ( הַבָּקָר / ha'ba'qar ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ DO (Verb) ( עָשָׂה / a'sah ) and~ he~ will~ GIVE (Verb) ( וַיִּתֵּן / wai'yi'teyn ) to~ FACE ~s ~them(m) ( לִפְנֵיהֶם / liph'ney'hem ) and~ HE ( וְהוּא / wê'hu ) STAND (Verb) ~ing(ms) ( עֹמֵד / o'meyd ) UPON ~them(m) ( עֲלֵיהֶם / a'ley'hem ) UNDER ( תַּחַת / ta'hhat ) the~ TREE ( הָעֵץ / ha'eyts ) and~ they(m)~ will~ EAT (Verb) ( וַיֹּאכֵלוּ / wai'yo'khey'lu )

and he took cheese and fat and a son of the cattle, which he made, and he gave it to their face, and he was standing upon them under the tree and they ate,

9 and~ they(m)~ will~ SAY (Verb) ( וַיֹּאמְרוּ / wai'yom'ru ) TO ~him ( אֵלָיו / ey'law ) WHERE ( אַיֵּה / ai'yeyh ) Sarah ( שָׂרָה / sa'rah ) WOMAN ~you(ms) ( אִשְׁתֶּךָ / ish'te'kha ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) LOOK ( הִנֵּה / hin'neyh ) in~ the~ TENT ( בָאֹהֶל / va'o'hel )

and they said to him, where is Sarah your woman, and he said, look, in the tent,

10 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) >~ TURN.BACK (Verb) ( שׁוֹב / shov ) I~ will~ TURN.BACK (Verb) ( אָשׁוּב / a'shuv ) TO ~you(ms) ( אֵלֶיךָ / ey'ley'kha ) like~ the~ APPOINTED.TIME ( כָּעֵת / ka'eyt ) LIVING ( חַיָּה / hhai'yah ) [ 127 ] and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) SON ( בֵן / veyn ) to~ Sarah ( לְשָׂרָה / lê'sa'rah ) WOMAN ~you(ms) ( אִשְׁתֶּךָ / ish'te'kha ) and~ Sarah ( וְשָׂרָה / wê'sa'rah ) HEAR (Verb) ~ing(fs) ( שֹׁמַעַת / sho'ma'at ) OPENING ( פֶּתַח / pe'tahh ) [ 128 ] the~ TENT ( הָאֹהֶל / ha'o'hel ) and~ HE ( וְהוּא / wê'hu ) AFTER ~him ( אַחֲרָיו / a'hha'raw )

and he said, I will surely turn back to you at the appointed time of living, and look, a son for Sarah your woman, and Sarah was hearing in the opening of the tent and he was behind him,

11 and~ Avraham ( וְאַבְרָהָם / wê'av'ra'ham ) and~ Sarah ( וְשָׂרָה / wê'sa'rah ) BEARD ~s ( זְקֵנִים / zê'qey'nim ) COME (Verb) ~ing(mp) ( בָּאִים / ba'im ) in~ the~ DAY ~s ( בַּיָּמִים / bai'ya'mim ) [ 129 ] he~ did~ TERMINATE (Verb) ( חָדַל / hha'dal ) to~ >~ EXIST (Verb) ( לִהְיוֹת / lih'yot ) to~ Sarah ( לְשָׂרָה / lê'sa'rah ) PATH ( אֹרַח / o'rahh ) like~ the~ WOMAN ~s ( כַּנָּשִׁים / ka'na'shim ) [ 130 ]

and Avraham and Sarah were bearded ones, coming in the days, the path like the women terminated to exist for Sarah,

12 and~ she~ will~ LAUGH (Verb) ( וַתִּצְחַק / wa'tits'hhaq ) Sarah ( שָׂרָה / sa'rah ) in~ INSIDE ~her ( בְּקִרְבָּהּ / bê'qir'bah ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) AFTER ( אַחֲרֵי / a'hha'rey ) I~ >~ WEAR.OUT (Verb) ( בְלֹתִי / vê'lo'ti ) [ 131 ] she~ did~ EXIST (Verb) ( הָיְתָה / hai'tah ) to~ ~me ( לִּי / li ) PLEASURE ( עֶדְנָה / ed'nah ) and~ LORD ~me ( וַאדֹנִי / wa'od'ni ) he~ did~ BE.OLD (Verb) ( זָקֵן / za'qeyn )

and Sarah laughed inside herself saying, after I am worn out, pleasure exists for me and my lord who is old,

13 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Avraham ( אַבְרָהָם / av'ra'ham ) to~ WHAT ( לָמָּה / la'mah ) THIS ( זֶּה / zeh ) she~ did~ LAUGH (Verb) ( צָחֲקָה / tsa'hha'qah ) Sarah ( שָׂרָה / sa'rah ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) ?~ MOREOVER ( הַאַף / ha'aph ) INDEED ( אֻמְנָם / um'nam ) [ 132 ] I~ will~ BRING.FORTH (Verb) ( אֵלֵד / ey'leyd ) and~ I ( וַאֲנִי / wa'ani ) I~ did~ BE.OLD (Verb) ( זָקַנְתִּי / za'qan'ti )

and YHWH said to Avraham, why is this, Sarah laughed saying, will I really bring forth when I am old.

14 ?~ he~ did~ be~ PERFORM (Verb) ( הֲיִפָּלֵא / ha'yi'pa'ley ) from~ YHWH ( מֵיְהוָה / mey'YHWH ) WORD ( דָּבָר / da'var ) [ 133 ] to~ the~ APPOINTED ( לַמּוֹעֵד / la'mo'eyd ) I~ will~ TURN.BACK (Verb) ( אָשׁוּב / a'shuv ) TO ~you(ms) ( אֵלֶיךָ / ey'ley'kha ) like~ the~ APPOINTED.TIME ( כָּעֵת / ka'eyt ) LIVING ( חַיָּה / hhai'yah ) [ 134 ] and~ to~ Sarah ( וּלְשָׂרָה / ul'sa'rah ) SON ( בֵן / veyn )

Is a word too difficult for YHWH, at the appointed time I will turn back to you, at the appointed time of life, and to Sarah will be a son,

15 and~ she~ will~ much~ DENY (Verb) ( וַתְּכַחֵשׁ / wat'kha'hheysh ) Sarah ( שָׂרָה / sa'rah ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) NOT ( לֹא / lo ) I~ did~ LAUGH (Verb) ( צָחַקְתִּי / tsa'hhaq'ti ) GIVEN.THAT ( כִּי / ki ) she~ did~ FEAR (Verb) ( יָרֵאָה / ya'rey'ah ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) NOT ( לֹא / lo ) GIVEN.THAT ( כִּי / ki ) you(fs)~ did~ LAUGH (Verb) ( צָחָקְתְּ / tsa'hhaq'te )

and Sarah lied, saying, I did not laugh, given that she feared, and he said, no, given that you did laugh,

16 and~ they(m)~ will~ RISE (Verb) ( וַיָּקֻמוּ / wai'ya'qu'mu ) from~ THERE ( מִשָּׁם / mi'sham ) the~ MAN ~s ( הָאֲנָשִׁים / ha'a'na'shim ) and~ they(m)~ will~ make~ LOOK.DOWN (Verb) ( וַיַּשְׁקִפוּ / wai'yash'qi'phu ) UPON ( עַל / al ) FACE ~s ( פְּנֵי / pê'ney ) Sedom ( סְדֹם / sê'dom ) and~ Avraham ( וְאַבְרָהָם / wê'av'ra'ham ) WALK (Verb) ~ing(ms) ( הֹלֵךְ / ho'leykh ) WITH ~them(m) ( עִמָּם / i'mam ) to~ >~ much~ SEND (Verb) ~them(m) ( לְשַׁלְּחָם / lê'shal'hham )

and the men rose from there and they looked down upon the face of Sedom, and Avraham was walking with them to send them off,

17 and~ YHWH ( וַיהוָה / wa'YHWH ) he~ did~ SAY (Verb) ( אָמָר / a'mar ) the~ much~ COVER.OVER (Verb) ~ing(ms) ( הַמְכַסֶּה / ham'kha'seh ) [ 135 ] I ( אֲנִי / a'ni ) from~ Avraham ( מֵאַבְרָהָם / mey'av'ra'ham ) WHICH ( אֲשֶׁר / a'sher ) I ( אֲנִי / a'ni ) DO (Verb) ~ing(ms) ( עֹשֶׂה / o'seh )

and YHWH had said, shall I cover over from Avraham what I am doing,

18 and~ Avraham ( וְאַבְרָהָם / wê'av'ra'ham ) >~ EXIST (Verb) ( הָיוֹ / hai'o ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) to~ NATION ( לְגוֹי / lê'goy ) GREAT ( גָּדוֹל / ga'dol ) and~ NUMEROUS ( וְעָצוּם / wê'a'tsum ) and~ they~ did~ be~ KNEEL (Verb) ( וְנִבְרְכוּ / wê'niv're'khu ) in~ ~him ( בוֹ / vo ) ALL ( כֹּל / kol ) NATION ~s ( גּוֹיֵי / go'yey ) the~ LAND ( הָאָרֶץ / ha'a'rets )

and Avraham will surely exist as a great and numerous nation, and all the nations of the land will be exalted with him.

Given that I knew him, so that he will direct his sons, and his house after him, and they will safeguard the road of YHWH to do steadfastness and decisions, so that YHWH will bring upon Avraham what he spoke upon him,

20 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) YELL ( זַעֲקַת / za'a'qat ) Sedom ( סְדֹם / sê'dom ) and~ Ghamorah ( וַעֲמֹרָה / wa'a'mo'rah ) GIVEN.THAT ( כִּי / ki ) she~ did~ INCREASE.IN.NUMBER (Verb) ( רָבָּה / ra'bah ) and~ FAILURE ~them(m) ( וְחַטָּאתָם / wê'hha'ta'tam ) GIVEN.THAT ( כִּי / ki ) she~ did~ BE.HEAVY (Verb) ( כָבְדָה / khav'dah ) MANY ( מְאֹד / mê'od )

and YHWH said, given that the yell of Sedom and Ghamorah had increased in number and given that their failure had become very heavy.

21 I~ will~ GO.DOWN (Verb) ~her ( אֵרְדָה / eyr'dah ) PLEASE ( נָא / na ) and~ I~ will~ SEE (Verb) ( וְאֶרְאֶה / wê'er'eh ) the~ like~ YELL ~her ( הַכְּצַעֲקָתָהּ / ha'ke'tsa'a'qa'tah ) the~ COME (Verb) ~ing(fs) ( הַבָּאָה / ha'ba'ah ) TO ~me ( אֵלַי / ey'lai ) they~ did~ DO (Verb) ( עָשׂוּ / a'su ) COMPLETION ( כָּלָה / ka'lah ) and~ IF ( וְאִם / wê'im ) NOT ( לֹא / lo ) I~ will~ KNOW (Verb) ~& ( אֵדָעָה / ey'da'ah )

I will go down to her please, and I will see whether her yell that is coming to me is a completion, and if not, I will know,

22 and~ they(m)~ will~ TURN (Verb) ( וַיִּפְנוּ / wai'yiph'nu ) from~ THERE ( מִשָּׁם / mi'sham ) the~ MAN ~s ( הָאֲנָשִׁים / ha'a'na'shim ) and~ they(m)~ will~ WALK (Verb) ( וַיֵּלְכוּ / wai'yeyl'khu ) Sedom ~unto ( סְדֹמָה / sê'do'mah ) and~ Avraham ( וְאַבְרָהָם / wê'av'ra'ham ) YET.AGAIN ~him ( עוֹדֶנּוּ / o'de'nu ) STAND (Verb) ~ing(ms) ( עֹמֵד / o'meyd ) to~ FACE ~s ( לִפְנֵי / liph'ney ) [ 136 ] YHWH ( יְהוָה / YHWH )

and the men turned from there and they walked unto Sedom, and yet again Avraham was standing to the face of YHWH,
