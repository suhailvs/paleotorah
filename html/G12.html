Genesis Chapter 12

1 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Avram ( אַבְרָם / av'ram ) !(ms)~ WALK (Verb) ( לֶךְ / lekh ) to~ ~you(ms) ( לְךָ / lê'kha ) from~ LAND ~you(ms) ( מֵאַרְצְךָ / mey'ar'tse'kha ) and~ from~ KINDRED ~you(ms) ( וּמִמּוֹלַדְתְּךָ / u'mi'mo'lad'te'kha ) and~ from~ HOUSE ( וּמִבֵּית / u'mi'beyt ) FATHER ~you(ms) ( אָבִיךָ / a'vi'kha ) TO ( אֶל / el ) the~ LAND ( הָאָרֶץ / ha'a'rets ) WHICH ( אֲשֶׁר / a'sher ) I~ will~ make~ SEE (Verb) ~you(ms) ( אַרְאֶךָּ / ar'e'ka )

and YHWH said to Avram, walk yourself from your land and from your kindred and from the house of your father to the land which I will show you,

2 and~ I~ will~ DO (Verb) ~you(ms) ( וְאֶעֶשְׂךָ / wê'e'es'kha ) to~ NATION ( לְגוֹי / lê'goy ) GREAT ( גָּדוֹל / ga'dol ) and~ I~ will~ much~ KNEEL (Verb) ~you(ms) ( וַאֲבָרֶכְךָ / wa'a'va'rekh'kha ) and~ I~ will~ MAGNIFY (Verb) ( וַאֲגַדְּלָה / wa'a'gad'lah ) TITLE ~you(ms) ( שְׁמֶךָ / shê'me'kha ) and~ !(ms)~ EXIST (Verb) ( וֶהְיֵה / weh'yey ) PRESENT ( בְּרָכָה / bê'ra'khah )

and I will make you a great nation and I will exalt you and I will magnify your title and exist as a present,

3 and~ I~ will~ much~ KNEEL (Verb) ~& ( וַאֲבָרְכָה / wa'a'var'khah ) much~ KNEEL (Verb) ~ing(mp) ~you(ms) ( מְבָרְכֶיךָ / mê'var'khey'kha ) and~ much~ BELITTLE (Verb) ~ing(mp) ~you(ms) ( וּמְקַלֶּלְךָ / um'qa'lel'kha ) I~ will~ SPIT.UPON (Verb) ( אָאֹר / a'or ) and~ they~ did~ be~ KNEEL (Verb) ( וְנִבְרְכוּ / wê'niv're'khu ) in~ ~you(ms) ( בְךָ / vê'kha ) ALL ( כֹּל / kol ) CLAN ~s ( מִשְׁפְּחֹת / mish'pe'hhot ) the~ GROUND ( הָאֲדָמָה / ha'a'da'mah )

and I will exalt ones exalting you and ones belittling you I will spit upon and all of the clans of the ground will be exalted with you,

4 and~ he~ will~ WALK (Verb) ( וַיֵּלֶךְ / wai'yey'lekh ) Avram ( אַבְרָם / av'ram ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) TO ~him ( אֵלָיו / ey'law ) YHWH ( יְהוָה / YHWH ) and~ he~ will~ WALK (Verb) ( וַיֵּלֶךְ / wai'yey'lekh ) AT ~him ( אִתּוֹ / i'to ) Lot ( לוֹט / lot ) and~ Avram ( וְאַבְרָם / wê'av'ram ) SON ( בֶּן / ben ) FIVE ( חָמֵשׁ / hha'meysh ) YEAR ~s ( שָׁנִים / sha'nim ) and~ SEVEN ~s ( וְשִׁבְעִים / wê'shiv'im ) YEAR ( שָׁנָה / sha'nah ) in~ >~ GO.OUT (Verb) ~him ( בְּצֵאתוֹ / bê'tsey'to ) from~ Hharan ( מֵחָרָן / mey'hha'ran )

and Avram walked just as YHWH spoke to him and Lot walked with him and Avram was a son of seventy-five years in his going out from Hharan,

5 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) Avram ( אַבְרָם / av'ram ) AT ( אֶת / et ) Sarai ( שָׂרַי / sa'rai ) WOMAN ~him ( אִשְׁתּוֹ / ish'to ) and~ AT ( וְאֶת / wê'et ) Lot ( לוֹט / lot ) SON ( בֶּן / ben ) BROTHER ~him ( אָחִיו / a'hhiw ) and~ AT ( וְאֶת / wê'et ) ALL ( כָּל / kol ) GOODS ~them(m) ( רְכוּשָׁם / rê'khu'sham ) WHICH ( אֲשֶׁר / a'sher ) they~ did~ ACCUMULATE (Verb) ( רָכָשׁוּ / ra'kha'shu ) and~ AT ( וְאֶת / wê'et ) the~ SOUL ( הַנֶּפֶשׁ / ha'ne'phesh ) WHICH ( אֲשֶׁר / a'sher ) they~ did~ DO (Verb) ( עָשׂוּ / a'su ) in~ Hharan ( בְחָרָן / vê'hha'ran ) and~ they(m)~ will~ GO.OUT (Verb) ( וַיֵּצְאוּ / wai'yeyts'u ) to~ >~ WALK (Verb) ( לָלֶכֶת / la'le'khet ) LAND ~unto ( אַרְצָה / ar'tsah ) Kena'an ( כְּנַעַן / kê'na'an ) and~ they(m)~ will~ COME (Verb) ( וַיָּבֹאוּ / wai'ya'vo'u ) LAND ~unto ( אַרְצָה / ar'tsah ) Kena'an ( כְּנָעַן / kê'na'an )

and Avram took Sarai his woman and Lot the son of his brother and all their goods which they accumulated and the souls which they did in Hharan and they went out to walk unto the land of Kena'an and they came unto the land of Kena'an,

6 and~ he~ will~ CROSS.OVER (Verb) ( וַיַּעֲבֹר / wai'ya'a'vor ) Avram ( אַבְרָם / av'ram ) in~ the~ LAND ( בָּאָרֶץ / ba'a'rets ) UNTIL ( עַד / ad ) AREA ( מְקוֹם / mê'qom ) Shekhem ( שְׁכֶם / shê'khem ) UNTIL ( עַד / ad ) GREAT.TREE ( אֵלוֹן / ey'lon ) Moreh ( מוֹרֶה / mo'reh ) and~ the~ Kena'an ~of ( וְהַכְּנַעֲנִי / wê'hak'na'a'ni ) AT.THAT.TIME ( אָז / az ) in~ the~ LAND ( בָּאָרֶץ / ba'a'rets )

and Avram crossed over in the land, as far as the area of Shekhem, as far as the great tree of Moreh and the one of Kena'an was at that time in the land,

7 and~ he~ will~ be~ SEE (Verb) ( וַיֵּרָא / wai'yey'ra ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Avram ( אַבְרָם / av'ram ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ SEED ~you(ms) ( לְזַרְעֲךָ / lê'zar'a'kha ) I~ will~ GIVE (Verb) ( אֶתֵּן / e'teyn ) AT ( אֶת / et ) the~ LAND ( הָאָרֶץ / ha'a'rets ) the~ THIS ( הַזֹּאת / ha'zot ) and~ he~ will~ BUILD (Verb) ( וַיִּבֶן / wai'yi'ven ) THERE ( שָׁם / sham ) ALTAR ( מִזְבֵּחַ / miz'bey'ahh ) to~ YHWH ( לַיהוָה / la'YHWH ) the~ be~ SEE (Verb) ~ing(ms) ( הַנִּרְאֶה / ha'nir'eh ) TO ~him ( אֵלָיו / ey'law )

and YHWH appeared to Avram and said, to your seed I will give this land, and he built there an altar to YHWH the one appearing to him,

8 and~ he~ will~ make~ ADVANCE (Verb) ( וַיַּעְתֵּק / wai'ya'teyq ) from~ THERE ( מִשָּׁם / mi'sham ) the~ HILL ~unto ( הָהָרָה / ha'ha'rah ) from~ EAST ( מִקֶּדֶם / mi'qe'dem ) to~ Beyt-El ( לְבֵית־אֵל / lê'veyt-eyl ) and~ he~ will~ EXTEND (Verb) ( וַיֵּט / wai'yeyt ) TENT ~her ( אָהֳלֹה / a'ha'loh ) [ 92 ] Beyt-El ( בֵּית־אֵל / beyt-eyl ) from~ SEA ( מִיָּם / mi'yam ) [ 93 ] and~ the~ Ay ( וְהָעַי / wê'ha'ai ) from~ EAST ( מִקֶּדֶם / mi'qe'dem ) and~ he~ will~ BUILD (Verb) ( וַיִּבֶן / wai'yi'ven ) THERE ( שָׁם / sham ) ALTAR ( מִזְבֵּחַ / miz'bey'ahh ) to~ YHWH ( לַיהוָה / la'YHWH ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) in~ TITLE ( בְּשֵׁם / bê'sheym ) YHWH ( יְהוָה / YHWH ) [ 94 ]

and he advanced from there unto the hill, from the east to Beyt-El and he stretched her tent, Beyt-El was from the sea and Ay was from the east and he built there an altar to YHWH and he called out in the title of YHWH,

9 and~ he~ will~ JOURNEY (Verb) ( וַיִּסַּע / wai'yi'sa ) Avram ( אַבְרָם / av'ram ) >~ WALK (Verb) ( הָלוֹךְ / ha'lokh ) and~ >~ JOURNEY (Verb) ( וְנָסוֹעַ / wê'na'so'a ) the~ SOUTH ~unto ( הַנֶּגְבָּה / ha'neg'bah )

and Avram journeyed, walking and journeying unto the south,

10 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) HUNGER ( רָעָב / ra'av ) in~ the~ LAND ( בָּאָרֶץ / ba'a'rets ) and~ he~ will~ GO.DOWN (Verb) ( וַיֵּרֶד / wai'yey'red ) Avram ( אַבְרָם / av'ram ) Mits'rayim ~unto ( מִצְרַיְמָה / mits'rai'mah ) to~ >~ IMMIGRATE (Verb) ( לָגוּר / la'gur ) THERE ( שָׁם / sham ) GIVEN.THAT ( כִּי / ki ) HEAVY ( כָבֵד / kha'veyd ) the~ HUNGER ( הָרָעָב / ha'ra'av ) in~ the~ LAND ( בָּאָרֶץ / ba'a'rets )

and a hunger existed in the land and Avram went down unto Mits'rayim to immigrate there, given that the hunger was heavy in the land,

11 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ make~ COME.NEAR (Verb) ( הִקְרִיב / hiq'riv ) to~ >~ COME (Verb) ( לָבוֹא / la'vo ) Mits'rayim ~unto ( מִצְרָיְמָה / mits'rai'mah ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) TO ( אֶל / el ) Sarai ( שָׂרַי / sa'rai ) WOMAN ~him ( אִשְׁתּוֹ / ish'to ) LOOK ( הִנֵּה / hin'neyh ) PLEASE ( נָא / na ) I~ did~ KNOW (Verb) ( יָדַעְתִּי / ya'da'ti ) GIVEN.THAT ( כִּי / ki ) WOMAN ( אִשָּׁה / i'shah ) BEAUTIFUL ( יְפַת / yê'phat ) APPEARANCE ( מַרְאֶה / mar'eh ) YOU(FS) ( אָתְּ / at )

and it came to pass just as he came near to come unto Mits'rayim and he said to Sarai his woman, please look, I know that you are a woman of beautiful appearance,

12 and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) GIVEN.THAT ( כִּי / ki ) they(m)~ will~ SEE (Verb) ( יִרְאוּ / yir'u ) AT ~you(fs) ( אֹתָךְ / o'takh ) the~ Mits'rayim ~s ( הַמִּצְרִים / ha'mits'rim ) and~ they~ did~ SAY (Verb) ( וְאָמְרוּ / wê'am'ru ) WOMAN ~him ( אִשְׁתּוֹ / ish'to ) THIS ( זֹאת / zot ) and~ they~ did~ KILL (Verb) ( וְהָרְגוּ / wê'har'gu ) AT ~me ( אֹתִי / o'ti ) and~ AT ~you(fs) ( וְאֹתָךְ / wê'o'takh ) they(m)~ will~ much~ LIVE (Verb) ( יְחַיּוּ / yê'hhai'yu )

and it will come to pass that the ones of Mits'rayim will see you and they will say this is his woman and they will kill me and they will keep you alive.

13 !(fs)~ SAY (Verb) ( אִמְרִי / im'ri ) PLEASE ( נָא / na ) SISTER ~me ( אֲחֹתִי / a'hho'ti ) YOU(FS) ( אָתְּ / at ) to~ THAT ( לְמַעַן / lê'ma'an ) he~ will~ DO.WELL (Verb) ( יִיטַב / yi'tav ) to~ ~me ( לִי / li ) in~ the~ CROSS.OVER (Verb) ~ed(ms) ~you(fs) ( בַעֲבוּרֵךְ / va'a'vu'reykh ) and~ she~ did~ LIVE (Verb) ( וְחָיְתָה / wê'hhai'tah ) SOUL ~me ( נַפְשִׁי / naph'shi ) in~ ON.ACCOUNT.OF ~you(fs) ( בִּגְלָלֵךְ / big'la'leykh )

Please say you are my sister so that it will go well for me with your crossing over, and my soul will live on account of you,

14 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) like~ >~ COME (Verb) ( כְּבוֹא / kê'vo ) Avram ( אַבְרָם / av'ram ) Mits'rayim ~unto ( מִצְרָיְמָה / mits'rai'mah ) and~ they(m)~ will~ SEE (Verb) ( וַיִּרְאוּ / wai'yir'u ) the~ Mits'rayim ~s ( הַמִּצְרִים / ha'mits'rim ) AT ( אֶת / et ) the~ WOMAN ( הָאִשָּׁה / ha'i'shah ) GIVEN.THAT ( כִּי / ki ) BEAUTIFUL ( יָפָה / ya'phah ) SHE ( הִוא / hi ) MANY ( מְאֹד / mê'od )

and it came to pass as Avram came unto Mits'rayim and the ones of Mits'rayim saw the woman given that she was very beautiful,

15 and~ they(m)~ will~ SEE (Verb) ( וַיִּרְאוּ / wai'yir'u ) AT ~her ( אֹתָהּ / o'tah ) NOBLE ~s ( שָׂרֵי / sa'rey ) Paroh ( פַרְעֹה / phar'oh ) and~ they(m)~ will~ much~ SHINE (Verb) ( וַיְהַלְלוּ / wa'ye'hal'lu ) AT ~her ( אֹתָהּ / o'tah ) TO ( אֶל / el ) Paroh ( פַּרְעֹה / par'oh ) and~ TAKE (Verb) ~ed(fs) ( וַתֻּקַּח / wa'tu'qahh ) the~ WOMAN ( הָאִשָּׁה / ha'i'shah ) HOUSE ( בֵּית / beyt ) [ 95 ] Paroh ( פַּרְעֹה / par'oh )

and the nobles of Paroh saw her and they endorsed her to Paroh and took the woman to the house of Paroh,

16 and~ to~ Avram ( וּלְאַבְרָם / ul'av'ram ) he~ did~ make~ DO.WELL (Verb) ( הֵיטִיב / hey'tiv ) in~ the~ CROSS.OVER (Verb) ~ed(ms) ~her ( בַּעֲבוּרָהּ / ba'a'vu'rah ) and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) to~ ~him ( לוֹ / lo ) FLOCKS ( צֹאן / tson ) and~ CATTLE ( וּבָקָר / u'va'qar ) and~ DONKEY ~s ( וַחֲמֹרִים / wa'hha'mo'rim ) and~ SERVANT ~s ( וַעֲבָדִים / wa'a'va'dim ) and~ MAID ~s ( וּשְׁפָחֹת / ush'pha'hhot ) and~ SHE.DONKEY ~s ( וַאֲתֹנֹת / wa'a'to'not ) and~ CAMEL ~s ( וּגְמַלִּים / ug'ma'lim )

and to Avram it was made well on account of her, and flocks existed for him and cattle and donkeys and servants and maids and she donkeys and camels,

17 and~ he~ will~ TOUCH (Verb) ( וַיְנַגַּע / wa'ye'na'ga ) YHWH ( יְהוָה / YHWH ) AT ( אֶת / et ) Paroh ( פַּרְעֹה / par'oh ) TOUCH ~s ( נְגָעִים / nê'ga'im ) GREAT ~s ( גְּדֹלִים / gê'do'lim ) and~ AT ( וְאֶת / wê'et ) HOUSE ~him ( בֵּיתוֹ / bey'to ) UPON ( עַל / al ) WORD ( דְּבַר / dê'var ) [ 96 ] Sarai ( שָׂרַי / sa'rai ) WOMAN ( אֵשֶׁת / ey'shet ) Avram ( אַבְרָם / av'ram )

and YHWH touched Paroh and his house with great plagues because of the word of Sarai, the woman of Avram,

18 and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) Paroh ( פַרְעֹה / phar'oh ) to~ Avram ( לְאַבְרָם / lê'av'ram ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) WHAT ( מַה / mah ) THIS ( זֹּאת / zot ) you(ms)~ did~ DO (Verb) ( עָשִׂיתָ / a'si'ta ) to~ ~me ( לִּי / li ) to~ WHAT ( לָמָּה / la'mah ) NOT ( לֹא / lo ) you(ms)~ did~ make~ BE.FACE.TO.FACE (Verb) ( הִגַּדְתָּ / hi'gad'ta ) to~ ~me ( לִּי / li ) GIVEN.THAT ( כִּי / ki ) WOMAN ~you(ms) ( אִשְׁתְּךָ / ish'te'kha ) SHE ( הִוא / hi )

and Paroh called out to Avram and he said, what is this you did to me, why did you not tell to me that she is your woman.

19 to~ WHAT ( לָמָה / la'mah ) you(ms)~ did~ SAY (Verb) ( אָמַרְתָּ / a'mar'ta ) SISTER ~me ( אֲחֹתִי / a'hho'ti ) SHE ( הִוא / hi ) and~ I~ will~ TAKE (Verb) ( וָאֶקַּח / wa'e'qahh ) AT ~her ( אֹתָהּ / o'tah ) to~ ~me ( לִי / li ) to~ WOMAN ( לְאִשָּׁה / lê'i'shah ) and~ NOW ( וְעַתָּה / wê'a'tah ) LOOK ( הִנֵּה / hin'neyh ) WOMAN ~you(ms) ( אִשְׁתְּךָ / ish'te'kha ) !(ms)~ TAKE (Verb) ( קַח / qahh ) and~ !(ms)~ WALK (Verb) ( וָלֵךְ / wa'leykh )

Why did you say she is my sister, and I took her for me for a woman and now look, take your woman and walk,

20 and~ he~ will~ much~ DIRECT (Verb) ( וַיְצַו / wai'tsaw ) UPON ~him ( עָלָיו / a'law ) Paroh ( פַּרְעֹה / par'oh ) MAN ~s ( אֲנָשִׁים / a'na'shim ) and~ they(m)~ will~ much~ SEND (Verb) ( וַיְשַׁלְּחוּ / wa'ye'shal'hhu ) AT ~him ( אֹתוֹ / o'to ) and~ AT ( וְאֶת / wê'et ) WOMAN ~him ( אִשְׁתּוֹ / ish'to ) and~ AT ( וְאֶת / wê'et ) ALL ( כָּל / kol ) WHICH ( אֲשֶׁר / a'sher ) to~ ~him ( לוֹ / lo )

and Paroh directed men concerning him and they sent him and his woman and all which belong to him,
