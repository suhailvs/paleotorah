3 and~ he~ will~ GO.OUT (Verb) ( וְיָצָא / wê'ya'tsa ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) TO ( אֶל / el ) from~ OUTSIDE ( מִחוּץ / mi'hhuts ) to~ the~ CAMP ( לַמַּחֲנֶה / la'ma'hha'neh ) and~ he~ did~ SEE (Verb) ( וְרָאָה / wê'ra'ah ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) he~ did~ be~ HEAL (Verb) ( נִרְפָּא / nir'pa ) TOUCH ( נֶגַע / ne'ga ) the~ INFECTION ( הַצָּרַעַת / ha'tsa'ra'at ) FROM ( מִן / min ) the~ INFECT (Verb) ~ed(ms) ( הַצָּרוּעַ / ha'tsa'ru'a )

4 and~ he~ did~ much~ DIRECT (Verb) ( וְצִוָּה / wê'tsi'wah ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) and~ he~ did~ TAKE (Verb) ( וְלָקַח / wê'la'qahh ) to~ the~ make~ BE.CLEAN (Verb) ~ing(ms) ( לַמִּטַּהֵר / la'mi'ta'heyr ) TWO ( שְׁתֵּי / shê'tey ) BIRD ~s ( צִפֳּרִים / tsi'pa'rim ) LIVING ~s ( חַיּוֹת / hhai'yot ) CLEAN ~s ( טְהֹרוֹת / tê'ho'rot ) and~ TREE ( וְעֵץ / wê'eyts ) CEDAR ( אֶרֶז / e'rez ) and~ SCARLET ( וּשְׁנִי / ush'ni ) KERMES ( תוֹלַעַת / to'al'at ) and~ HYSSOP ( וְאֵזֹב / wê'ey'zov )

and the administrator will direct, and the one being clean will take two living clean birds and cedar wood and scarlet of kermes and hyssop,

5 and~ he~ did~ much~ DIRECT (Verb) ( וְצִוָּה / wê'tsi'wah ) [ 701 ] the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) and~ he~ did~ SLAY (Verb) ( וְשָׁחַט / wê'sha'hhat ) AT ( אֶת / et ) the~ BIRD ( הַצִּפּוֹר / ha'tsi'por ) the~ UNIT ( הָאֶחָת / ha'e'hhat ) TO ( אֶל / el ) UTENSIL ( כְּלִי / kê'li ) CLAY ( חֶרֶשׂ / hhe'res ) [ 702 ] UPON ( עַל / al ) WATER ~s2 ( מַיִם / ma'yim ) LIVING ~s ( חַיִּים / hhai'yim ) [ 703 ]

He will take the living bird and the cedar wood and the scarlet kermes and the hyssop and he will dip them and the living bird in blood of the slain bird upon the living waters,

7 and~ he~ did~ make~ SPATTER (Verb) ( וְהִזָּה / wê'hi'zah ) UPON ( עַל / al ) to~ the~ make~ BE.CLEAN (Verb) ~ing(ms) ( הַמִּטַּהֵר / ha'mi'ta'heyr ) FROM ( מִן / min ) the~ INFECTION ( הַצָּרַעַת / ha'tsa'ra'at ) SEVEN ( שֶׁבַע / she'va ) FOOTSTEP ~s ( פְּעָמִים / pê'a'mim ) and~ he~ did~ much~ BE.CLEAN (Verb) ~him ( וְטִהֲרוֹ / wê'ti'ha'ro ) and~ he~ did~ much~ SEND (Verb) ( וְשִׁלַּח / wê'shi'lahh ) AT ( אֶת / et ) the~ BIRD ( הַצִּפֹּר / ha'tsi'por ) the~ LIVING ( הַחַיָּה / ha'hha'yah ) UPON ( עַל / al ) FACE ~s ( פְּנֵי / pê'ney ) the~ FIELD ( הַשָּׂדֶה / ha'sa'deh )

and he will spatter upon the ones being made clean from the infection seven times, then he will declare him clean and he will send the living bird upon the face of the field,

11 and~ he~ did~ make~ STAND (Verb) ( וְהֶעֱמִיד / wê'he'e'mid ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) the~ much~ BE.CLEAN (Verb) ~ing(ms) ( הַמְטַהֵר / ham'ta'heyr ) AT ( אֵת / eyt ) the~ MAN ( הָאִישׁ / ha'ish ) to~ the~ make~ BE.CLEAN (Verb) ~ing(ms) ( הַמִּטַּהֵר / ha'mi'ta'heyr ) and~ AT ~them(m) ( וְאֹתָם / wê'o'tam ) to~ FACE ~s ( לִפְנֵי / liph'ney ) YHWH ( יְהוָה / YHWH ) OPENING ( פֶּתַח / pe'tahh ) TENT ( אֹהֶל / o'hel ) APPOINTED ( מוֹעֵד / mo'eyd )

and the administrator will take the one sheep and he will make him come near for guilt, and the log of oil, and he will wave them as a waving to the face of YHWH,
