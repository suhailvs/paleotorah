Numbers Chapter 30

1 (40) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Mosheh ( מֹשֶׁה / mo'sheh ) TO ( אֶל / el ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) like~ ALL ( כְּכֹל / kê'khol ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ DIRECT (Verb) ( צִוָּה / tsi'wah ) YHWH ( יְהוָה / YHWH ) AT ( אֶת / et ) Mosheh ( מֹשֶׁה / mo'sheh )

and Mosheh said to the sons of Yisra'eyl, according to all which YHWH directed Mosheh,

2 (1) and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) Mosheh ( מֹשֶׁה / mo'sheh ) TO ( אֶל / el ) HEAD ~s ( רָאשֵׁי / ra'shey ) the~ BRANCH ( הַמַּטּוֹת / ha'ma'tot ) to~ SON ~s ( לִבְנֵי / liv'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) THIS ( זֶה / zeh ) the~ WORD ( הַדָּבָר / ha'da'var ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ DIRECT (Verb) ( צִוָּה / tsi'wah ) YHWH ( יְהוָה / YHWH )

and Mosheh spoke to the heads of the branches of the sons of Yisra'eyl saying, this is the word which YHWH directed.

3 (2) MAN ( אִישׁ / ish ) GIVEN.THAT ( כִּי / ki ) he~ will~ MAKE.A.VOW (Verb) ( יִדֹּר / yi'dor ) VOW ( נֶדֶר / ne'der ) to~ YHWH ( לַיהוָה / la'YHWH ) OR ( אוֹ / o ) >~ be~ SWEAR (Verb) ( הִשָּׁבַע / hi'sha'va ) SWEARING ( שְׁבֻעָה / shê'vu'ah ) to~ >~ TIE.UP (Verb) ( לֶאְסֹר / le'sor ) BOND ( אִסָּר / i'sar ) UPON ( עַל / al ) SOUL ~him ( נַפְשׁוֹ / naph'sho ) NOT ( לֹא / lo ) he~ will~ make~ DRILL (Verb) ( יַחֵל / ya'hheyl ) WORD ~him ( דְּבָרוֹ / dê'va'ro ) like~ ALL ( כְּכָל / kê'khol ) the~ GO.OUT (Verb) ~ing(ms) ( הַיֹּצֵא / hai'yo'tsey ) from~ MOUTH ~him ( מִפִּיו / mi'piw ) he~ will~ DO (Verb) ( יַעֲשֶׂה / ya'a'seh )

A man that will make a vow to YHWH, or swear a swearing, to tie up a bond upon his soul, he will not defile his word, according to all the goings out from his mouth he will do,

4 (3) and~ WOMAN ( וְאִשָּׁה / wê'i'shah ) GIVEN.THAT ( כִּי / ki ) you(ms)~ will~ MAKE.A.VOW (Verb) ( תִדֹּר / ti'dor ) VOW ( נֶדֶר / ne'der ) to~ YHWH ( לַיהוָה / la'YHWH ) and~ she~ did~ TIE.UP (Verb) ( וְאָסְרָה / wê'as'rah ) BOND ( אִסָּר / i'sar ) in~ HOUSE ( בְּבֵית / bê'veyt ) FATHER ~her ( אָבִיהָ / a'vi'ah ) in~ YOUNG.AGE ~her ( בִּנְעֻרֶיהָ / bin'u'rey'ah )

and a woman that will make a vow to YHWH, and she will tie up a bond in the house of her father in her young age,

5 (4) and~ he~ did~ HEAR (Verb) ( וְשָׁמַע / wê'sha'ma ) FATHER ~her ( אָבִיהָ / a'vi'ah ) AT ( אֶת / et ) VOW ~her ( נִדְרָהּ / nid'rah ) and~ BOND ~her ( וֶאֱסָרָהּ / we'e'sa'rah ) WHICH ( אֲשֶׁר / a'sher ) she~ did~ TIE.UP (Verb) ( אָסְרָה / as'rah ) UPON ( עַל / al ) SOUL ~her ( נַפְשָׁהּ / naph'shah ) and~ he~ did~ make~ KEEP.SILENT (Verb) ( וְהֶחֱרִישׁ / wê'he'hhe'rish ) to~ ~her ( לָהּ / lah ) FATHER ~her ( אָבִיהָ / a'vi'ah ) and~ they~ did~ RISE (Verb) ( וְקָמוּ / wê'qa'mu ) ALL ( כָּל / kol ) VOW ~s ~her ( נְדָרֶיהָ / nê'da'rey'ah ) and~ ALL ( וְכָל / wê'khol ) BOND ( אִסָּר / i'sar ) WHICH ( אֲשֶׁר / a'sher ) she~ did~ TIE.UP (Verb) ( אָסְרָה / as'rah ) UPON ( עַל / al ) SOUL ~her ( נַפְשָׁהּ / naph'shah ) he~ will~ RISE (Verb) ( יָקוּם / ya'qum )

and her father will hear her vow and her bond which she tied up upon her soul, and her father will make her keep silent, and all her vows will rise, and every bond which she tied up upon her soul will rise,

6 (5) and~ IF ( וְאִם / wê'im ) he~ did~ make~ FORBID (Verb) ( הֵנִיא / hey'ni ) FATHER ~her ( אָבִיהָ / a'vi'ah ) AT ~her ( אֹתָהּ / o'tah ) in~ DAY ( בְּיוֹם / bê'yom ) >~ HEAR (Verb) ~him ( שָׁמְעוֹ / sham'o ) ALL ( כָּל / kol ) VOW ~s ~her ( נְדָרֶיהָ / nê'da'rey'ah ) and~ BOND ~s ~her ( וֶאֱסָרֶיהָ / we'e'sa'rey'ah ) WHICH ( אֲשֶׁר / a'sher ) she~ did~ TIE.UP (Verb) ( אָסְרָה / as'rah ) UPON ( עַל / al ) SOUL ~her ( נַפְשָׁהּ / naph'shah ) NOT ( לֹא / lo ) he~ will~ RISE (Verb) ( יָקוּם / ya'qum ) and~ YHWH ( וַיהוָה / wa'YHWH ) he~ will~ FORGIVE (Verb) ( יִסְלַח / yis'lahh ) to~ ~her ( לָהּ / lah ) GIVEN.THAT ( כִּי / ki ) he~ did~ make~ FORBID (Verb) ( הֵנִיא / hey'ni ) FATHER ~her ( אָבִיהָ / a'vi'ah ) AT ~her ( אֹתָהּ / o'tah )

and if her father forbid her in the day of his hearing, all her vows and her bonds which she tied up upon her soul will not rise, and YHWH will forgive her given that her father forbid her,

7 (6) and~ IF ( וְאִם / wê'im ) >~ EXIST (Verb) ( הָיוֹ / hai'o ) she~ will~ EXIST (Verb) ( תִהְיֶה / tih'yeh ) to~ MAN ( לְאִישׁ / lê'ish ) and~ VOW ~s ~her ( וּנְדָרֶיהָ / un'da'rey'ah ) UPON ~her ( עָלֶיהָ / a'ley'ah ) OR ( אוֹ / o ) UTTERANCE ( מִבְטָא / miv'ta ) LIP ~s2 ~her ( שְׂפָתֶיהָ / sê'pha'tey'ah ) WHICH ( אֲשֶׁר / a'sher ) she~ did~ TIE.UP (Verb) ( אָסְרָה / as'rah ) UPON ( עַל / al ) SOUL ~her ( נַפְשָׁהּ / naph'shah )

and if she will exist for a man, and her vows upon her or the utterance of her lips which she tied up upon her soul,

8 (7) and~ he~ did~ HEAR (Verb) ( וְשָׁמַע / wê'sha'ma ) MAN ~her ( אִישָׁהּ / i'shah ) in~ DAY ( בְּיוֹם / bê'yom ) >~ HEAR (Verb) ~him ( שָׁמְעוֹ / sham'o ) and~ he~ did~ make~ KEEP.SILENT (Verb) ( וְהֶחֱרִישׁ / wê'he'hhe'rish ) to~ ~her ( לָהּ / lah ) and~ they~ did~ RISE (Verb) ( וְקָמוּ / wê'qa'mu ) VOW ~s ~her ( נְדָרֶיהָ / nê'da'rey'ah ) and~ BOND ~her ( וֶאֱסָרֶהָ / we'e'sa're'ah ) WHICH ( אֲשֶׁר / a'sher ) she~ did~ TIE.UP (Verb) ( אָסְרָה / as'rah ) UPON ( עַל / al ) SOUL ~her ( נַפְשָׁהּ / naph'shah ) they(m)~ will~ RISE (Verb) ( יָקֻמוּ / ya'qu'mu )

and her man heard in the day of his hearing, and he will make her keep silent, then her vows will rise, and her bond which she tied up upon her soul will rise,

9 (8) and~ IF ( וְאִם / wê'im ) in~ DAY ( בְּיוֹם / bê'yom ) >~ HEAR (Verb) ( שְׁמֹעַ / shê'mo'a ) MAN ~her ( אִישָׁהּ / i'shah ) he~ will~ make~ FORBID (Verb) ( יָנִיא / ya'ni ) SIGN ~her ( אוֹתָהּ / otah ) and~ he~ did~ make~ BREAK (Verb) ( וְהֵפֵר / wê'hey'pheyr ) AT ( אֶת / et ) VOW ~her ( נִדְרָהּ / nid'rah ) WHICH ( אֲשֶׁר / a'sher ) UPON ~her ( עָלֶיהָ / a'ley'ah ) and~ AT ( וְאֵת / wê'eyt ) UTTERANCE ( מִבְטָא / miv'ta ) LIP ~s2 ~her ( שְׂפָתֶיהָ / sê'pha'tey'ah ) WHICH ( אֲשֶׁר / a'sher ) she~ did~ TIE.UP (Verb) ( אָסְרָה / as'rah ) UPON ( עַל / al ) SOUL ~her ( נַפְשָׁהּ / naph'shah ) and~ YHWH ( וַיהוָה / wa'YHWH ) he~ will~ FORGIVE (Verb) ( יִסְלַח / yis'lahh ) to~ ~her ( לָהּ / lah )

and if in the day her man hears, he will forbid her sign, and he will break her vow which was upon her, and the utterance of her lips which she tied up upon her soul, then YHWH will forgive her,

10 (9) and~ VOW ( וְנֵדֶר / wê'ney'der ) WIDOW ( אַלְמָנָה / al'ma'nah ) and~ CAST.OUT (Verb) ~ed(fs) ( וּגְרוּשָׁה / ug'ru'shah ) ALL ( כֹּל / kol ) WHICH ( אֲשֶׁר / a'sher ) she~ did~ TIE.UP (Verb) ( אָסְרָה / as'rah ) UPON ( עַל / al ) SOUL ~her ( נַפְשָׁהּ / naph'shah ) he~ will~ RISE (Verb) ( יָקוּם / ya'qum ) UPON ~her ( עָלֶיהָ / a'ley'ah )

and the vow of a widow and the casted out ones, all which she tied up upon her soul will rise upon her,

11 (10) and~ IF ( וְאִם / wê'im ) HOUSE ( בֵּית / beyt ) [ 898 ] MAN ~her ( אִישָׁהּ / i'shah ) she~ did~ MAKE.A.VOW (Verb) ( נָדָרָה / na'da'rah ) OR ( אוֹ / o ) she~ did~ TIE.UP (Verb) ( אָסְרָה / as'rah ) BOND ( אִסָּר / i'sar ) UPON ( עַל / al ) SOUL ~her ( נַפְשָׁהּ / naph'shah ) in~ SWEARING ( בִּשְׁבֻעָה / bish'vu'ah )

and if the house of her man, she made a vow, or she tied up a bond upon her soul in swearing.,

12 (11) and~ he~ did~ HEAR (Verb) ( וְשָׁמַע / wê'sha'ma ) MAN ~her ( אִישָׁהּ / i'shah ) and~ he~ did~ make~ KEEP.SILENT (Verb) ( וְהֶחֱרִשׁ / wê'he'hhe'rish ) to~ ~her ( לָהּ / lah ) NOT ( לֹא / lo ) he~ did~ make~ FORBID (Verb) ( הֵנִיא / hey'ni ) AT ~her ( אֹתָהּ / o'tah ) and~ they~ did~ RISE (Verb) ( וְקָמוּ / wê'qa'mu ) ALL ( כָּל / kol ) VOW ~s ~her ( נְדָרֶיהָ / nê'da'rey'ah ) and~ ALL ( וְכָל / wê'khol ) BOND ( אִסָּר / i'sar ) WHICH ( אֲשֶׁר / a'sher ) she~ did~ TIE.UP (Verb) ( אָסְרָה / as'rah ) UPON ( עַל / al ) SOUL ~her ( נַפְשָׁהּ / naph'shah ) he~ will~ RISE (Verb) ( יָקוּם / ya'qum )

and her man heard, and he kept silent at her, he did not forbid her, and all her vows will rise, and every bond which she tied up upon her soul will rise.,

13 (12) and~ IF ( וְאִם / wê'im ) >~ make~ BREAK (Verb) ( הָפֵר / ha'pheyr ) he~ will~ make~ BREAK (Verb) ( יָפֵר / ya'pheyr ) AT ~them(m) ( אֹתָם / o'tam ) MAN ~her ( אִישָׁהּ / i'shah ) in~ DAY ( בְּיוֹם / bê'yom ) >~ HEAR (Verb) ~him ( שָׁמְעוֹ / sham'o ) ALL ( כָּל / kol ) GOING.OUT ( מוֹצָא / mo'tsa ) LIP ~s2 ~her ( שְׂפָתֶיהָ / sê'pha'tey'ah ) to~ VOW ~s ~her ( לִנְדָרֶיהָ / lin'da'rey'ah ) and~ to~ BOND ( וּלְאִסַּר / ul'i'sar ) SOUL ~her ( נַפְשָׁהּ / naph'shah ) NOT ( לֹא / lo ) he~ will~ RISE (Verb) ( יָקוּם / ya'qum ) MAN ~her ( אִישָׁהּ / i'shah ) he~ did~ make~ BREAK (Verb) ~them(m) ( הֲפֵרָם / ha'phey'ram ) and~ YHWH ( וַיהוָה / wa'YHWH ) he~ will~ FORGIVE (Verb) ( יִסְלַח / yis'lahh ) to~ ~her ( לָהּ / lah )

and if her man will certainly make them broken in the day of his hearing, everything going out of her lips concerning her vow and to the bond of her soul will not rise, and her man made them broken, and YHWH will forgive her.

14 (13) ALL ( כָּל / kol ) VOW ( נֵדֶר / ney'der ) and~ ALL ( וְכָל / wê'khol ) SWEARING ( שְׁבֻעַת / shê'vu'at ) BOND ( אִסָּר / i'sar ) to~ >~ much~ AFFLICT (Verb) ( לְעַנֹּת / lê'a'not ) SOUL ( נָפֶשׁ / na'phesh ) MAN ~her ( אִישָׁהּ / i'shah ) he~ will~ make~ RISE (Verb) ~him ( יְקִימֶנּוּ / yê'qi'me'nu ) and~ MAN ~her ( וְאִישָׁהּ / wê'i'shah ) he~ will~ make~ BREAK (Verb) ~him ( יְפֵרֶנּוּ / yê'phey're'nu ) [ 899 ]

Every vow and every swearing of a bond to afflict the soul, her man will make him rise or her man will make him broken,

15 (14) and~ IF ( וְאִם / wê'im ) >~ make~ KEEP.SILENT (Verb) ( הַחֲרֵשׁ / ha'hha'reysh ) he~ will~ make~ KEEP.SILENT (Verb) ( יַחֲרִישׁ / ya'hha'rish ) to~ ~her ( לָהּ / lah ) MAN ~her ( אִישָׁהּ / i'shah ) from~ DAY ( מִיּוֹם / mi'yom ) TO ( אֶל / el ) DAY ( יוֹם / yom ) and~ he~ did~ make~ RISE (Verb) ( וְהֵקִים / wê'hey'qim ) AT ( אֶת / et ) ALL ( כָּל / kol ) VOW ~s ~her ( נְדָרֶיהָ / nê'da'rey'ah ) OR ( אוֹ / o ) AT ( אֶת / et ) ALL ( כָּל / kol ) BOND ~s ~her ( אֱסָרֶיהָ / e'sa'rey'ah ) WHICH ( אֲשֶׁר / a'sher ) UPON ~her ( עָלֶיהָ / a'ley'ah ) he~ did~ make~ RISE (Verb) ( הֵקִים / hey'qim ) AT ~them(m) ( אֹתָם / o'tam ) GIVEN.THAT ( כִּי / ki ) he~ did~ make~ KEEP.SILENT (Verb) ( הֶחֱרִשׁ / he'hhe'rish ) to~ ~her ( לָהּ / lah ) in~ DAY ( בְּיוֹם / bê'yom ) >~ HEAR (Verb) ~him ( שָׁמְעוֹ / sham'o )

but if her man will certainly keep silent at her, from day to day, then he will make all of her vows rise, or all of her bonds which are upon her, he will make them rise, given that he kept silent at her in the day of his hearing,

16 (15) and~ IF ( וְאִם / wê'im ) >~ make~ BREAK (Verb) ( הָפֵר / ha'pheyr ) he~ will~ make~ BREAK (Verb) ( יָפֵר / ya'pheyr ) AT ~them(m) ( אֹתָם / o'tam ) AFTER ( אַחֲרֵי / a'hha'rey ) >~ HEAR (Verb) ~him ( שָׁמְעוֹ / sham'o ) and~ he~ did~ LIFT.UP (Verb) ( וְנָשָׂא / wê'na'sa ) AT ( אֶת / et ) TWISTEDNESS ~her ( עֲוֹנָהּ / a'o'nah )

but if he will make them broken after his hearing, then he will lift up her twistedness.

17 (16) THESE ( אֵלֶּה / ey'leh ) the~ CUSTOM ~s ( הַחֻקִּים / ha'hhu'qim ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ DIRECT (Verb) ( צִוָּה / tsi'wah ) YHWH ( יְהוָה / YHWH ) AT ( אֶת / et ) Mosheh ( מֹשֶׁה / mo'sheh ) BETWEEN ( בֵּין / beyn ) MAN ( אִישׁ / ish ) to~ WOMAN ~him ( לְאִשְׁתּוֹ / lê'ish'to ) BETWEEN ( בֵּין / beyn ) FATHER ( אָב / av ) to~ DAUGHTER ~him ( לְבִתּוֹ / lê'vi'to ) in~ YOUNG.AGE ~her ( בִּנְעֻרֶיהָ / bin'u'rey'ah ) HOUSE ( בֵּית / beyt ) [ 900 ] FATHER ~her ( אָבִיהָ / a'vi'ah )
