and to Mosheh he said, go up to YHWH, you and Aharon, Nadav and Aviyhu, and seventy from the bearded ones of Yisra'eyl, and you will bend yourself down at a distance,

2 and~ he~ did~ be~ DRAW.NEAR (Verb) ( וְנִגַּשׁ / wê'ni'gash ) Mosheh ( מֹשֶׁה / mo'sheh ) to~ STRAND ~him ( לְבַדּוֹ / lê'va'do ) TO ( אֶל / el ) YHWH ( יְהוָה / YHWH ) and~ THEY(m) ( וְהֵם / wê'heym ) NOT ( לֹא / lo ) they(m)~ will~ DRAW.NEAR (Verb) ( יִגָּשׁוּ / yi'ga'shu ) and~ the~ PEOPLE ( וְהָעָם / wê'ha'am ) NOT ( לֹא / lo ) they(m)~ will~ GO.UP (Verb) ( יַעֲלוּ / ya'a'lu ) WITH ~him ( עִמּוֹ / i'mo )

3 and~ he~ will~ COME (Verb) ( וַיָּבֹא / wai'ya'vo ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ he~ will~ much~ COUNT (Verb) ( וַיְסַפֵּר / wai'sa'peyr ) to~ the~ PEOPLE ( לָעָם / la'am ) AT ( אֵת / eyt ) ALL ( כָּל / kol ) WORD ~s ( דִּבְרֵי / div'rey ) YHWH ( יְהוָה / YHWH ) and~ AT ( וְאֵת / wê'eyt ) ALL ( כָּל / kol ) the~ DECISION ~s ( הַמִּשְׁפָּטִים / ha'mish'pa'tim ) and~ he~ will~ ANSWER (Verb) ( וַיַּעַן / wai'ya'an ) ALL ( כָּל / kol ) the~ PEOPLE ( הָעָם / ha'am ) VOICE ( קוֹל / qol ) UNIT ( אֶחָד / e'hhad ) and~ they(m)~ will~ SAY (Verb) ( וַיֹּאמְרוּ / wai'yom'ru ) ALL ( כָּל / kol ) the~ WORD ~s ( הַדְּבָרִים / ha'de'va'rim ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) YHWH ( יְהוָה / YHWH ) we~ will~ DO (Verb) ( נַעֲשֶׂה / na'a'seh )

and Mosheh came and he recounted to the people all the words of YHWH and all the decisions, and all the people of one voice answered, and they said, all the words which YHWH spoke, we will do,

4 and~ he~ will~ WRITE (Verb) ( וַיִּכְתֹּב / wai'yikh'tov ) Mosheh ( מֹשֶׁה / mo'sheh ) AT ( אֵת / eyt ) ALL ( כָּל / kol ) WORD ~s ( דִּבְרֵי / div'rey ) YHWH ( יְהוָה / YHWH ) and~ he~ will~ make~ DEPART.EARLY (Verb) ( וַיַּשְׁכֵּם / wai'yash'keym ) in~ the~ MORNING ( בַּבֹּקֶר / ba'bo'qer ) and~ he~ will~ BUILD (Verb) ( וַיִּבֶן / wai'yi'ven ) ALTAR ( מִזְבֵּחַ / miz'bey'ahh ) UNDER ( תַּחַת / ta'hhat ) [ 439 ] the~ HILL ( הָהָר / ha'har ) and~ TWO ( וּשְׁתֵּים / ush'teym ) TEN ( עֶשְׂרֵה / es'reyh ) MONUMENT ( מַצֵּבָה / ma'tsey'vah ) to~ TWO ( לִשְׁנֵים / lish'neym ) TEN ( עָשָׂר / a'sar ) STAFF ~s ( שִׁבְטֵי / shiv'tey ) [ 440 ] Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl )

5 and~ he~ will~ SEND (Verb) ( וַיִּשְׁלַח / wai'yish'lahh ) AT ( אֶת / et ) YOUNG.MAN ~s ( נַעֲרֵי / na'a'rey ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) and~ they(m)~ will~ make~ GO.UP (Verb) ( וַיַּעֲלוּ / wai'ya'a'lu ) ASCENSION.OFFERING ~s ( עֹלֹת / o'lot ) and~ they(m)~ will~ SACRIFICE (Verb) ( וַיִּזְבְּחוּ / wai'yiz'be'hhu ) SACRIFICE ~s ( זְבָחִים / zê'va'hhim ) OFFERING.OF.RESTITUTION ~s ( שְׁלָמִים / shê'la'mim ) to~ YHWH ( לַיהוָה / la'YHWH ) BULL ~s ( פָּרִים / pa'rim )

and he sent the young men of the sons of Yisra'eyl, and they made ascension offerings go up, and they sacrificed sacrifices, offerings of restitutions of bulls to YHWH,

6 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) Mosheh ( מֹשֶׁה / mo'sheh ) HALF ( חֲצִי / hha'tsi ) the~ BLOOD ( הַדָּם / ha'dam ) and~ he~ will~ PLACE (Verb) ( וַיָּשֶׂם / wai'ya'sem ) in~ the~ GOBLET ~s ( בָּאַגָּנֹת / ba'a'ga'not ) and~ HALF ( וַחֲצִי / wa'hha'tsi ) the~ BLOOD ( הַדָּם / ha'dam ) he~ did~ SPRINKLE (Verb) ( זָרַק / za'raq ) UPON ( עַל / al ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh )

and Mosheh took half of the blood and he placed it in the goblets, and half of the blood he sprinkled upon the altar,

7 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) SCROLL ( סֵפֶר / sey'pher ) the~ COVENANT ( הַבְּרִית / ha'be'rit ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) [ 441 ] in~ EAR ~s2 ( בְּאָזְנֵי / bê'az'ney ) the~ PEOPLE ( הָעָם / ha'am ) and~ they(m)~ will~ SAY (Verb) ( וַיֹּאמְרוּ / wai'yom'ru ) ALL ( כֹּל / kol ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ SPEAK (Verb) ( דִּבֶּר / di'ber ) YHWH ( יְהוָה / YHWH ) we~ will~ DO (Verb) ( נַעֲשֶׂה / na'a'seh ) and~ we~ will~ HEAR (Verb) ( וְנִשְׁמָע / wê'nish'ma )

and he took the scroll of the covenant and he called it out in the ears of the people, and they said, all which YHWH spoke, we will do and we will hear,

8 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) Mosheh ( מֹשֶׁה / mo'sheh ) AT ( אֶת / et ) the~ BLOOD ( הַדָּם / ha'dam ) and~ he~ will~ SPRINKLE (Verb) ( וַיִּזְרֹק / wai'yiz'roq ) UPON ( עַל / al ) the~ PEOPLE ( הָעָם / ha'am ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) LOOK ( הִנֵּה / hin'neyh ) BLOOD ( דַם / dam ) the~ COVENANT ( הַבְּרִית / ha'be'rit ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ CUT (Verb) ( כָּרַת / ka'rat ) YHWH ( יְהוָה / YHWH ) WITH ~you(mp) ( עִמָּכֶם / i'ma'khem ) UPON ( עַל / al ) ALL ( כָּל / kol ) the~ WORD ~s ( הַדְּבָרִים / ha'de'va'rim ) the~ THESE ( הָאֵלֶּה / ha'ey'leh )

and Mosheh took the blood and he sprinkled it upon the people, and he said, look, the blood of the covenant, which YHWH cut with you concerning all these words,

9 and~ he~ will~ make~ GO.UP (Verb) ( וַיַּעַל / wai'ya'al ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ Aharon ( וְאַהֲרֹן / wê'a'ha'ron ) Nadav ( נָדָב / na'dav ) and~ Aviyhu ( וַאֲבִיהוּא / wa'a'vi'hu ) and~ SEVEN ~s ( וְשִׁבְעִים / wê'shiv'im ) from~ BEARD ~s ( מִזִּקְנֵי / mi'ziq'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl )

and Mosheh went up, and Aharon, Nadav and Aviyhu, and seventy from the bearded ones of Yisra'eyl,

and they saw Elohiym of Yisra'eyl, and under his feet was like a work of brick of the lapis-lazuli, and like a bone of the skies for cleanliness,

11 and~ TO ( וְאֶל / wê'el ) LEADER ~s ( אֲצִילֵי / a'tsi'ley ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) NOT ( לֹא / lo ) he~ did~ SEND (Verb) ( שָׁלַח / sha'lahh ) HAND ~him ( יָדוֹ / ya'do ) and~ they(m)~ will~ PERCEIVE (Verb) ( וַיֶּחֱזוּ / wai'ye'hhe'zu ) AT ( אֶת / et ) the~ Elohiym ( הָאֱלֹהִים / ha'e'lo'him ) and~ they(m)~ will~ EAT (Verb) ( וַיֹּאכְלוּ / wai'yokh'lu ) and~ they(m)~ will~ GULP (Verb) ( וַיִּשְׁתּוּ / wai'yish'tu )

and to the leaders of the sons of Yisra'eyl he did not send his hand, and they perceived the Elohiym, and they ate and they gulped,

12 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) !(ms)~ GO.UP (Verb) ( עֲלֵה / a'leyh ) TO ~me ( אֵלַי / ey'lai ) the~ HILL ~unto ( הָהָרָה / ha'ha'rah ) and~ !(ms)~ EXIST (Verb) ( וֶהְיֵה / weh'yey ) THERE ( שָׁם / sham ) and~ I~ will~ GIVE (Verb) ~& ( וְאֶתְּנָה / wê'et'nah ) to~ ~you(ms) ( לְךָ / lê'kha ) AT ( אֶת / et ) SLAB ~s ( לֻחֹת / lu'hhot ) the~ STONE ( הָאֶבֶן / ha'e'ven ) and~ the~ TEACHING ( וְהַתּוֹרָה / wê'ha'to'rah ) and~ the~ DIRECTIVE ( וְהַמִּצְוָה / wê'ha'mits'wah ) WHICH ( אֲשֶׁר / a'sher ) I~ did~ WRITE (Verb) ( כָּתַבְתִּי / ka'tav'ti ) to~ >~ make~ THROW (Verb) ~them(m) ( לְהוֹרֹתָם / lê'ho'ro'tam )

and YHWH said to Mosheh, go up to me unto the hill and exist there, and I will give to you slabs of the stone, and the teaching and the directive which I wrote to teach them,

13 and~ he~ will~ RISE (Verb) ( וַיָּקָם / wai'ya'qam ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ Yehoshu'a ( וִיהוֹשֻׁעַ / wi'ho'shu'a ) much~ MINISTER (Verb) ~ing(ms) ~him ( מְשָׁרְתוֹ / mê'shar'to ) and~ he~ will~ make~ GO.UP (Verb) ( וַיַּעַל / wai'ya'al ) Mosheh ( מֹשֶׁה / mo'sheh ) TO ( אֶל / el ) HILL ( הַר / har ) the~ Elohiym ( הָאֱלֹהִים / ha'e'lo'him )

and Mosheh rose, and Yehoshu'a his minister, and Mosheh went up to the hill of the Elohiym,

14 and~ TO ( וְאֶל / wê'el ) the~ BEARD ~s ( הַזְּקֵנִים / haz'qey'nim ) he~ did~ SAY (Verb) ( אָמַר / a'mar ) !(mp)~ SETTLE (Verb) ( שְׁבוּ / shê'vu ) to~ ~us ( לָנוּ / la'nu ) in~ THIS ( בָזֶה / va'zeh ) UNTIL ( עַד / ad ) WHICH ( אֲשֶׁר / a'sher ) we~ will~ TURN.BACK (Verb) ( נָשׁוּב / na'shuv ) TO ~you(mp) ( אֲלֵיכֶם / a'ley'khem ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) Aharon ( אַהֲרֹן / a'ha'ron ) and~ Hhur ( וְחוּר / wê'hhur ) WITH ~you(mp) ( עִמָּכֶם / i'ma'khem ) WHO ( מִי / mi ) MASTER ( בַעַל / va'al ) WORD ~s ( דְּבָרִים / dê'va'rim ) [ 444 ] he~ will~ DRAW.NEAR (Verb) ( יִגַּשׁ / yi'gash ) TO ~them(m) ( אֲלֵהֶם / a'ley'hem )

and to the bearded ones he said, settle for us here, until we turn back to you, and look, Aharon and Hhur are with you, whoever is a master of words will draw near to them,

15 and~ he~ will~ make~ GO.UP (Verb) ( וַיַּעַל / wai'ya'al ) Mosheh ( מֹשֶׁה / mo'sheh ) TO ( אֶל / el ) the~ HILL ( הָהָר / ha'har ) and~ he~ will~ much~ COVER.OVER (Verb) ( וַיְכַס / wai'khas ) the~ CLOUD ( הֶעָנָן / he'a'nan ) AT ( אֶת / et ) the~ HILL ( הָהָר / ha'har )

and Mosheh went up to the hill, and the cloud covered over the hill,

16 and~ he~ will~ DWELL (Verb) ( וַיִּשְׁכֹּן / wai'yish'kon ) ARMAMENT ( כְּבוֹד / kê'vod ) YHWH ( יְהוָה / YHWH ) UPON ( עַל / al ) HILL ( הַר / har ) Sinai ( סִינַי / si'nai ) and~ he~ will~ much~ COVER.OVER (Verb) ~him ( וַיְכַסֵּהוּ / wai'kha'sey'hu ) the~ CLOUD ( הֶעָנָן / he'a'nan ) SIX ( שֵׁשֶׁת / shey'shet ) DAY ~s ( יָמִים / ya'mim ) and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) in~ the~ DAY ( בַּיּוֹם / ba'yom ) the~ SEVENTH ( הַשְּׁבִיעִי / hash'vi'i ) from~ MIDST ( מִתּוֹךְ / mi'tokh ) the~ CLOUD ( הֶעָנָן / he'a'nan )

17 and~ APPEARANCE ( וּמַרְאֵה / u'mar'eyh ) ARMAMENT ( כְּבוֹד / kê'vod ) YHWH ( יְהוָה / YHWH ) like~ FIRE ( כְּאֵשׁ / kê'eysh ) EAT (Verb) ~ing(fs) ( אֹכֶלֶת / o'khe'let ) in~ HEAD ( בְּרֹאשׁ / bê'rosh ) the~ HILL ( הָהָר / ha'har ) [ 445 ] to~ EYE ~s2 ( לְעֵינֵי / lê'ey'ney ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl )

and the appearance of the armament of YHWH was like a fire eating in the head of the hill to the eyes of the sons of Yisra'eyl,
