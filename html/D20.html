Given that you will go out to battle, upon your attackers, and you will see the horse and vehicle, a people more abundant than you, you will not fear them, given that YHWH your Elohiym is with you, bringing you up from the land of Mits'rayim,

2 and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) like~ >~ COME.NEAR (Verb) ~you(mp) ( כְּקָרָבְכֶם / kê'qa'rav'khem ) TO ( אֶל / el ) the~ BATTLE ( הַמִּלְחָמָה / ha'mil'hha'mah ) and~ he~ did~ be~ DRAW.NEAR (Verb) ( וְנִגַּשׁ / wê'ni'gash ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) and~ he~ did~ much~ SPEAK (Verb) ( וְדִבֶּר / wê'di'ber ) TO ( אֶל / el ) the~ PEOPLE ( הָעָם / ha'am )

and it will come to pass, as you come near to battle, and the administrator will be drawn near, and he will speak to the people,

and he will say to them, listen Yisra'eyl, you are near the day to battle upon your attackers, your heart will not be soft, you will not fear, and you will not hasten, and you will not be terrified from their faces.

4 GIVEN.THAT ( כִּי / ki ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(mp) ( אֱלֹהֵיכֶם / e'lo'hey'khem ) the~ WALK (Verb) ~ing(ms) ( הַהֹלֵךְ / ha'ho'leykh ) WITH ~you(mp) ( עִמָּכֶם / i'ma'khem ) to~ >~ be~ FIGHT (Verb) ( לְהִלָּחֵם / lê'hi'la'hheym ) to~ ~you(mp) ( לָכֶם / la'khem ) WITH ( עִם / im ) ATTACK (Verb) ~ing(mp) ~you(mp) ( אֹיְבֵיכֶם / oy'vey'khem ) to~ >~ make~ RESCUE (Verb) ( לְהוֹשִׁיעַ / lê'ho'shi'a ) AT ~you(mp) ( אֶתְכֶם / et'khem )

Given that YHWH your Elohiym is the one walking with you to fight for you with your attackers to rescue you,

5 and~ they~ did~ much~ SPEAK (Verb) ( וְדִבְּרוּ / wê'di'be'ru ) the~ DOMINATE (Verb) ~ing(mp) ( הַשֹּׁטְרִים / ha'shot'rim ) TO ( אֶל / el ) the~ PEOPLE ( הָעָם / ha'am ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) WHO ( מִי / mi ) the~ MAN ( הָאִישׁ / ha'ish ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ BUILD (Verb) ( בָּנָה / ba'nah ) HOUSE ( בַיִת / va'yit ) NEW ( חָדָשׁ / hha'dash ) and~ NOT ( וְלֹא / wê'lo ) he~ did~ DEVOTE (Verb) ~him ( חֲנָכוֹ / hha'na'kho ) he~ will~ WALK (Verb) ( יֵלֵךְ / yey'leykh ) and~ he~ will~ TURN.BACK (Verb) ( וְיָשֹׁב / wê'ya'shov ) to~ HOUSE ~him ( לְבֵיתוֹ / lê'vey'to ) OTHERWISE ( פֶּן / pen ) he~ will~ DIE (Verb) ( יָמוּת / ya'mut ) in~ the~ BATTLE ( בַּמִּלְחָמָה / ba'mil'hha'mah ) and~ MAN ( וְאִישׁ / wê'ish ) OTHER ( אַחֵר / a'hheyr ) he~ will~ DEVOTE (Verb) ~him ( יַחְנְכֶנּוּ / yahh'ne'khe'nu )

and the dominators will speak to the people saying, who is the man which built a new house and did not devote himself, he will walk and he will turn back to his house, otherwise he will die in battle, and the other man will devote himself,

6 and~ WHO ( וּמִי / u'mi ) the~ MAN ( הָאִישׁ / ha'ish ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ PLANT (Verb) ( נָטַע / na'ta ) VINEYARD ( כֶּרֶם / ke'rem ) and~ NOT ( וְלֹא / wê'lo ) he~ did~ much~ DRILL (Verb) ~him ( חִלְּלוֹ / hhil'lo ) he~ will~ WALK (Verb) ( יֵלֵךְ / yey'leykh ) and~ he~ will~ TURN.BACK (Verb) ( וְיָשֹׁב / wê'ya'shov ) to~ HOUSE ~him ( לְבֵיתוֹ / lê'vey'to ) OTHERWISE ( פֶּן / pen ) he~ will~ DIE (Verb) ( יָמוּת / ya'mut ) in~ the~ BATTLE ( בַּמִּלְחָמָה / ba'mil'hha'mah ) and~ MAN ( וְאִישׁ / wê'ish ) OTHER ( אַחֵר / a'hheyr ) he~ will~ much~ DRILL (Verb) ~him ( יְחַלְּלֶנּוּ / yê'hhal'le'nu )

and who is the man which planted a vineyard, and he will not begin him, he will walk and he will turn back to his house, otherwise he will die in battle and the other man will begin him,

7 and~ WHO ( וּמִי / u'mi ) the~ MAN ( הָאִישׁ / ha'ish ) WHICH ( אֲשֶׁר / a'sher ) he~ did~ much~ BETROTH (Verb) ( אֵרַשׂ / ey'ras ) WOMAN ( אִשָּׁה / i'shah ) and~ NOT ( וְלֹא / wê'lo ) he~ did~ TAKE (Verb) ~her ( לְקָחָהּ / lê'qa'hhah ) he~ will~ WALK (Verb) ( יֵלֵךְ / yey'leykh ) and~ he~ will~ TURN.BACK (Verb) ( וְיָשֹׁב / wê'ya'shov ) to~ HOUSE ~him ( לְבֵיתוֹ / lê'vey'to ) OTHERWISE ( פֶּן / pen ) he~ will~ DIE (Verb) ( יָמוּת / ya'mut ) in~ the~ BATTLE ( בַּמִּלְחָמָה / ba'mil'hha'mah ) and~ MAN ( וְאִישׁ / wê'ish ) OTHER ( אַחֵר / a'hheyr ) he~ will~ TAKE (Verb) ~her ( יִקָּחֶנָּה / yi'qa'hhe'nah )

and who is the man which will betroth a woman and he will not take her, he will walk and he will turn back to his house, otherwise he will die in battle, and the other man will take her,

8 and~ they~ did~ ADD (Verb) ( וְיָסְפוּ / wê'yas'phu ) the~ DOMINATE (Verb) ~ing(mp) ( הַשֹּׁטְרִים / ha'shot'rim ) to~ >~ much~ SPEAK (Verb) ( לְדַבֵּר / lê'da'beyr ) TO ( אֶל / el ) the~ PEOPLE ( הָעָם / ha'am ) and~ they~ did~ SAY (Verb) ( וְאָמְרוּ / wê'am'ru ) WHO ( מִי / mi ) the~ MAN ( הָאִישׁ / ha'ish ) the~ FEAR (Verb) ~ing(ms) ( הַיָּרֵא / hai'ya'rey ) and~ TENDER ( וְרַךְ / wê'rakh ) the~ HEART ( הַלֵּבָב / ha'ley'vav ) he~ will~ WALK (Verb) ( יֵלֵךְ / yey'leykh ) and~ he~ will~ TURN.BACK (Verb) ( וְיָשֹׁב / wê'ya'shov ) to~ HOUSE ~him ( לְבֵיתוֹ / lê'vey'to ) and~ NOT ( וְלֹא / wê'lo ) he~ will~ be~ MELT.AWAY (Verb) ( יִמַּס / yi'mas ) AT ( אֶת / et ) HEART ( לְבַב / lê'vav ) BROTHER ~s ~him ( אֶחָיו / e'hhaw ) like~ HEART ~him ( כִּלְבָבוֹ / kil'va'vo )

and the dominators will add to speak to the people, and they will say, who is the man fearing and the tender heart will walk and he will turn back to his house, and the heart of his brothers will not be melted away like his heart,

9 and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) like~ >~ much~ FINISH (Verb) ( כְּכַלֹּת / kê'kha'lot ) the~ DOMINATE (Verb) ~ing(mp) ( הַשֹּׁטְרִים / ha'shot'rim ) to~ >~ much~ SPEAK (Verb) ( לְדַבֵּר / lê'da'beyr ) TO ( אֶל / el ) the~ PEOPLE ( הָעָם / ha'am ) and~ they~ did~ REGISTER (Verb) ( וּפָקְדוּ / u'phaq'du ) NOBLE ~s ( שָׂרֵי / sa'rey ) ARMY ~s ( צְבָאוֹת / tsê'va'ot ) in~ HEAD ( בְּרֹאשׁ / bê'rosh ) the~ PEOPLE ( הָעָם / ha'am )

and it will come to pass, as the dominators finish speaking to the people, and they will register the nobles of the armies by the head of the people.

10 GIVEN.THAT ( כִּי / ki ) you(ms)~ will~ COME.NEAR (Verb) ( תִקְרַב / tiq'rav ) TO ( אֶל / el ) CITY ( עִיר / ir ) to~ >~ be~ FIGHT (Verb) ( לְהִלָּחֵם / lê'hi'la'hheym ) UPON ~her ( עָלֶיהָ / a'ley'ah ) and~ you(ms)~ did~ CALL.OUT (Verb) ( וְקָרָאתָ / wê'qa'ra'ta ) TO ~her ( אֵלֶיהָ / ey'ley'ah ) to~ COMPLETENESS ( לְשָׁלוֹם / lê'sha'lom )

Given that you will come near to the city to fight upon her, and you will call out to her for completeness,

11 and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) IF ( אִם / im ) COMPLETENESS ( שָׁלוֹם / sha'lom ) you(ms)~ will~ AFFLICT (Verb) ~you(ms) ( תַּעַנְךָ / ta'an'kha ) and~ she~ did~ OPEN (Verb) ( וּפָתְחָה / u'phat'hhah ) to~ ~you(fs) ( לָךְ / lakh ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) ALL ( כָּל / kol ) the~ PEOPLE ( הָעָם / ha'am ) the~ be~ FIND (Verb) ~ing(ms) ( הַנִּמְצָא / ha'nim'tsa ) in~ ~her ( בָהּ / vah ) they(m)~ will~ EXIST (Verb) ( יִהְיוּ / yih'yu ) to~ ~you(ms) ( לְךָ / lê'kha ) to~ the~ TASK.WORK ( לָמַס / la'mas ) and~ they~ did~ SERVE (Verb) ~you(ms) ( וַעֲבָדוּךָ / wa'a'va'du'kha )

and it will come to pass, if completeness, she will afflict you, and she will open to you, and it will come to pass, all the people being found in her, they will exist for you for task work, and they will serve you,

12 and~ IF ( וְאִם / wê'im ) NOT ( לֹא / lo ) she~ will~ make~ MAKE.RESTITUTION (Verb) ( תַשְׁלִים / tash'lim ) WITH ~you(fs) ( עִמָּךְ / i'makh ) and~ she~ did~ DO (Verb) ( וְעָשְׂתָה / wê'as'tah ) WITH ~you(ms) ( עִמְּךָ / im'kha ) BATTLE ( מִלְחָמָה / mil'hha'mah ) and~ you(ms)~ did~ SMACK (Verb) ( וְצַרְתָּ / wê'tsar'ta ) UPON ~her ( עָלֶיהָ / a'ley'ah )

and if not, she will make restitution with you, and she will make battle with you, and you will smack upon her,

13 and~ he~ did~ GIVE (Verb) ~her ( וּנְתָנָהּ / un'ta'nah ) YHWH ( יְהוָה / YHWH ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) in~ HAND ~you(ms) ( בְּיָדֶךָ / bê'ya'de'kha ) and~ you(ms)~ did~ make~ HIT (Verb) ( וְהִכִּיתָ / wê'hi'ki'ta ) AT ( אֶת / et ) ALL ( כָּל / kol ) MEN ~her ( זְכוּרָהּ / zê'khu'rah ) to~ MOUTH ( לְפִי / lê'phi ) [ 1013 ] SWORD ( חָרֶב / hha'rev )

and YHWH your Elohiym will give her in your hand, and you will attack all her men by the mouth of the sword.
