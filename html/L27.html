Leviticus Chapter 27

1 and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) to~ >~ SAY (Verb) ( לֵּאמֹר / ley'mor )

and YHWH spoke to Mosheh saying,

2 !(ms)~ much~ SPEAK (Verb) ( דַּבֵּר / da'beyr ) TO ( אֶל / el ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) and~ you(ms)~ did~ SAY (Verb) ( וְאָמַרְתָּ / wê'a'mar'ta ) TO ~them(m) ( אֲלֵהֶם / a'ley'hem ) MAN ( אִישׁ / ish ) GIVEN.THAT ( כִּי / ki ) he~ will~ make~ PERFORM (Verb) ( יַפְלִא / yaph'li ) VOW ( נֶדֶר / ne'der ) in~ ARRANGEMENT ~you(ms) ( בְּעֶרְכְּךָ / bê'er'ke'kha ) SOUL ~s ( נְפָשֹׁת / nê'pha'shot ) to~ YHWH ( לַיהוָה / la'YHWH )

Speak to the sons of Yisra'eyl and you will say to them, a man that will perform a vow, by your valuation souls belong to YHWH,

3 and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) ARRANGEMENT ~you(ms) ( עֶרְכְּךָ / er'ke'kha ) the~ MALE ( הַזָּכָר / ha'za'khar ) from~ SON ( מִבֶּן / mi'ben ) TEN ~s ( עֶשְׂרִים / es'rim ) YEAR ( שָׁנָה / sha'nah ) and~ UNTIL ( וְעַד / wê'ad ) SON ( בֶּן / ben ) SIX ~s ( שִׁשִּׁים / shi'shim ) YEAR ( שָׁנָה / sha'nah ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) ARRANGEMENT ~you(ms) ( עֶרְכְּךָ / er'ke'kha ) FIVE ~s ( חֲמִשִּׁים / hha'mi'shim ) SHEQEL ( שֶׁקֶל / she'qel ) SILVER ( כֶּסֶף / ke'seph ) in~ SHEQEL ( בְּשֶׁקֶל / bê'she'qel ) the~ SPECIAL ( הַקֹּדֶשׁ / ha'qo'desh )

and your valuation will exist of the male, from a son of twenty years and until a son of sixty years, and your arrangement will exist of fifty sheqels of silver, by the special sheqel,

4 and~ IF ( וְאִם / wê'im ) FEMALE ( נְקֵבָה / nê'qey'vah ) SHE ( הִוא / hi ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) ARRANGEMENT ~you(ms) ( עֶרְכְּךָ / er'ke'kha ) THREE ~s ( שְׁלֹשִׁים / shê'lo'shim ) SHEQEL ( שָׁקֶל / sha'qel )

and if she is a female, your valuation will exist of thirty sheqels,

5 and~ IF ( וְאִם / wê'im ) from~ SON ( מִבֶּן / mi'ben ) FIVE ( חָמֵשׁ / hha'meysh ) YEAR ~s ( שָׁנִים / sha'nim ) and~ UNTIL ( וְעַד / wê'ad ) SON ( בֶּן / ben ) TEN ~s ( עֶשְׂרִים / es'rim ) YEAR ( שָׁנָה / sha'nah ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) ARRANGEMENT ~you(ms) ( עֶרְכְּךָ / er'ke'kha ) the~ MALE ( הַזָּכָר / ha'za'khar ) TEN ~s ( עֶשְׂרִים / es'rim ) SHEQEL ~s ( שְׁקָלִים / shê'qa'lim ) and~ to~ the~ FEMALE ( וְלַנְּקֵבָה / wê'lan'qey'vah ) TEN ( עֲשֶׂרֶת / a'se'ret ) SHEQEL ~s ( שְׁקָלִים / shê'qa'lim )

and if from a son of five years and until a son of twenty years your valuation will exist of the male of twenty sheqels,

6 and~ IF ( וְאִם / wê'im ) from~ SON ( מִבֶּן / mi'ben ) NEW.MOON ( חֹדֶשׁ / hho'desh ) and~ UNTIL ( וְעַד / wê'ad ) SON ( בֶּן / ben ) FIVE ( חָמֵשׁ / hha'meysh ) YEAR ~s ( שָׁנִים / sha'nim ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) ARRANGEMENT ~you(ms) ( עֶרְכְּךָ / er'ke'kha ) the~ MALE ( הַזָּכָר / ha'za'khar ) FIVE ( חֲמִשָּׁה / hha'mi'shah ) SHEQEL ~s ( שְׁקָלִים / shê'qa'lim ) SILVER ( כָּסֶף / ka'seph ) and~ to~ the~ FEMALE ( וְלַנְּקֵבָה / wê'lan'qey'vah ) ARRANGEMENT ~you(ms) ( עֶרְכְּךָ / er'ke'kha ) THREE ( שְׁלֹשֶׁת / shê'lo'shet ) SHEQEL ~s ( שְׁקָלִים / shê'qa'lim ) SILVER ( כָּסֶף / ka'seph )

and if from a son of a new moon and until a son of five years, then your valuation will exist of the male of five sheqels of silver, and to the female your valuation is three sheqels of silver,

7 and~ IF ( וְאִם / wê'im ) from~ SON ( מִבֶּן / mi'ben ) SIX ~s ( שִׁשִּׁים / shi'shim ) YEAR ( שָׁנָה / sha'nah ) and~ UPWARD ~unto ( וָמַעְלָה / wa'ma'lah ) IF ( אִם / im ) MALE ( זָכָר / za'khar ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) ARRANGEMENT ~you(ms) ( עֶרְכְּךָ / er'ke'kha ) FIVE ( חֲמִשָּׁה / hha'mi'shah ) TEN ( עָשָׂר / a'sar ) SHEQEL ( שָׁקֶל / sha'qel ) and~ to~ the~ FEMALE ( וְלַנְּקֵבָה / wê'lan'qey'vah ) TEN ( עֲשָׂרָה / a'sa'rah ) SHEQEL ~s ( שְׁקָלִים / shê'qa'lim )

and if from a son of sixty years and upward, if a male, then your valuation will exist of fifteen sheqels, and for the female ten sheqels,

8 and~ IF ( וְאִם / wê'im ) he~ did~ BE.LOW (Verb) ( מָךְ / makh ) HE ( הוּא / hu ) from~ ARRANGEMENT ~you(ms) ( מֵעֶרְכֶּךָ / mey'er'ke'kha ) and~ he~ did~ make~ STAND (Verb) ~him ( וְהֶעֱמִידוֹ / wê'he'e'mi'do ) to~ FACE ~s ( לִפְנֵי / liph'ney ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) and~ he~ did~ make~ ARRANGE (Verb) ( וְהֶעֱרִיךְ / wê'he'e'rikh ) AT ~him ( אֹתוֹ / o'to ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) UPON ( עַל / al ) MOUTH ( פִּי / pi ) [ 762 ] WHICH ( אֲשֶׁר / a'sher ) she~ will~ make~ OVERTAKE (Verb) ( תַּשִּׂיג / ta'sig ) HAND ( יַד / yad ) the~ MAKE.A.VOW (Verb) ~ing(ms) ( הַנֹּדֵר / ha'no'deyr ) he~ will~ make~ ARRANGE (Verb) ~him ( יַעֲרִיכֶנּוּ / ya'a'ri'khe'nu ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn )

and if he be low from your valuation, then he will make him stand to the face of the administrator, and the administrator will value him by the mouth of what the hand making a vow will overtake, the administrator will value him,

9 and~ IF ( וְאִם / wê'im ) BEAST ( בְּהֵמָה / bê'hey'mah ) WHICH ( אֲשֶׁר / a'sher ) they(m)~ will~ make~ COME.NEAR (Verb) ( יַקְרִיבוּ / yaq'ri'vu ) FROM ~her ( מִמֶּנָּה / mi'me'nah ) DONATION ( קָרְבָּן / qar'ban ) to~ YHWH ( לַיהוָה / la'YHWH ) ALL ( כֹּל / kol ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ GIVE (Verb) ( יִתֵּן / yi'teyn ) FROM ~him ( מִמֶּנּוּ / mi'me'nu ) to~ YHWH ( לַיהוָה / la'YHWH ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) SPECIAL ( קֹּדֶשׁ / qo'desh )

and if a beast that they will bring near, a donation for YHWH, all that he will give from him belong to YHWH, he will exist special.

10 NOT ( לֹא / lo ) he~ will~ make~ PASS.OVER (Verb) ~him ( יַחֲלִיפֶנּוּ / ya'hha'li'phe'nu ) and~ NOT ( וְלֹא / wê'lo ) he~ will~ make~ CONVERT (Verb) ( יָמִיר / ya'mir ) AT ~him ( אֹתוֹ / o'to ) FUNCTIONAL ( טוֹב / tov ) in~ DYSFUNCTIONAL ( בְּרָע / bê'ra ) OR ( אוֹ / o ) DYSFUNCTIONAL ( רַע / ra ) in~ FUNCTIONAL ( בְּטוֹב / bê'tov ) and~ IF ( וְאִם / wê'im ) >~ make~ CONVERT (Verb) ( הָמֵר / ha'meyr ) he~ will~ make~ CONVERT (Verb) ( יָמִיר / ya'mir ) BEAST ( בְּהֵמָה / bê'hey'mah ) in~ BEAST ( בִּבְהֵמָה / biv'hey'mah ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) HE ( הוּא / hu ) and~ EXCHANGE ~him ( וּתְמוּרָתוֹ / ut'mu'ra'to ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) SPECIAL ( קֹּדֶשׁ / qo'desh )

He will not pass him over, and he will not convert him, a functional for a dysfunctional or a dysfunction for a functional, and if he convert beast for a beast, and he will exist, he and his exchange will exist special,

11 and~ IF ( וְאִם / wê'im ) ALL ( כָּל / kol ) BEAST ( בְּהֵמָה / bê'hey'mah ) DIRTY ( טְמֵאָה / tê'mey'ah ) WHICH ( אֲשֶׁר / a'sher ) NOT ( לֹא / lo ) they(m)~ will~ make~ COME.NEAR (Verb) ( יַקְרִיבוּ / yaq'ri'vu ) FROM ~her ( מִמֶּנָּה / mi'me'nah ) DONATION ( קָרְבָּן / qar'ban ) to~ YHWH ( לַיהוָה / la'YHWH ) and~ he~ did~ make~ STAND (Verb) ( וְהֶעֱמִיד / wê'he'e'mid ) AT ( אֶת / et ) the~ BEAST ( הַבְּהֵמָה / ha'be'hey'mah ) to~ FACE ~s ( לִפְנֵי / liph'ney ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn )

and if any dirty beast, which they will not bring her near for a donation to YHWH, then he will make the beast stand to the face of the administrators,

12 and~ he~ did~ make~ ARRANGE (Verb) ( וְהֶעֱרִיךְ / wê'he'e'rikh ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) AT ~her ( אֹתָהּ / o'tah ) BETWEEN ( בֵּין / beyn ) FUNCTIONAL ( טוֹב / tov ) and~ BETWEEN ( וּבֵין / u'veyn ) DYSFUNCTIONAL ( רָע / ra ) like~ ARRANGEMENT ~you(ms) ( כְּעֶרְכְּךָ / kê'er'ke'kha ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) SO ( כֵּן / keyn ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh )

and the administrator will value her between the functional and the dysfunctional, the administrator, so he will exist,

13 and~ IF ( וְאִם / wê'im ) >~ REDEEM (Verb) ( גָּאֹל / ga'ol ) he~ will~ REDEEM (Verb) ~her ( יִגְאָלֶנָּה / yig'a'le'nah ) and~ he~ will~ ADD (Verb) ( וְיָסַף / wê'ya'saph ) FIFTH ~him ( חֲמִישִׁתוֹ / hha'mi'shi'to ) UPON ( עַל / al ) ARRANGEMENT ~you(ms) ( עֶרְכֶּךָ / er'ke'kha )

but if he will surely redeem her, and he will add a fifth of him upon your valuation,

14 and~ MAN ( וְאִישׁ / wê'ish ) GIVEN.THAT ( כִּי / ki ) he~ will~ make~ SET.APART (Verb) ( יַקְדִּשׁ / yaq'dish ) AT ( אֶת / et ) HOUSE ~him ( בֵּיתוֹ / bey'to ) SPECIAL ( קֹדֶשׁ / qo'desh ) to~ YHWH ( לַיהוָה / la'YHWH ) and~ he~ did~ make~ ARRANGE (Verb) ~him ( וְהֶעֱרִיכוֹ / wê'he'e'ri'kho ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) BETWEEN ( בֵּין / beyn ) FUNCTIONAL ( טוֹב / tov ) and~ BETWEEN ( וּבֵין / u'veyn ) DYSFUNCTIONAL ( רָע / ra ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ will~ make~ ARRANGE (Verb) ( יַעֲרִיךְ / ya'a'rikh ) AT ~him ( אֹתוֹ / o'to ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) SO ( כֵּן / keyn ) he~ will~ RISE (Verb) ( יָקוּם / ya'qum )

and a man that will make his house set apart as special for YHWH, and the administrator will value him between the functional and dysfunctional, just as the administrator will value him, so he will rise,

15 and~ IF ( וְאִם / wê'im ) the~ make~ SET.APART (Verb) ~ing(ms) ( הַמַּקְדִּישׁ / ha'maq'dish ) he~ will~ REDEEM (Verb) ( יִגְאַל / yig'al ) AT ( אֶת / et ) HOUSE ~him ( בֵּיתוֹ / bey'to ) and~ he~ will~ ADD (Verb) ( וְיָסַף / wê'ya'saph ) FIFTH ( חֲמִישִׁית / hha'mi'shit ) SILVER ( כֶּסֶף / ke'seph ) ARRANGEMENT ~you(ms) ( עֶרְכְּךָ / er'ke'kha ) UPON ~him ( עָלָיו / a'law ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) to~ ~him ( לוֹ / lo )

and if the one making set apart will redeem his house, then he will add a fifth of silver of your valuation upon him, and he will exist for him,

16 and~ IF ( וְאִם / wê'im ) from~ FIELD ( מִשְּׂדֵה / mis'deyh ) HOLDINGS ~him ( אֲחֻזָּתוֹ / a'hhu'za'to ) he~ will~ make~ SET.APART (Verb) ( יַקְדִּישׁ / yaq'dish ) MAN ( אִישׁ / ish ) to~ YHWH ( לַיהוָה / la'YHWH ) and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) ARRANGEMENT ~you(ms) ( עֶרְכְּךָ / er'ke'kha ) to~ MOUTH ( לְפִי / lê'phi ) [ 763 ] SEED ~him ( זַרְעוֹ / zar'o ) SEED ( זֶרַע / ze'ra ) HHOMER ( חֹמֶר / hho'mer ) BARLEY ~s ( שְׂעֹרִים / sê'o'rim ) in~ the~ FIVE ~s ( בַּחֲמִשִּׁים / ba'hha'mi'shim ) SHEQEL ( שֶׁקֶל / she'qel ) SILVER ( כָּסֶף / ka'seph )

and if from the field of his holdings a man will make set apart for YHWH, then your valuation will exist by the mouth of his seed, the seed of a hhomer of barleys in fifty sheqels of silver.

17 IF ( אִם / im ) from~ YEAR ( מִשְּׁנַת / mi'she'nat ) the~ JUBILEE ( הַיֹּבֵל / hai'yo'veyl ) he~ will~ make~ SET.APART (Verb) ( יַקְדִּישׁ / yaq'dish ) FIELD ~him ( שָׂדֵהוּ / sa'dey'hu ) like~ ARRANGEMENT ~you(ms) ( כְּעֶרְכְּךָ / kê'er'ke'kha ) he~ will~ RISE (Verb) ( יָקוּם / ya'qum )

If from the year of the jubilee, he will make his field set apart, like your valuation he will rise,

18 and~ IF ( וְאִם / wê'im ) AFTER ( אַחַר / a'hhar ) the~ JUBILEE ( הַיֹּבֵל / hai'yo'veyl ) he~ will~ make~ SET.APART (Verb) ( יַקְדִּישׁ / yaq'dish ) FIELD ~him ( שָׂדֵהוּ / sa'dey'hu ) and~ he~ did~ much~ THINK (Verb) ( וְחִשַּׁב / wê'hhi'shav ) to~ ~him ( לוֹ / lo ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) AT ( אֶת / et ) the~ SILVER ( הַכֶּסֶף / ha'ke'seph ) UPON ( עַל / al ) MOUTH ( פִּי / pi ) [ 764 ] the~ YEAR ~s ( הַשָּׁנִים / ha'sha'nim ) the~ be~ LEAVE.BEHIND (Verb) ~ing(fp) ( הַנּוֹתָרֹת / ha'no'ta'rot ) UNTIL ( עַד / ad ) YEAR ( שְׁנַת / shê'nat ) the~ JUBILEE ( הַיֹּבֵל / hai'yo'veyl ) and~ he~ did~ be~ TAKE.AWAY (Verb) ( וְנִגְרַע / wê'nig'ra ) from~ ARRANGEMENT ~you(ms) ( מֵעֶרְכֶּךָ / mey'er'ke'kha )

but if after the jubilee, he will make his field set apart, then the administrator will plan for him, the silver by the mouth of the years being left behind until the year of the jubilee, then he will be taken away from your valuation,

19 and~ IF ( וְאִם / wê'im ) >~ REDEEM (Verb) ( גָּאֹל / ga'ol ) he~ will~ REDEEM (Verb) ( יִגְאַל / yig'al ) AT ( אֶת / et ) the~ FIELD ( הַשָּׂדֶה / ha'sa'deh ) the~ make~ SET.APART (Verb) ~ing(ms) ( הַמַּקְדִּישׁ / ha'maq'dish ) AT ~him ( אֹתוֹ / o'to ) and~ he~ will~ ADD (Verb) ( וְיָסַף / wê'ya'saph ) FIVE ( חֲמִשִׁית / hha'mi'shit ) SILVER ( כֶּסֶף / ke'seph ) ARRANGEMENT ~you(ms) ( עֶרְכְּךָ / er'ke'kha ) UPON ~him ( עָלָיו / a'law ) and~ he~ did~ RISE (Verb) ( וְקָם / wê'qam ) to~ ~him ( לוֹ / lo )

and if he will surely redeem the field, the one making him set apart, then he will add a fifth of the silver of your valuation, and he will rise for him,

20 and~ IF ( וְאִם / wê'im ) NOT ( לֹא / lo ) he~ will~ REDEEM (Verb) ( יִגְאַל / yig'al ) AT ( אֶת / et ) the~ FIELD ( הַשָּׂדֶה / ha'sa'deh ) and~ IF ( וְאִם / wê'im ) he~ did~ SELL (Verb) ( מָכַר / ma'khar ) AT ( אֶת / et ) the~ FIELD ( הַשָּׂדֶה / ha'sa'deh ) to~ MAN ( לְאִישׁ / lê'ish ) OTHER ( אַחֵר / a'hheyr ) NOT ( לֹא / lo ) he~ will~ be~ REDEEM (Verb) ( יִגָּאֵל / yi'ga'eyl ) YET.AGAIN ( עוֹד / od )

and if he will not redeem the field, or if he will sell the field to another man, he will not be redeemed again,

21 and~ he~ did~ EXIST (Verb) ( וְהָיָה / wê'hai'yah ) the~ FIELD ( הַשָּׂדֶה / ha'sa'deh ) in~ >~ GO.OUT (Verb) ~him ( בְּצֵאתוֹ / bê'tsey'to ) in~ the~ JUBILEE ( בַיֹּבֵל / vai'yo'veyl ) SPECIAL ( קֹדֶשׁ / qo'desh ) to~ YHWH ( לַיהוָה / la'YHWH ) like~ FIELD ( כִּשְׂדֵה / kis'deyh ) the~ ASSIGNED ( הַחֵרֶם / ha'hhey'rem ) to~ the~ ADMINISTRATOR ( לַכֹּהֵן / la'ko'heyn ) she~ will~ EXIST (Verb) ( תִּהְיֶה / tih'yeh ) HOLDINGS ~him ( אֲחֻזָּתוֹ / a'hhu'za'to )

and the field will exist in his going out in the jubilee, special to YHWH, like the assigned field, his holdings will exist for the administrator,
