1 and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) to~ >~ SAY (Verb) ( לֵּאמֹר / ley'mor )

Speak to the sons of Yisra'eyl and they will turn back and they will camp in front of Piy-Hahhiyrot, between Migdol and the sea, to the face of Ba'al-Tsephon, in front of him you will camp, upon the sea,

3 and~ he~ did~ SAY (Verb) ( וְאָמַר / wê'a'mar ) Paroh ( פַּרְעֹה / par'oh ) to~ SON ~s ( לִבְנֵי / liv'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) be~ ENTANGLED (Verb) ~ing(mp) ( נְבֻכִים / nê'vu'khim ) THEY(m) ( הֵם / heym ) in~ the~ LAND ( בָּאָרֶץ / ba'a'rets ) he~ did~ SHUT (Verb) ( סָגַר / sa'gar ) UPON ~them(m) ( עֲלֵיהֶם / a'ley'hem ) the~ WILDERNESS ( הַמִּדְבָּר / ha'mid'bar )

and Paroh will say to the sons of Yisra'eyl, they are being entangled in the land, the wilderness shut in upon them,

4 and~ I~ did~ much~ SEIZE (Verb) ( וְחִזַּקְתִּי / wê'hhi'zaq'ti ) AT ( אֶת / et ) HEART ( לֵב / leyv ) Paroh ( פַּרְעֹה / par'oh ) and~ he~ did~ PURSUE (Verb) ( וְרָדַף / wê'ra'daph ) AFTER ~them(m) ( אַחֲרֵיהֶם / a'hha'rey'hem ) and~ I~ will~ be~ BE.HEAVY (Verb) ~& ( וְאִכָּבְדָה / wê'i'kav'dah ) in~ Paroh ( בְּפַרְעֹה / bê'phar'oh ) and~ in~ ALL ( וּבְכָל / uv'khol ) FORCE ~him ( חֵילוֹ / hhey'lo ) and~ they~ did~ KNOW (Verb) ( וְיָדְעוּ / wê'yad'u ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) GIVEN.THAT ( כִּי / ki ) I ( אֲנִי / a'ni ) YHWH ( יְהוָה / YHWH ) and~ they(m)~ will~ DO (Verb) ( וַיַּעֲשׂוּ / wai'ya'a'su ) SO ( כֵן / kheyn )

and I will strengthen the heart of Paroh, and he will pursue after them, and I will be heavy in Paroh, and in all his forces, and Mits'rayim will know that I am YHWH, and they will do so,

5 and~ he~ will~ be~ make~ BE.FACE.TO.FACE (Verb) ( וַיֻּגַּד / wai'yu'gad ) to~ KING ( לְמֶלֶךְ / lê'me'lekh ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) GIVEN.THAT ( כִּי / ki ) he~ did~ FLEE.AWAY (Verb) ( בָרַח / va'rahh ) the~ PEOPLE ( הָעָם / ha'am ) and~ he~ will~ be~ OVERTURN (Verb) ( וַיֵּהָפֵךְ / wai'yey'ha'pheykh ) HEART ( לְבַב / lê'vav ) Paroh ( פַּרְעֹה / par'oh ) and~ SERVANT ~s ~him ( וַעֲבָדָיו / wa'a'va'daw ) TO ( אֶל / el ) the~ PEOPLE ( הָעָם / ha'am ) and~ they(m)~ will~ SAY (Verb) ( וַיֹּאמְרוּ / wai'yom'ru ) WHAT ( מַה / mah ) THIS ( זֹּאת / zot ) we~ did~ DO (Verb) ( עָשִׂינוּ / a'si'nu ) GIVEN.THAT ( כִּי / ki ) we~ did~ much~ SEND (Verb) ( שִׁלַּחְנוּ / shi'lahh'nu ) AT ( אֶת / et ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) from~ >~ SERVE (Verb) ~us ( מֵעָבְדֵנוּ / mey'av'dey'nu )

and it was told to the king of Mits'rayim that the people fled away, and the heart of Paroh, and his servants, were overturned to the people, and they said, what is this we did, given that we sent Yisra'eyl from serving us,

6 and~ he~ will~ TIE.UP (Verb) ( וַיֶּאְסֹר / wai'ye'sor ) [ 327 ] AT ( אֶת / et ) VEHICLE ~him ( רִכְבּוֹ / rikh'bo ) and~ AT ( וְאֶת / wê'et ) PEOPLE ~him ( עַמּוֹ / a'mo ) he~ did~ TAKE (Verb) ( לָקַח / la'qahh ) WITH ~him ( עִמּוֹ / i'mo )

and he tied up his vehicle, and he took his people with him,

7 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) SIX ( שֵׁשׁ / sheysh ) HUNDRED ~s ( מֵאוֹת / mey'ot ) VEHICLE ( רֶכֶב / re'khev ) CHOOSE (Verb) ~ed(ms) ( בָּחוּר / ba'hhur ) and~ ALL ( וְכֹל / wê'khol ) VEHICLE ( רֶכֶב / re'khev ) Mits'rayim ( מִצְרָיִם / mits'ra'yim ) and~ LIEUTENANT ~s ( וְשָׁלִשִׁם / wê'sha'li'shim ) UPON ( עַל / al ) ALL ~him ( כֻּלּוֹ / ku'lo )

and he took six hundred chosen vehicles, and all the vehicles of Mits'rayim, and the lieutenants over all of them,

8 and~ he~ will~ much~ SEIZE (Verb) ( וַיְחַזֵּק / wai'hha'zeyq ) YHWH ( יְהוָה / YHWH ) AT ( אֶת / et ) HEART ( לֵב / leyv ) Paroh ( פַּרְעֹה / par'oh ) KING ( מֶלֶךְ / me'lekh ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) and~ he~ will~ PURSUE (Verb) ( וַיִּרְדֹּף / wai'yir'doph ) AFTER ( אַחֲרֵי / a'hha'rey ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) and~ SON ~s ( וּבְנֵי / uv'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) GO.OUT (Verb) ~ing(mp) ( יֹצְאִים / yots'im ) in~ HAND ( בְּיָד / bê'yad ) RAISED ~ing(fs) ( רָמָה / ra'mah ) [ 328 ]

and YHWH strengthened the heart of Paroh, the king of Mits'rayim, and he pursued after the sons of Yisra'eyl, and the sons of Yisra'eyl were going out with the hand raising,

9 and~ they(m)~ will~ PURSUE (Verb) ( וַיִּרְדְּפוּ / wai'yir'de'phu ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) AFTER ~them(m) ( אַחֲרֵיהֶם / a'hha'rey'hem ) and~ they(m)~ will~ make~ OVERTAKE (Verb) ( וַיַּשִּׂיגוּ / wai'ya'si'gu ) AT ~them(m) ( אוֹתָם / o'tam ) CAMP (Verb) ~ing(fp) ( חֹנִים / hho'nim ) UPON ( עַל / al ) the~ SEA ( הַיָּם / hai'yam ) ALL ( כָּל / kol ) HORSE ( סוּס / sus ) VEHICLE ( רֶכֶב / re'khev ) Paroh ( פַּרְעֹה / par'oh ) and~ HORSEMAN ~s ~him ( וּפָרָשָׁיו / u'pha'ra'shaw ) and~ FORCE ~him ( וְחֵילוֹ / wê'hhey'lo ) UPON ( עַל / al ) Piy-Hahhiyrot ( פִּי־הַחִירֹת / pi-ha'hhi'rot ) to~ FACE ~s ( לִפְנֵי / liph'ney ) Ba'al-Tsephon ( בַּעַל־צְפֹן / ba'al-tse'phon )

and Mits'rayim pursued after them, and all the horses of the vehicles of Paroh, and his horsemen, and his forces overtook them camping upon the sea, upon Piy-Hahhiyrot, to the face of Ba'al-Tsephon,

10 and~ Paroh ( וּפַרְעֹה / u'phar'oh ) he~ did~ make~ COME.NEAR (Verb) ( הִקְרִיב / hiq'riv ) and~ they(m)~ will~ LIFT.UP (Verb) ( וַיִּשְׂאוּ / wai'yis'u ) SON ~s ( בְנֵי / vê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) AT ( אֶת / et ) EYE ~s2 ~them(m) ( עֵינֵיהֶם / ay'ney'hem ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) JOURNEY (Verb) ~ing(ms) ( נֹסֵעַ / no'sey'a ) AFTER ~them(m) ( אַחֲרֵיהֶם / a'hha'rey'hem ) and~ they(m)~ will~ FEAR (Verb) ( וַיִּירְאוּ / wai'yir'u ) MANY ( מְאֹד / mê'od ) and~ they(m)~ will~ CRY.OUT (Verb) ( וַיִּצְעֲקוּ / wai'yits'a'qu ) SON ~s ( בְנֵי / vê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) TO ( אֶל / el ) YHWH ( יְהוָה / YHWH )

and Paroh had come near, and the sons of Yisra'eyl lifted up their eyes and saw Mits'rayim journeying after them, and they feared greatly, and the sons of Yisra'eyl cried out to YHWH,

11 and~ they(m)~ will~ SAY (Verb) ( וַיֹּאמְרוּ / wai'yom'ru ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) ?~ from~ UNAWARE ( הֲמִבְּלִי / ha'mi'be'li ) WITHOUT ( אֵין / eyn ) [ 329 ] GRAVE ~s ( קְבָרִים / qê'va'rim ) in~ Mits'rayim ( בְּמִצְרַיִם / bê'mits'ra'yim ) you(ms)~ did~ TAKE (Verb) ~us ( לְקַחְתָּנוּ / lê'qahh'ta'nu ) to~ >~ DIE (Verb) ( לָמוּת / la'mut ) in~ the~ WILDERNESS ( בַּמִּדְבָּר / ba'mid'bar ) WHAT ( מַה / mah ) THIS ( זֹּאת / zot ) you(ms)~ did~ DO (Verb) ( עָשִׂיתָ / a'si'ta ) to~ ~us ( לָּנוּ / la'nu ) to~ >~ make~ GO.OUT (Verb) ~us ( לְהוֹצִיאָנוּ / lê'ho'tsi'a'nu ) from~ Mits'rayim ( מִמִּצְרָיִם / mi'mits'ra'yim )

and they said to Mosheh, is it from a lack of graves in Mits'rayim that you took us to die in the wilderness, what is this you did to us, to make us go out from Mits'rayim.

12 ?~ NOT ( הֲלֹא / ha'lo ) THIS ( זֶה / zeh ) the~ WORD ( הַדָּבָר / ha'da'var ) WHICH ( אֲשֶׁר / a'sher ) we~ did~ much~ SPEAK (Verb) ( דִּבַּרְנוּ / di'bar'nu ) TO ~you(ms) ( אֵלֶיךָ / ey'ley'kha ) in~ Mits'rayim ( בְמִצְרַיִם / vê'mits'ra'yim ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) !(ms)~ TERMINATE (Verb) ( חֲדַל / hha'dal ) FROM ~us ( מִמֶּנּוּ / mi'me'nu ) and~ we~ will~ SERVE (Verb) ~& ( וְנַעַבְדָה / wê'na'av'dah ) AT ( אֶת / et ) Mits'rayim ( מִצְרָיִם / mits'ra'yim ) GIVEN.THAT ( כִּי / ki ) FUNCTIONAL ( טוֹב / tov ) to~ ~us ( לָנוּ / la'nu ) >~ SERVE (Verb) ( עֲבֹד / a'vod ) AT ( אֶת / et ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) from~ >~ DIE (Verb) ~us ( מִמֻּתֵנוּ / mi'mu'tey'nu ) in~ the~ WILDERNESS ( בַּמִּדְבָּר / ba'mid'bar )

Is not this the word which we spoke to you in Mits'rayim saying, terminate from us and we will serve Mits'rayim, given that it is functional for us to serve Mits'rayim rather than us dying in the wilderness,
