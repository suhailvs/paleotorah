Leviticus Chapter 21

1 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) !(ms)~ SAY (Verb) ( אֱמֹר / e'mor ) TO ( אֶל / el ) the~ ADMINISTRATOR ~s ( הַכֹּהֲנִים / ha'ko'ha'nim ) SON ~s ( בְּנֵי / bê'ney ) Aharon ( אַהֲרֹן / a'ha'ron ) and~ you(ms)~ did~ SAY (Verb) ( וְאָמַרְתָּ / wê'a'mar'ta ) TO ~them(m) ( אֲלֵהֶם / a'ley'hem ) to~ SOUL ( לְנֶפֶשׁ / lê'ne'phesh ) NOT ( לֹא / lo ) he~ will~ BE.DIRTY (Verb) ( יִטַּמָּא / yi'ta'ma ) in~ PEOPLE ~s ~him ( בְּעַמָּיו / bê'a'maw )

and YHWH said to Mosheh, say to the administrators, the sons of Aharon, and you will say to them, he will not be dirty for a soul in his people.

2 GIVEN.THAT ( כִּי / ki ) IF ( אִם / im ) to~ KIN ~him ( לִשְׁאֵרוֹ / lish'ey'ro ) the~ NEAR ( הַקָּרֹב / ha'qa'rov ) TO ~him ( אֵלָיו / ey'law ) to~ MOTHER ~him ( לְאִמּוֹ / lê'i'mo ) and~ to~ FATHER ~him ( וּלְאָבִיו / ul'a'viw ) and~ to~ SON ~him ( וְלִבְנוֹ / wê'liv'no ) and~ to~ DAUGHTER ~him ( וּלְבִתּוֹ / ul'vi'to ) and~ to~ BROTHER ~him ( וּלְאָחִיו / ul'a'hhiw )

Instead, for his kin, the one near him, for his mother and for his father and for his son and for his daughter and for his brother,

3 and~ to~ SISTER ~him ( וְלַאֲחֹתוֹ / wê'la'a'hho'to ) the~ VIRGIN ( הַבְּתוּלָה / ha'be'tu'lah ) the~ NEAR ( הַקְּרוֹבָה / haq'ro'vah ) TO ~him ( אֵלָיו / ey'law ) WHICH ( אֲשֶׁר / a'sher ) NOT ( לֹא / lo ) she~ did~ EXIST (Verb) ( הָיְתָה / hai'tah ) to~ MAN ( לְאִישׁ / lê'ish ) to~ ~her ( לָהּ / lah ) he~ will~ BE.DIRTY (Verb) ( יִטַּמָּא / yi'ta'ma )

and for his sister the virgin, the one near to him, who does not have a man, for her he will be dirty.

4 NOT ( לֹא / lo ) he~ will~ BE.DIRTY (Verb) ( יִטַּמָּא / yi'ta'ma ) MASTER ( בַּעַל / ba'al ) in~ PEOPLE ~s ~him ( בְּעַמָּיו / bê'a'maw ) to~ >~ make~ DRILL (Verb) ~him ( לְהֵחַלּוֹ / lê'hey'hha'lo )

He will not be dirty, a master in his people, to defile himself.

5 NOT ( לֹא / lo ) he~ will~ make~ MAKE.BALD (Verb) ~her ( יִקְרְחָה / yiq're'hhah ) [ 739 ] BALD.SPOT ( קָרְחָה / qar'hhah ) in~ HEAD ~them(m) ( בְּרֹאשָׁם / bê'ro'sham ) and~ EDGE ( וּפְאַת / uph'at ) BEARD ~them(m) ( זְקָנָם / zê'qa'nam ) NOT ( לֹא / lo ) they(m)~ will~ much~ SHAVE (Verb) ( יְגַלֵּחוּ / yê'ga'ley'hhu ) and~ in~ FLESH ~them(m) ( וּבִבְשָׂרָם / u'viv'sa'ram ) NOT ( לֹא / lo ) they(m)~ will~ SLICE (Verb) ( יִשְׂרְטוּ / yis're'tu ) SLICING ( שָׂרָטֶת / sa'ra'tet )

They will not make bald a bald spot on their head, and the edge of their beard they will not shave, and in their flesh they will not slice a slicing.

6 UNIQUE ~s ( קְדֹשִׁים / qê'do'shim ) they(m)~ will~ EXIST (Verb) ( יִהְיוּ / yih'yu ) to~ Elohiym ~them(m) ( לֵאלֹהֵיהֶם / ley'lo'hey'hem ) and~ NOT ( וְלֹא / wê'lo ) they(m)~ will~ much~ DRILL (Verb) ( יְחַלְּלוּ / yê'hhal'lu ) TITLE ( שֵׁם / sheym ) Elohiym ~them(m) ( אֱלֹהֵיהֶם / e'lo'hey'hem ) GIVEN.THAT ( כִּי / ki ) AT ( אֶת / et ) FIRE.OFFERING ~s ( אִשֵּׁי / i'shey ) YHWH ( יְהוָה / YHWH ) BREAD ( לֶחֶם / le'hhem ) Elohiym ~them(m) ( אֱלֹהֵיהֶם / e'lo'hey'hem ) THEY(m) ( הֵם / heym ) make~ COME.NEAR (Verb) ~ing(mp) ( מַקְרִיבִם / maq'ri'vim ) and~ they~ did~ EXIST (Verb) ( וְהָיוּ / wê'hai'u ) SPECIAL ( קֹדֶשׁ / qo'desh )

They will exist as unique ones to their Elohiym, and they will not defile the title of their Elohiym, given that the fire offerings of YHWH, the bread of their Elohiym, they are bringing near, and they will be special.

7 WOMAN ( אִשָּׁה / i'shah ) BE.A.HARLOT (Verb) ~ing(fs) ( זֹנָה / zo'nah ) and~ DRILLED ( וַחֲלָלָה / wa'hha'la'lah ) NOT ( לֹא / lo ) they(m)~ will~ TAKE (Verb) ( יִקָּחוּ / yi'qa'hhu ) and~ WOMAN ( וְאִשָּׁה / wê'i'shah ) CAST.OUT (Verb) ~ed(fs) ( גְּרוּשָׁה / gê'ru'shah ) from~ MAN ~her ( מֵאִישָׁהּ / mey'i'shah ) NOT ( לֹא / lo ) they(m)~ will~ TAKE (Verb) ( יִקָּחוּ / yi'qa'hhu ) GIVEN.THAT ( כִּי / ki ) UNIQUE ( קָדֹשׁ / qa'dosh ) HE ( הוּא / hu ) to~ Elohiym ~him ( לֵאלֹהָיו / ley'lo'haw )

A woman being a harlot and drilled they will not take, and a woman casted out from her man they will not take, given that he is unique to his Elohiym,

8 and~ you(ms)~ did~ much~ SET.APART (Verb) ~him ( וְקִדַּשְׁתּוֹ / wê'qi'dash'to ) GIVEN.THAT ( כִּי / ki ) AT ( אֶת / et ) BREAD ( לֶחֶם / le'hhem ) Elohiym ~you(ms) ( אֱלֹהֶיךָ / e'lo'hey'kha ) HE ( הוּא / hu ) make~ COME.NEAR (Verb) ~ing(ms) ( מַקְרִיב / maq'riv ) UNIQUE ( קָדֹשׁ / qa'dosh ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) to~ ~you(fs) ( לָּךְ / lakh ) GIVEN.THAT ( כִּי / ki ) UNIQUE ( קָדוֹשׁ / qa'dosh ) I ( אֲנִי / a'ni ) YHWH ( יְהוָה / YHWH ) much~ SET.APART (Verb) ~ing(ms) ~you(ms) ( מְקַדִּשְׁכֶם / mê'qa'dish'khem )

and you will set him apart, given that the bread of your Elohiym he is bringing near, he exists unique for you, given that unique am I, YHWH, the one setting you apart,

9 and~ DAUGHTER ( וּבַת / u'vat ) MAN ( אִישׁ / ish ) ADMINISTRATOR ( כֹּהֵן / ko'heyn ) GIVEN.THAT ( כִּי / ki ) she~ will~ be~ DRILL (Verb) ( תֵחֵל / tey'hheyl ) to~ >~ BE.A.HARLOT (Verb) ( לִזְנוֹת / liz'not ) AT ( אֶת / et ) FATHER ~her ( אָבִיהָ / a'vi'ah ) SHE ( הִיא / hi ) much~ DRILL (Verb) ~ing(fs) ( מְחַלֶּלֶת / mê'hha'le'let ) in~ the~ FIRE ( בָּאֵשׁ / ba'eysh ) you(ms)~ will~ CREMATE (Verb) ( תִּשָּׂרֵף / ti'sa'reyph )

and the daughter of each administrator that will be defiled by being a harlot, she is defiling her father, you will cremate in the fire,

10 and~ the~ ADMINISTRATOR ( וְהַכֹּהֵן / wê'ha'ko'heyn ) the~ GREAT ( הַגָּדוֹל / ha'ga'dol ) from~ BROTHER ~s ~him ( מֵאֶחָיו / mey'e'hhaw ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ be~ make~ POUR.DOWN (Verb) ( יוּצַק / yu'tsaq ) UPON ( עַל / al ) HEAD ~him ( רֹאשׁוֹ / ro'sho ) OIL ( שֶׁמֶן / she'men ) the~ OINTMENT ( הַמִּשְׁחָה / ha'mish'hhah ) and~ he~ did~ much~ FILL (Verb) ( וּמִלֵּא / u'mi'ley ) [ 740 ] AT ( אֶת / et ) HAND ~him ( יָדוֹ / ya'do ) to~ >~ WEAR (Verb) ( לִלְבֹּשׁ / lil'bosh ) AT ( אֶת / et ) the~ GARMENT ~s ( הַבְּגָדִים / ha'be'ga'dim ) AT ( אֶת / et ) HEAD ~him ( רֹאשׁוֹ / ro'sho ) NOT ( לֹא / lo ) he~ will~ LOOSE (Verb) ( יִפְרָע / yiph'ra ) and~ GARMENT ~s ~him ( וּבְגָדָיו / uv'ga'daw ) NOT ( לֹא / lo ) he~ will~ RIP (Verb) ( יִפְרֹם / yiph'rom )

and the great administrator from his brothers, which will have poured the oil of the ointment down upon his head, and he will fill his hand to wear the garments, he will not loose his head, and his garments he will not rip,

11 and~ UPON ( וְעַל / wê'al ) ALL ( כָּל / kol ) SOUL ~s ( נַפְשֹׁת / naph'shot ) DIE (Verb) ~ing(ms) ( מֵת / meyt ) NOT ( לֹא / lo ) he~ will~ COME (Verb) ( יָבֹא / ya'vo ) to~ FATHER ~him ( לְאָבִיו / lê'a'viw ) and~ to~ MOTHER ~him ( וּלְאִמּוֹ / ul'i'mo ) NOT ( לֹא / lo ) he~ will~ BE.DIRTY (Verb) ( יִטַּמָּא / yi'ta'ma )

and he will not come upon any soul of the dying, for his father and for his mother he will not be dirty,

12 and~ FROM ( וּמִן / u'min ) the~ SANCTUARY ( הַמִּקְדָּשׁ / ha'miq'dash ) NOT ( לֹא / lo ) he~ will~ GO.OUT (Verb) ( יֵצֵא / yey'tsey ) and~ NOT ( וְלֹא / wê'lo ) he~ will~ much~ DRILL (Verb) ( יְחַלֵּל / yê'hha'leyl ) AT ( אֵת / eyt ) SANCTUARY ( מִקְדַּשׁ / miq'dash ) Elohiym ~him ( אֱלֹהָיו / e'lo'haw ) GIVEN.THAT ( כִּי / ki ) DEDICATION ( נֵזֶר / ney'zer ) OIL ( שֶׁמֶן / she'men ) OINTMENT ( מִשְׁחַת / mish'hhat ) Elohiym ~him ( אֱלֹהָיו / e'lo'haw ) UPON ~him ( עָלָיו / a'law ) I ( אֲנִי / a'ni ) YHWH ( יְהוָה / YHWH )

and from the sanctuary he will not go out, and he will not defile the sanctuary of his Elohiym, given that the dedication of oil of ointment of his Elohiym is upon him, I am YHWH,

13 and~ HE ( וְהוּא / wê'hu ) WOMAN ( אִשָּׁה / i'shah ) in~ VIRGINITY ~s ~her ( בִבְתוּלֶיהָ / viv'tu'ley'ah ) he~ will~ TAKE (Verb) ( יִקָּח / yi'qahh )

and he, he will take a woman in her virginity.

14 WIDOW ( אַלְמָנָה / al'ma'nah ) and~ CAST.OUT (Verb) ~ed(fs) ( וּגְרוּשָׁה / ug'ru'shah ) and~ DRILLED ( וַחֲלָלָה / wa'hha'la'lah ) BE.A.HARLOT (Verb) ~ing(fs) ( זֹנָה / zo'nah ) AT ( אֶת / et ) THESE ( אֵלֶּה / ey'leh ) NOT ( לֹא / lo ) he~ will~ TAKE (Verb) ( יִקָּח / yi'qahh ) GIVEN.THAT ( כִּי / ki ) IF ( אִם / im ) VIRGIN ( בְּתוּלָה / bê'tu'lah ) from~ PEOPLE ~s ~him ( מֵעַמָּיו / mey'a'maw ) he~ will~ TAKE (Verb) ( יִקַּח / yi'qahh ) WOMAN ( אִשָּׁה / i'shah )

A widow and a casted out one and a drilled one, one being a harlot, he will not take these, but if there is a virgin from his people he will take a woman,

15 and~ NOT ( וְלֹא / wê'lo ) he~ will~ much~ DRILL (Verb) ( יְחַלֵּל / yê'hha'leyl ) SEED ~him ( זַרְעוֹ / zar'o ) in~ PEOPLE ~s ~him ( בְּעַמָּיו / bê'a'maw ) GIVEN.THAT ( כִּי / ki ) I ( אֲנִי / a'ni ) YHWH ( יְהוָה / YHWH ) much~ SET.APART (Verb) ~ing(ms) ~him ( מְקַדְּשׁוֹ / mê'qad'sho )

and he will not defile his seed in his people, given that I am YHWH setting him apart,

16 and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) to~ >~ SAY (Verb) ( לֵּאמֹר / ley'mor )

and YHWH spoke to Mosheh saying,

17 !(ms)~ much~ SPEAK (Verb) ( דַּבֵּר / da'beyr ) TO ( אֶל / el ) Aharon ( אַהֲרֹן / a'ha'ron ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) MAN ( אִישׁ / ish ) from~ SEED ~you(ms) ( מִזַּרְעֲךָ / mi'zar'a'kha ) to~ GENERATION ~s ~them(m) ( לְדֹרֹתָם / lê'do'ro'tam ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) in~ ~him ( בוֹ / vo ) BLEMISH ( מוּם / mum ) NOT ( לֹא / lo ) he~ will~ COME.NEAR (Verb) ( יִקְרַב / yiq'rav ) to~ >~ make~ COME.NEAR (Verb) ( לְהַקְרִיב / lê'haq'riv ) BREAD ( לֶחֶם / le'hhem ) Elohiym ~him ( אֱלֹהָיו / e'lo'haw )

Speak to Aharon saying, a man from your seed to their generation who will exist in him a blemish, he will not come near to bring near the bread of his Elohiym.

18 GIVEN.THAT ( כִּי / ki ) ALL ( כָל / khol ) MAN ( אִישׁ / ish ) WHICH ( אֲשֶׁר / a'sher ) in~ ~him ( בּוֹ / bo ) BLEMISH ( מוּם / mum ) NOT ( לֹא / lo ) he~ will~ COME.NEAR (Verb) ( יִקְרָב / yiq'rav ) MAN ( אִישׁ / ish ) BLIND ( עִוֵּר / i'weyr ) OR ( אוֹ / o ) LAME ( פִסֵּחַ / phi'sey'ahh ) OR ( אוֹ / o ) PERFORATE (Verb) ~ed(ms) ( חָרֻם / hha'rum ) [ 741 ] OR ( אוֹ / o ) BE.SUPERFLUOUS (Verb) ~ed(ms) ( שָׂרוּעַ / sa'ru'a )

Given that every man who in him is a blemish, he will not come near, a blind man or a lame one or a perforated one or one being superfluous.

19 OR ( אוֹ / o ) MAN ( אִישׁ / ish ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) in~ ~him ( בוֹ / vo ) SHATTERING ( שֶׁבֶר / she'ver ) FOOT ( רָגֶל / ra'gel ) OR ( אוֹ / o ) SHATTERING ( שֶׁבֶר / she'ver ) HAND ( יָד / yad )

Or a man who exists in him a shattering of the foot or a shattering of the hand.

20 OR ( אוֹ / o ) HUNCHBACK ( גִבֵּן / gi'beyn ) OR ( אוֹ / o ) SCRAWNY ( דַק / daq ) OR ( אוֹ / o ) CATARACT ( תְּבַלֻּל / tê'va'lul ) in~ EYE ~him ( בְּעֵינוֹ / bê'ey'no ) OR ( אוֹ / o ) IRRITATION ( גָרָב / ga'rav ) OR ( אוֹ / o ) SKIN.SORE ( יַלֶּפֶת / ya'le'phet ) OR ( אוֹ / o ) CRUMBLED ( מְרוֹחַ / mê'ro'ahh ) TESTICLES ( אָשֶׁךְ / a'shekh )

Or a hunchback or scrawny or a cataract in his eye or an irritation or a skin sore or crumbled testicles.

21 ALL ( כָּל / kol ) MAN ( אִישׁ / ish ) WHICH ( אֲשֶׁר / a'sher ) in~ ~him ( בּוֹ / bo ) BLEMISH ( מוּם / mum ) from~ SEED ( מִזֶּרַע / mi'ze'ra ) Aharon ( אַהֲרֹן / a'ha'ron ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) NOT ( לֹא / lo ) he~ will~ DRAW.NEAR (Verb) ( יִגַּשׁ / yi'gash ) to~ >~ make~ COME.NEAR (Verb) ( לְהַקְרִיב / lê'haq'riv ) AT ( אֶת / et ) FIRE.OFFERING ~s ( אִשֵּׁי / i'shey ) YHWH ( יְהוָה / YHWH ) BLEMISH ( מוּם / mum ) in~ ~him ( בּוֹ / bo ) AT ( אֵת / eyt ) BREAD ( לֶחֶם / le'hhem ) Elohiym ~him ( אֱלֹהָיו / e'lo'haw ) NOT ( לֹא / lo ) he~ will~ DRAW.NEAR (Verb) ( יִגַּשׁ / yi'gash ) to~ >~ make~ COME.NEAR (Verb) ( לְהַקְרִיב / lê'haq'riv )

Every man, which is in him a blemish, from the seed of Aharon the administrator, will not draw near to bring near the fire offerings of YHWH, a blemish is in him, the bread of his Elohiym he will not draw near to bring near.

22 BREAD ( לֶחֶם / le'hhem ) Elohiym ~him ( אֱלֹהָיו / e'lo'haw ) from~ SPECIAL ~s ( מִקָּדְשֵׁי / mi'qad'shey ) the~ SPECIAL ~s ( הַקֳּדָשִׁים / ha'qa'da'shim ) and~ FROM ( וּמִן / u'min ) the~ SPECIAL ~s ( הַקֳּדָשִׁים / ha'qa'da'shim ) he~ will~ EAT (Verb) ( יֹאכֵל / yo'kheyl )

The bread of his Elohiym, from the very special ones, and from the special ones, he will eat.

23 SURELY ( אַךְ / akh ) TO ( אֶל / el ) the~ TENT.CURTAIN ( הַפָּרֹכֶת / ha'pa'ro'khet ) NOT ( לֹא / lo ) he~ will~ COME (Verb) ( יָבֹא / ya'vo ) and~ TO ( וְאֶל / wê'el ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh ) NOT ( לֹא / lo ) he~ will~ DRAW.NEAR (Verb) ( יִגַּשׁ / yi'gash ) GIVEN.THAT ( כִּי / ki ) BLEMISH ( מוּם / mum ) in~ ~him ( בּוֹ / bo ) and~ NOT ( וְלֹא / wê'lo ) he~ will~ much~ DRILL (Verb) ( יְחַלֵּל / yê'hha'leyl ) AT ( אֶת / et ) SANCTUARY ~s ~me ( מִקְדָּשַׁי / miq'da'shai ) GIVEN.THAT ( כִּי / ki ) I ( אֲנִי / a'ni ) YHWH ( יְהוָה / YHWH ) from~ >~ much~ SET.APART (Verb) ~them(m) ( מְקַדְּשָׁם / mê'qad'sham )

Surely, to the tent curtain he will not come and to the altar he will not draw near, given that a blemish is in him, and he will not defile my sanctuaries, given that I am YHWH setting them apart,

24 and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) Mosheh ( מֹשֶׁה / mo'sheh ) TO ( אֶל / el ) Aharon ( אַהֲרֹן / a'ha'ron ) and~ TO ( וְאֶל / wê'el ) SON ~s ~him ( בָּנָיו / ba'naw ) and~ TO ( וְאֶל / wê'el ) ALL ( כָּל / kol ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl )
