Leviticus Chapter 1

1 and~ he~ will~ CALL.OUT (Verb) ( וַיִּקְרָא / wai'yiq'ra ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) YHWH ( יְהוָה / YHWH ) TO ~him ( אֵלָיו / ey'law ) from~ TENT ( מֵאֹהֶל / meyohel ) APPOINTED ( מוֹעֵד / mo'eyd ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor )

and he called out to Mosheh, and YHWH spoke to him from the appointed tent saying,

2 !(ms)~ much~ SPEAK (Verb) ( דַּבֵּר / da'beyr ) TO ( אֶל / el ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) and~ you(ms)~ did~ SAY (Verb) ( וְאָמַרְתָּ / wê'a'mar'ta ) TO ~them(m) ( אֲלֵהֶם / a'ley'hem ) HUMAN ( אָדָם / a'dam ) GIVEN.THAT ( כִּי / ki ) he~ will~ make~ COME.NEAR (Verb) ( יַקְרִיב / yaq'riv ) from~ ~you(mp) ( מִכֶּם / mi'kem ) DONATION ( קָרְבָּן / qar'ban ) to~ YHWH ( לַיהוָה / la'YHWH ) FROM ( מִן / min ) the~ BEAST ( הַבְּהֵמָה / ha'be'hey'mah ) FROM ( מִן / min ) the~ CATTLE ( הַבָּקָר / ha'ba'qar ) and~ FROM ( וּמִן / u'min ) the~ FLOCKS ( הַצֹּאן / ha'tson ) you(mp)~ will~ make~ COME.NEAR (Verb) ( תַּקְרִיבוּ / taq'ri'vu ) AT ( אֶת / et ) DONATION ~you(mp) ( קָרְבַּנְכֶם / qar'ban'khem )

Speak to the sons of Yisra'eyl, and you will say to them, each that will bring from you a donation to YHWH, from the beast, from the cattle, from the flocks, you will bring near your donation.

3 IF ( אִם / im ) ASCENSION.OFFERING ( עֹלָה / o'lah ) DONATION ~him ( קָרְבָּנוֹ / qar'ba'no ) FROM ( מִן / min ) the~ CATTLE ( הַבָּקָר / ha'ba'qar ) MALE ( זָכָר / za'khar ) WHOLE ( תָּמִים / ta'mim ) he~ will~ make~ COME.NEAR (Verb) ~him ( יַקְרִיבֶנּוּ / yaq'riy've'nu ) TO ( אֶל / el ) OPENING ( פֶּתַח / pe'tahh ) TENT ( אֹהֶל / o'hel ) APPOINTED ( מוֹעֵד / mo'eyd ) he~ will~ make~ COME.NEAR (Verb) ( יַקְרִיב / yaq'riv ) AT ~him ( אֹתוֹ / o'to ) to~ SELF-WILL ~him ( לִרְצֹנוֹ / lir'tso'no ) to~ FACE ~s ( לִפְנֵי / liph'ney ) YHWH ( יְהוָה / YHWH )

If his donation is a ascension offering from the cattle, it will be a whole male, he will bring him near to the opening of the appointed tent, he will bring him near for his self-will, to the face of YHWH,

4 and~ he~ did~ SUPPORT (Verb) ( וְסָמַךְ / wê'sa'makh ) HAND ~him ( יָדוֹ / ya'do ) UPON ( עַל / al ) HEAD ( רֹאשׁ / rosh ) the~ ASCENSION.OFFERING ( הָעֹלָה / ha'o'lah ) and~ he~ did~ be~ ACCEPT (Verb) ( וְנִרְצָה / wê'nir'tsah ) to~ ~him ( לוֹ / lo ) [ 648 ] to~ >~ much~ COVER (Verb) ( לְכַפֵּר / lê'kha'peyr ) UPON ~him ( עָלָיו / a'law )

and he will support his hand upon the head of the ascension offering, and he will be acceptable to him, for making a covering upon him,

5 and~ he~ did~ SLAY (Verb) ( וְשָׁחַט / wê'sha'hhat ) AT ( אֶת / et ) SON ( בֶּן / ben ) the~ CATTLE ( הַבָּקָר / ha'ba'qar ) to~ FACE ~s ( לִפְנֵי / liph'ney ) YHWH ( יְהוָה / YHWH ) and~ they~ did~ make~ COME.NEAR (Verb) ( וְהִקְרִיבוּ / wê'hiq'ri'vu ) SON ~s ( בְּנֵי / bê'ney ) Aharon ( אַהֲרֹן / a'ha'ron ) the~ ADMINISTRATOR ~s ( הַכֹּהֲנִים / ha'ko'ha'nim ) AT ( אֶת / et ) the~ BLOOD ( הַדָּם / ha'dam ) and~ they~ did~ SPRINKLE (Verb) ( וְזָרְקוּ / wê'zar'qu ) AT ( אֶת / et ) the~ BLOOD ( הַדָּם / ha'dam ) UPON ( עַל / al ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh ) ALL.AROUND ( סָבִיב / sa'viv ) WHICH ( אֲשֶׁר / a'sher ) OPENING ( פֶּתַח / pe'tahh ) TENT ( אֹהֶל / o'hel ) APPOINTED ( מוֹעֵד / mo'eyd )

and he will slay the son of the cattle to the face of YHWH, and the sons of Aharon, the administrators, will bring near the blood, and they will sprinkle the blood upon the altar all around where the opening of the appointed tent is,

6 and~ he~ did~ make~ STRIP.OFF (Verb) ( וְהִפְשִׁיט / wê'hiph'shit ) AT ( אֶת / et ) the~ ASCENSION.OFFERING ( הָעֹלָה / ha'o'lah ) and~ he~ did~ much~ DIVIDE.INTO.PIECES (Verb) ( וְנִתַּח / wê'ni'tahh ) AT ~her ( אֹתָהּ / o'tah ) to~ PIECE ~s ~her ( לִנְתָחֶיהָ / lin'ta'hhey'ah )

and he will strip off the ascension offering, and he will divide her into pieces to her pieces,

7 and~ they~ did~ GIVE (Verb) ( וְנָתְנוּ / wê'nat'nu ) SON ~s ( בְּנֵי / bê'ney ) Aharon ( אַהֲרֹן / a'ha'ron ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) FIRE ( אֵשׁ / eysh ) UPON ( עַל / al ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh ) and~ they~ did~ ARRANGE (Verb) ( וְעָרְכוּ / wê'ar'khu ) TREE ~s ( עֵצִים / ey'tsim ) UPON ( עַל / al ) the~ FIRE ( הָאֵשׁ / ha'eysh )

and the sons of Aharon, the administrators, will give fire upon the altar, and they will arrange the wood upon the fire,

8 and~ they~ did~ ARRANGE (Verb) ( וְעָרְכוּ / wê'ar'khu ) SON ~s ( בְּנֵי / bê'ney ) Aharon ( אַהֲרֹן / a'ha'ron ) the~ ADMINISTRATOR ~s ( הַכֹּהֲנִים / ha'ko'ha'nim ) AT ( אֵת / eyt ) the~ PIECE ~s ( הַנְּתָחִים / han'ta'hhim ) AT ( אֶת / et ) the~ HEAD ( הָרֹאשׁ / ha'rosh ) and~ AT ( וְאֶת / wê'et ) the~ SUET ( הַפָּדֶר / ha'pa'der ) UPON ( עַל / al ) the~ TREE ~s ( הָעֵצִים / ha'eytsim ) WHICH ( אֲשֶׁר / a'sher ) UPON ( עַל / al ) the~ FIRE ( הָאֵשׁ / ha'eysh ) WHICH ( אֲשֶׁר / a'sher ) UPON ( עַל / al ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh )

and the sons of Aharon, the administrators, will arrange the pieces, the head, the suet, upon the wood, which is upon the fire, which is upon the altar,

9 and~ INSIDE ~him ( וְקִרְבּוֹ / wê'qir'bo ) and~ LEG ~s ~him ( וּכְרָעָיו / ukh'ra'aw ) he~ will~ BATHE (Verb) ( יִרְחַץ / yir'hhats ) in~ the~ WATER ~s2 ( בַּמָּיִם / ba'ma'yim ) and~ he~ did~ make~ BURN.INCENSE (Verb) ( וְהִקְטִיר / wê'hiq'tir ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) AT ( אֶת / et ) the~ ALL ( הַכֹּל / ha'kol ) the~ ALTAR ~unto ( הַמִּזְבֵּחָה / ha'miz'bey'hhah ) [ 649 ] ASCENSION.OFFERING ( עֹלָה / o'lah ) FIRE.OFFERING ( אִשֵּׁה / i'sheyh ) AROMA ( רֵיחַ / rey'ahh ) SWEET ( נִיחוֹחַ / ni'hho'ahh ) to~ YHWH ( לַיהוָה / la'YHWH )

and he will bathe his insides and his legs in the waters, and the administrator will burn it all as incense upon the altar, a ascension offering of a fire offering, a sweet aroma for YHWH,

10 and~ IF ( וְאִם / wê'im ) FROM ( מִן / min ) the~ FLOCKS ( הַצֹּאן / ha'tson ) DONATION ~him ( קָרְבָּנוֹ / qar'ba'no ) FROM ( מִן / min ) the~ SHEEP ~s ( הַכְּשָׂבִים / hak'sa'vim ) OR ( אוֹ / o ) FROM ( מִן / min ) the~ GOAT ~s ( הָעִזִּים / ha'i'zim ) to~ ASCENSION.OFFERING ( לְעֹלָה / lê'lah ) MALE ( זָכָר / za'khar ) WHOLE ( תָּמִים / ta'mim ) he~ will~ make~ COME.NEAR (Verb) ~him ( יַקְרִיבֶנּוּ / yaq'riy've'nu )

and if the donation is from the flocks, from the sheep or from the goats, for a ascension offering, it will be a whole male, he will bring him near,

11 and~ he~ did~ SLAY (Verb) ( וְשָׁחַט / wê'sha'hhat ) AT ~him ( אֹתוֹ / o'to ) UPON ( עַל / al ) MIDSECTION ( יֶרֶךְ / ye'rekh ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh ) NORTH ~unto ( צָפֹנָה / tsa'pho'nah ) to~ FACE ~s ( לִפְנֵי / liph'ney ) YHWH ( יְהוָה / YHWH ) and~ they~ did~ SPRINKLE (Verb) ( וְזָרְקוּ / wê'zar'qu ) SON ~s ( בְּנֵי / bê'ney ) Aharon ( אַהֲרֹן / a'ha'ron ) the~ ADMINISTRATOR ~s ( הַכֹּהֲנִים / ha'ko'ha'nim ) AT ( אֶת / et ) BLOOD ~him ( דָּמוֹ / da'mo ) UPON ( עַל / al ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh ) ALL.AROUND ( סָבִיב / sa'viv )

and he will slay him upon the midsection of the altar unto the north, to the face of YHWH, and the sons of Aharon, the administrators, will sprinkle his blood upon the altar all around,

12 and~ he~ did~ much~ DIVIDE.INTO.PIECES (Verb) ( וְנִתַּח / wê'ni'tahh ) AT ~him ( אֹתוֹ / o'to ) to~ PIECE ~s ~him ( לִנְתָחָיו / lin'ta'hhaw ) and~ AT ( וְאֶת / wê'et ) HEAD ~him ( רֹאשׁוֹ / ro'sho ) and~ AT ( וְאֶת / wê'et ) SUET ~him ( פִּדְרוֹ / pid'ro ) and~ he~ did~ ARRANGE (Verb) ( וְעָרַךְ / wê'a'rekh ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) AT ~them(m) ( אֹתָם / o'tam ) UPON ( עַל / al ) the~ TREE ~s ( הָעֵצִים / ha'eytsim ) WHICH ( אֲשֶׁר / a'sher ) UPON ( עַל / al ) the~ FIRE ( הָאֵשׁ / ha'eysh ) WHICH ( אֲשֶׁר / a'sher ) UPON ( עַל / al ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh )

and he will divide him into pieces to his pieces, and his head and his suet, and the administrator will arrange them upon the wood, which is upon the fire, which is upon the altar,

13 and~ the~ INSIDE ( וְהַקֶּרֶב / wê'ha'qe'rev ) and~ the~ LEG ~s2 ( וְהַכְּרָעַיִם / wê'hak'ra'a'im ) he~ will~ BATHE (Verb) ( יִרְחַץ / yir'hhats ) in~ the~ WATER ~s2 ( בַּמָּיִם / ba'ma'yim ) and~ he~ did~ make~ COME.NEAR (Verb) ( וְהִקְרִיב / wê'hiq'riv ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) AT ( אֶת / et ) the~ ALL ( הַכֹּל / ha'kol ) and~ he~ did~ make~ BURN.INCENSE (Verb) ( וְהִקְטִיר / wê'hiq'tir ) the~ ALTAR ~unto ( הַמִּזְבֵּחָה / ha'miz'bey'hhah ) ASCENSION.OFFERING ( עֹלָה / o'lah ) HE ( הוּא / hu ) FIRE.OFFERING ( אִשֵּׁה / i'sheyh ) AROMA ( רֵיחַ / rey'ahh ) SWEET ( נִיחֹחַ / ni'hho'ahh ) to~ YHWH ( לַיהוָה / la'YHWH )

and he will bathe the insides and the legs in the waters, and the administrator will bring near all, and he will burn it as incense upon the altar, a ascension offering, he is a fire offering, a sweet aroma for YHWH,

14 and~ IF ( וְאִם / wê'im ) FROM ( מִן / min ) the~ FLYER ( הָעוֹף / ha'oph ) ASCENSION.OFFERING ( עֹלָה / o'lah ) DONATION ~him ( קָרְבָּנוֹ / qar'ba'no ) to~ YHWH ( לַיהוָה / la'YHWH ) and~ he~ did~ make~ COME.NEAR (Verb) ( וְהִקְרִיב / wê'hiq'riv ) FROM ( מִן / min ) the~ TURTLEDOVE ~s ( הַתֹּרִים / ha'to'rim ) OR ( אוֹ / o ) FROM ( מִן / min ) SON ~s ( בְּנֵי / bê'ney ) the~ DOVE ( הַיּוֹנָה / hai'yo'nah ) AT ( אֶת / et ) DONATION ~him ( קָרְבָּנוֹ / qar'ba'no )

and if the ascension offering is from the flyers, his donation to YHWH, then he will bring near from the turtledoves or from the sons of the doves his donation,

15 and~ he~ did~ make~ COME.NEAR (Verb) ~him ( וְהִקְרִיבוֹ / wê'hiq'ri'vo ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn ) TO ( אֶל / el ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh ) and~ he~ did~ SNAP.OFF (Verb) ( וּמָלַק / u'ma'laq ) AT ( אֶת / et ) HEAD ~him ( רֹאשׁוֹ / ro'sho ) and~ he~ did~ make~ BURN.INCENSE (Verb) ( וְהִקְטִיר / wê'hiq'tir ) the~ ALTAR ~unto ( הַמִּזְבֵּחָה / ha'miz'bey'hhah ) and~ he~ did~ be~ DRAIN (Verb) ( וְנִמְצָה / wê'nim'tsah ) BLOOD ~him ( דָמוֹ / da'mo ) UPON ( עַל / al ) WALL ( קִיר / qir ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh )

and the administrator will bring him near to the altar, and he will snap off his head, and he will burn it as incense upon the altar, and his blood will be drained upon the wall of the altar,

16 and~ he~ did~ make~ TURN.ASIDE (Verb) ( וְהֵסִיר / wê'hey'sir ) AT ( אֶת / et ) CROP ~him ( מֻרְאָתוֹ / mur'a'to ) [ 650 ] in~ PLUMAGE ~her ( בְּנֹצָתָהּ / bê'no'tsa'tah ) and~ he~ did~ make~ THROW.OUT (Verb) ( וְהִשְׁלִיךְ / wê'hish'likh ) AT ~her ( אֹתָהּ / o'tah ) BESIDE ( אֵצֶל / ey'tsel ) the~ ALTAR ( הַמִּזְבֵּחַ / ha'miz'bey'ahh ) EAST ~unto ( קֵדְמָה / qeyd'mah ) TO ( אֶל / el ) AREA ( מְקוֹם / mê'qom ) the~ FATNESS ( הַדָּשֶׁן / ha'da'shen )

and he will split him in two by his wings, he will not separate, and the administrator will burn him as incense upon the altar, upon the wood, which is upon the fire, a ascension offering, he is a fire offering, a sweet aroma for YHWH,
