Genesis Chapter 23

1 and~ they(m)~ will~ EXIST (Verb) ( וַיִּהְיוּ / wai'yih'yu ) LIVING ~s ( חַיֵּי / hhai'yey ) Sarah ( שָׂרָה / sa'rah ) HUNDRED ( מֵאָה / mey'ah ) YEAR ( שָׁנָה / sha'nah ) and~ TEN ~s ( וְעֶשְׂרִים / wê'es'rim ) YEAR ( שָׁנָה / sha'nah ) and~ SEVEN ( וְשֶׁבַע / wê'she'va ) YEAR ~s ( שָׁנִים / sha'nim ) YEAR ~s ( שְׁנֵי / shê'ney ) LIVING ~s ( חַיֵּי / hhai'yey ) Sarah ( שָׂרָה / sa'rah )

2 and~ she~ will~ DIE (Verb) ( וַתָּמָת / wa'ta'mat ) Sarah ( שָׂרָה / sa'rah ) in~ Qiryat-Arba ( בְּקִרְיַת־אַרְבַּע / bê'qir'yat-ar'ba ) SHE ( הִוא / hi ) Hhevron ( חֶבְרוֹן / hhev'ron ) in~ LAND ( בְּאֶרֶץ / bê'e'rets ) Kena'an ( כְּנָעַן / kê'na'an ) and~ he~ will~ COME (Verb) ( וַיָּבֹא / wai'ya'vo ) Avraham ( אַבְרָהָם / av'ra'ham ) to~ >~ LAMENT (Verb) ( לִסְפֹּד / lis'pod ) to~ Sarah ( לְשָׂרָה / lê'sa'rah ) and~ to~ >~ WEEP (Verb) ~her ( וְלִבְכֹּתָהּ / wê'liv'ko'tah )

3 and~ he~ will~ RISE (Verb) ( וַיָּקָם / wai'ya'qam ) Avraham ( אַבְרָהָם / av'ra'ham ) from~ UPON ( מֵעַל / mey'al ) FACE ~s ( פְּנֵי / pê'ney ) DIE (Verb) ~ing(ms) ~him ( מֵתוֹ / mey'to ) and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) TO ( אֶל / el ) SON ~s ( בְּנֵי / bê'ney ) Hhet ( חֵת / hheyt ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor )

and Avraham rose from upon the face of his dead and he spoke to the sons of Hhet saying,

4 IMMIGRANT ( גֵּר / geyr ) and~ SETTLER ( וְתוֹשָׁב / wê'to'shav ) I ( אָנֹכִי / a'no'khi ) WITH ~you(mp) ( עִמָּכֶם / i'ma'khem ) !(mp)~ GIVE (Verb) ( תְּנוּ / tê'nu ) to~ ~me ( לִי / li ) HOLDINGS ( אֲחֻזַּת / a'hhu'zat ) GRAVE ( קֶבֶר / qe'ver ) WITH ~you(mp) ( עִמָּכֶם / i'ma'khem ) and~ I~ will~ BURY (Verb) ~& ( וְאֶקְבְּרָה / wê'eq'be'rah ) DIE (Verb) ~ing(ms) ~me ( מֵתִי / mey'ti ) from~ to~ FACE ~s ~me ( מִלְּפָנָי / mil'pha'nai )

I am an immigrant and settler with you, give to me a holdings of a grave with you and I will bury my dead from before my face,

5 and~ they(m)~ will~ ANSWER (Verb) ( וַיַּעֲנוּ / wai'ya'a'nu ) SON ~s ( בְנֵי / vê'ney ) Hhet ( חֵת / hheyt ) AT ( אֶת / et ) Avraham ( אַבְרָהָם / av'ra'ham ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) to~ ~him ( לוֹ / lo )

and the sons of Hhet answered Avraham saying to him.

6 !(ms)~ HEAR (Verb) ~us ( שְׁמָעֵנוּ / shê'ma'ey'nu ) LORD ~me ( אֲדֹנִי / a'do'ni ) CAPTAIN ( נְשִׂיא / nê'si ) Elohiym ( אֱלֹהִים / e'lo'him ) YOU(MS) ( אַתָּה / a'tah ) in~ MIDST ~us ( בְּתוֹכֵנוּ / bê'to'khey'nu ) in~ CHOSEN ( בְּמִבְחַר / bê'miv'hhar ) GRAVE ~s ~us ( קְבָרֵינוּ / qê'va'rey'nu ) !(ms)~ BURY (Verb) ( קְבֹר / qê'vor ) AT ( אֶת / et ) DIE (Verb) ~ing(ms) ~you(ms) ( מֵתֶךָ / mey'te'kha ) MAN ( אִישׁ / ish ) FROM ~us ( מִמֶּנּוּ / mi'me'nu ) AT ( אֶת / et ) GRAVE ~him ( קִבְרוֹ / qiv'ro ) NOT ( לֹא / lo ) he~ will~ RESTRICT (Verb) ( יִכְלֶה / yikh'leh ) FROM ~you(ms) ( מִמְּךָ / mim'kha ) from~ >~ BURY (Verb) ( מִקְּבֹר / mi'qe'vor ) DIE (Verb) ~ing(ms) ~you(ms) ( מֵתֶךָ / mey'te'kha )

Hear us my lord you are a captain of Elohiym in the midst of us, with the chosen of our graves bury your dead, not a man from us will restrict his grave from you from burying your dead,

7 and~ he~ will~ RISE (Verb) ( וַיָּקָם / wai'ya'qam ) Avraham ( אַבְרָהָם / av'ra'ham ) and~ he~ will~ self~ BEND.DOWN (Verb) ( וַיִּשְׁתַּחוּ / wai'yish'ta'hhu ) to~ PEOPLE ( לְעַם / lê'am ) the~ LAND ( הָאָרֶץ / ha'a'rets ) to~ SON ~s ( לִבְנֵי / liv'ney ) Hhet ( חֵת / hheyt )

and Avraham rose and bent himself down to the people of the land, to the sons of Hhet,

8 and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) AT ~them(m) ( אִתָּם / i'tam ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) IF ( אִם / im ) THERE.IS ( יֵשׁ / yesh ) AT ( אֶת / et ) SOUL ~you(ms) ( נַפְשְׁכֶם / naph'she'khem ) to~ >~ BURY (Verb) ( לִקְבֹּר / liq'bor ) AT ( אֶת / et ) DIE (Verb) ~ing(ms) ~me ( מֵתִי / mey'ti ) from~ to~ FACE ~s ~me ( מִלְּפָנַי / mi'le'pha'nai ) !(mp)~ HEAR (Verb) ~me ( שְׁמָעוּנִי / shê'ma'u'ni ) and~ !(mp)~ REACH (Verb) ( וּפִגְעוּ / u'phig'u ) to~ ~me ( לִי / li ) in~ Ephron ( בְּעֶפְרוֹן / bê'eph'ron ) SON ( בֶּן / ben ) Tsohhar ( צֹחַר / tso'hhar )

and he spoke to them saying, if it is your soul to bury my dead from before my face, hear me and reach for me with Ephron the son of Tsohhar,

9 and~ he~ will~ GIVE (Verb) ( וְיִתֶּן / wê'yi'ten ) to~ ~me ( לִי / li ) AT ( אֶת / et ) CAVE ( מְעָרַת / mê'a'rat ) the~ Makhpelah ( הַמַּכְפֵּלָה / ha'makh'pey'lah ) WHICH ( אֲשֶׁר / a'sher ) to~ ~him ( לוֹ / lo ) WHICH ( אֲשֶׁר / a'sher ) in~ EXTREMITY ( בִּקְצֵה / biq'tseyh ) FIELD ~him ( שָׂדֵהוּ / sa'dey'hu ) in~ SILVER ( בְּכֶסֶף / bê'khe'seph ) FULL ( מָלֵא / ma'ley ) he~ will~ GIVE (Verb) ~her ( יִתְּנֶנָּה / yit'ne'nah ) to~ ~me ( לִּי / li ) in~ MIDST ~you(mp) ( בְּתוֹכְכֶם / bê'tokh'khem ) to~ HOLDINGS ( לַאֲחֻזַּת / la'a'hhu'zat ) GRAVE ( קָבֶר / qa'ver )

and he will give to me the cave Makhpelah which belongs to him which is in the far end of his field, with full silver he will give to me in your midst for holdings of a grave,

10 and~ Ephron ( וְעֶפְרוֹן / wê'eph'ron ) SETTLE (Verb) ~ing(ms) ( יֹשֵׁב / yo'sheyv ) in~ MIDST ( בְּתוֹךְ / bê'tokh ) SON ~s ( בְּנֵי / bê'ney ) Hhet ( חֵת / hheyt ) and~ he~ will~ ANSWER (Verb) ( וַיַּעַן / wai'ya'an ) Ephron ( עֶפְרוֹן / eph'ron ) the~ Hhet ~of ( הַחִתִּי / ha'hhi'ti ) AT ( אֶת / et ) Avraham ( אַבְרָהָם / av'ra'ham ) in~ EAR ~s2 ( בְּאָזְנֵי / bê'az'ney ) SON ~s ( בְנֵי / vê'ney ) Hhet ( חֵת / hheyt ) to~ ALL ( לְכֹל / lê'khol ) COME (Verb) ~ing(mp) ( בָּאֵי / ba'ey ) GATE ( שַׁעַר / sha'ar ) [ 152 ] CITY ~him ( עִירוֹ / i'ro ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor )

No my lord hear me, the field I give to you and the cave which is in him I give her to you to the eyes of the sons of my people I give her to you, bury your dead,

12 and~ he~ will~ self~ BEND.DOWN (Verb) ( וַיִּשְׁתַּחוּ / wai'yish'ta'hhu ) Avraham ( אַבְרָהָם / av'ra'ham ) to~ FACE ~s ( לִפְנֵי / liph'ney ) PEOPLE ( עַם / am ) the~ LAND ( הָאָרֶץ / ha'a'rets )

and Avraham bent himself down to the face of the people of the land,

13 and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) TO ( אֶל / el ) Ephron ( עֶפְרוֹן / eph'ron ) in~ EAR ~s2 ( בְּאָזְנֵי / bê'az'ney ) PEOPLE ( עַם / am ) the~ LAND ( הָאָרֶץ / ha'a'rets ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) SURELY ( אַךְ / akh ) IF ( אִם / im ) YOU(MS) ( אַתָּה / a'tah ) WOULD.THAT ( לוּ / lu ) !(ms)~ HEAR (Verb) ~me ( שְׁמָעֵנִי / shê'ma'ey'ni ) I~ did~ GIVE (Verb) ( נָתַתִּי / na'ta'ti ) SILVER ( כֶּסֶף / ke'seph ) the~ FIELD ( הַשָּׂדֶה / ha'sa'deh ) !(ms)~ TAKE (Verb) ( קַח / qahh ) FROM ~me ( מִמֶּנִּי / mi'me'ni ) and~ I~ will~ BURY (Verb) ~& ( וְאֶקְבְּרָה / wê'eq'be'rah ) AT ( אֶת / et ) DIE (Verb) ~ing(ms) ~me ( מֵתִי / mey'ti ) THERE ~unto ( שָׁמָּה / sha'mah )

and he spoke to Ephron in the ears of the people of the land saying, surely, if you would hear me, I give the silver of the field, take from me and I will bury my dead unto there,

14 and~ he~ will~ ANSWER (Verb) ( וַיַּעַן / wai'ya'an ) Ephron ( עֶפְרוֹן / eph'ron ) AT ( אֶת / et ) Avraham ( אַבְרָהָם / av'ra'ham ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) to~ ~him ( לוֹ / lo )

and Ephron answered Avraham saying to him.

and Avraham heard Ephron and Avraham weighed to Ephron the silver which he spoke in the ears of the sons of Hhet, four hundred sheqels of silver, a crossing over for the trading,

and afterward, Avraham buried Sarah his woman to the cave of the field of Makhpelah upon the face of Mamre, she is Hhevron, in the land of Kena'an,

and the field rose, and the cave which is in him, belonging to Avraham for holdings of a grave from the sons of Hhet,
