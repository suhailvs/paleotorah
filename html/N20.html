Numbers Chapter 20

1 and~ they(m)~ will~ COME (Verb) ( וַיָּבֹאוּ / wai'ya'vo'u ) SON ~s ( בְנֵי / vê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) ALL ( כָּל / kol ) the~ COMPANY ( הָעֵדָה / ha'ey'dah ) WILDERNESS ( מִדְבַּר / mid'bar ) [ 854 ] Tsin ( צִן / tsin ) in~ the~ NEW.MOON ( בַּחֹדֶשׁ / ba'hho'desh ) the~ FIRST ( הָרִאשׁוֹן / ha'ri'shon ) and~ he~ will~ SETTLE (Verb) ( וַיֵּשֶׁב / wai'yey'shev ) the~ PEOPLE ( הָעָם / ha'am ) in~ Qadesh ( בְּקָדֵשׁ / bê'qa'deysh ) and~ she~ will~ DIE (Verb) ( וַתָּמָת / wa'ta'mat ) THERE ( שָׁם / sham ) Mir'yam ( מִרְיָם / mir'yam ) and~ she~ will~ be~ BURY (Verb) ( וַתִּקָּבֵר / wa'ti'qa'veyr ) THERE ( שָׁם / sham )

and the sons of Yisra'eyl, all the company, came to the wilderness of Tsin in the first new moon, and the people settled in Qadesh, and there Mir'yam died, and she was buried there,

2 and~ NOT ( וְלֹא / wê'lo ) he~ did~ EXIST (Verb) ( הָיָה / hai'yah ) WATER ~s2 ( מַיִם / ma'yim ) to~ the~ COMPANY ( לָעֵדָה / la'ey'dah ) and~ they(m)~ will~ be~ ASSEMBLE (Verb) ( וַיִּקָּהֲלוּ / wai'yi'qa'ha'lu ) UPON ( עַל / al ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ UPON ( וְעַל / wê'al ) Aharon ( אַהֲרֹן / a'ha'ron )

and waters did not exist for the company and they assembled upon Mosheh and upon Aharon,

3 and~ he~ will~ DISPUTE (Verb) ( וַיָּרֶב / wai'ya'rev ) the~ PEOPLE ( הָעָם / ha'am ) WITH ( עִם / im ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ they(m)~ will~ SAY (Verb) ( וַיֹּאמְרוּ / wai'yom'ru ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) and~ WOULD.THAT ( וְלוּ / wê'lu ) we~ did~ EXPIRE (Verb) ( גָוַעְנוּ / ga'wa'nu ) in~ EXPIRE (Verb) ( בִּגְוַע / big'wa ) BROTHER ~s ~us ( אַחֵינוּ / a'hhey'nu ) to~ FACE ~s ( לִפְנֵי / liph'ney ) YHWH ( יְהוָה / YHWH )

and the people disputed with Mosheh and they said, saying, and would that we expired with the expiring of our brothers to the face of YHWH,

4 and~ to~ WHAT ( וְלָמָה / wê'la'mah ) you(mp)~ did~ make~ COME (Verb) ( הֲבֵאתֶם / ha'vey'tem ) AT ( אֶת / et ) ASSEMBLY ( קְהַל / qê'hal ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) the~ WILDERNESS ( הַמִּדְבָּר / ha'mid'bar ) the~ THIS ( הַזֶּה / ha'zeh ) to~ >~ DIE (Verb) ( לָמוּת / la'mut ) THERE ( שָׁם / sham ) WE ( אֲנַחְנוּ / a'nahh'nu ) and~ CATTLE ~us ( וּבְעִירֵנוּ / uv'i'rey'nu )

and why did you bring the assembly of YHWH to this wilderness, to die there, we and our cattle,

5 and~ to~ WHAT ( וְלָמָה / wê'la'mah ) you(mp)~ did~ make~ GO.UP (Verb) ~us ( הֶעֱלִיתֻנוּ / he'e'li'tu'nu ) from~ Mits'rayim ( מִמִּצְרַיִם / mi'mits'ra'yim ) to~ >~ make~ COME (Verb) ( לְהָבִיא / lê'ha'vi ) AT ~us ( אֹתָנוּ / o'ta'nu ) TO ( אֶל / el ) the~ AREA ( הַמָּקוֹם / ha'ma'qom ) the~ DYSFUNCTIONAL ( הָרָע / ha'ra ) the~ THIS ( הַזֶּה / ha'zeh ) NOT ( לֹא / lo ) AREA ( מְקוֹם / mê'qom ) SEED ( זֶרַע / ze'ra ) and~ FIG ( וּתְאֵנָה / ut'ey'nah ) and~ GRAPEVINE ( וְגֶפֶן / wê'ge'phen ) and~ POMEGRANATE ( וְרִמּוֹן / wê'ri'mon ) and~ WATER ~s2 ( וּמַיִם / u'ma'yim ) WITHOUT ( אַיִן / a'yin ) to~ >~ GULP (Verb) ( לִשְׁתּוֹת / lish'tot )

and why did you bring us up from Mits'rayim, to bring us to this area of dysfunction, not an area of seed and fig and grapevine and pomegranate and without waters to gulp,

6 and~ he~ will~ COME (Verb) ( וַיָּבֹא / wai'ya'vo ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ Aharon ( וְאַהֲרֹן / wê'a'ha'ron ) from~ FACE ~s ( מִפְּנֵי / mip'ney ) the~ ASSEMBLY ( הַקָּהָל / ha'qa'hal ) TO ( אֶל / el ) OPENING ( פֶּתַח / pe'tahh ) TENT ( אֹהֶל / o'hel ) APPOINTED ( מוֹעֵד / mo'eyd ) and~ they(m)~ will~ FALL (Verb) ( וַיִּפְּלוּ / wai'yip'lu ) UPON ( עַל / al ) FACE ~s ~them(m) ( פְּנֵיהֶם / pê'ney'hem ) and~ he~ will~ be~ SEE (Verb) ( וַיֵּרָא / wai'yey'ra ) ARMAMENT ( כְבוֹד / khê'vod ) YHWH ( יְהוָה / YHWH ) TO ~them(m) ( אֲלֵיהֶם / a'ley'hem )

and Mosheh came, and Aharon, from the face of the assembly to the opening of the appointed tent, and they fell upon their faces, and the armament of YHWH appeared to them,

7 and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) to~ >~ SAY (Verb) ( לֵּאמֹר / ley'mor )

and YHWH spoke to Mosheh saying,

8 !(ms)~ TAKE (Verb) ( קַח / qahh ) AT ( אֶת / et ) the~ BRANCH ( הַמַּטֶּה / ha'ma'teh ) and~ !(ms)~ make~ ASSEMBLE (Verb) ( וְהַקְהֵל / wê'haq'heyl ) AT ( אֶת / et ) the~ COMPANY ( הָעֵדָה / ha'ey'dah ) YOU(MS) ( אַתָּה / a'tah ) and~ Aharon ( וְאַהֲרֹן / wê'a'ha'ron ) BROTHER ~you(ms) ( אָחִיךָ / a'hhi'kha ) and~ you(mp)~ did~ much~ SPEAK (Verb) ( וְדִבַּרְתֶּם / wê'di'bar'tem ) TO ( אֶל / el ) the~ CLIFF ( הַסֶּלַע / ha'se'la ) to~ EYE ~s2 ~them(m) ( לְעֵינֵיהֶם / lê'ey'ney'hem ) and~ he~ did~ GIVE (Verb) ( וְנָתַן / wê'na'tan ) WATER ~s2 ~him ( מֵימָיו / mey'maw ) and~ you(ms)~ did~ make~ GO.OUT (Verb) ( וְהוֹצֵאתָ / wê'ho'tsey'ta ) to~ ~them(m) ( לָהֶם / la'hem ) WATER ~s2 ( מַיִם / ma'yim ) FROM ( מִן / min ) the~ CLIFF ( הַסֶּלַע / ha'se'la ) and~ you(ms)~ did~ make~ DRINK (Verb) ( וְהִשְׁקִיתָ / wê'hish'qi'ta ) AT ( אֶת / et ) the~ COMPANY ( הָעֵדָה / ha'ey'dah ) and~ AT ( וְאֶת / wê'et ) CATTLE ~them(m) ( בְּעִירָם / bê'i'ram )

Take the branch and assemble the company, you and Aharon your brother, and you will speak to the cliff before their eyes, and he will give his waters, and you will bring to them the waters from the cliff, and you will make the company and their cattle drink,

9 and~ he~ will~ TAKE (Verb) ( וַיִּקַּח / wai'yi'qahh ) Mosheh ( מֹשֶׁה / mo'sheh ) AT ( אֶת / et ) the~ BRANCH ( הַמַּטֶּה / ha'ma'teh ) from~ to~ FACE ~s ( מִלִּפְנֵי / mi'liph'ney ) YHWH ( יְהוָה / YHWH ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ did~ much~ DIRECT (Verb) ~him ( צִוָּהוּ / tsi'wa'hu )

and Mosheh took the branch from before the face of YHWH, just as he directed him,

10 and~ they(m)~ will~ make~ ASSEMBLE (Verb) ( וַיַּקְהִלוּ / wai'yaq'hi'lu ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ Aharon ( וְאַהֲרֹן / wê'a'ha'ron ) AT ( אֶת / et ) the~ ASSEMBLY ( הַקָּהָל / ha'qa'hal ) TO ( אֶל / el ) FACE ~s ( פְּנֵי / pê'ney ) the~ CLIFF ( הַסָּלַע / ha'sa'la ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ ~them(m) ( לָהֶם / la'hem ) !(mp)~ HEAR (Verb) ( שִׁמְעוּ / shim'u ) PLEASE ( נָא / na ) the~ DISOBEY (Verb) ~ing(mp) ( הַמֹּרִים / ha'mo'rim ) ?~ FROM ( הֲמִן / ha'min ) the~ CLIFF ( הַסֶּלַע / ha'se'la ) the~ THIS ( הַזֶּה / ha'zeh ) we~ will~ make~ GO.OUT (Verb) ( נוֹצִיא / no'tsi ) to~ ~you(mp) ( לָכֶם / la'khem ) WATER ~s2 ( מָיִם / ma'yim )

and Mosheh and Aharon made the assembly assemble to the face of the cliff, and he said to them, the ones disobeying, please hear, will we bring out to you waters from this cliff,

11 and~ he~ will~ make~ RAISE.UP (Verb) ( וַיָּרֶם / wai'ya'rem ) Mosheh ( מֹשֶׁה / mo'sheh ) AT ( אֶת / et ) HAND ~him ( יָדוֹ / ya'do ) and~ he~ will~ make~ HIT (Verb) ( וַיַּךְ / wai'yakh ) AT ( אֶת / et ) the~ CLIFF ( הַסֶּלַע / ha'se'la ) in~ BRANCH ~him ( בְּמַטֵּהוּ / bê'ma'tey'hu ) FOOTSTEP ~s2 ( פַּעֲמָיִם / pa'a'ma'yim ) [ 855 ] and~ they(m)~ will~ GO.OUT (Verb) ( וַיֵּצְאוּ / wai'yeyts'u ) WATER ~s2 ( מַיִם / ma'yim ) ABUNDANT ~s ( רַבִּים / ra'bim ) and~ she~ will~ GULP (Verb) ( וַתֵּשְׁתְּ / wa'teysh'te ) the~ COMPANY ( הָעֵדָה / ha'ey'dah ) and~ CATTLE ~them(m) ( וּבְעִירָם / uv'i'ram )

and Mosheh raised up his hand and he hit the cliff with his branch two times, and abundant waters came out, and the company and the cattle gulped,

12 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ TO ( וְאֶל / wê'el ) Aharon ( אַהֲרֹן / a'ha'ron ) SEEING.AS ( יַעַן / ya'an ) NOT ( לֹא / lo ) you(mp)~ did~ make~ SECURE (Verb) ( הֶאֱמַנְתֶּם / he'e'man'tem ) in~ ~me ( בִּי / bi ) to~ >~ make~ SET.APART (Verb) ~me ( לְהַקְדִּישֵׁנִי / lê'haq'di'shey'ni ) to~ EYE ~s2 ( לְעֵינֵי / lê'ey'ney ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) to~ SO ( לָכֵן / la'kheyn ) NOT ( לֹא / lo ) you(mp)~ will~ make~ COME (Verb) ( תָבִיאוּ / ta'vi'u ) AT ( אֶת / et ) the~ ASSEMBLY ( הַקָּהָל / ha'qa'hal ) the~ THIS ( הַזֶּה / ha'zeh ) TO ( אֶל / el ) the~ LAND ( הָאָרֶץ / ha'a'rets ) WHICH ( אֲשֶׁר / a'sher ) I~ did~ GIVE (Verb) ( נָתַתִּי / na'ta'ti ) to~ ~them(m) ( לָהֶם / la'hem )

and YHWH said to Mosheh and to Aharon, seeing as you did not support me, to make me set apart to the eyes of the sons of Yisra'eyl, because of this you will not bring this assembly to the land which I gave to them.

13 THEY(m) ( הֵמָּה / hey'mah ) WATER ~s2 ( מֵי / mey ) Meriyvah ( מְרִיבָה / mê'ri'vah ) WHICH ( אֲשֶׁר / a'sher ) they~ did~ DISPUTE (Verb) ( רָבוּ / ra'vu ) SON ~s ( בְנֵי / vê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) AT ( אֶת / et ) YHWH ( יְהוָה / YHWH ) and~ he~ will~ be~ SET.APART (Verb) ( וַיִּקָּדֵשׁ / wai'yi'qa'deysh ) in~ ~them(m) ( בָּם / bam )

This is the waters of Meriyvah, because the sons of Yisra'eyl disputed with YHWH, and he was set apart in them,

14 and~ he~ will~ SEND (Verb) ( וַיִּשְׁלַח / wai'yish'lahh ) Mosheh ( מֹשֶׁה / mo'sheh ) MESSENGER ~s ( מַלְאָכִים / mal'a'khim ) from~ Qadesh ( מִקָּדֵשׁ / mi'qa'deysh ) TO ( אֶל / el ) KING ( מֶלֶךְ / me'lekh ) Edom ( אֱדוֹם / e'dom ) IN.THIS.WAY ( כֹּה / koh ) he~ did~ SAY (Verb) ( אָמַר / a'mar ) BROTHER ~you(ms) ( אָחִיךָ / a'hhi'kha ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) YOU(MS) ( אַתָּה / a'tah ) you(ms)~ did~ KNOW (Verb) ( יָדַעְתָּ / ya'da'ta ) AT ( אֵת / eyt ) ALL ( כָּל / kol ) the~ TROUBLE ( הַתְּלָאָה / hat'la'ah ) WHICH ( אֲשֶׁר / a'sher ) she~ did~ FIND (Verb) ~us ( מְצָאָתְנוּ / mê'tsa'at'nu )

and Mosheh sent messengers from Qadesh to the king of Edom, in this way your brother Yisra'eyl said, you, you know all the trouble which has found us,

15 and~ they(m)~ will~ GO.DOWN (Verb) ( וַיֵּרְדוּ / wai'yeyr'du ) FATHER ~s ~us ( אֲבֹתֵינוּ / a'vo'tey'nu ) Mits'rayim ~unto ( מִצְרַיְמָה / mits'rai'mah ) and~ we~ will~ SETTLE (Verb) ( וַנֵּשֶׁב / wa'ney'shev ) in~ Mits'rayim ( בְּמִצְרַיִם / bê'mits'ra'yim ) DAY ~s ( יָמִים / ya'mim ) ABUNDANT ~s ( רַבִּים / ra'bim ) and~ they(m)~ will~ make~ BE.DYSFUNCTIONAL (Verb) ( וַיָּרֵעוּ / wai'ya'rey'u ) to~ ~us ( לָנוּ / la'nu ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) and~ to~ FATHER ~s ~us ( וְלַאֲבֹתֵינוּ / wê'la'a'vo'tey'nu )

and our fathers went down unto Mits'rayim, and we settled in Mits'rayim an abundance of days, and Mits'rayim did dysfunction to us and to our fathers,

16 and~ we~ will~ CRY.OUT (Verb) ( וַנִּצְעַק / wa'nits'aq ) TO ( אֶל / el ) YHWH ( יְהוָה / YHWH ) and~ he~ will~ HEAR (Verb) ( וַיִּשְׁמַע / wai'yish'ma ) VOICE ~us ( קֹלֵנוּ / qo'ley'nu ) and~ he~ will~ SEND (Verb) ( וַיִּשְׁלַח / wai'yish'lahh ) MESSENGER ( מַלְאָךְ / mal'akh ) and~ he~ will~ make~ GO.OUT (Verb) ~us ( וַיֹּצִאֵנוּ / wai'yo'tsi'ey'nu ) from~ Mits'rayim ( מִמִּצְרָיִם / mi'mits'ra'yim ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) WE ( אֲנַחְנוּ / a'nahh'nu ) in~ Qadesh ( בְקָדֵשׁ / vê'qa'deysh ) CITY ( עִיר / ir ) EXTREMITY ( קְצֵה / qê'tseyh ) BORDER ~you(ms) ( גְבוּלֶךָ / gê'vu'le'kha )

and we cried out to YHWH and he heard our voice, and he sent a messenger, and he brought us out from Mits'rayim, and look, we are in Qadesh, a city at the extremity of your border.

17 we~ will~ CROSS.OVER (Verb) ~& ( נַעְבְּרָה / na'be'rah ) PLEASE ( נָּא / na ) in~ LAND ~you(ms) ( בְאַרְצֶךָ / vê'ar'tse'kha ) NOT ( לֹא / lo ) we~ will~ CROSS.OVER (Verb) ( נַעֲבֹר / na'a'vor ) in~ FIELD ( בְּשָׂדֶה / bê'sa'deh ) and~ in~ VINEYARD ( וּבְכֶרֶם / uv'khe'rem ) and~ NOT ( וְלֹא / wê'lo ) we~ will~ GULP (Verb) ( נִשְׁתֶּה / nish'teh ) WATER ~s2 ( מֵי / mey ) WELL ( בְאֵר / ve'eyr ) ROAD ( דֶּרֶךְ / de'rekh ) the~ KING ( הַמֶּלֶךְ / ha'me'lekh ) we~ will~ WALK (Verb) ( נֵלֵךְ / ney'leykh ) NOT ( לֹא / lo ) we~ will~ EXTEND (Verb) ( נִטֶּה / ni'teh ) RIGHT.HAND ( יָמִין / ya'min ) and~ LEFT.HAND ( וּשְׂמֹאול / us'mo'ul ) [ 856 ] UNTIL ( עַד / ad ) WHICH ( אֲשֶׁר / a'sher ) we~ will~ CROSS.OVER (Verb) ( נַעֲבֹר / na'a'vor ) BORDER ~you(ms) ( גְּבוּלֶךָ / gê'vu'le'kha )

We will cross over please in your land, we will not cross over in the field or in the vineyard, and we will not gulp waters of a well, the road of the king we will walk, we will not extend the right hand or the left hand until when we cross over your border,

18 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) TO ~him ( אֵלָיו / ey'law ) Edom ( אֱדוֹם / e'dom ) NOT ( לֹא / lo ) you(ms)~ will~ CROSS.OVER (Verb) ( תַעֲבֹר / ta'a'vor ) in~ ~me ( בִּי / bi ) OTHERWISE ( פֶּן / pen ) in~ the~ SWORD ( בַּחֶרֶב / ba'hhe'rev ) I~ will~ GO.OUT (Verb) ( אֵצֵא / ey'tsey ) to~ >~ MEET (Verb) ~you(ms) ( לִקְרָאתֶךָ / liq'ra'te'kha )

and Edom said to him, you will not cross over in me, otherwise with the sword I will go out to meet you,
