and Ya'aqov settled in the land of the immigration of his father, in the land of Kena'an.

2 THESE ( אֵלֶּה / ey'leh ) BIRTHING ~s ( תֹּלְדוֹת / tol'dot ) Ya'aqov ( יַעֲקֹב / ya'a'qov ) Yoseph ( יוֹסֵף / yo'seyph ) SON ( בֶּן / ben ) SEVEN ( שְׁבַע / shê'va ) TEN ( עֶשְׂרֵה / es'reyh ) YEAR ( שָׁנָה / sha'nah ) he~ did~ EXIST (Verb) ( הָיָה / hai'yah ) FEED (Verb) ~ing(ms) ( רֹעֶה / ro'eh ) AT ( אֶת / et ) BROTHER ~s ~him ( אֶחָיו / e'hhaw ) in~ the~ FLOCKS ( בַּצֹּאן / ba'tson ) and~ HE ( וְהוּא / wê'hu ) YOUNG.MAN ( נַעַר / na'ar ) AT ( אֶת / et ) SON ~s ( בְּנֵי / bê'ney ) Bilhah ( בִלְהָה / vil'hah ) and~ AT ( וְאֶת / wê'et ) SON ~s ( בְּנֵי / bê'ney ) Zilpah ( זִלְפָּה / zil'pah ) WOMAN ~s ( נְשֵׁי / nê'shey ) FATHER ~him ( אָבִיו / a'viw ) and~ he~ will~ make~ COME (Verb) ( וַיָּבֵא / wai'ya'vey ) Yoseph ( יוֹסֵף / yo'seyph ) AT ( אֶת / et ) SLANDER ~them(m) ( דִּבָּתָם / di'ba'tam ) DYSFUNCTIONAL ( רָעָה / ra'ah ) TO ( אֶל / el ) FATHER ~them(m) ( אֲבִיהֶם / a'vi'hem )

These are the birthings of Ya'aqov, Yoseph, the son of seventeen years, he existed the feeder with his brothers in the flocks and he was a young man with the sons of Bilhah and with the sons of Zilpah, the women of his father, and Yoseph brought their dysfunctional slander to their father,

and Yisra'eyl had loved Yoseph out of all his sons given that he was the son of his extreme old age and he made for him a tunic of wrists,

and he said to them, please hear this dream which I dreamed,

7 and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) WE ( אֲנַחְנוּ / a'nahh'nu ) much~ BIND.UP (Verb) ~ing(mp) ( מְאַלְּמִים / mê'al'mim ) BOUND.SHEAF ~s ( אֲלֻמִּים / a'lu'mim ) in~ MIDST ( בְּתוֹךְ / bê'tokh ) the~ FIELD ( הַשָּׂדֶה / ha'sa'deh ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) she~ did~ RISE (Verb) ( קָמָה / qa'mah ) BOUND.SHEAF ~me ( אֲלֻמָּתִי / a'lu'ma'ti ) and~ ALSO ( וְגַם / wê'gam ) she~ did~ be~ STAND.UP (Verb) ( נִצָּבָה / ni'tsa'vah ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) they(f)~ did~ GO.AROUND (Verb) ( תְסֻבֶּינָה / tê'su'bey'nah ) BOUND.SHEAF ~s ~you(mp) ( אֲלֻמֹּתֵיכֶם / a'lu'mo'tey'khem ) and~ they(f)~ will~ self~ BEND.DOWN (Verb) ( וַתִּשְׁתַּחֲוֶיןָ / wa'tish'ta'hha'wey'na ) to~ BOUND.SHEAF ~me ( לַאֲלֻמָּתִי / la'a'lu'ma'ti )

and he recounted it to his father and to his brothers and his father rebuked him and he said to him, what is this dream which you dreamed, will I and your mother and your brothers come to bend ourselves down to you unto the land,
