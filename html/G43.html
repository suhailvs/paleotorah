and if you will not send, we will not go down given that the man had said to us, you will not see my face except your brother be with you,

6 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) to~ WHAT ( לָמָה / la'mah ) you(mp)~ did~ make~ BE.DYSFUNCTIONAL (Verb) ( הֲרֵעֹתֶם / ha'rey'o'tem ) to~ ~me ( לִי / li ) to~ >~ make~ BE.FACE.TO.FACE (Verb) ( לְהַגִּיד / lê'ha'gid ) to~ the~ MAN ( לָאִישׁ / la'ish ) ?~ YET.AGAIN ( הַעוֹד / ha'od ) to~ ~you(mp) ( לָכֶם / la'khem ) BROTHER ( אָח / ahh )

and Yisra'eyl said, why did you make me dysfunctional, telling to the man you had another brother,

and they said, the man greatly inquired about us and about our kindred saying, is your father yet alive and is there to you a brother, and we told to him by the mouth of these words, could we certainly know that he would say, bring down your brother,

8 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) Yehudah ( יְהוּדָה / yê'hu'dah ) TO ( אֶל / el ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) FATHER ~him ( אָבִיו / a'viw ) !(ms)~ SEND (Verb) ~& ( שִׁלְחָה / shil'hhah ) the~ YOUNG.MAN ( הַנַּעַר / ha'na'ar ) AT ~me ( אִתִּי / i'ti ) and~ we~ will~ RISE (Verb) ~& ( וְנָקוּמָה / wê'na'qu'mah ) and~ we~ will~ WALK (Verb) ~& ( וְנֵלֵכָה / wê'ney'ley'khah ) and~ we~ will~ LIVE (Verb) ( וְנִחְיֶה / wê'nihh'yeh ) and~ NOT ( וְלֹא / wê'lo ) we~ will~ DIE (Verb) ( נָמוּת / na'mut ) ALSO ( גַּם / gam ) WE ( אֲנַחְנוּ / a'nahh'nu ) ALSO ( גַם / gam ) YOU(MS) ( אַתָּה / a'tah ) ALSO ( גַּם / gam ) BABIES ~us ( טַפֵּנוּ / ta'pey'nu )

and Yehudah said to Yisra'eyl his father, send the young man with me and we will rise and we will walk and we will live and we will not die, also us, also you, also our babies.

and take double the silver in your hand and the returned silver in the mouth of your bag you will return in your hand, possibly he is a mistake,
