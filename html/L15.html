Leviticus Chapter 15

1 and~ he~ will~ much~ SPEAK (Verb) ( וַיְדַבֵּר / wai'da'beyr ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) Mosheh ( מֹשֶׁה / mo'sheh ) and~ TO ( וְאֶל / wê'el ) Aharon ( אַהֲרֹן / a'ha'ron ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor )

and YHWH spoke to Mosheh and to Aharon saying,

2 !(mp)~ much~ SPEAK (Verb) ( דַּבְּרוּ / da'be'ru ) TO ( אֶל / el ) SON ~s ( בְּנֵי / bê'ney ) Yisra'eyl ( יִשְׂרָאֵל / yis'ra'eyl ) and~ you(mp)~ did~ SAY (Verb) ( וַאֲמַרְתֶּם / wa'a'mar'tem ) TO ~them(m) ( אֲלֵהֶם / a'ley'hem ) MAN ( אִישׁ / ish ) MAN ( אִישׁ / ish ) GIVEN.THAT ( כִּי / ki ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) ISSUE (Verb) ~ing(ms) ( זָב / zav ) from~ FLESH ~him ( מִבְּשָׂרוֹ / mi'be'sa'ro ) DISCHARGE ~him ( זוֹבוֹ / zo'vo ) DIRTY ( טָמֵא / ta'mey ) HE ( הוּא / hu )

Speak to the sons of Yisra'eyl and you will say to them, each man that exists with an issuing from a discharge of his flesh, he is dirty,

3 and~ THIS ( וְזֹאת / wê'zot ) she~ will~ EXIST (Verb) ( תִּהְיֶה / tih'yeh ) DIRTY ~him ( טֻמְאָתוֹ / tu'm'a'to ) in~ DISCHARGE ~him ( בְּזוֹבוֹ / bê'zo'vo ) he~ did~ FLOW.OUT (Verb) ( רָר / rar ) FLESH ~him ( בְּשָׂרוֹ / bê'sar'o ) AT ( אֶת / et ) DISCHARGE ~him ( זוֹבוֹ / zo'vo ) OR ( אוֹ / o ) he~ did~ make~ SEAL (Verb) ( הֶחְתִּים / hehh'tim ) FLESH ~him ( בְּשָׂרוֹ / bê'sar'o ) from~ DISCHARGE ~him ( מִזּוֹבוֹ / mi'zo'vo ) DIRTY ~him ( טֻמְאָתוֹ / tu'm'a'to ) SHE ( הִוא / hi )

and this will be his dirtiness with his discharge, his flesh flowed out with his discharge or his flesh was sealed from his discharge, she is his dirtiness.

4 ALL ( כָּל / kol ) the~ LYING.PLACE ( הַמִּשְׁכָּב / ha'mish'kav ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ LIE.DOWN (Verb) ( יִשְׁכַּב / yish'kav ) UPON ~him ( עָלָיו / a'law ) the~ ISSUE (Verb) ~ing(ms) ( הַזָּב / ha'zav ) [ 710 ] he~ will~ BE.DIRTY (Verb) ( יִטְמָא / yit'ma ) and~ ALL ( וְכָל / wê'khol ) the~ UTENSIL ( הַכְּלִי / hak'li ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ SETTLE (Verb) ( יֵשֵׁב / yey'sheyv ) UPON ~him ( עָלָיו / a'law ) he~ will~ BE.DIRTY (Verb) ( יִטְמָא / yit'ma )

Every lying place where he lies down upon with the issuing, will be dirty, and every utensil which he settles upon will be dirty,

5 and~ MAN ( וְאִישׁ / wê'ish ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ TOUCH (Verb) ( יִגַּע / yiga ) in~ LYING.PLACE ~him ( בְּמִשְׁכָּבוֹ / bê'mish'ka'vo ) he~ will~ WASH (Verb) ( יְכַבֵּס / yê'kha'beys ) GARMENT ~s ~him ( בְּגָדָיו / bê'ga'daw ) and~ he~ did~ BATHE (Verb) ( וְרָחַץ / wê'ra'hhats ) in~ the~ WATER ~s2 ( בַּמַּיִם / ba'ma'yim ) and~ he~ did~ BE.DIRTY (Verb) ( וְטָמֵא / wê'ta'mey ) UNTIL ( עַד / ad ) the~ EVENING ( הָעָרֶב / ha'a'rev )

and a man which touches his lying place will wash his garments and he will bathe in the waters and he will be dirty until evening,

6 and~ the~ SETTLE (Verb) ~ing(ms) ( וְהַיֹּשֵׁב / wê'hai'yo'sheyv ) UPON ( עַל / al ) the~ UTENSIL ( הַכְּלִי / hak'li ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ SETTLE (Verb) ( יֵשֵׁב / yey'sheyv ) UPON ~him ( עָלָיו / a'law ) the~ ISSUE (Verb) ~ing(ms) ( הַזָּב / ha'zav ) he~ will~ WASH (Verb) ( יְכַבֵּס / yê'kha'beys ) GARMENT ~s ~him ( בְּגָדָיו / bê'ga'daw ) and~ he~ did~ BATHE (Verb) ( וְרָחַץ / wê'ra'hhats ) in~ the~ WATER ~s2 ( בַּמַּיִם / ba'ma'yim ) and~ he~ did~ BE.DIRTY (Verb) ( וְטָמֵא / wê'ta'mey ) UNTIL ( עַד / ad ) the~ EVENING ( הָעָרֶב / ha'a'rev )

and the one settling upon the utensil, who settles upon him with an issuing, he will wash his garments and he will bathe in the waters and he will be dirty until the evening,

7 and~ the~ TOUCH (Verb) ~ing(ms) ( וְהַנֹּגֵעַ / wê'ha'no'gey'a ) in~ FLESH ( בִּבְשַׂר / biv'sar ) the~ ISSUE (Verb) ~ing(ms) ( הַזָּב / ha'zav ) he~ will~ WASH (Verb) ( יְכַבֵּס / yê'kha'beys ) GARMENT ~s ~him ( בְּגָדָיו / bê'ga'daw ) and~ he~ did~ BATHE (Verb) ( וְרָחַץ / wê'ra'hhats ) in~ the~ WATER ~s2 ( בַּמַּיִם / ba'ma'yim ) and~ he~ did~ BE.DIRTY (Verb) ( וְטָמֵא / wê'ta'mey ) UNTIL ( עַד / ad ) the~ EVENING ( הָעָרֶב / ha'a'rev )

and the one touching the flesh with the issuing, he will wash his garments and he will bathe in the waters and he will be dirty until the evening,

8 and~ GIVEN.THAT ( וְכִי / wê'khi ) >~ SPIT (Verb) ( יָרֹק / ya'roq ) the~ ISSUE (Verb) ~ing(ms) ( הַזָּב / ha'zav ) in~ the~ CLEAN ( בַּטָּהוֹר / ba'ta'hor ) [ 711 ] and~ he~ did~ much~ WASH (Verb) ( וְכִבֶּס / wê'khi'bes ) GARMENT ~s ~him ( בְּגָדָיו / bê'ga'daw ) and~ he~ did~ BATHE (Verb) ( וְרָחַץ / wê'ra'hhats ) in~ the~ WATER ~s2 ( בַּמַּיִם / ba'ma'yim ) and~ he~ did~ BE.DIRTY (Verb) ( וְטָמֵא / wê'ta'mey ) UNTIL ( עַד / ad ) the~ EVENING ( הָעָרֶב / ha'a'rev )

and if the issuing spit on the clean one, then he will wash his garments and he will bathe in the waters and he will be dirty until the evening,

9 and~ ALL ( וְכָל / wê'khol ) the~ SADDLE ( הַמֶּרְכָּב / ha'mer'kav ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ RIDE (Verb) ( יִרְכַּב / yir'kav ) UPON ~him ( עָלָיו / a'law ) the~ ISSUE (Verb) ~ing(ms) ( הַזָּב / ha'zav ) he~ will~ BE.DIRTY (Verb) ( יִטְמָא / yit'ma )

and every saddle which he rides upon with the issuing will be dirty,

10 and~ ALL ( וְכָל / wê'khol ) the~ TOUCH (Verb) ~ing(ms) ( הַנֹּגֵעַ / ha'no'gey'a ) in~ ALL ( בְּכֹל / bê'khol ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ EXIST (Verb) ( יִהְיֶה / yih'yeh ) UNDER ~s ~him ( תַחְתָּיו / tahh'taw ) he~ will~ BE.DIRTY (Verb) ( יִטְמָא / yit'ma ) UNTIL ( עַד / ad ) the~ EVENING ( הָעָרֶב / ha'a'rev ) and~ the~ LIFT.UP (Verb) ~ing(ms) ( וְהַנּוֹשֵׂא / wê'ha'no'sey ) AT ~them(m) ( אוֹתָם / o'tam ) he~ will~ WASH (Verb) ( יְכַבֵּס / yê'kha'beys ) GARMENT ~s ~him ( בְּגָדָיו / bê'ga'daw ) and~ he~ did~ BATHE (Verb) ( וְרָחַץ / wê'ra'hhats ) in~ the~ WATER ~s2 ( בַּמַּיִם / ba'ma'yim ) and~ he~ did~ BE.DIRTY (Verb) ( וְטָמֵא / wê'ta'mey ) UNTIL ( עַד / ad ) the~ EVENING ( הָעָרֶב / ha'a'rev )

and every one touching anything which will exist under him will be dirty until the evening, and the one lifting them up will wash his garments and he will bathe in the waters and he will be dirty until the evening,

11 and~ ALL ( וְכֹל / wê'khol ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ TOUCH (Verb) ( יִגַּע / yiga ) in~ ~him ( בּוֹ / bo ) the~ ISSUE (Verb) ~ing(ms) ( הַזָּב / ha'zav ) and~ HAND ~s2 ~him ( וְיָדָיו / wê'ya'daw ) NOT ( לֹא / lo ) he~ did~ FLUSH (Verb) ( שָׁטַף / sha'taph ) in~ the~ WATER ~s2 ( בַּמָּיִם / ba'ma'yim ) and~ he~ did~ much~ WASH (Verb) ( וְכִבֶּס / wê'khi'bes ) GARMENT ~s ~him ( בְּגָדָיו / bê'ga'daw ) and~ he~ did~ BATHE (Verb) ( וְרָחַץ / wê'ra'hhats ) in~ the~ WATER ~s2 ( בַּמַּיִם / ba'ma'yim ) and~ he~ did~ BE.DIRTY (Verb) ( וְטָמֵא / wê'ta'mey ) UNTIL ( עַד / ad ) the~ EVENING ( הָעָרֶב / ha'a'rev )

and every one he touches with the issuing and did not flush his hands in the waters, then he will wash his garments and he will bath in the waters and he will be dirty until the evening,

12 and~ UTENSIL ( וּכְלִי / ukh'li ) CLAY ( חֶרֶשׂ / hhe'res ) WHICH ( אֲשֶׁר / a'sher ) he~ will~ TOUCH (Verb) ( יִגַּע / yiga ) in~ ~him ( בּוֹ / bo ) the~ ISSUE (Verb) ~ing(ms) ( הַזָּב / ha'zav ) he~ will~ be~ CRACK (Verb) ( יִשָּׁבֵר / yi'sha'veyr ) and~ ALL ( וְכָל / wê'khol ) UTENSIL ( כְּלִי / kê'li ) TREE ( עֵץ / eyts ) he~ will~ be~ FLUSH (Verb) ( יִשָּׁטֵף / yi'sha'teyph ) in~ the~ WATER ~s2 ( בַּמָּיִם / ba'ma'yim )

and a utensil of clay which he touches with the issuing will be cracked and every utensil of wood will be flushed in the waters,

and if the one with the issuing will be clean from his discharge, then he will count to himself seven days for his cleanness, then he will wash his garments and he will bathe his flesh in living waters and he will be clean,

14 and~ in~ the~ DAY ( וּבַיּוֹם / u'vai'yom ) the~ EIGHTH ( הַשְּׁמִינִי / hash'mini ) he~ will~ TAKE (Verb) ( יִקַּח / yi'qahh ) to~ ~him ( לוֹ / lo ) TWO ( שְׁתֵּי / shê'tey ) TURTLEDOVE ~s ( תֹרִים / to'rim ) OR ( אוֹ / o ) TWO ( שְׁנֵי / shê'ney ) SON ~s ( בְּנֵי / bê'ney ) DOVE ( יוֹנָה / yo'nah ) and~ he~ did~ COME (Verb) ( וּבָא / u'va ) to~ FACE ~s ( לִפְנֵי / liph'ney ) YHWH ( יְהוָה / YHWH ) TO ( אֶל / el ) OPENING ( פֶּתַח / pe'tahh ) TENT ( אֹהֶל / o'hel ) APPOINTED ( מוֹעֵד / mo'eyd ) and~ he~ did~ GIVE (Verb) ~them(m) ( וּנְתָנָם / un'ta'nam ) TO ( אֶל / el ) the~ ADMINISTRATOR ( הַכֹּהֵן / ha'ko'heyn )

and in the eighth day he will take for himself two turtledoves or two sons of the dove, and he will come to the face of YHWH, to the opening of the appointed tent, and he will give them to the administrator,

and the administrator will do them, one is the failure and the other one is the ascension offering, and the administrator will make a covering upon him, to the face of YHWH because of his discharge,
