Genesis Chapter 40

1 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) AFTER ( אַחַר / a'hhar ) the~ WORD ~s ( הַדְּבָרִים / ha'de'va'rim ) the~ THESE ( הָאֵלֶּה / ha'ey'leh ) they~ did~ FAIL (Verb) ( חָטְאוּ / hhat'u ) make~ DRINK (Verb) ~ing(ms) ( מַשְׁקֵה / mash'qeyh ) KING ( מֶלֶךְ / me'lekh ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) and~ the~ BAKE (Verb) ~ing(ms) ( וְהָאֹפֶה / wê'ha'o'pheh ) to~ LORD ~s ~them(m) ( לַאֲדֹנֵיהֶם / la'a'do'ney'hem ) [ 195 ] to~ KING ( לְמֶלֶךְ / lê'me'lekh ) Mits'rayim ( מִצְרָיִם / mits'ra'yim )

and it came to pass after these words, the drinker of the king of Mits'rayim and the baker failed their lord, to the king of Mits'rayim,

2 and~ he~ will~ SNAP (Verb) ( וַיִּקְצֹף / wai'yiq'tsoph ) Paroh ( פַּרְעֹה / par'oh ) UPON ( עַל / al ) TWO ( שְׁנֵי / shê'ney ) EUNUCH ~s ~him ( סָרִיסָיו / sa'ri'saw ) UPON ( עַל / al ) NOBLE ( שַׂר / sar ) the~ make~ DRINK (Verb) ~ing(mp) ( הַמַּשְׁקִים / ha'mash'qim ) and~ UPON ( וְעַל / wê'al ) NOBLE ( שַׂר / sar ) the~ BAKE (Verb) ~ing(mp) ( הָאוֹפִים / ha'o'phim )

and Paroh snapped upon his two eunuchs, upon the noble of the drinkers and upon the noble of the bakers,

3 and~ he~ will~ GIVE (Verb) ( וַיִּתֵּן / wai'yi'teyn ) AT ~them(m) ( אֹתָם / o'tam ) in~ CUSTODY ( בְּמִשְׁמַר / bê'mish'mar ) HOUSE ( בֵּית / beyt ) NOBLE ( שַׂר / sar ) the~ SLAUGHTERING ~s ( הַטַּבָּחִים / ha'ta'ba'hhim ) TO ( אֶל / el ) HOUSE ( בֵּית / beyt ) the~ PRISON ( הַסֹּהַר / ha'so'har ) AREA ( מְקוֹם / mê'qom ) WHICH ( אֲשֶׁר / a'sher ) Yoseph ( יוֹסֵף / yo'seyph ) TIE.UP (Verb) ~ed(ms) ( אָסוּר / a'sur ) THERE ( שָׁם / sham )

and he gave them in the custody of the house of the noble of the slaughtering ones, to the house of the prison, the place where Yoseph was tied up there,

4 and~ he~ will~ REGISTER (Verb) ( וַיִּפְקֹד / wai'yiph'qod ) NOBLE ( שַׂר / sar ) the~ SLAUGHTERING ~s ( הַטַּבָּחִים / ha'ta'ba'hhim ) AT ( אֶת / et ) Yoseph ( יוֹסֵף / yo'seyph ) AT ~them(m) ( אִתָּם / i'tam ) [ 196 ] and~ he~ will~ much~ MINISTER (Verb) ( וַיְשָׁרֶת / wa'ye'sha'ret ) AT ~them(m) ( אֹתָם / o'tam ) and~ they(m)~ will~ EXIST (Verb) ( וַיִּהְיוּ / wai'yih'yu ) DAY ~s ( יָמִים / ya'mim ) in~ CUSTODY ( בְּמִשְׁמָר / bê'mish'mar )

and the noble of the slaughtering ones set Yoseph over them and he ministered them and they existed days in custody,

5 and~ they(m)~ will~ DREAM (Verb) ( וַיַּחַלְמוּ / wai'ya'hhal'mu ) DREAM ( חֲלוֹם / hha'lom ) TWO ~them(m) ( שְׁנֵיהֶם / shê'ney'hem ) MAN ( אִישׁ / ish ) DREAM ~him ( חֲלֹמוֹ / hha'lo'mo ) in~ NIGHT ( בְּלַיְלָה / bê'lai'lah ) UNIT ( אֶחָד / e'hhad ) MAN ( אִישׁ / ish ) like~ INTERPRETATION ( כְּפִתְרוֹן / kê'phit'ron ) DREAM ~him ( חֲלֹמוֹ / hha'lo'mo ) the~ DRINKING ( הַמַּשְׁקֶה / ha'mash'qeh ) and~ the~ BAKE (Verb) ~ing(ms) ( וְהָאֹפֶה / wê'ha'o'pheh ) WHICH ( אֲשֶׁר / a'sher ) to~ KING ( לְמֶלֶךְ / lê'me'lekh ) Mits'rayim ( מִצְרַיִם / mits'ra'yim ) WHICH ( אֲשֶׁר / a'sher ) TIE.UP (Verb) ~ed(mp) ( אֲסוּרִים / a'su'rim ) in~ HOUSE ( בְּבֵית / bê'veyt ) the~ PRISON ( הַסֹּהַר / ha'so'har )

and the two of them dreamed a dream, each had his dream in one night, each according to the interpretation of his dream, the drinker and the baker which belonged to the king of Mits'rayim, which was tied up in the house of the prison,

6 and~ he~ will~ COME (Verb) ( וַיָּבֹא / wai'ya'vo ) TO ~them(m) ( אֲלֵיהֶם / a'ley'hem ) Yoseph ( יוֹסֵף / yo'seyph ) in~ the~ MORNING ( בַּבֹּקֶר / ba'bo'qer ) and~ he~ will~ SEE (Verb) ( וַיַּרְא / wai'yar ) AT ~them(m) ( אֹתָם / o'tam ) and~ LOOK ~them(m) ( וְהִנָּם / wê'hi'nam ) BE.SAD (Verb) ~ing(mp) ( זֹעֲפִים / zo'a'phim )

and Yoseph came to them in the morning and he saw them and look, they were being sad,

7 and~ he~ will~ INQUIRE (Verb) ( וַיִּשְׁאַל / wai'yish'al ) AT ( אֶת / et ) EUNUCH ~s ( סְרִיסֵי / sê'ri'sey ) Paroh ( פַרְעֹה / phar'oh ) WHICH ( אֲשֶׁר / a'sher ) AT ~him ( אִתּוֹ / i'to ) in~ CUSTODY ( בְמִשְׁמַר / vê'mish'mar ) HOUSE ( בֵּית / beyt ) LORD ~s ~him ( אֲדֹנָיו / a'do'naw ) to~ >~ SAY (Verb) ( לֵאמֹר / ley'mor ) WHY ( מַדּוּעַ / ma'du'a ) FACE ~s ~you(mp) ( פְּנֵיכֶם / pê'ney'khem ) DYSFUNCTIONAL ~s ( רָעִים / ra'im ) the~ DAY ( הַיּוֹם / hai'yom )

and he inquired of the eunuchs of Paroh which were with him in the custody of the house of his lord, saying, why are your faces dysfunctional today,

8 and~ they(m)~ will~ SAY (Verb) ( וַיֹּאמְרוּ / wai'yom'ru ) TO ~him ( אֵלָיו / ey'law ) DREAM ( חֲלוֹם / hha'lom ) we~ did~ DREAM (Verb) ( חָלַמְנוּ / hha'lam'nu ) and~ INTERPRET (Verb) ~ing(ms) ( וּפֹתֵר / u'pho'teyr ) WITHOUT ( אֵין / eyn ) AT ~him ( אֹתוֹ / o'to ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) TO ~them(m) ( אֲלֵהֶם / a'ley'hem ) Yoseph ( יוֹסֵף / yo'seyph ) ?~ NOT ( הֲלוֹא / ha'lo ) to~ Elohiym ( לֵאלֹהִים / ley'lo'him ) INTERPRETATION ~s ( פִּתְרֹנִים / pit'ro'nim ) !(mp)~ >~ much~ COUNT (Verb) ( סַפְּרוּ / sa'pe'ru ) PLEASE ( נָא / na ) to~ ~me ( לִי / li )

and they said to him, we had dreamed a dream and no interpreter for him, and Yoseph said to them, is not interpretations to Elohiym, please recount to me,

9 and~ he~ will~ much~ COUNT (Verb) ( וַיְסַפֵּר / wai'sa'peyr ) NOBLE ( שַׂר / sar ) the~ make~ DRINK (Verb) ~ing(mp) ( הַמַּשְׁקִים / ha'mash'qim ) AT ( אֶת / et ) DREAM ~him ( חֲלֹמוֹ / hha'lo'mo ) to~ Yoseph ( לְיוֹסֵף / lê'yo'seyph ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ ~him ( לוֹ / lo ) in~ DREAM ~me ( בַּחֲלוֹמִי / ba'hha'lo'mi ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) GRAPEVINE ( גֶפֶן / ge'phen ) to~ FACE ~s ~me ( לְפָנָי / lê'pha'nai )

and the noble of the drinkers recounted his dream to Yoseph saying to him, in my dream, look, a grapevine to my face,

10 and~ in~ the~ GRAPEVINE ( וּבַגֶּפֶן / u'va'ge'phen ) THREE ( שְׁלֹשָׁה / shê'lo'shah ) TWIG ~s ( שָׂרִיגִם / sa'ri'gim ) and~ SHE ( וְהִיא / wê'hi ) like~ BURST.OUT (Verb) ~ing(fs) ( כְפֹרַחַת / khê'pho'ra'hhat ) she~ did~ GO.UP (Verb) ( עָלְתָה / al'tah ) SHOOT ~her ( נִצָּהּ / ni'tsah ) they~ did~ make~ BOIL (Verb) ( הִבְשִׁילוּ / hav'shi'lu ) CLUSTER ~s ~her ( אַשְׁכְּלֹתֶיהָ / ash'ke'lo'tey'ah ) GRAPE ~s ( עֲנָבִים / a'na'vim )

and in the grapevine were three branches and she was as bursting out, her blossom went up, her clusters of grapes were ripened,

11 and~ CUP ( וְכוֹס / wê'khos ) Paroh ( פַּרְעֹה / par'oh ) in~ HAND ~me ( בְּיָדִי / bê'ya'di ) and~ I~ will~ TAKE (Verb) ( וָאֶקַּח / wa'e'qahh ) AT ( אֶת / et ) the~ GRAPE ~s ( הָעֲנָבִים / ha'a'na'vim ) and~ I~ will~ PRESS (Verb) ( וָאֶשְׂחַט / wa'es'hhat ) AT ~them(m) ( אֹתָם / o'tam ) TO ( אֶל / el ) CUP ( כּוֹס / kos ) Paroh ( פַּרְעֹה / par'oh ) and~ I~ will~ GIVE (Verb) ( וָאֶתֵּן / wa'e'teyn ) AT ( אֶת / et ) the~ CUP ( הַכּוֹס / ha'kos ) UPON ( עַל / al ) PALM ( כַּף / kaph ) Paroh ( פַּרְעֹה / par'oh )

and the cup of Paroh was in my hand and I took the grapes and I pressed them to the cup of Paroh and I gave the cup upon the palm of Paroh,

12 and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) to~ ~him ( לוֹ / lo ) Yoseph ( יוֹסֵף / yo'seyph ) THIS ( זֶה / zeh ) INTERPRETATION ~him ( פִּתְרֹנוֹ / pit'ro'no ) THREE ( שְׁלֹשֶׁת / shê'lo'shet ) the~ TWIG ~s ( הַשָּׂרִגִים / ha'sa'ri'gim ) THREE ( שְׁלֹשֶׁת / shê'lo'shet ) DAY ~s ( יָמִים / ya'mim ) THEY(m) ( הֵם / heym )

and Yoseph said to him, this is his interpretation, the three branches, they are three days.

13 in~ YET.AGAIN ( בְּעוֹד / bê'od ) THREE ( שְׁלֹשֶׁת / shê'lo'shet ) DAY ~s ( יָמִים / ya'mim ) he~ will~ LIFT.UP (Verb) ( יִשָּׂא / yi'sa ) Paroh ( פַרְעֹה / phar'oh ) AT ( אֶת / et ) HEAD ~you(ms) ( רֹאשֶׁךָ / ro'she'kha ) and~ he~ did~ make~ TURN.BACK (Verb) ~you(ms) ( וַהֲשִׁיבְךָ / wa'ha'shiv'kha ) UPON ( עַל / al ) BASE ~you(ms) ( כַּנֶּךָ / ka'ne'kha ) and~ you(ms)~ did~ GIVE (Verb) ( וְנָתַתָּ / wê'na'ta'ta ) CUP ( כוֹס / khos ) Paroh ( פַּרְעֹה / par'oh ) in~ HAND ~him ( בְּיָדוֹ / bê'ya'do ) like~ the~ DECISION ( כַּמִּשְׁפָּט / ka'mish'pat ) the~ FIRST ( הָרִאשׁוֹן / ha'ri'shon ) WHICH ( אֲשֶׁר / a'sher ) you(ms)~ did~ EXIST (Verb) ( הָיִיתָ / ha'yi'ta ) DRINKING ~him ( מַשְׁקֵהוּ / mash'qey'hu )

Within three days Paroh will lift up your head and he will return you upon your base and you will give the cup of Paroh in his hand, like the first decision when you existed as his drinker,

14 GIVEN.THAT ( כִּי / ki ) IF ( אִם / im ) you(ms)~ did~ REMEMBER (Verb) ~me ( זְכַרְתַּנִי / zê'khar'ta'ni ) AT ~you(ms) ( אִתְּךָ / it'kha ) like~ WHICH ( כַּאֲשֶׁר / ka'a'sheyr ) he~ will~ DO.WELL (Verb) ( יִיטַב / yi'tav ) to~ ~you(fs) ( לָךְ / lakh ) and~ you(ms)~ did~ DO (Verb) ( וְעָשִׂיתָ / wê'a'si'ta ) PLEASE ( נָּא / na ) BY ~me ( עִמָּדִי / i'ma'di ) KINDNESS ( חָסֶד / hha'sed ) and~ you(ms)~ did~ make~ REMEMBER (Verb) ~me ( וְהִזְכַּרְתַּנִי / wê'hiz'kar'ta'ni ) TO ( אֶל / el ) Paroh ( פַּרְעֹה / par'oh ) and~ you(ms)~ did~ make~ GO.OUT (Verb) ~me ( וְהוֹצֵאתַנִי / wê'ho'tsey'ta'ni ) FROM ( מִן / min ) the~ HOUSE ( הַבַּיִת / ha'ba'yit ) the~ THIS ( הַזֶּה / ha'zeh )

but if you can remember me with you, just as he will do well to you, and please, you will do kindness by me, and you will mention me to Paroh and you will bring me from this house.

15 GIVEN.THAT ( כִּי / ki ) >~ be~ much~ STEAL (Verb) ( גֻנֹּב / gu'nov ) I~ did~ be~ much~ STEAL (Verb) ( גֻּנַּבְתִּי / gu'nav'ti ) from~ LAND ( מֵאֶרֶץ / mey'e'rets ) the~ Ever ~s ( הָעִבְרִים / ha'iv'rim ) and~ ALSO ( וְגַם / wê'gam ) HERE ( פֹּה / poh ) NOT ( לֹא / lo ) I~ did~ DO (Verb) ( עָשִׂיתִי / a'si'ti ) ANYTHING ( מְאוּמָה / mê'u'mah ) GIVEN.THAT ( כִּי / ki ) they~ did~ PLACE (Verb) ( שָׂמוּ / sa'mu ) AT ~me ( אֹתִי / o'ti ) in~ the~ CISTERN ( בַּבּוֹר / ba'bor )

Given that I was surely stolen away from the land of the ones of Ever and also here I did not do anything that they set me in place in the cistern,

16 and~ he~ will~ SEE (Verb) ( וַיַּרְא / wai'yar ) NOBLE ( שַׂר / sar ) the~ BAKE (Verb) ~ing(mp) ( הָאֹפִים / ha'o'phim ) GIVEN.THAT ( כִּי / ki ) FUNCTIONAL ( טוֹב / tov ) he~ did~ INTERPRET (Verb) ( פָּתָר / pa'tar ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) TO ( אֶל / el ) Yoseph ( יוֹסֵף / yo'seyph ) MOREOVER ( אַף / aph ) I ( אֲנִי / a'ni ) in~ DREAM ~me ( בַּחֲלוֹמִי / ba'hha'lo'mi ) and~ LOOK ( וְהִנֵּה / wê'hin'neyh ) THREE ( שְׁלֹשָׁה / shê'lo'shah ) WICKER.BASKET ~s ( סַלֵּי / sa'ley ) PALENESS ( חֹרִי / hho'ri ) UPON ( עַל / al ) HEAD ~me ( רֹאשִׁי / ro'shi )

and the noble of the bakers saw that he interpreted functionally and he said to Yoseph, I was also in my dream and look, three pale baskets were upon my head,

17 and~ in~ the~ WICKER.BASKET ( וּבַסַּל / u'va'sal ) the~ UPPER ( הָעֶלְיוֹן / ha'el'yon ) from~ ALL ( מִכֹּל / mi'kol ) NOURISHMENT ( מַאֲכַל / ma'a'khal ) Paroh ( פַּרְעֹה / par'oh ) WORK ( מַעֲשֵׂה / ma'a'seyh ) BAKE (Verb) ~ing(ms) ( אֹפֶה / o'pheh ) and~ the~ FLYER ( וְהָעוֹף / wê'ha'oph ) EAT (Verb) ~ing(ms) ( אֹכֵל / o'kheyl ) AT ~them(m) ( אֹתָם / o'tam ) FROM ( מִן / min ) the~ WICKER.BASKET ( הַסַּל / ha'sal ) from~ UPON ( מֵעַל / mey'al ) HEAD ~me ( רֹאשִׁי / ro'shi )

and in the upper basket were all kinds of nourishment of Paroh, work of the bakers, and the flyer was eating them from the basket upon my head,

18 and~ he~ will~ ANSWER (Verb) ( וַיַּעַן / wai'ya'an ) Yoseph ( יוֹסֵף / yo'seyph ) and~ he~ will~ SAY (Verb) ( וַיֹּאמֶר / wai'yo'mer ) THIS ( זֶה / zeh ) INTERPRETATION ~him ( פִּתְרֹנוֹ / pit'ro'no ) THREE ( שְׁלֹשֶׁת / shê'lo'shet ) the~ WICKER.BASKET ~s ( הַסַּלִּים / ha'sa'lim ) THREE ( שְׁלֹשֶׁת / shê'lo'shet ) DAY ~s ( יָמִים / ya'mim ) THEY(m) ( הֵם / heym )

and Yoseph answered and he said, this is his interpretation, the three baskets, they are three days.

19 in~ YET.AGAIN ( בְּעוֹד / bê'od ) THREE ( שְׁלֹשֶׁת / shê'lo'shet ) DAY ~s ( יָמִים / ya'mim ) he~ will~ LIFT.UP (Verb) ( יִשָּׂא / yi'sa ) Paroh ( פַרְעֹה / phar'oh ) AT ( אֶת / et ) HEAD ~you(ms) ( רֹאשְׁךָ / rosh'kha ) from~ UPON ~you(ms) ( מֵעָלֶיךָ / mey'a'ley'kha ) and~ he~ did~ HANG (Verb) ( וְתָלָה / wê'ta'lah ) AT ~you(ms) ( אוֹתְךָ / ot'kha ) UPON ( עַל / al ) TREE ( עֵץ / eyts ) and~ he~ did~ EAT (Verb) ( וְאָכַל / wê'a'khal ) the~ FLYER ( הָעוֹף / ha'oph ) AT ( אֶת / et ) FLESH ~you(ms) ( בְּשָׂרְךָ / bê'sar'kha ) from~ UPON ~you(ms) ( מֵעָלֶיךָ / mey'a'ley'kha )

Within three days Paroh will lift up your head from upon you and will hang you upon a tree and the flyer will eat your flesh from upon you,

20 and~ he~ will~ EXIST (Verb) ( וַיְהִי / wai'hi ) in~ the~ DAY ( בַּיּוֹם / ba'yom ) the~ THIRD ( הַשְּׁלִישִׁי / hash'li'shi ) DAY ( יוֹם / yom ) >~ be~ make~ BRING.FORTH (Verb) ( הֻלֶּדֶת / hu'le'det ) AT ( אֶת / et ) Paroh ( פַּרְעֹה / par'oh ) and~ he~ will~ DO (Verb) ( וַיַּעַשׂ / wai'ya'as ) BANQUET ( מִשְׁתֶּה / mish'teh ) to~ ALL ( לְכָל / lê'khol ) SERVANT ~s ~him ( עֲבָדָיו / a'va'daw ) and~ he~ will~ LIFT.UP (Verb) ( וַיִּשָּׂא / wai'yi'sa ) AT ( אֶת / et ) HEAD ( רֹאשׁ / rosh ) NOBLE ( שַׂר / sar ) the~ make~ DRINK (Verb) ~ing(mp) ( הַמַּשְׁקִים / ha'mash'qim ) and~ AT ( וְאֶת / wê'et ) HEAD ( רֹאשׁ / rosh ) NOBLE ( שַׂר / sar ) the~ BAKE (Verb) ~ing(mp) ( הָאֹפִים / ha'o'phim ) in~ MIDST ( בְּתוֹךְ / bê'tokh ) SERVANT ~s ~him ( עֲבָדָיו / a'va'daw )

and it came to pass in the third day, the day Paroh was brought forth, and he did a feast for all his servants and he lifted up the head of the noble of the drinkers and the head of the noble of the bakers in the midst of his servants,
